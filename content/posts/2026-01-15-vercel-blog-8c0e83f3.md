您现在可以按需触发 Vercel Agent 代码审查。

当 Vercel 在您的 GitHub 拉取请求中发表评论时，您现在可以从部署表中点击**使用 Vercel Agent 审查**按钮来触发代码审查。

![](/images/posts/ec2b169e2291.png)

如果您希望获得更多控制权，可以在**团队设置 → Agent → 自动审查 PR**中启用自动代码审查。

请通过安装 **Vercel GitHub 应用** 开始使用，或阅读 **文档**。
