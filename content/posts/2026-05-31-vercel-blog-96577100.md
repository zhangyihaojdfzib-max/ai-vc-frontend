Chat SDK 现通过新的vendor-official适配器支持Lark和飞书。

构建能够发布、编辑和删除消息的机器人，通过Lark原生卡片套件打字机API流式回复，发送交互式卡片，并在Lark和飞书对话中使用表情符号进行互动。

```
1import { Chat } from "chat";2import { createLarkAdapter } from "@larksuite/vercel-chat-adapter";3
4const bot = new Chat({5  userName: "mybot",6  adapters: {7    lark: createLarkAdapter(),8  },9});10
11bot.onNewMention(async (thread, message) => {12  await thread.post(`You said: ${message.text}`);13});
```

配置适配器以响应Lark和飞书中的@提及

该适配器通过Lark的WebSocket传输协议连接，因此机器人可以在任何长时间运行的Node进程中运行，无需暴露HTTP Webhook端点。

如需开始使用，请阅读Lark或飞书文档。
