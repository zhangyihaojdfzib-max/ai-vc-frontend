Bun 运行时现已在 Vercel Functions 中开放公测。

您可以在 Node.js 和 Bun 之间选择作为项目的运行时，为工作负载选择最佳方案。

基准测试显示，与 Node.js 相比，Bun 将 CPU 密集型 Next.js 渲染的平均延迟降低了 28%。

要在 Vercel Functions 中使用 Bun，请在项目的 `vercel.json` 中全局设置运行时：

```
1{2  "bunVersion": "1.x"3}
```

通过在 vercel.json 中添加 bunVersion 来启用应用对 Bun 运行时的支持。

我们目前支持以下框架，更多框架即将推出：

- Next.js
- Hono
- Express
- Nitro
