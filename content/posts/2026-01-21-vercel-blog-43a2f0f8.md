技能支持现已在 `bash-tool` 中提供，因此您的 AI SDK Agent（智能体）可以在文件系统上下文、Bash 执行和沙盒运行时访问中使用技能模式。

这为您的 Agent（智能体）提供了一种一致的方式，来为任务引入正确的上下文，其使用的隔离执行模型与支持基于文件系统的上下文检索的模型相同。

这使得您的 Agent（智能体）能够访问**种类繁多的公开可用技能**，或者您可以编写自己的专有技能，并在您的 Agent（智能体）中私下使用它们。

```
1import {2  experimental_createSkillTool as createSkillTool,3  createBashTool,4} from "bash-tool";5import { ToolLoopAgent } from "ai";6
7// Discover skills and get files to upload8const { skill, files, instructions } = await createSkillTool({9  skillsDirectory: "./skills",10});11
12// Create bash tool with skill files13const { tools } = await createBashTool({14  files,15  extraInstructions: instructions,16});17
18// Use both tools with an agent19const agent = new ToolLoopAgent({20  model,21  tools: { skill, ...tools },22});
```
