skills.sh API 现已可用。使用您项目的 Vercel OIDC Token 进行身份验证，即可开始查询来自开源生态系统中超过 60 万项技能。

搜索技能、获取任意技能的详细信息、查看其安全审计报告等。

Vercel 会颁发一个限定于您团队和项目的短期 Token，并自动轮换，因此不存在需要泄露或轮换的长期密钥。每次请求时，skills.sh 会验证该 Token，并对每个团队和项目应用每分钟 600 次请求的速率限制。

```
1import { getVercelOidcToken } from '@vercel/oidc';2
3const token = await getVercelOidcToken();4const res = await fetch('https://skills.sh/api/v1/skills?per_page=10', {5  headers: { Authorization: `Bearer ${token}` },6});
```
