Chat SDK 现已支持 WhatsApp，通过新的 WhatsApp 适配器，将其单一代码库的方法扩展至 Slack、Discord、GitHub、Teams 和 Telegram。

团队可以构建支持消息、回应、自动分块和已读回执的机器人。该适配器处理多媒体下载（例如图片、语音消息、贴纸），并支持通过 Google Maps URL 分享位置。

立即尝试 WhatsApp 适配器：

```
1import { Chat } from "chat";2import { createWhatsAppAdapter } from "@chat-adapter/whatsapp";3
4const bot = new Chat({5  userName: "mybot",6  adapters: {7    whatsapp: createWhatsAppAdapter(),8  },9});10
11bot.onNewMention(async (thread, message) => {12  await thread.post(`You said: ${message.text}`);13});
```

该适配器不支持消息检索、编辑或删除。卡片会渲染为最多包含三个选项的交互式回复按钮，并会回退为格式化文本。此外，WhatsApp 强制执行 24 小时消息窗口；机器人只能在用户最后一条消息的 24 小时内进行回复。

阅读文档以开始使用，或浏览适配器目录。
