AI Gateway 现已支持 Claude Opus 4.6 的快速模式。

快速模式是一种优质高速选项，能在保持相同模型智能水平的同时，将输出 Token 速度提升 2.5 倍。这是一项早期的实验性功能。

快速模式提升的输出 Token 速度使得新的应用场景成为可能，尤其适用于人机协同工作流。您可以运行大型编码任务而无需切换上下文，并在无需长时间等待的情况下获得规划结果。

要启用快速模式，请在 AI SDK 的 Anthropic 提供者选项中传递 `speed: 'fast'`：

```
1import { streamText } from "ai";2
3const { text } = await streamText({4  model: 'anthropic/claude-opus-4.6',5  prompt:6   `Analyze this codebase structure and create a step-by-step plan7    to add user authentication.`,8  providerOptions: {9    anthropic: {10      speed: 'fast',11    },12  },13});
```

您可以通过在 shell 配置文件或 `~/.claude/settings.json` 中设置 `CLAUDE_CODE_SKIP_FAST_MODE_ORG_CHECK` 变量，在 AI Gateway 上对 Claude Code 使用快速模式。

```
1export CLAUDE_CODE_SKIP_FAST_MODE_ORG_CHECK=1
```

```
1{2  "env": {3    "CLAUDE_CODE_SKIP_FAST_MODE_ORG_CHECK": "1"4  }5}
```

您可以直接在 AI Gateway 的 Opus 4.6 游乐场中尝试快速模式。
