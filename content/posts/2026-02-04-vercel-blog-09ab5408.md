您现在可以在整个Vercel平台上使用Parallel专为LLM优化的网络搜索及其他工具。

## AI Gateway

与那些仅适用于特定模型的供应商专用网络搜索工具不同，Parallel的网络搜索工具可普遍适用于所有供应商。这意味着您可以为任何模型添加网络搜索功能，而无需更改您的实现方式。

要通过AI SDK使用，请在`tools`中设置`parallel_search: gateway.tools.parallelSearch()`。

```
1import { gateway, streamText } from 'ai';2
3const result = streamText({4  model: 'moonshotai/kimi-k2.5', 5  prompt: 'What are the best new restaurants in San Francisco?',6  tools: {7    parallel_search: gateway.tools.parallelSearch(),8  },9});
```

Parallel网络搜索从网页中提取相关摘要，使其非常适合Agent任务和实时信息检索。为了获得更多控制，您还可以配置该工具以使用特定参数。

```
1import { gateway, streamText } from 'ai';2
3const result = streamText({4  model: 'moonshotai/kimi-k2.5',5  prompt: 'What new AI model releases have been announced this month?',6  tools: {7    parallel_search: gateway.tools.parallelSearch({8      mode: 'one-shot',9      maxResults: 10,10      includeDomains: ['anthropic.com', 'moonshot.ai', 'deepmind.google'],11      afterDate: '2026-01-01',12    }),13  },14});
```

对于Agent工作流，使用`mode: 'agentic'`来获得简洁、节省Token的搜索结果，这些结果在多步推理中表现良好。

对于时间敏感的查询，可以通过`maxAgeSeconds`控制缓存新鲜度，而特定域搜索则允许您将结果限制在可信来源或排除嘈杂的域名。

Parallel网络搜索请求的计费标准与Parallel API完全相同。每1,000次请求5美元（每次请求最多包含10个结果），超过10个的额外结果按每1,000个额外结果1美元收费。阅读文档以获取更多信息及如何使用该工具的详细信息。

## AI SDK

AI SDK支持将Parallel作为网络搜索和提取的工具。要使用，只需安装`parallel-web`工具包。

```
1pnpm install @parallel-web/ai-sdk-tools
```

查看文档以获取有关如何利用这些工具的更多详细信息。

## Vercel Marketplace

您可以在Vercel Agent Marketplace中使用所有Parallel产品：搜索、提取、任务、查找和监控，并通过Vercel进行集中计费，使用单一API密钥。要开始使用，请前往Parallel集成并连接您的账户，或部署Next.js模板，以在实践中将Parallel的网络研究API与Vercel集成。

通过AI Gateway、AI SDK工具包或Vercel Marketplace，为您的AI应用程序开启Parallel之旅。
