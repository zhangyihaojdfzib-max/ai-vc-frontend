## 链接到标题我们为何构建 AI Gateway

当前 AI 发展速度迅猛，且只会越来越快。

几乎每周都有新的最先进模型发布。令人沮丧的是，这意味着开发者一直被锁定在应用代码中的特定供应商或模型 API 上。我们希望帮助开发者快速交付，跟上 AI 发展步伐，而无需管理 10 个不同的 API 密钥和供应商账户。

生产级 AI 应用常因高需求而遇到容量问题或速率限制。基础设施供应商迅速行动，将模型上线以满足需求，但这可能以牺牲性能或可用性为代价。

AI Gateway 将允许您在供应商之间进行负载均衡，或在供应商出现停机或性能下降时进行故障转移。模型推理成本持续下降，供应商在质量、性能和价格上展开竞争。Gateway 帮助您快速利用这些成本优势。

我们通过快速负载均衡和切换混合供应商组合，将扩展 v0 至数百万用户过程中积累的经验，转化为 AI Gateway 基础设施。

## 链接到标题与 AI SDK 的集成

我们构建了 AI SDK，为文本、图像和音频等模态的 AI 模型 API 创建了通用抽象层。

AI SDK 免费且开源，适用于任何模型或基础设施供应商。AI Gateway 是 Vercel 在 AI SDK 之上构建的独立产品。

我们以高内聚、低耦合的方式构建这些产品。Gateway 将充分利用 AI SDK 的功能，如工具调用、函数参数、流式传输、重试、附件和结构化输出。

## 链接到标题定价

在 AI Gateway Alpha 测试期间，使用免费，但根据您的 Vercel 套餐层级设有速率限制。这些速率限制与当前的 AI SDK Playground 类似。

我们计划在 Gateway 正式发布时支持按量付费定价。模型定价将遵循供应商的市场价格，并定期更新。我们还计划在未来探索自带密钥模式。
