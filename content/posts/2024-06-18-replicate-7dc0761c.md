# 在 Apple Silicon Mac 上运行 Stable Diffusion 3

- zsxkib

Stable Diffusion 3 (SD3) 是能将文本转换为图像的最新开源 AI 模型。您可以在您的 Apple Silicon Mac 上本地运行它，并在几分钟内开始制作令人惊叹的图片。观看此视频了解实际效果：

## 先决条件

- 配备 M 系列 Apple Silicon 芯片的 Mac
- Git

## 克隆代码仓库并设置环境

运行以下命令克隆 SD3 代码仓库：

```
git clone https://github.com/zsxkib/sd3-on-apple-silicon.git
cd sd3-on-apple-silicon
```

然后，创建一个包含 SD3 所需软件包的新虚拟环境：

```
python3 -m venv sd3-env
source sd3-env/bin/activate
pip install -r requirements.txt
```

## 运行它！

现在，您可以生成您的第一张 SD3 图像：

```
python sd3-on-mps.py
```

首次运行将下载 SD3 模型和权重文件，大小约为 15.54 GB。后续运行将使用已下载的文件。

不到一分钟，您的目录中就会出现一张名为 `sd3-output-mps.png` 的新图像。它看起来会像这样：

就是这样！

## 自定义您的图像

您可以调整 `sd3-on-mps.py` 中的各种设置来生成不同的图像：

- `seed`：设置一个特定数字以重现相同的图像，或将其保留为 `None` 以每次生成独特的图像。
- `prompt`：修改此文本以生成不同的图像（例如，“A sunset over the mountains”）。
- `height` 和 `width`：调整这些值以更改图像尺寸（例如，1024 用于更大的图像）。
- `num_inference_steps`：增加此值以获得更详细的图像。步骤越多意味着质量越好，但生成速度越慢。
- `guidance_scale`：更改此值以控制图像与提示词的匹配程度。值越高，图像越准确，但可能看起来不那么自然。

尝试调整这些选项，创造出各种酷炫的图像！

## 另一个选择：DiffusionKit

DiffusionKit 是一个 CLI 工具，可让您使用 Python MLX 运行 SD3。这是运行 SD3 的另一种方式，就像我们上面所做的那样，但无需克隆 SD3 仓库并手动设置环境。以下是使其运行的方法：

1.  设置一个新的虚拟环境并安装 DiffusionKit：

```
python3 -m venv diffusionkit-env
source diffusionkit-env/bin/activate
cd /path/to/diffusionkit/repo
pip install -e .
```

2.  使用您的 READ token 登录 Hugging Face Hub：

```
huggingface-cli login --token YOUR_HF_HUB_TOKEN
```

3.  使用您想要的设置运行 DiffusionKit：

```
diffusionkit-cli \
--prompt "A cat holding a sign that says hello world" \
--height 512 \
--width 512 \
--steps 28 \
--cfg 7.0 \
--output-path "/path/to/output/sd3-output-diffusionkit.png" \
--a16 \
--w16
```

以下是其中一些标志的作用：

- `--a16`：通过以 float16 精度运行模型来减少内存使用。
- `--w16`：通过以 float16 精度加载模型来节省更多内存。
- `--t5`：提高文本理解能力，允许更长的提示词（>77 个 Token），但会使用更多内存。

查看 [DiffusionKit 仓库](https://github.com/zsxkib/diffusionkit) 以获取所有可用选项的更多详细信息。
