Chat SDK 现在通过新的 `transcripts` 和 `identity` 选项支持跨平台对话历史记录。用户对话记录会在每个平台适配器之间持久化，使得同一用户无论在哪里与你的机器人对话，都能保留他们的消息历史。

```
1import { Chat } from "chat";
2
3const bot = new Chat({
4  userName: "mybot",
5  identity: ({ author }) => author.email ?? null,
6  transcripts: {
7    retention: "30d",
8    maxPerUser: 200,
9  },
10  
11});
```

通过电子邮件识别用户，并为每个用户保留最多 200 条消息，保留 30 天

`bot.transcripts` 暴露了四个方法，由你现有的状态适配器支持：

- `append`：持久化一条入站消息或机器人回复
- `list`：按时间顺序返回带有过滤条件的条目
- `count`：存储的某个用户的总条目数
- `delete`：删除某个用户的所有条目
