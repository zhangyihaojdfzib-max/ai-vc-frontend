您现在可以使用 `vercel/ai-action@v2` GitHub Action 来访问 AI SDK 和 AI Gateway，通过在您的工作流中指定提示词、模型和 API 密钥，直接生成文本或结构化的 JSON。

此集成为 GitHub Actions 开启了新的 AI 驱动用例，例如总结版本发布内容、进行轻量级的 PR 代码审查、评论审核，或查找重复或相关的议题。例如，您可以像这样用它来对议题进行分类：

```

```
