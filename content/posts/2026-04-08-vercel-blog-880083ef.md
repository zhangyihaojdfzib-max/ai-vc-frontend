Vercel Sandbox 现已支持为企业客户创建配备高达 32 个 vCPU 和 64 GB 内存的沙盒环境。这使得运行那些受 CPU 限制或需要大量内存的大型、资源密集型应用成为可能。

您可以通过在 SDK 中设置 `resources.vcpus` 选项来开始使用：

```
1import { Sandbox } from "@vercel/sandbox";2
3const sandbox = await Sandbox.create({4  resources: { vcpus: 32 },5});
```

或者在 CLI 中使用 `--vcpus` 选项：

```
1sandbox create --connect --vcpus 32
```

欲了解更多关于 Sandbox 的信息，请查阅文档。
