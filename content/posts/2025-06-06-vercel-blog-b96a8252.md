# Claude Code 和 Cursor Agent 不再需要团队席位

我们已更新构建逻辑，确保由 Claude Code 或 Cursor Agent 创建的 Git 提交能够触发 Vercel 上的部署。不再需要团队席位。

如果您的 Agent 在 Vercel 上构建时遇到任何问题，请联系我们。
