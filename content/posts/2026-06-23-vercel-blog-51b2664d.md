现在，您可以在Vercel上零配置部署Node.js服务器。

除了现有的零配置后端（如Express、Koa和NestJS）之外，Vercel还会检测项目根目录或src/server.ts路径下的server.ts文件，并将其作为Node.js应用进行部署：

```
import { createServer } from 'node:http'
const server = createServer((req, res) => {  res.end('Hello from Node.js on Vercel!')})
server.listen(process.env.PORT ?? 3000)
```

一个部署在Vercel上的Node应用

Vercel CLI可以处理本地开发和部署：

```
vc dev
vc deploy
```

这两个命令都会自动识别server.ts文件，无需任何配置文件。

Vercel上的后端由Fluid compute（弹性计算）和Active CPU pricing（活跃CPU计费）提供支持。
