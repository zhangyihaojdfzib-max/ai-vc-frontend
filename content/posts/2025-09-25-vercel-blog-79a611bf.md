FastAPI 是一个用于使用 Python 构建 API 的现代化、高性能 Web 框架，现已支持零配置部署。

```
1from fastapi import FastAPI 2
3app = FastAPI() 4
5@app.get("/") 6def read_root(): 7    return {"Hello": "World"}
```

一个在 Vercel 上的 "Hello World" FastAPI 应用

Vercel 的框架定义基础设施现已能够识别并深度理解 FastAPI 应用程序。此次更新消除了在 `vercel.json` 中配置重定向或使用 `/api` 文件夹的需要。

Vercel 上的后端默认使用具备动态 CPU 定价的流体计算。这意味着您的 FastAPI 应用将根据流量自动扩缩容，并且您只需为实际使用的资源付费。
