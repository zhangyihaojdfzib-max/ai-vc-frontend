xAI 最新、最强大的 Grok 3 模型现已通过 Vercel Marketplace 提供，将最先进的 AI 能力引入您的 Vercel 项目。

要开始使用，您可以在项目中通过 AI SDK 的 xAI 提供者进行调用：


然后，通过 Vercel CLI（或从控制台）安装 xAI Marketplace 集成：

```
vercel install xai
```

一旦您接受相关条款，即可在项目中使用 Grok 模型，无需额外步骤。

为帮助您快速上手，我们还提供了一个可立即部署的 Next.js xAI 入门模板。要了解更多关于 Vercel 上 xAI 的信息，请阅读我们的公告和文档。
