xAI 的音频模型现已上线 AI Gateway。实时语音、文本转语音和语音转文本功能均可通过 AI SDK 使用，并具备与其他模型相同的路由、可观测性和费用控制功能。

这些能力已在 AI SDK 7 版本中提供。

```
1npm install ai @ai-sdk/react @ai-sdk/gateway
```

## 链接到标题实时

一个语音 Agent（智能体）包含两部分：一个用于生成短期令牌的服务端路由（确保你的 API 密钥不会到达客户端），以及一个与之连接的浏览器组件。

添加令牌路由：以下示例将模型设置为 xai/grok-voice-think-fast-1.0：

```
1import { gateway } from '@ai-sdk/gateway';2
3export async function POST() {4  const { token, url } = await gateway.experimental_realtime.getToken({5    model: 'xai/grok-voice-think-fast-1.0',6  });7  return Response.json({ token, url, tools: [] });8}
```

然后从浏览器进行连接。来自 @ai-sdk/react 的 useRealtime 钩子会获取该路由，并管理 WebSocket 连接、麦克风采集和音频播放：

```
1'use client';2import { experimental_useRealtime as useRealtime } from '@ai-sdk/react';3import { gateway } from '@ai-sdk/gateway';4
56const { status, connect, startAudioCapture } = useRealtime({7  model: gateway.experimental_realtime('xai/grok-voice-think-fast-1.0'),8  api: { token: '/api/realtime/token' },9  sessionConfig: { turnDetection: { type: 'server-vad' } },10});11
```

## 链接到标题文本转语音

使用 generateSpeech 从文本生成语音音频。传入语音和输出格式，然后使用 xai/grok-tts 将结果写入文件：

```
1import { generateSpeech } from 'ai';2import { writeFile } from 'node:fs/promises';3
4const result = await generateSpeech({5  model: 'xai/grok-tts',6  text: 'Thanks for trying out AI Gateway.',7  voice: 'eve',8  outputFormat: 'mp3',9});10await writeFile('speech.mp3', result.audio.uint8Array);
```

## 链接到标题语音转文本

使用 transcribe 将录音转录为文本。以下示例使用 xai/grok-stt：

```
1import { transcribe } from 'ai';2import { readFile } from 'node:fs/promises';3
4const result = await transcribe({5  model: 'xai/grok-stt',6  audio: await readFile('audio.mp3'),7});8console.log(result.text);
```
