v0 API 现已支持连接任何自定义 MCP 服务器。团队可以通过提供必要的端点与认证信息，以编程方式配置新的服务器。

```
1import { v0 } from "v0-sdk"2
3const server = await v0.mcpServers.create({4  name: "Vercel",5  url: "https://mcp.vercel.com",6})
```

配置完成后，您可以在 v0 聊天会话中直接引用服务器 ID 来启用这些自定义服务器：

```
1import { v0 } from "v0-sdk"2
3await v0.chats.create({4  message: "Build a heatmap of my team's Vercel deployments.",5  mcpServerIds: [server.id],6})
```
