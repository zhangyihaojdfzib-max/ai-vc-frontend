# 使用 Grep 搜索任意公开 GitHub 仓库

![](/images/posts/bf64d58e8cb5.jpg)

![](/images/posts/7fbecbc4d80a.jpg)

您现在可以使用 Grep 搜索 GitHub 上的任何公开仓库，不再局限于 100 多万个预索引仓库。

要搜索特定仓库，请使用 grep.app/[所有者]/[仓库名]。

例如：访问 grep.app/vercel/ai 并开始输入搜索查询（试试 streamText）。

无需任何设置，即可在仓库内进行快速、全文和正则表达式搜索。
