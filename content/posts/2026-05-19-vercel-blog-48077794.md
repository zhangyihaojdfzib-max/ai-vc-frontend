Nuxt MCP 工具包现已支持 MCP 应用。你的 Agent 工具可以返回交互式 HTML 响应，这些响应会在 Claude 和 ChatGPT 等 MCP 客户端中内联渲染，而非纯文本响应。

使用 `defineMcpApp` 宏声明一个工具，然后通过 `useMcpApp` 组合式函数读取预填充的数据、触发后续提示词，或从 UI 内部调用其他工具。该工具包会在构建时将每个 Vue SFC 打包成独立的 HTML 文件，并通过你的 MCP 端点提供服务。

```
1<script setup lang="ts">2import { z } from 'zod'3
4defineMcpApp({5  name: 'weather',6  description: '显示某个城市的天气预报',7  inputSchema: {8    city: z.string().describe('要获取天气预报的城市'),9  },10  handler: async ({ city }) => ({11    structuredContent: await $fetch(`/api/weather?city=${city}`),12  }),13})14
15const { data, callTool } = useMcpApp<{ city: string, summary: string, tempC: number }>()16</script>17
18<template>19  <article>20    <h1>{{ data?.city }}</h1>21    <p>{{ data?.summary }}, {{ data?.tempC }}°C</p>22    <button @click="callTool('weather', { city: 'London' })">23      查看伦敦天气24    </button>25  </article>26</template>
```

一个在宿主 UI 中内联渲染，并能通过新城市名称调用自身的天气工具

阅读 Nuxt MCP 工具包文档开始使用。
