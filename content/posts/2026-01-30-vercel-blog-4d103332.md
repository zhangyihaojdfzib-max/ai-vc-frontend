Vercel Sandboxes现已全面开放，提供了一种用于安全执行不可信代码的临时计算原语。

它让团队能够运行AI Agent（智能体）生成的输出、未经验证的用户上传内容和第三方代码，而无需暴露生产系统。

每个沙箱都在Firecracker微虚拟机内运行，与您的基础设施隔离，因此沙箱中运行的代码无法访问环境变量、数据库连接和云资源。

包括v0、Blackbox AI和RooCode在内的团队已在生产环境中使用沙箱。

要创建一个简单的Node.js应用程序来启动Vercel沙箱，请使用以下代码：

```
1import { Sandbox } from '@vercel/sandbox';2
3const sandbox = await Sandbox.create();4
5await sandbox.runCommand({6  cmd: 'node', 7  args: ["-e", 'console.log("Hello from Vercel Sandbox!")'],8  stdout: process.stdout,9});10
11await sandbox.stop();
```

或者，通过打开交互式shell使用CLI开始：

```
1npx sandbox create --connect
```

查阅文档以开始使用，并查看开源SDK和CLI。
