# Rogo 借助 OpenAI o1 扩展 AI 驱动的金融研究

Rogo⁠(opens in a new window) 是一个企业级 AI 金融平台，正在改变投资银行、私募股权公司和资产管理公司的工作方式。

通过对 OpenAI 的模型进行微调，并整合包括标普全球、Crunchbase 和 FactSet 在内的广泛金融数据集，Rogo 为数以千计的金融专业人士提供实时金融情报，将他们的工作重心从手动劳动转向高价值的决策制定。

自 2024 年从秘密运营状态公开以来，Rogo 已经：

*   为上市投资银行和大型私募股权公司的**超过 5,000 名银行家**提供服务。
*   每周为分析师在会议准备、公司概况分析和市场研究等任务上**节省 10 小时以上**。
*   利用 OpenAI 的模型驱动其解决方案，使其**年度经常性收入增长了 27 倍**。

“几十年来，还没有一家公司能颠覆金融工作流程，”Rogo 的联合创始人兼首席技术官 Tumas Rackaitis 说。“我们打造 Rogo，旨在成为每位投资银行家日常依赖的必备工具——就像彭博终端一样，但专注于深度的金融洞察。”

## 为现代金融公司释放时间与洞察力

Rogo 使用多种 OpenAI 模型，提供安全的解决方案，直接集成到金融专业人士的日常工作流程中，从而释放时间，让他们专注于创收和高杠杆的工作。

在众多其他应用场景中，OpenAI 的 API 使 Rogo 能够创建一个提供以下功能的平台：

1.  **准确、实时的洞察**：用户可以在几秒钟内从申报文件、文字记录和演示文稿中提取可操作的见解，并生成端到端、可直接用于演示的材料。
2.  **自动化尽职调查**：Rogo 整合私人数据室，生成定制化问题清单，提供写作协助，并帮助跟踪客户互动，每周为分析师节省多达 10 小时，帮助团队更快地为会议做好准备。
3.  **协作工作流**：初级和高级专业人士都依赖 Rogo 来简化构建市场地图或进行竞争分析等任务，使其成为合伙人及分析师之间的共享资源。

“在使用 Rogo 之前，我需要花费数小时仔细研究几十份文档和文件。有了 Rogo，我可以立即获得所需的所有数据，”一家领先的公共投资银行的分析师表示。

## 微调 OpenAI 模型以实现金融工作自动化

Rogo 对 OpenAI 的安全模型进行了调整，以应对金融工作的独特挑战。通过对 OpenAI 的模型进行微调，并整合标普全球、Crunchbase 和 FactSet 等海量数据集，该公司能够搜索和分析超过 5000 万份金融文档。

他们的模型架构采用分层设计，以平衡性能、成本和用例需求：

*   **GPT‑4o** 驱动 Rogo 基于聊天的问答，并处理深入的财务分析。
*   **o1‑mini** 用于对金融数据进行情境化和结构化，以实现有效搜索。
*   **o1** 则用于评估、合成数据生成和高级推理工作流。

这种方法使 Rogo 能够通过将常规任务分配给较小的模型来优化成本，同时将最先进的能力保留给高风险场景。一个由前银行家和投资者组成的团队还会审查和标注数据集，以确保对金融终端用户的准确性和相关性。

Rogo 机器学习引擎的核心是一个 Agent（智能体）框架，旨在处理复杂的金融工作流，包括多步骤查询规划与理解、上下文管理和高效搜索。其输出可在桌面端、移动端和平板电脑上访问，使用户能够随时随地轻松获取洞察。

“我们使用 OpenAI 的最新模型来获得独特深度和复杂的金融洞察，这对私募股权公司和对冲基金等投资基金更具相关性，”Rackaitis 说。“然后，这些洞察可以被提炼到更小的模型中，从而提供卓越的速度和效率，这对于投资银行等高节奏环境至关重要。”

Rogo 团队选择 OpenAI 是因为其卓越的推理能力、稳健的 API 以及处理广泛任务和特定领域任务的灵活性。

“OpenAI 提供的工具生态系统——微调、函数调用和多模态能力——确保我们现在和未来都能满足客户的复杂需求，”Rackaitis 说。

双方团队也共享一种实验文化。Rogo 的部署团队由前银行家和投资者组成，他们与客户合作实时完善功能，而 OpenAI 持续不断的模型进步使 Rogo 能够快速迭代。
