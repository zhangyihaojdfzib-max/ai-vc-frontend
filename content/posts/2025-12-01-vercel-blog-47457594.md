您现在可以通过 Vercel 的 **AI Gateway** 访问 DeepSeek 的最新模型 DeepSeek V3.2 和 DeepSeek V3.2 Speciale，无需其他提供商账户。

DeepSeek V3.2 支持组合思维与工具使用，可在推理和非推理模式下处理 Agent（智能体）风格的操作（工具调用）。DeepSeek V3.2 Speciale 针对极致推理性能进行了优化，适用于复杂的任务场景，但需要更高的 Token 使用量且不支持工具使用。

要使用 DeepSeek V3.2 模型，请在 **AI SDK** 中将 `model` 设置为以下值：

*   非思维模式：`deepseek/deepseek-v3.2`
*   思维模式：`deepseek/deepseek-v3.2-thinking`
*   Speciale 模式：`deepseek/deepseek-v3.2-speciale`

非思维模式：`deepseek/deepseek-v3.2`

思维模式：`deepseek/deepseek-v3.2-thinking`

Speciale 模式：`deepseek/deepseek-v3.2-speciale`

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'deepseek/deepseek-v3.2-speciale',5  prompt:6  `Design a self-contained, step-by-step solution to a novel math–algorithm7   hybrid problem: prove correctness, derive complexity, and construct8   an optimal implementation for the general case.`,9});
```

AI Gateway 提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商水平的运行时间。它包含内置的**可观测性**、**自带密钥支持**以及具有自动重试功能的智能**提供商路由**。

阅读**文档**，查看 **AI Gateway 模型排行榜**，或直接在我们的**模型游乐场**中使用 DeepSeek V3.2 模型。

AI Gateway：按使用量追踪顶级 AI 模型

AI Gateway 模型排行榜根据通过网关的所有流量的总 Token 量，对一段时间内使用最多的模型进行排名。榜单定期更新。
