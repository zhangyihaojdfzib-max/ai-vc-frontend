GPT-5.4 和 GPT-5.4 Pro 现已在 AI Gateway 上可用。

该模型将 GPT-5.3-Codex 的 Agent（智能体）与推理能力飞跃带到了所有领域。这除了编码之外，还包括报告、电子表格、演示文稿和分析等知识工作。它能更可靠地处理复杂的多步骤工作流，包括涉及工具使用、研究以及从多个来源提取信息的任务。与之前的迭代版本（GPT-5.2）相比，GPT-5.4 速度更快，Token 使用效率也更高。GPT-5.4 Pro 面向需要在最复杂任务上获得极致性能的开发者。

要使用此模型，请在 AI SDK 中将模型设置为 `openai/gpt-5.4` 或 `openai/gpt-5.4-pro`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'openai/gpt-5.4',5  prompt:6    `Refactor our API layer to use the new auth SDK, update the7     affected tests and docs, then generate a migration guide8     for the rest of the team.`,9});
```

AI Gateway 提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于供应商水平的运行时间。它包含内置的可观测性、自带密钥支持以及具有自动重试功能的智能供应商路由。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
