Builds 和 Functions 现已支持 Python 3.13 和 Python 3.14，同时继续支持先前已支持的 Python 3.12。未指定 Python 版本的项目默认继续使用 Python 3.12。

默认版本将在未来几个月内切换至 Python 3.14。如需继续使用 Python 3.12，请在您的项目清单（pyproject.toml 或 Pipfile）中指定一个上限版本，如下方示例所示。

```
1[project]2...3requires-python = "~=3.12.0"
```

在 pyproject.toml 中指定 Python 版本

```
1[requires]2...3python_version = "3.12"
```

在 Pipfile 中指定 Python 版本

请参阅Python 文档以了解更多关于 Vercel 上 Python 支持的信息。
