next-forge 是一个面向 Next.js 应用的生产级 Turborepo 模板，旨在为新应用提供一个全面且包含预设技术栈的起点。

此主要版本带来了多项开发者体验改进、一项 Agent（智能体）技能，以及新的快速入门、Docker 和迁移路径指南。

next-forge 技能

现在，您可以将 next-forge 技能安装到您首选的 Agent（智能体）中，使其具备关于 next-forge 架构、软件包和常见任务的结构化知识。

```
1npx skills add vercel/next-forge
```

默认使用 Bun

默认的包管理器现已改为 Bun。CLI 初始化脚本会在提示前检测您当前使用的包管理器，同时通过初始化流程仍支持 pnpm、npm 和 yarn。

优雅降级

现在，当缺少环境变量时，每个可选的集成都会静默降级，而不是抛出错误。Stripe、PostHog、BaseHub 和功能标志都会返回安全的默认值。启动项目唯一必需的环境变量是 `DATABASE_URL`。

新指南

通过快速入门指南，您可以在几分钟内仅使用 Clerk 和一个 Postgres 数据库就启动一个运行中的开发服务器。

同时，还新增了 Docker 部署指南，并为 Appwrite（认证、数据库、存储）、Convex（数据库）和 Novu（通知）提供了迁移指南。

请阅读文档以开始使用。
