您现在可以通过 Vercel 的 **AI Gateway** 直接使用 Black Forest Labs 的最新图像模型 **FLUX.2 Pro**，无需其他供应商账户。

FLUX.2 Pro 是一个全新训练的基础模型，专为先进的视觉智能设计，提供更高分辨率的输出（最高达 4MP）、更丰富的现实世界知识，以及对光照和空间构图的精确控制。与 FLUX.1 系列模型相比，它引入了多参考输入、增强的角色与产品一致性、精确的色彩匹配以及更丰富的控制选项。

FLUX.2 Pro 与当前 AI Gateway 中其他具备图像生成能力的模型有根本性不同。该模型是一个纯粹的、专注于图像的整流流 Transformer 模型，这与已集成的多模态 LLM 形成对比。要使用此模型，请在 **AI SDK** 中将 `model` 参数设置为 `bfl/flux-2-pro`。此模型支持 `generateImage`。

```
1import { experimental_generateImage as generateImage } from 'ai';2
3const result = await generateImage({4  model: 'bfl/flux-2-pro',5  prompt:6    `Render a turquoise throated puffleg hummingbird7     resting on a branch covered with dew at sunrise.`,8  size: '1024x1024',9});
```

AI Gateway 提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于供应商水平的运行时间。它包含内置的**可观测性**、**自带密钥（BYOK）** 支持以及具备自动重试功能的智能**供应商路由**。

请阅读**文档**以获取关于如何通过 AI SDK 和 OpenAI 兼容 API 使用 FLUX.2 Pro 的更详细示例，查看 **AI Gateway 模型排行榜**，或直接在**模型游乐场**中试用这些模型。

AI Gateway：按使用量追踪顶级 AI 模型

AI Gateway 模型排行榜根据通过网关的所有流量的总 Token 量，对一段时间内使用最多的模型进行排名。榜单定期更新。
