# NVIDIA L40S GPU 现已上线

- zeke

今天，我们将 NVIDIA L40S GPU 添加到了我们支持的硬件类型中。这些新 GPU 的速度比 A40 GPU 快约 40%。

同时，我们将停止对 A40 GPU 的支持。我们将在未来几周内开始将所有现有模型和部署从 A40 GPU 迁移到 L40S GPU。您的私有模型和部署将继续按原价计费，但如果您在 A40 GPU 上使用公共模型或训练模型，费用可能会增加。

您现在可以为任何新模型、现有模型或部署运行 L40S GPU。要了解如何更改模型和部署的硬件类型，请查阅文档。

从今天起，您可以选择将任何现有模型和部署切换到 L40S GPU，但这不是强制要求。如果您选择不切换，您的模型和部署将在未来几周内继续在 A40 GPU 上运行，直到我们将它们迁移到 L40S GPU。

## 迁移时间表

为了给您提供更高的性价比，我们将在未来几周内开始将所有现有模型和部署从 A40 GPU 迁移到 L40S GPU：

*   **2024-12-02：公共模型迁移。** 我们将开始迁移所有在 A40 GPU 上运行的公共模型，并在 2024-12-04 前完成所有模型的迁移。
*   **2024-12-09：私有模型和部署迁移。** 我们将开始迁移所有在 A40 GPU 上运行的私有模型和部署，并在 2024-12-11 前完成所有模型的迁移。在此日期之后，A40 GPU 将不再可用。

## 速度与成本

L40S GPU 每秒的成本高于 A40 GPU，但速度也更快。

我们对当前使用率最高的 A40 模型进行的基准测试显示，从 A40 GPU 迁移到 L40S GPU 时，**中位速度提升约为 40%**。更多详情，请参阅我们的 Observable notebook，其中涵盖了我们的基准测试方法、数据收集方式和结果。

您的模型迁移到 L40S GPU 后的费用取决于您的使用场景：

*   **私有模型和部署：** 如果您当前在 A40 GPU 上运行私有模型或部署，这完全是个好消息：在我们将其迁移到 L40S GPU 后，您将继续支付与现在相同的价格。由于这些新 GPU 速度更快，您的账单很可能会下降，最坏情况也是保持不变。
*   **公共模型和训练：** 如果您在 A40 GPU 上使用公共模型或训练模型，迁移到 L40S GPU 后，您将按每小时 3.51 美元的新价格付费。
    *   如果您使用的是标准 A40，L40S GPU 的每小时价格高出 70%，但速度大约快 40%，因此您的账单预计大致保持不变。(1.7 × 0.6 ≈ 1)
    *   如果您使用的是大容量 A40，L40S GPU 的每小时价格高出 34%，但速度大约快 40%，因此您的账单预计会下降约 20%。(1.34 × 0.6 ≈ 0.8)

要比较我们所有可用硬件类型的性能和定价，请访问 replicate.com/pricing。

## 更新您的部署

如果您有使用 A40 GPU 的部署，您需要减少其最小实例数量，以避免产生更高费用。

例如，如果您的部署在 A40 GPU 上运行 **10 个最小实例**，那么在切换到 L40S GPU 时，您应将部署配置更改为 **6 个最小实例**，因为 L40S GPU 的速度大约快 40%。

您可以通过网页编辑您的部署配置或使用 HTTP API。

如果您不确定如何最佳配置您的部署，请发送邮件至 support@replicate.com 联系我们。
