Vercel Sandbox 现已支持创建仅配备 1 个 vCPU 和 2 GB 内存的沙盒环境。这对于单线程或轻量级工作负载非常理想，因为它们无法从额外的系统资源中获益。若未指定配置，默认仍为 2 个 vCPU 和 4 GB 内存。

您可以通过在 SDK 中设置 `resources.vcpus` 选项来开始使用：

```
1import { Sandbox } from "@vercel/sandbox";2
3const sandbox = await Sandbox.create({4  resources: { vcpus: 1 },5});
```

或在 CLI 中使用 `--vcpus` 选项：

```
1sandbox create --connect --vcpus 1
```

欲了解更多关于 Sandbox 的信息，请查阅文档。
