## Llama Nemotron Nano VL 简介

Llama Nemotron Nano VL 是 **NVIDIA Nemotron** 模型家族的最新成员，是一款旨在突破智能文档处理和光学字符识别边界的视觉语言模型。凭借其高精度、低模型占用和多模态能力，Llama Nemotron Nano VL 能够从复杂文档中无缝提取和理解信息。这包括 PDF、图像、表格、图表、公式和图示，使其成为自动化金融、医疗保健、法律和政府等各行业文档工作流的理想解决方案。

## 模型架构与创新

Llama Nemotron Nano VL 基于 **Llama-3.1-8B-Instruct** 和 **C-RADIOv2-VLM-H** 构建，后者是一个作为视觉特征提取骨干的 Vision Transformer。这使得模型能够处理文档中的各种视觉元素，包括图表、图形和其他复杂的视觉表示。

### 核心技术

#### 强大的视觉基础

**C-RADIOv2-VLM-H Vision Transformer**：模型的核心视觉理解组件，C-RADIO 允许对包含复杂视觉元素的文档进行高分辨率处理。它作为视觉骨干，在多个视觉领域表现出色，并使模型能够以高分辨率处理多图像理解。这项技术支撑了模型处理包含图像、图示、图表和表格等视觉元素的复杂文档的能力。

C-RADIO 使用多种蒸馏技术在多分辨率数据上进行训练。在训练期间对权重应用了乘法噪声以提高泛化能力。

Llama Nemotron VL 进一步采用了一种动态聚合编码补丁特征的设计，从而能够在保持空间连续性的同时支持高分辨率输入。该策略能高效处理任意纵横比的文档，同时保留局部细节和全局上下文。它使得对密集视觉内容（如小字体、多栏布局和复杂图表）进行细粒度分析成为可能，而不会影响计算效率或覆盖范围。由于高分辨率分块技术的创新，模型还能更好地保留信息并减少失真。

通过为 Llama-3.1 8B LLM 配备这一强大的视觉基础，Llama Nemotron Nano VL 在文档解析和解释方面提供了无与伦比的准确性。

#### 用于文档智能的高质量数据

Llama Nemotron Nano VL 使用多个开源数据集以及来自 NVIDIA 基于 VLM 的 OCR 解决方案 **NeMo Retriever Parse** 的数据进行训练。这提供了文本和表格解析以及定位能力，使 Llama Nemotron Nano VL 在文档理解任务中达到行业领先水平。用于训练此 OCR 解决方案的合成表格提取数据集也用于训练 Llama Nemotron Nano VL 8B VLM，以实现更优化的表格理解和提取。

Llama Nemotron Nano VL 在文本识别和视觉推理等任务中表现出色，并展示了先进的图表和图示理解能力。Llama Nemotron Nano VL 允许预测归一化空间中的边界框坐标，以实现类似定位的任务和文本引用。

这种强大的性能得益于高质量领域内数据与跨文档类型、语言和布局的多样化训练分布的结合。稳健的数据策略通过选择性筛选、针对性增强和阐明任务意图、减少歧义的格式化技术，确保了在具有挑战性的用例中的覆盖范围，从而产生了能够有效泛化到实际应用的模型。

##### 预训练

Llama Nemotron Nano VL 经历了两阶段的训练方案：预训练，然后是监督微调。
初始的预训练阶段侧重于实现语言和视觉领域之间的跨模态对齐。这是通过训练一个多层感知机连接器来实现的，该连接器充当两种模态之间的接口。

在训练过程中，Llama Nemotron Nano VL 利用了全面且多样化的数据集集合。这个汇总的数据集总共包含约 150 万个样本，包括公开可用的、合成的以及内部整理的数据集。图 1 展示了预训练阶段使用的数据集构成摘要。

##### 监督微调

在监督微调阶段，Llama-Nemotron-Nano-VL 在合成、公开和内部整理的数据集组合上进行端到端训练。数据涵盖广泛的任务，包括但不限于：OCR、文本定位、表格解析和基于文档的通用视觉问答。

Llama Nemotron Nano VL 的文档理解能力很大程度上归功于以 OCR 为重点的 SFT 数据混合。除了简单的 OCR，许多数据集还涉及预测正确的阅读顺序、重建 Markdown 格式以及语义类别（如标题、标题、章节标题）和单个文本块的边界框等任务。该模型还被训练用于解析 LaTeX 格式的数学公式，并根据提示以 LaTeX、HTML 或 Markdown 格式提取表格。

为确保模型在不同领域的鲁棒性，我们对文档图像应用了仿射变换和光度增强。为进一步提升表格和图表解析性能，我们启用了跨数据集间全页文档内嵌表格与图表的互换机制。这使得模型能够处理多样化的文档版式与结构。

内部创建的数据集大部分基于Nemo Retriever Parse的训练数据。其中包括：NVPDFTex——包含带真实标签的arXiv文档集合，标签内容为按阅读顺序排列的格式化文本及其边界框与语义类别，以及LaTeX表格和公式；由人工标注的Common Crawl PDF文件；来自Wikipedia的带Markdown格式和表格的渲染文本；以及多个旨在提升表格解析能力和密集OCR性能的合成数据集。此外，训练混合数据还包含多个公开数据集，如DocLayNet、FinTabNet和PubTables-1M，我们对这些数据集的真实标签进行了优化。
