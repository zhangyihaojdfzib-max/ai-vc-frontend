您现在可以通过 Vercel Sandbox CLI 为正在运行的沙盒开启安全、交互式的 shell 会话。

```
1pnpm i -g sandbox2sandbox login3sandbox create # 如果您没有正在运行的沙盒可供 SSH 连接4sandbox ssh <sandbox-id>
```

注意：在连接期间，沙盒的超时时间会自动以 5 分钟为单位延长，以避免意外断开连接，最长可延长至 5 小时。

了解更多信息，请查阅 Sandbox CLI 文档。
