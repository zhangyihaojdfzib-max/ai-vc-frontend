# 扩展Project Glasswing

![扩展Project Glasswing](/images/posts/c353b28cff0a.jpg)

Project Glasswing是我们为保护全球最重要软件而开展的协作努力。四月初，我们宣布约50家初始合作伙伴已获得Claude Mythos Preview的访问权限，此后，他们一直在部署该模型以扫描其代码库中的漏洞。我们最近描述了这些合作伙伴迄今已发现超过10,000个高危或严重级别的安全缺陷。

我们现在正在扩展Project Glasswing。经过与我们的Project Glasswing合作伙伴、安全行业、开源软件维护者以及美国政府数周的紧密合作，我们正在将合作伙伴关系扩展至约150家新组织。每家组织在获得访问权限前都需要满足我们的安全要求。

这一新群体中的组织分布在超过15个国家，其中大多数为更多国家提供关键基础设施。（未来，我们计划进一步扩大地理覆盖范围。）该群体涵盖了我们初始群体中代表性不足的几个行业，如电力、水务、医疗、通信和硬件。许多新合作伙伴是供应商——即维护着全球众多其他组织（包括政府）所依赖代码库的公司或非营利组织。

每位合作伙伴的共同点是，对其代码库的成功攻击可能造成灾难性后果。对于大多数合作伙伴，我们估计一次重大攻击可能影响超过1亿人，并对全球和国家安全产生重要影响。

此次扩展是我们迈向长期目标的下一步：让AI使所有软件更加安全，并帮助行业适应AI可能改变网络安全许多核心假设的方式。

## Project Glasswing的角色

Project Glasswing和Claude Mythos Preview的能力引发了关于AI如何改变网络安全的广泛讨论——无论是在软件行业内部还是与政府之间。这些讨论影响了我们扩展该项目的方式，也塑造了我们对Project Glasswing根本目的的思考。

具备强大网络能力的廉价、快速AI模型即将到来。我们希望Project Glasswing能促使各机构建立反映这一现实的运营规范。

Mythos Preview延续了我们一段时间以来一直警告的长期趋势：在未来6到12个月内，我们预计许多其他AI公司将拥有Mythos级别的模型，并且它们可能会在缺乏防止滥用的保障措施的情况下发布这些模型。在那个世界里，网络攻击可能更频繁地发生，且形式更加不可预测。网络防御者必须适应以跟上步伐。

我们将自身角色视为双重的。首先，通过安全地提供更广泛访问更优模型、工具和通用基础设施，帮助软件行业适应。其次，稳步转变我们提供的支持，从发现漏洞转向披露、修复和部署修补后的软件。我们现在将依次讨论这两点。

### 支持网络防御者

迄今为止，公司、非营利组织、维护者和研究人员行动迅速。在Project Glasswing启动的最初几周内，每位成员就开始大规模使用Mythos Preview，与其他合作伙伴分享信息和最佳实践，并与第三方合作对模型的发现进行分类。这些组织适应新工具的方法可以且应该被广泛复制到数百万易受网络攻击的组织和开发者中。

为支持这一点，我们最近发布了Claude Security，该产品使用我们最新的公开前沿模型（如Claude Opus 4.8）来扫描代码库并提出补丁建议。我们还应要求向可信的安全团队发布了我们开发的工具，以帮助Project Glasswing的合作伙伴更快地发现漏洞。

我们计划走得更远：我们的长期目标是支持行业为强大网络模型时代创建新的倡议、标准和基础设施。

### 加速修补及其他安全工作

正如我们之前讨论的，网络安全的瓶颈现在在于验证、披露和修补Mythos级别模型能够发现的大量漏洞。

Mythos Preview本身可以提供帮助。Project Glasswing的许多合作伙伴现在使用该模型编写补丁，以及进行发布前检查，从而从一开始就防止漏洞出现。像Mythos Preview这样的模型还可用于渗透测试（模拟网络攻击以识别漏洞可能如何被利用）、自动化威胁检测和响应，以及用内存安全语言重建遗留代码库等众多防御任务。

我们正在与第三方讨论如何大规模提升开源软件漏洞的审查和修补工作。我们还在分享向开源维护者披露漏洞的想法和最佳实践，旨在使这些报告更易于分类和处理。

## 未来之路

为应对即将到来的挑战规模，数十万组织、研究人员和维护者可能需要访问最先进的网络能力和可用工具。

我们正在尽可能快地安全地以通用访问方式发布Mythos级别能力。为此，我们需要高度稳健的保障措施来防止模型的网络能力被滥用——这些保障措施我们（以及据我们所知，所有其他AI开发者）尚未开发出来。由于网络安全既有有益用途也有破坏性用途，制作既强大又足够精确的保障措施是一项重大挑战。

与此同时，我们计划进一步扩展Project Glasswing——优先考虑更多关键基础设施提供商、关键开源软件维护者和安全测试人员。我们打算像本次扩展一样，未来扩展涵盖美国和海外的组织。我们还计划扩大我们的网络验证计划，该计划将为更多组织授予Mythos级别能力用于特定网络防御任务。

未来，前沿模型的发布将变得越来越高风险。能力将在所有领域持续提升，包括许多像网络安全这样既能赋能攻击者也能赋能防御者的领域。这不会是我们最后一次需要面对这样的挑战。但Project Glasswing教会了我们很多关于当模型跨越重要能力阈值时如何应对的知识。如果我们成功，我们希望为防御者创造持久的优势。
