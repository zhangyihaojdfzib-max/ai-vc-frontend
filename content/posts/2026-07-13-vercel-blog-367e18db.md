## 复制链接到标题AI 投资持续攀升，平均 Token 价格趋于平稳

6 月，Token 量和支出以几乎相同的速度增长，分别为 29% 和 27%，而每 Token 价格保持稳定。前一个月，支出增长速度是量的两倍多，推动平均 Token 成本上涨近 20%。

市场总体支出增加，但每 Token 支出并未增加。自 4 月以来，开源权重模型已从所有 Token 量的九分之一攀升至近三分之一，而 Gateway 上的平均 Token 价格约为十分之一。仅此一项本应拉低每 Token 价格。但随着 6 月廉价量的上升，闭源前沿模型的价格也随之上涨，每 Token 上涨约 12%。它们相互抵消，因此每 Token 平均价格持平。

这证明了 6 月报告所记录的路由纪律，现在在总量中可见：高量工作流向低成本模型，高风险工作留在前沿。对 AI 的投资仍在快速增长，企业正更有策略地平衡各模型之间的支出。

## 复制链接到标题开源权重运行了近三分之一的生产 Token

开源权重模型在 6 月运行了 29% 的 Gateway Token，支出仅占不到 4%。这相当于以二十五分之一的成本获得了近三分之一的 Token。

自 4 月以来，开源权重 Token 份额几乎增长了两倍。其中大部分量来自 DeepSeek 的持续增长，目前以 22.6% 的份额成为 Gateway 上第三大 Token 来源，仅次于 Anthropic 和 Google。随着 4 月的激增消退，Google 的 Token 量份额下滑至 24%，使 DeepSeek 距离第二名不到两个百分点。

大约八分之一的企业客户现在在生产中运行开源权重模型。按照当前趋势，一个开源权重实验室很快将成为 AI Gateway 上按量计算的第二大实验室。

最新进入者显示了这一模式的加速。Z.ai 发布了 GLM 5.2，一个 MIT 许可的开源权重模型，旨在以约 Opus 4.8 五分之一的价格进行长期 Agent 工作。从 6 月 16 日 API 可用到月底，其每日 Token 量增长了约 50 倍。它在最后一周按 Token 量在 AI Gateway 上排名第 11，单日最高排名第 7。它在短短两周内占据了其家族 6 月 Token 量的 76%。Gemini 3.1 Pro，我们之前记录的最快家族内迁移，用了第二个月才达到类似份额。

## 复制链接到标题前沿实验室几乎保留了所有支出

6 月，美国四大前沿实验室占据了 AI Gateway 支出的 95%。仅 Anthropic 就以 32% 的 Token 量占据了 61% 的支出，低于 5 月的 65%，但与 4 月持平。在后果最严重的用例中，即错误代价高昂的领域：编码 Agent、后台 Agent 和应用生成，它占据了 72% 或更多的支出。

OpenAI 的 Token 份额从 12.5% 降至 10.3%，而其支出份额从 13.3% 升至 16.1%，推动其每 Token 成本在一个月内相对于市场上涨约 50%。与 DeepSeek 相反，客户向 OpenAI 发送了更少的量但成本更高的工作，这种差异只出现在生产数据中。

## 复制链接到标题每种模态都有不同的领导者

图像生成在 AI Gateway 上是两个实验室的竞赛。OpenAI 的 GPT Image 家族在 6 月生成了 53% 的图像，并占据了 52% 的图像支出。Google 的 Nano Banana 生成了近 39% 的图像，并占据了 43% 的支出。没有其他家族在任一指标上超过 5%。

视频是中国实验室目前占据高端市场的唯一模态。字节跳动的 Seedance 以 49% 的支出领先，仅生成了三分之一的视频。xAI 的 Grok Imagine 以 19% 的支出生成了 42% 的视频，与 DeepSeek 在文本中的量-折扣地位相同。中国实验室，包括 Seedance、Kling 和阿里巴巴的 Wan，共同占据了约三分之二的视频支出。该类别尚处于早期，因此我们预计这些份额会波动。

Anthropic 在文本领域领先，OpenAI 在图像领域领先，xAI 在视频量上领先，字节跳动在视频支出上领先。要在每种模态中运行领先模型，需要跨至少三个实验室进行路由。
