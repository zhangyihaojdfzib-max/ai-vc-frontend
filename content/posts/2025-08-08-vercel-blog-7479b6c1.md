# 批量升级已弃用的 Node.js 版本

![Vercel Upgrade Deprecated Node.js Version modal](/images/posts/5581a13f350c.jpg)

![Vercel Upgrade Deprecated Node.js Version modal](/images/posts/40d69b2bf97a.jpg)

团队所有者和成员现在可以在 Vercel 仪表板中一键将所有使用 Node.js 18 或更早版本的项目升级到 Node.js 22。

此操作会更新项目设置中的 `Node.js` 版本。如果你的项目还在 `package.json` 中定义了版本，则需要手动更新。现有部署不受影响。

立即查看并升级已弃用的 Node.js 项目。
