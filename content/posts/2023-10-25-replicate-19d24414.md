# 在Mac上使用潜在一致性模型一秒生成图像

- fofr

潜在一致性模型（LCMs）基于Stable Diffusion构建，但能更快生成图像，仅需4到8步即可获得优质图像（相比25到50步）。通过在M1或M2芯片的Mac上运行LCM，您可以每秒生成一张512x512的图像。

Simian Luo等人发布了首个Stable Diffusion蒸馏模型。该模型通过将无分类器引导融入模型输入，从Dreamshaper微调版本蒸馏而来。目前仅蒸馏出一个模型，但更多版本即将发布。论文作者正在对Stable Diffusion 2.1和SDXL进行适配。

您可以在Replicate云端运行首个潜在一致性模型，也可在本地运行。除了生成预测结果，您还可以进行修改、调整并构建新应用。

我们编写了本指南帮助您入门。

## 环境要求

您需要：
- 配备M1或M2芯片的Mac
- 16GB或更高内存
- macOS 12.3或更高版本
- Python 3.10或更高版本

我们发现配备32GB内存的M1 Max或M2芯片能在1秒内生成图像。16GB内存的M1 Pro芯片需要2到4秒。请在GitHub仓库分享您的测试数据。

## 配置Python

需要Python 3.10或更高版本。运行`python -V`查看已安装版本：

$ python3 -V
Python 3.10.6

若版本为3.10或更高（如此处所示），可直接进入下一步。

否则需要安装Python 3.10。最简单的方式是通过Homebrew安装。若未安装Homebrew，请先安装。

然后安装最新版Python：

brew update
brew install python

现在运行`python3 -V`应显示3.10或更高版本。可能需要重启终端才能生效。

## 克隆仓库并安装依赖

运行以下命令从GitHub克隆LCM脚本：

git clonehttps://github.com/replicate/latent-consistency-model.gitcd latent-consistency-model

然后创建虚拟环境安装依赖：

python3 -m pip install virtualenv
python3 -m virtualenv venv

激活虚拟环境：

source venv/bin/activate

（每次运行脚本时都需要重新执行此命令）

接着安装依赖：

pip install -r requirements.txt

## 运行模型！

现在可以运行潜在一致性模型了。脚本将自动从HuggingFace下载`SimianLuo/LCM_Dreamshaper_v7`（3.44 GB）模型和安全检测器（1.22 GB）。

```
python main.py \
  "a beautiful apple floating in outer space, like a planet" \
  --steps 4 --width 512 --height 512
```

您将看到如下输出：

```
Output image saved to: output/out-20231026-144506.png
Using seed: 48404
100%|███████████████████████████| 4/4 [00:00<00:00,  5.54it/s]
```

我们还添加了`--continuous`参数，可连续生成图像直至硬盘存满。首次生成后的速度会更快。

```
python main.py \
  "a beautiful apple floating in outer space, like a planet" \
  --steps 4 --width 512 --height 512 --continuous
```

完成！
