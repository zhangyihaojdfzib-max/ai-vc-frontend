# 使用 ComfyUI 在本地机器上运行 Stable Diffusion 3

- zsxkib
- zeke

Stable Diffusion 3 (SD3) 刚刚发布，你可以在 Replicate 上云端运行它，但同样也可以在你自己的配备 GPU 的机器上，使用 ComfyUI 在本地运行它。

## 什么是 Stable Diffusion 3？

Stable Diffusion 3 (SD3) 是 Stability AI 根据文本生成图像工具的最新版本。它比旧版本速度更快，生成的图像质量更好。SD3 非常擅长根据提示词生成包含大量细节的复杂场景和清晰的文字。

![一位巫师手持两个标牌，左手标牌写着‘SD3’，右手标牌写着‘COMFY’“](/images/posts/4a7b3a5d34b4.webp)

## 什么是 ComfyUI？

ComfyUI 是一个为 SD3 等 Stable Diffusion 模型设计的图形用户界面 (GUI)。它允许你将不同的 AI 模型（称为节点）连接在一起以创建自定义图像，就像拼接乐高积木一样。你无需了解任何代码即可使用它。ComfyUI 让尝试不同组合（称为“工作流”）和制作独特图片变得有趣而简单。

特别感谢 `comfyanonymous` 创建了 ComfyUI，以及 `fofr` 创建了其 Cog 兼容版本。

## 前提条件

要遵循本指南，你需要：

- 一台配备 GPU 的机器（推荐使用至少 8GB 显存的 NVIDIA GPU）
- 能够在终端中复制和粘贴 😉

## 步骤 1：安装 Cog

首先安装 `Cog`。这是一个工具，用于在容器环境中设置打包和运行机器学习模型所需的一切。打开终端并运行以下命令：

sudo curl -o /usr/local/bin/cog -L “https://github.com/replicate/cog/releases/latest/download/cog_$(uname-s)_$(uname -m)”
sudo chmod +x /usr/local/bin/cog

## 步骤 2：克隆 ComfyUI 仓库

接下来，从 GitHub 克隆 ComfyUI 代码：

git clone —recurse-submodulesgit@github.com:fofr/cog-comfyui-sd3.git
cd cog-comfyui-sd3

## 步骤 3：启动 ComfyUI

启动 ComfyUI。运行此命令以下载权重并启动 ComfyUI Web 服务器：

sudo cog run —use-cog-base-image -p 8188 /bin/bash -c “python scripts/get_weights.py workflow_api.json && cd ComfyUI/ && python main.py —listen 0.0.0.0”

这可能需要一些时间，请放松并等待其完成。

## 步骤 4：打开 ComfyUI Web 界面

ComfyUI 启动后，打开你的 Web 浏览器并访问 `http://<你的IP地址>:8188`。如果你不知道机器的 IP 地址，请在终端中运行此命令：

hostname -I

就是这样！你现在应该可以看到准备就绪的 ComfyUI 界面了。

## 步骤 5：加载 SD3 工作流

如果你没有 SD3 工作流文件，不用担心。你可以使用以下命令从 GitHub 下载：

curl -O “https://raw.githubusercontent.com/fofr/cog-comfyui-sd3/main/workflow_api.json”

下载 `workflow_api.json` 文件后，打开 ComfyUI GUI，点击“Load”，然后选择 `workflow_api.json` 文件。

## 步骤 6：生成你的第一张图片

转到“CLIP Text Encode (Prompt)”节点，该节点初始没有文本，输入你想看到的内容。例如，`"cat on a fridge"`。

“CLIP Text Encode (Negative Prompt)”节点已经预先填充了你不想在图像中出现的内容列表，但你可以随意更改。

最后，点击“Queue Prompt”按钮来生成你的第一张图片。现在，你可以开始创作令人惊叹的视觉作品了！
