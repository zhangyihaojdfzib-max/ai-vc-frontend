# GPT‑5.6 现已成为 Microsoft 365 Copilot 中的首选模型

今天，OpenAI 发布了 GPT‑5.6，该模型将成为 Microsoft 365 Copilot（包括 Word、Excel、PowerPoint、Chat 和 Cowork）中的新首选模型。对于 Microsoft 365 客户而言，此次更新将 OpenAI 最新的旗舰模型系列引入人们日常使用的生产力工具中，帮助他们在各项工作流程中借助更强大的 AI 辅助进行创作、分析和协作。GPT‑5.6 是 OpenAI 最新的旗舰模型系列，能够从每个 Token 中产出更多有用的工作成果，在每美元成本下提供更强的性能，并具备应对最复杂任务的按需能力。

借助 GPT‑5.6，Microsoft 365 用户将能够在他们已依赖的各类应用中，以更少的精力创建更高质量的工作成果：

- 在 Word 中，GPT‑5.6 可以帮助用户起草、编辑和润色文档，减少提示词轮次。
- 在 Excel 中，它可以在更高效使用 Token 的同时支持更深入的分析，帮助用户更快地从数据中获得洞察。
- 在 PowerPoint 中，它可以帮助用户将初步想法转化为更精致、视觉上更具吸引力的演示文稿，减少手动指导。
- 在 Cowork 中，它可以帮助用户完成复杂的跨职能工作，并以更少的手动协调产出更高质量的成果。

“我们迫不及待地想让客户看到 GPT‑5.6 在 Microsoft 365 中的表现，让他们能够在日常使用的工具中更有效地与 AI 协同工作，”Microsoft 的 Copilot 与 Agent 核心部门总裁 Nitin Agrawal 表示。“借助由 OpenAI 最新模型驱动的 Copilot，客户将能够在 Word、Excel、PowerPoint、Cowork 和 Copilot Chat 中产出更精致的成果，无论是起草文档、分析数据、创建演示文稿，还是跨团队协作。我们很高兴能继续与 OpenAI 合作，为全球的个人和组织带来更强大的 AI 体验。”

“Microsoft 365 是数百万人每天写作、分析、创作和协作的地方，”OpenAI 的 API 产品负责人 Nikunj Handa 表示。“通过 OpenAI API 将 GPT‑5.6 引入 Microsoft 365 Copilot，我们正在帮助组织从每个 Token 中产出更多有用的工作成果，并从他们已使用的工具中获得更多 AI 价值。”

除了原生提供这些模型外，Microsoft 还将直接通过 API 访问 OpenAI 模型，将 GPT‑5.6 带给 Microsoft 365 客户。

我们与 Microsoft 的合作始终致力于将先进 AI 的益处带给更多个人和组织，我们很高兴能继续在这一共同承诺的基础上不断前行。
