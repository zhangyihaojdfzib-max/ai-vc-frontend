# Anthropic 承诺向加拿大人工智能研究投入 1000 万加元

加拿大机构和研究人员在现代人工智能时代发挥了关键作用。在人们对神经网络研究普遍持怀疑态度的时期，多伦多大学和蒙特利尔大学是为数不多培育这一关键研究方向的机构中的两家，而阿尔伯塔大学的研究人员在强化学习方面做出了开创性工作。在 2010 年代初期，加拿大研究机构率先证明，随着强大的新计算资源——特别是通用 GPU——的到来，深度神经网络能够大规模成功，开启了现代时代。

如今，加拿大国内外的研究人员继续在人工智能研究、安全和政策领域发挥主导作用——包括在 Anthropic。这就是为什么我们承诺向加拿大研究机构提供 1000 万加元，用于资助下一代研究工作。我们还基于 Anthropic 经济指数发布了首份加拿大国家简报，概述了加拿大人如何将 Claude 应用于工作。

### 投资加拿大研究

我们承诺的 1000 万加元将用于资助有益且负责任的人工智能应用研究。作为其中的一部分，我们宣布与加拿大三大区域人工智能研究所——埃德蒙顿的阿尔伯塔机器智能研究所（Amii）、蒙特利尔的 Mila 和多伦多的 Vector 研究所——以及 CHEO、成瘾与精神健康中心（CAMH）、拉瓦尔大学、多伦多大学和萨斯喀彻温大学建立合作伙伴关系，未来几个月还将有更多合作伙伴加入：

- 位于埃德蒙顿的 **Amii** 将为其研究和工程团队提供 Claude 积分，以推进他们在强化学习、人工智能可信度与安全等领域的工作，并促进加拿大关键经济领域对人工智能的采用。
- **Mila**（魁北克人工智能研究所，拥有全球最集中的学术深度学习研究人员）将向其社区提供 Claude，以支持他们在负责任人工智能、健康、可持续性、多智能体系统和机器人等领域的研究。Mila 还将使用 Claude 开发人工智能助手，帮助研究人员发现和评估科学突破。
- 多伦多的 **Vector 研究所** 将使用 Claude 积分推进人工智能在可信度与安全、健康与科学以及人工智能独特优势能够帮助加拿大人解决的更广泛挑战方面的研究。
- **CHEO 和 CHEO 研究所** 将使用 Claude 积分开发和评估基于人工智能的方法，旨在改善儿童、青少年及其家庭的健康结果和护理体验，并研究如何最负责任地将人工智能应用于儿童健康领域。
- **CAMH** 将把 Claude 积分用于其研究、教育项目和临床项目中。例如，CAMH 的 Krembil 神经信息学中心将使用 Claude 进行计算精神健康研究，包括开发和评估针对精神健康疾病患者的治疗预测模型，以及进行精神病学人工智能系统公平性的大规模评估。积分还将用于帮助扩大 CAMH 全球学习学院的影响力，该学院正在开发多语言、基于人工智能的精神健康教育。
- **拉瓦尔大学智能与数据研究所** 将与 Claude 合作，加深研究人员对 LLM 在不同文化背景下行为方式的理解，以及对低资源语言和方言（如魁北克法语和原住民语言）的理解。
- **萨斯喀彻温大学** 将使用 Claude 推进其在生物医学进展、食品与水资源安全、公共卫生、量子计算和公共服务等领域的研究。
- **多伦多大学数据科学研究所** 将通过基于科学评审的流程支持一系列研究项目，以获取 Claude API 积分。

除了这些捐赠，今年夏天 Anthropic 还将把 Amii、Mila 和 Vector 纳入 **Anthropic for Startups 计划**，该计划为基于 Claude 构建的创始人提供社区和资源支持其成长。数百家隶属于这些机构的加拿大初创公司将各获得至少 5000 美元的 API 积分，以继续发展其业务。

"现代人工智能的一些基础来自多伦多、蒙特利尔和埃德蒙顿——同样引人注目的是，许多最致力于确保其安全的研究人员也来自这些地方，"Anthropic 联合创始人 Chris Olah 表示。"我深受这种文化的影响，我为 Anthropic 能够支持下一篇章感到自豪。"
