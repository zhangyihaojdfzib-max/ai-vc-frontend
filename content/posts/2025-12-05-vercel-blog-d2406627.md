1 分钟阅读
您现在可以使用今日发布的全新 Vercel for Platforms 产品构建平台，轻松代表您的用户创建和运行客户项目。

平台提供两种模式：多租户模式与多项目模式，允许您通过单一代码库或多个代码库，在任意数量的域名上进行部署。

### 多租户平台
运行单一代码库，服务众多客户，功能包括：
*   通配符域名 (`*.yourapp.com`) 支持，具备自动路由和 SSL 功能。
*   通过 SDK 支持自定义域名，包括 DNS 验证和证书管理。
*   用于边缘主机名解析和客户识别的路由中间件。
*   单一部署模式：一次部署，变更适用于所有租户。

数秒内为您的应用添加自定义域名：
```javascript
import { addDomain, getDomainStatus,} from "@/components/vercel-platform/src/actions/add-custom-domain";

const added = await addDomain("test.com");
if (added.status === "Valid Configuration") { // do something}
const config = await getDomainStatus("test.com");
config.dnsRecordsToSet; // show this in a table
```

### 多项目平台
为每位客户创建独立的 Vercel 项目，功能包括：
*   使用 Vercel SDK 以编程方式创建项目。
*   每位客户的构建、函数、环境变量和设置相互隔离。
*   每个项目支持不同的框架。

数秒内将客户代码部署到独立项目中：
```javascript
import { deployFiles } from "@/components/vercel-platform/actions/deploy-files";

// automatically detects the framework & build commands
await deployFiles([], { // optionally assign a custom domian domain: "site.myapp.com",});
```

今日我们还推出了 Platform Elements，这是一个新的库，旨在让平台构建更加便捷。

立即开始使用我们的多租户或多项目平台快速入门进行构建。
