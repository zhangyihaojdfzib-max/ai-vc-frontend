与总收入或新客户增长不同，NRR反映的是公司从现有客户基础上增长收入的能力。

NRR大于100%意味着升级、增购和交叉销售超过了客户流失、取消和降级，这表明收入增长具有可持续性。NRR小于100%则表明客户群中产生收入的活动在收缩。

NRR被认为是B2B SaaS和基于订阅的B2C中最重要的指标之一，因为它衡量的是随时间推移的扩展收入，而不仅仅是客户留存。

你也会听到NRR被称为净美元留存率或简称为净收入留存率。

## NRR与总留存率（GRR）

很容易将NRR与总留存率（GRR）混淆：

- GRR显示的是在考虑流失、降级和取消后，你保留的订阅收入，**不**计入扩展收入。
- NRR则**包含**扩展收入，以更全面地反映你的净留存公式。

由于GRR忽略扩展收入，它永远无法超过100%。而NRR的设计使其可以远高于这个阈值。

| 指标 | 为何使用 | 典型上限 |
| :--- | :--- | :--- |
| GRR | 用于诊断留存问题、发现流失模式并评估客户是否持续获得价值。它隔离了取消和降级的影响。 | 最高100% |
| NRR | 用于了解来自现有客户群的总体收入持久性和增长，包括扩展收入。它显示增购、交叉销售和升级是否抵消了流失。 | 可超过100% |

在实际应用中，每个指标都有其独特的优势：

- GRR有助于发现留存问题，并在客户群中提供一致、积极的体验，而
- NRR有助于评估增销效果、产品市场契合度以及销售、客户成功和产品团队的综合影响。

## 如何计算NRR

NRR公式很直接：

NRR = (期初MRR + 扩展收入 – 降级 – 流失 – 取消) ÷ 期初MRR

其中：

- **期初MRR** = 特定时期开始时同一客户群的经常性收入
- **扩展收入** = 同一时期的升级、增购和交叉销售
- **收缩** = 同一时期的降级、流失和取消

示例：

- 期初MRR：100,000美元
- 扩展收入：20,000美元
- 降级 + 流失：10,000美元
- 期末MRR：110,000美元

NRR = 110,000美元 ÷ 100,000美元 = **110%**

| 因素 | 对NRR的影响 | 示例 |
| :--- | :--- | :--- |
| 升级 / 增购 | 提高NRR | 客户升级到更高层级 |
| 交叉销售 | | 客户增加另一个产品线 |
| 降级 | 降低NRR | 客户切换到更便宜的方案 |
| 流失 / 取消 | | 客户完全离开 |

更高的NRR表明你的留存和扩展策略正在发挥作用，并且你的现有客户随着时间的推移发现你的产品套件更有价值。

## 为什么NRR对SaaS公司很重要

对于SaaS公司来说，NRR是衡量财务健康、盈利能力和增长潜力的**关键指标**。获取新客户成本高昂，而扩展收入效率更高。它也表明客户正在从你的产品或产品套件中获得价值。

良好的净收入留存率告诉投资者你的商业模式是可持续的：即使没有持续获取新客户，你的基础收入也在增长。运营者利用它来制定定价、产品路线图和客户成功组织架构。

## 什么是好的NRR评级？

基准因细分市场、公司阶段和行业而异。早期公司在寻找产品市场契合度时，可能难以达到80%。在规模化阶段，最好的SaaS公司通常能持续实现**高于100%**的NRR。例如，在B2B SaaS领域：

- **100% NRR** = 基线（客户续约，无扩展）
- **高于100%** = 扩展抵消了流失；是强大SaaS业务的标志
- **120%+** = 常见于Snowflake或Zoom等品类领导者

## 提高NRR的策略

提高NRR与其说是追逐新客户，不如说是**最大化客户群的生命周期价值**：

- **减少流失**—— 强大的引导流程、主动的客户成功和支持循环，以阻止可预防的损失。
- **推动扩展收入**—— 将升级、增购和交叉销售纳入你的路线图。
- **优化定价策略**—— 年度订阅和分层模式鼓励长期合同，并为扩展创造空间。
- **投资客户留存**—— 让产品和客户成功团队围绕解决实际问题保持一致，而不仅仅是获取客户。

像Persona的Rick Song这样的创始人，依靠NRR而非纯粹的客户获取，建立了价值数十亿美元的公司。

而在AI等市场，像Writer的May Habib这样的领导者展示了企业扩展如何推动NRR。

## NRR的实际应用

在最佳状态下，NRR是一个**现实世界的运营指南**，而不仅仅是电子表格上的一个数字。它直接关系到财务健康、估值和规划，同时展示了业务重要部分的实力。

像任何指标一样，如果使用不当，NRR可能会产生误导。例如，具有强劲扩展的NRR可能掩盖新客户的“漏桶”情况。

当正确使用时，NRR成为健康增长业务的关键指标，显示你的收入流是否随着客户的增长而自然扩展。
