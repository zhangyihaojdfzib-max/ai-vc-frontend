# DeepInfra 现已登陆 Hugging Face 推理提供商 🔥

我们非常激动地宣布，DeepInfra 现已成为 Hugging Face Hub 上受支持的推理提供商！

DeepInfra 加入了我们不断发展的生态系统，增强了 Hub 模型页面上无服务器推理的广度和能力。推理提供商也已无缝集成到我们的客户端 SDK（适用于 JS 和 Python）中，让您能够轻松地使用偏好的提供商来调用各种模型。

DeepInfra 是一个无服务器 AI 推理平台，提供行业内最具性价比的每 Token 定价。凭借超过 100 个模型的目录，DeepInfra 使开发者能够以最少的设置，轻松将广泛的 AI 能力集成到他们的应用中。

DeepInfra 支持多种模型类型——从 LLM（大语言模型）到文生图、文生视频、嵌入/向量等。作为此次初始集成的一部分，DeepInfra 在 Hugging Face 上推出了对**对话和文本生成任务**的支持，使您能够访问流行的开源权重 LLM（大语言模型），例如 DeepSeek V4、Kimi-K2.6、GLM-5.1 等。**对其他任务**（文生图、文生视频、嵌入/向量等）的支持将很快推出！

请在其专门的**文档页面**中阅读更多关于如何使用 DeepInfra 作为推理提供商的信息。

在此处查看 DeepInfra 支持的完整模型列表。

在 Hugging Face 上关注 DeepInfra：https://huggingface.co/DeepInfra。

## 工作原理

### 在网站 UI 中

1. 在您的用户账户设置中，您可以：

- 为您已注册的提供商设置您自己的 API 密钥。如果未设置自定义密钥，您的请求将通过 HF 路由。
- 按偏好排序提供商。这适用于模型页面中的小部件和代码片段。

![推理提供商](/images/posts/0339ff6b511c.png)

1. 如前所述，调用推理提供商时有两种模式：

- 自定义密钥（请求直接发送到推理提供商，使用您自己的对应推理提供商的 API 密钥）
- 由 HF 路由（在这种情况下，您不需要提供商的 Token，费用将直接计入您的 HF 账户，而非提供商的账户）

![推理提供商](/images/posts/d2717fd3c654.png)

1. 模型页面展示第三方推理提供商（与当前模型兼容的提供商，按用户偏好排序）

### 从客户端 SDK

DeepInfra 可通过 Hugging Face SDK 使用——Python 的 `huggingface_hub`（>= 1.11.2）和 JavaScript 的 `@huggingface/inference`。

以下示例展示了如何通过 DeepInfra 使用 DeepSeek V4 Pro。使用 Hugging Face Token 进行身份验证——请求将自动路由到 DeepInfra。

#### 从您喜爱的 Agent（智能体）框架

Hugging Face 推理提供商已集成到大多数 Agent（智能体）框架中——包括 Pi、OpenCode、Hermes Agents、OpenClaw 等。这意味着您可以将 DeepInfra 托管的模型直接插入您喜爱的工具中，无需任何额外的胶水代码。在此处浏览完整的集成列表。

#### 从 Python

```python
import os
from openai import OpenAI

client = OpenAI(
    base_url="https://router.huggingface.co/v1",
    api_key=os.environ["HF_TOKEN"],
)

completion = client.chat.completions.create(
    model="deepseek-ai/DeepSeek-V4-Pro:deepinfra",
    messages=[
        {
            "role": "user",
            "content": "Write a Python function that returns the nth Fibonacci number using memoization."
        }
    ],
)

print(completion.choices[0].message)

```

#### 从 JS

```js
import { OpenAI } from "openai";

const client = new OpenAI({
    baseURL: "https://router.huggingface.co/v1",
    apiKey: process.env.HF_TOKEN,
});

const chatCompletion = await client.chat.completions.create({
    model: "deepseek-ai/DeepSeek-V4-Pro:deepinfra",
    messages: [
        {
            role: "user",
            content: "Write a Python function that returns the nth Fibonacci number using memoization.",
        },
    ],
});

console.log(chatCompletion.choices[0].message);

```

## 计费

对于直接请求，即当您使用推理提供商的密钥时，您由相应的提供商计费。例如，如果您使用 DeepInfra API 密钥，费用将计入您的 DeepInfra 账户。

对于路由请求，即当您通过 Hugging Face Hub 进行身份验证时，您只需支付标准的提供商 API 费率。我们不会额外加价；我们只是直接传递提供商的成本。（未来，我们可能会与我们的提供商合作伙伴建立收入分成协议。）

重要提示‼️ PRO 用户每月可获得价值 2 美元的推理积分。您可以在不同提供商之间使用它们。🔥

我们还为已登录的免费用户提供少量配额的免费推理，但如果可能，请升级到 PRO！
