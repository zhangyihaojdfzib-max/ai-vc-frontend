AI视频生成已经变得非常出色。

这种动态风格迁移是该实现独有的：其他仅在图像上训练的视频模型无法捕捉到这一点。

## 先决条件

*   一个Replicate账户
*   用作训练数据的视频或YouTube URL

## 第1步：创建您的训练数据

要训练视频模型，您需要一个包含视频片段和描述每个视频的文本字幕的数据集。

这个过程可能很耗时，因此我们创建了一个模型来简化它：[zsxkib/create-video-dataset](https://replicate.com/zsxkib/create-video-dataset)接收视频文件或YouTube URL作为输入，将其切片成较小的片段，并为每个片段生成字幕。

以下是如何在浏览器中仅需几次点击即可创建训练数据：

1.  找到您想用于训练的YouTube URL（或视频文件）。
2.  访问[replicate.com/zsxkib/create-video-dataset](https://replicate.com/zsxkib/create-video-dataset)
3.  粘贴您的视频URL，或从您的计算机上传视频文件。
4.  选择一个独特的触发词，如`RCKRLL`。避免使用已有实际关联的真实词汇。
5.  点击`运行`并下载生成的ZIP文件。

可选：如果您想查看每个片段的自动生成字幕，可以查看训练运行的日志。

## 第2步：训练您的模型

现在，您将使用刚刚编译的训练数据创建自己的微调视频生成模型。

1.  访问[replicate.com/zsxkib/hunyuan-video-lora/train](https://replicate.com/zsxkib/hunyuan-video-lora/train)
2.  为您的模型选择一个名称。
3.  对于`input_videos`输入，上传您刚刚下载的ZIP文件。
4.  输入您之前使用的相同触发词，例如`RCKRLL`
5.  调整训练设置（我们建议从2个轮次开始）
6.  点击`创建训练`

使用默认设置，训练通常需要大约5-10分钟，但具体时间取决于片段的大小和数量。

## 第3步：运行您的模型

训练完成后，您可以通过以下几种方式生成新视频：

*   直接在您的模型页面于浏览器中运行模型。
*   在Replicate的`Playground`中运行您的模型：转到“管理模型”并输入您的模型名称。
*   使用API：转到您的模型页面并点击`API`选项卡获取代码片段。

您只需几行代码即可将您的模型作为API运行。

以下是使用`replicate-javascript`客户端的示例：

```
import Replicate from "replicate"

const replicate = new Replicate()

const model = "your-username/your-model:your-model-version"
const prompt = "A lion dancing on a subway train the style of RCKRLL"
const output = await replicate.run(model, {input: { prompt }})
console.log(output)
```

## 第4步：实验以获得最佳结果

视频微调还相当新，所以我们仍在探索什么方法最有效。

以下是一些早期提示：

*   使用一个独特的`触发词`，该词不与真实词汇有关联。
*   尝试不同的训练设置：
    *   更多轮次 == 质量更好但训练时间更长
    *   调整LoRA秩
    *   增加批次大小以加速训练
    *   使用`max_steps`精确控制训练时长
*   如果训练看起来需要几个小时，请取消它并尝试：
    *   减少轮次数量
    *   降低秩
    *   增加批次大小
*   查看[GitHub README](https://github.com/replicate/musubi-tuner)获取详细的参数解释。

## 附加内容：以编程方式训练新模型

如果您想自动化此过程或构建应用程序，可以使用我们的API。

以下是如何使用Replicate Python客户端以编程方式训练新模型的示例：

```
import replicate
import time

# 从视频创建训练数据集
dataset = replicate.run(
    "zsxkib/create-video-dataset:4eb83cc8ba563da7032933374444a9a7a6f630b5b1e4f219cf9088f6a4acc138",
    input={
        "video_url": "YOUR_VIDEO_URL",
        "trigger_word": "UNIQUE_TRIGGER",
        "start_time": 10,
        "end_time": 40,
        "num_segments": 8,
        "autocaption": True,
        "autocaption_prefix": "a video of UNIQUE_TRIGGER,"
    }
)

# 创建一个新模型来存储训练结果
model = replicate.models.create(
    owner="your-username",
    name="your-model-name",
    visibility="public",
    hardware="gpu-t4"
)

# 使用处理后的视频开始训练
training = replicate.trainings.create(
    model="zsxkib/hunyuan-video-lora",
    version="04279caf015c30a635cabc4077b5bd82c5c706262eb61797a48db139444bcca9",  # 当前模型版本ID
    input={
        "input_videos": dataset.url,
        "trigger_word": "UNIQUE_TRIGGER",
        "epochs": 2,
        "batch_size": 8,
    },
    destination="your-username/your-model-name",  # 推送训练后模型的位置
)

# 等待训练完成
while training.status not in ["succeeded", "failed", "canceled"]:
    training.reload()
    time.sleep(10)  # 每次检查之间等待10秒

if training.status != "succeeded":
    raise Exception(f"Training failed: {training.error}")

# 使用您的微调模型生成新视频
output = replicate.run(
    training.output['version'],
    input={
        "prompt": "A video of UNIQUE_TRIGGER in a cyberpunk city",
        "num_frames": 45,
        "frame_rate": 24
    }
)
```
