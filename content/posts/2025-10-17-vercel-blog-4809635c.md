Vercel 现已支持 NestJS 应用程序。NestJS 是一个用于构建高效、可扩展 Node.js 服务器端应用程序的流行框架，并且支持零配置部署。

```
1import 'reflect-metadata';2import { Controller, Get, Module } from '@nestjs/common';3import { NestFactory } from '@nestjs/core';4
5@Controller()6class AppController {7  @Get()8  getHello() {9    return 'Hello, world!';10  }11}12
13@Module({14  controllers: [AppController],15})16class AppModule {}17
18NestFactory.create(AppModule).then(app => app.listen(3000));19

```

一个在 Vercel 上运行的 "Hello World" NestJS 应用

在 Vercel 上，后端默认使用**动态计算**和**活跃 CPU 计费**。这意味着您的 NestJS 应用将根据流量自动扩缩容，并且您只需为实际使用的资源付费。

[在 Vercel 上部署 NestJS](https://vercel.com/new) 或访问 [Vercel 上的 NestJS 文档](https://vercel.com/docs/frameworks/nestjs)
