Chat SDK 现在通过新的 `vendor-official` 适配器支持 AgentPhone（智能体电话）。

为您的机器人分配一个专属电话号码，使其能够使用您已编写的相同处理器来处理语音通话和短信。通话结束后，通话记录将以消息形式传递，让您的机器人能够通过短信或 iMessage 进行回复。

```
1import { Chat } from "chat";2import { createAgentPhoneAdapter } from "@agentphone/chat-sdk-adapter";3
4const bot = new Chat({5  userName: "mybot",6  adapters: {7    agentphone: createAgentPhoneAdapter(),8  },9});10
11bot.onNewMention(async (thread, message) => {12  await thread.post(`You said: ${message.text}`);13});
```

配置适配器以响应 AgentPhone（智能体电话）对话中的 @-提及

发送到同一号码的通话和消息通过单个 Webhook 到达，因此您可以从单个端点处理两者，而无需为每个渠道设置单独的管道。它们还会归入同一线程，因此您的机器人能够在整个对话中保持上下文。

阅读 AgentPhone（智能体电话）文档以开始使用。
