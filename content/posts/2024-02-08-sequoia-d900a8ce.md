# 新设奖学金：红杉资本如何支持开源事业

FastAPI 创作者塞巴斯蒂安·拉米雷斯成为首位红杉开源奖学金获得者。

开源是软件的生命线。最受欢迎的开源软件包下载量已达数万亿次且持续快速增长，当今大多数商业软件都采用了由全球开源贡献者优化完善的库和程序。红杉资本长期与 MongoDB、Confluent 等开源软件商业化先驱企业合作，近年来与开源企业的合作步伐也在加快，包括 Chainguard、Deno、dbt labs、Infracost 和 Temporal Technologies 等。但我们同样认识到，未商业化的开源项目对我们的投资组合公司及更广泛社区仍是至关重要的工具。

去年我们有幸结识了广受欢迎的开源 Python 框架 FastAPI 的创作者塞巴斯蒂安·拉米雷斯。该框架应用场景遍及宝马、美国运通、罗氏等《财富》全球 500 强企业，学术界，以及 NASA 和红十字会等非营利组织。塞巴斯蒂安对改进现有成果充满创意，但由于即使最受欢迎的开源项目也难以维持生计，他不得不在 FastAPI 与兼职工作间分配时间。

他的处境促使我们思考如何支持他及其他为科技生态做出巨大贡献的开源创作者。今天，我们很高兴宣布塞巴斯蒂安成为首位红杉开源学者。

每年我们将邀请少数顶尖开源贡献者加入这项新设奖学金计划，提供为期 6 至 12 个月的生活津贴，使他们能全职投入开源工作。他们还将加入红杉社区，与创始人建立联系并提供指导（塞巴斯蒂安近期协助我们举办了种子阶段催化器 Arc 的闭幕活动）。简而言之，我们将相互支持。我们相信红杉能从学者的专业见解中获益良多，这有助于我们更深入理解开源生态，除此之外没有任何附加条件。我们的目标是为塞巴斯蒂安及像他这样的建设者提供创新所需的时间与空间，无论这对他们意味着什么。

近期我们与塞巴斯蒂安深入探讨了 FastAPI、他目前的奖学金体验以及未来规划。

**对于不熟悉 FastAPI 的人，能否介绍它的用途及开发初衷？**

塞巴斯蒂安：FastAPI 极大便利了开发者使用 Python（最流行的编程语言之一，常用于机器学习、数据科学和 AI）开发云端应用。通过 FastAPI，开发者可在统一环境中利用代码编辑器的自动补全、行内错误提示等全部功能，同时通过简洁代码默认实现应用最佳实践，从而在保证极高效率的同时获得最优质量与性能。

我曾参与数据和机器学习产品开发团队，不断遇到相同的问题和错误。我尝试了所有能找到的工具，但都无法满足全部需求——最终我决定将其他框架中所有优秀特性整合到一个项目中。原本没抱太大期望，但它成功了！2018 年 12 月 24 日，在支持 FastAPI 的 Python 版本发布两年后，我推出了首个公开版本，至今下载量已超 80 万次。它甚至被用于分享詹姆斯·韦伯太空望远镜采集的数据——这堪称高光时刻。

**为何认为这项奖学金适合您？**

塞巴斯蒂安：自创建 FastAPI 以来，我始终尝试优化工作安排以投入更多时间到 FastAPI 及其他开源项目（如 Typer 和 SQLModel）。但由于项目增长迅猛，仅维护就耗费大量时间，改进完善更需要额外投入。我曾辞职从事兼职工作，但时间仍不够用。因此全职投入开源极具吸引力——不仅是时间总量，更是时间的连续性。我知道深度专注能带来更大进展。

我也希望学习企业构建之道，因为我认为 FastAPI 在未来某个阶段可能需要自给自足的模式。它能帮助众多个人和组织提升效率，这本身就蕴含价值。但我希望首先专注于将其打造成最优秀的工具。

**目前的体验如何？**

非常棒。说实话，我没想到会如此顺利。项目启动前的沟通就很顺畅，书面方案也很完善。实际执行完全符合当初的沟通。我们会讨论公司和项目，除此之外我可以全身心投入工作。

一个不那么明显但极具价值的是：我能专注于最具影响力的工作，即使这些对大多数用户不可见——比如 FastAPI 的底层构建模块，甚至它所依赖的其他开源项目。最近我甚至在改进 Python 内部代码（参考）API 的文档规范，这类工作通常很难分配时间和资源。但这对于 FastAPI 用户及所有人都会极具价值。

**在奖学金期间及未来，您最期待什么？**

有很多期待！最明显的是对新版 Pydantic（现也属于红杉社区）的集成支持。这将显著提升 FastAPI 及其他工具的性能，并增强对边界情况的处理。我正在完善 FastAPI 内部代码文档——虽然已有大量教程，但缺少词典式的代码 API 参考手册，而这正是用户最迫切的需求之一。同时我正在改进 FastAPI 本身以及 Typer、SQLModel、Asyncer 等工具，这将简化在 FastAPI 应用中构建完整技术栈的过程。

我也期待延续所有已建立的关系，包括与二位的联系！我曾对 VC 心存畏惧。融资创业的理念对我并不直观。但这次经历改变了我的观念——让我看到这确实可以是积极的。我们是朋友！这让我感觉：“是的，这条路确实可行。”

红杉开源奖学金目前仅接受邀请提名；我们旨在支持成熟的创作者，助力他们在项目持续成长过程中规划发展路径。如果您希望了解更多信息，请通过此链接联系我们。
