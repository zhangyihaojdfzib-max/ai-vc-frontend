# Stable Diffusion 3.5 现已发布

-   deepfates

![](/images/posts/1acc79ffe0c8.webp)

我们很高兴地宣布，Stability AI 最新、最强大的文生图模型 **Stable Diffusion 3.5** 现已在 Replicate 平台上线。它在图像质量、提示词理解方面带来了显著提升，并支持广泛的艺术风格。

Stable Diffusion 3.5 包含三个变体：

-   **Stable Diffusion 3.5 Large**：生成最高质量的图像
-   **Stable Diffusion 3.5 Large Turbo**：图像质量几乎同样出色，但速度更快
-   **Stable Diffusion 3.5 Medium**：一个较小的模型，可在云端以及消费级 GPU 上运行

您可以立即使用 Stable Diffusion 3.5 生成图像。在 Python 中尝试：

```
import replicate

output = replicate.run(
    "stability-ai/stable-diffusion-3.5-large",
    input={"prompt": "A watercolor painting of a futuristic city skyline at dawn"}
)
print(output.url)
```

或者使用 JavaScript：

```
import Replicate from "replicate";

const replicate = new Replicate({
  auth: process.env.REPLICATE_API_TOKEN,
});

const [output] = await replicate.run("stability-ai/stable-diffusion-3.5-large", {
  input: {
    prompt: "A watercolor painting of a futuristic city skyline at dawn",
  },
});
console.log(output.url());
```

您也可以**直接在浏览器中体验 Stable Diffusion 3.5**。

## Stable Diffusion 3.5 Large 的新特性

-   **增强的图像质量**：生成分辨率更高、细节更精细的图像，从而获得更具照片真实感和视觉吸引力的输出。
-   **更丰富的输出多样性**：得益于查询-键归一化技术，模型能从相同的提示词中产生更广泛的输出。例如，输入“一个人”并配合不同的随机种子，将产生不同性别、种族和外貌的多样结果。
-   **改进的提示词遵循能力**：更好地理解和呈现复杂的提示词，允许根据您的描述生成更准确、更详细的图像。
-   **多样的风格**：能够生成多种风格的图像，如水彩画、像素艺术、3D 渲染、线条艺术等。

## 同时推出 Stable Diffusion 3.5 Large Turbo

Stability AI 还发布了 **Stable Diffusion 3.5 Large Turbo**。这个蒸馏版本仅需 **4 步**即可生成高质量图像，提供更快的推理速度和更低的成本——非常适合对速度和效率要求苛刻的应用场景。

以下是如何在 Python 中使用它：

```
import replicate

output = replicate.run(
    "stability-ai/stable-diffusion-3.5-large-turbo",
    input={"prompt": "A pixel art dragon in a mystical forest"}
)
print(output.url)
```

## Stable 3.5 Medium 可在消费级硬件上运行

Stable Diffusion 3.5 Medium 是一个拥有 25 亿参数的模型，既可以在云端运行，也可以在消费级硬件上运行。

要在 Replicate API 上使用 Python 运行它：

```
import replicate

output = replicate.run(
    "stability-ai/stable-diffusion-3.5-medium",
    input={"prompt": "A baroque-style painting of aliens attending a royal ball"}
)
print(output.url)
```

如果您想在本地 GPU 上运行该模型，权重文件可在 HuggingFace 获取：https://huggingface.co/stabilityai/stable-diffusion-3.5-medium
