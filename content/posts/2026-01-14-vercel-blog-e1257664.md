您现在可以通过 Vercel 的 **AI Gateway** 访问 GPT 5.2 Codex，无需其他供应商账户。GPT 5.2 Codex 结合了 GPT 5.2 在专业知识工作方面的优势与 GPT 5.1 Codex Max 的智能编码能力。

与之前的版本相比，GPT 5.2 Codex 更擅长处理长期运行的编码任务，并能更可靠地处理大型重构和迁移等更复杂的任务。该模型具备更强的视觉性能，能更准确地处理编码过程中分享的截图和图表。GPT 5.2 Codex 在网络能力方面也超越了 GPT 5.1 Codex Max，并在 OpenAI 的 **Professional Capture-the-Flag (CTF)** 网络安全评估中表现优于前代模型。

要使用 GPT 5.2 Codex，可通过 **AI SDK**，将模型设置为 `openai/gpt-5.2-codex`：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'openai/gpt-5.2-codex',5  prompt:6    `Take the attached prototypes, diagram, and reference screenshots7     to build a production app for customer analytics dashboards.`8});
```

AI Gateway 提供了一个统一的 API 用于调用模型、跟踪使用情况和成本，并可配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它内置了**可观测性**、**自带密钥（Bring Your Own Key）** 支持以及智能的**供应商路由**和自动重试功能。

了解更多关于 **AI Gateway** 的信息，查看 **AI Gateway 模型排行榜** 或在我们的 **模型游乐场** 中试用。

AI Gateway：按使用量追踪顶级 AI 模型

AI Gateway 模型排行榜根据通过 Gateway 的所有流量的总 Token 量，对长期使用最多的模型进行排名。榜单定期更新。
