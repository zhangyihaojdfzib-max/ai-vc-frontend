# 推出 Claude 2.1

![](/images/posts/d99b44283103.jpg)

我们的最新模型 Claude 2.1 现已通过我们控制台的 API 提供，并为我们 claude.ai 的聊天体验提供支持。Claude 2.1 在企业关键能力方面取得了进步——包括业界领先的 20 万 Token 上下文窗口、模型幻觉率显著降低、系统提示词以及我们新的测试版功能：工具使用。我们也在更新我们的定价，以提高客户在不同模型上的成本效益。

**20 万 Token 上下文窗口**
自今年早些时候发布以来，已有数百万人将 Claude 用于广泛的应用——从翻译学术论文到起草商业计划和分析复杂合同。在与用户的讨论中，他们要求在处理长文档时拥有更大的上下文窗口和更准确的输出。

为此，我们将您能传递给 Claude 的信息量增加了一倍，上限为 20 万个 Token，相当于大约 15 万个单词，或超过 500 页的材料。我们的用户现在可以上传整个代码库等技术文档、S-1 表格等财务报表，甚至是《伊利亚特》或《奥德赛》等长篇文学作品。通过能够与大量内容或数据对话，Claude 可以进行总结、执行问答、预测趋势、比较和对比多个文档等等。

处理 20 万长度的消息是一项复杂的壮举，也是业界首创。虽然我们很高兴能将这一强大的新功能交到用户手中，但通常需要数小时人力才能完成的任务，Claude 可能需要几分钟。我们预计随着技术的进步，延迟将大幅减少。

**幻觉率降低 2 倍**
Claude 2.1 在诚实性方面也取得了显著进步，与之前的 Claude 2.0 模型相比，错误陈述减少了 2 倍。这使得企业能够构建高性能的 AI 应用程序来解决具体的业务问题，并以更高的信任度和可靠性在其运营中部署 AI。

我们通过策划大量复杂的、事实性的问题来测试 Claude 2.1 的诚实性，这些问题旨在探究当前模型的已知弱点。使用一个区分错误主张（"玻利维亚第五大城市是蒙特罗"）和承认不确定性（"我不确定玻利维亚第五大城市是哪个"）的评分标准，Claude 2.1 明显更倾向于表示异议，而不是提供错误信息。

![](/images/posts/d61440e59342.jpg)

Claude 2.1 在理解和总结方面也取得了有意义的改进，特别是对于那些需要高度准确性的长而复杂的文档，如法律文件、财务报告和技术规范。在我们的评估中，Claude 2.1 的错误答案减少了 30%，错误地断定文档支持特定主张的比率降低了 3-4 倍。

![](/images/posts/54913da96368.jpg)

虽然我们对这些准确性改进感到鼓舞，但提高输出结果的精确性和可靠性仍然是我们的产品和研究团队的首要任务。

**API 工具使用**
应广大用户的要求，我们还增加了工具使用功能，这是一项新的测试版功能，允许 Claude 与用户现有的流程、产品和 API 集成。这种扩展的互操作性旨在使 Claude 在用户的日常操作中更加有用。

Claude 现在可以跨开发者定义的函数或 API 进行编排、搜索网络资源，并从私有知识库中检索信息。用户可以定义一组供 Claude 使用的工具并指定请求。然后，模型将决定完成任务需要哪个工具，并代表用户执行操作，例如：

*   使用计算器进行复杂的数值推理
*   将自然语言请求转换为结构化的 API 调用
*   通过搜索数据库或使用网络搜索 API 来回答问题
*   通过私有 API 在软件中执行简单操作
*   连接产品数据集以提供建议并帮助用户完成购买

工具使用功能目前处于早期开发阶段——我们正在构建开发者功能和提示词指南，以便更轻松地集成到您的应用程序中。我们鼓励用户分享关于工具使用的反馈，以帮助塑造和改进产品。

**开发者体验**
我们一直在努力简化 Claude API 用户的开发者控制台体验，同时使其更容易测试新的提示词以加快学习速度。我们的新产品 Workbench 使开发者能够在类似游乐场的体验中迭代提示词，并访问新的模型设置以优化 Claude 的行为。他们可以为不同的项目创建多个提示词并在其间导航，并且修订版本会随着进度保存以保留历史上下文。开发者还可以生成代码片段，以便在我们的某个 SDK 中直接使用他们的提示词。

我们还引入了系统提示词，允许用户向 Claude 提供自定义指令以提高性能。系统提示词设置了有用的上下文，增强了 Claude 承担特定个性和角色，或以更可定制、更符合用户需求的一致方式构建响应的能力。

Claude 2.1 现已通过我们的 API 提供，并且也为我们 claude.ai 的免费版和 Pro 版的聊天界面提供支持。20 万 Token 上下文窗口的使用权限保留给 Claude Pro 用户，他们现在可以上传比以往更大的文件。我们迫不及待地想看到这些新功能激发出的用例，同时我们将努力构建业界最安全、技术最先进的 AI 系统。
