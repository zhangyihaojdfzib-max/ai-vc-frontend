Gemini 3.5 Flash 现已在 Vercel AI Gateway 上可用。

与之前的 Flash 版本相比，该模型提升了编码能力与并行 Agent（智能体）执行循环。它还在核心推理、指令遵循和多轮对话连贯性方面有所改进，在复杂任务上表现更强，并在思考模式下提供更高质量的推理轨迹。

3.5 Flash 默认使用中等思考级别，在响应质量与更快、更具成本效益的生成之间取得平衡。

要使用 Gemini 3.5 Flash，请在 AI SDK 中将模型设置为 `google/gemini-3.5-flash`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'google/gemini-3.5-flash',5  prompt: '重构此服务以并行运行 API 调用。',6  providerOptions: {7    google: { 8      thinkingConfig: {9        thinkingLevel: 'high',10        includeThoughts: true,11    },12  },13});
```

请注意，该模型不支持 `temperature`、`topP`、`topK` 和 `thinking_budget`。

AI Gateway 提供统一的 API 用于调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商正常运行时间的可用性。它包含内置的自定义报告、可观测性、自带密钥支持，以及具有自动重试功能的智能提供商路由。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
