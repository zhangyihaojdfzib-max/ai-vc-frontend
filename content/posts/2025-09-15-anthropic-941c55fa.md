# Claude 现已全面登陆 Xcode

开发者现在可以将他们的 Claude 账户连接到 Xcode 26，以利用 Claude Sonnet 4 为编码智能功能提供支持。Xcode 是苹果的集成开发环境（IDE），提供了为苹果平台开发、测试和分发应用程序所需的工具。此集成让开发者能够直接在他们的工作流程中使用 Claude 的编码能力，从而加速苹果平台的应用程序开发。

![Xcode 中 Claude Sonnet 4 的放大视图](/images/posts/827a9172ce9b.jpg)

## 在 Xcode 中使用 Claude

当您在 Xcode 中登录您的 Claude 账户后，您将可以使用由 Claude Sonnet 4 驱动的编码智能功能：

- **编码助手**：使用自然语言与您的代码交互。Claude 会自动从您的项目中收集上下文，并支持对话历史和文件附件，以帮助您调试、重构和构建功能。
- **编码工具**：为您的代码生成文档，获取高亮部分的解释，创建 SwiftUI 预览和 Playground，并直接在编辑器中内联修改代码。

您的 Claude 订阅使用限额在多个平台间共享，其中一部分可通过 Xcode 使用。

## 开始使用

要在 Xcode 中开始使用 Claude：

1. 从 Mac App Store 下载 Xcode 26。
2. 打开 Xcode 偏好设置，并导航至“智能”设置。
3. 使用您的 Claude 账户登录。

![Xcode 中 Claude 登录界面视图](/images/posts/7eeb21d10a5e.jpg)

Xcode 中的 Claude 适用于所有包含 Claude Code 功能的套餐——Pro 和 Max，以及 Team 和 Enterprise 套餐的高级席位。

欲了解更多详情，请参阅 Xcode 26 发布说明 和 Apple 关于在 Xcode 中使用智能编写代码的文档。
