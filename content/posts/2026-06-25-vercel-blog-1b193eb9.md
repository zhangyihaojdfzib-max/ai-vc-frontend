AI SDK Harness 让你能够通过一个统一的接口运行已有的编码 Agent 运行时，从而无需修改应用代码即可切换运行时。今天，我们新增了两个适配器：Deep Agents 和 OpenCode，两者均在 Vercel Sandbox 中运行。

## 链接到标题 Deep Agents

@ai-sdk/harness-deepagents 适配了 LangChain 的 deepagents 运行时，内置了文件和 Shell 工具、技能、主机工具、多轮会话、附加与恢复功能，以及内置的工具审批机制。

```
1import { HarnessAgent } from '@ai-sdk/harness/agent';2import { deepAgents } from '@ai-sdk/harness-deepagents';3const agent = new HarnessAgent({4  harness: deepAgents,5});
```

使用 HarnessAgent 运行 Deep Agents 的基本示例

阅读 Deep Agents harness 文档以开始使用。

## 链接到标题 OpenCode

@ai-sdk/harness-opencode 通过 @opencode-ai/sdk 在沙箱内启动一个真实的 OpenCode 服务器，并通过 harness 流式传输其会话事件。它暴露了 OpenCode 的内置工具，支持内置和主机工具审批，并允许你选择模型、提供商和推理变体。

```
1import { HarnessAgent } from '@ai-sdk/harness/agent';2import { openCode } from '@ai-sdk/harness-opencode';3const agent = new HarnessAgent({4  harness: openCode,5});
```

使用 HarnessAgent 运行 OpenCode 的基本示例

阅读 OpenCode harness 文档以开始使用。

目前支持的完整 harness 列表包括：Claude Code、Codex、Deep Agents、OpenCode、Pi，更多功能即将推出。
