# Hugging Face 与 FriendliAI 合作，在 Hub 上加速模型部署

FriendliAI 的推理基础设施现已集成到 Hugging Face Hub 中，作为"部署此模型"按钮的一个选项，简化并加速了生成式 AI 模型的服务。

## 推动 AI 创新的合作

Hugging Face 赋能开发者、研究人员和企业进行 AI 创新。我们的共同重点是建立有影响力的合作伙伴关系，简化工作流程，为 AI 社区提供尖端工具。

今天，我们很高兴宣布 Hugging Face 与加速生成式 AI 推理领域的领导者 FriendliAI 达成合作，以提升开发者部署和管理 AI 模型的方式。此次集成将 FriendliAI Endpoints 作为 Hugging Face Hub 中的一个部署选项引入，为开发者提供直接访问高性能、高性价比推理基础设施的途径。

FriendliAI 被 Artificial Analysis 评为最快的基于 GPU 的生成式 AI 推理提供商，其突破性技术包括连续批处理、原生量化和一流的自动扩缩。凭借这些技术，FriendliAI 不断推进 AI 推理服务性能的标准，为大规模部署生成式 AI 模型提供更快的处理速度、更低的延迟和显著的成本节约。通过这次合作，Hugging Face 用户和 FriendliAI 客户可以轻松部署开源或自定义生成式 AI 模型，获得无与伦比的效率和可靠性。

## 简化模型部署

去年，FriendliAI 推出了 Hugging Face 集成，使用户能够在 Friendli Suite 平台内无缝部署 Hugging Face 模型。通过这一集成，用户可以访问 Hugging Face 上数千个受支持的开源模型，并能够轻松部署私有模型。FriendliAI 当前支持的模型架构列表可在此处查看。

今天，我们进一步推进这一集成，直接在 Hugging Face Hub 中实现相同功能，提供一键部署的无缝用户体验。您可以使用 Friendli Suite 账户直接从 Hugging Face Hub 的模型卡片部署模型。

![Hugging Face 中的 Friendli 推理部署选项](/images/posts/1695169cee97.png)

选择 Friendli Endpoints 将跳转至 FriendliAI 的模型部署页面。在此页面，您可以在 NVIDIA H100 GPU 上部署模型，同时与优化的开源模型进行交互。部署页面提供直观的界面，用于设置 Friendli Dedicated Endpoints（生成式 AI 推理的托管服务）。此外，在部署处理过程中，您可以直接在页面上与开源模型聊天，轻松探索和测试其能力。

### 在 Friendli Dedicated Endpoints 中使用 NVIDIA H100 部署模型

借助 FriendliAI 先进的 GPU 优化推理引擎，Dedicated Endpoints 作为托管服务提供快速且高性价比的推理。开发者只需在模型部署页面上点击"立即部署"，即可使用 Friendli Dedicated Endpoints 在 NVIDIA H100 GPU 上轻松部署开源或自定义模型。

H100 GPU 性能强大，但大规模运行成本较高。通过 FriendliAI 的优化服务，您可以在保持峰值性能的同时减少所需 GPU 数量，从而显著降低成本。除了成本效益，Dedicated Endpoints 还简化了基础设施管理的复杂性。

![在模型部署页面部署 Hugging Face 模型](/images/posts/ecdde1ca7c6f.gif)

### 使用 Friendli Serverless Endpoints 推理开源模型

Friendli Serverless Endpoints 是希望高效推理开源模型的开发者的理想解决方案。该服务为 FriendliAI 优化的模型提供用户友好的 API，确保以低成本实现高性能。您可以直接在模型部署页面上与这些强大的开源模型聊天。

![在模型部署页面试用 Serverless Endpoints](/images/posts/9000a9845182.gif)
