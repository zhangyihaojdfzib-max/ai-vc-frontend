# 在 Replicate 上微调 Llama 2

- cbh123

Llama 2 是首个与 OpenAI 模型水平相当的开源语言模型，由于它是开源的，你可以对其进行改造，实现 GPT-4 无法完成的新功能。

比如成为更好的诗人。像霍默·辛普森那样说话。编写 Midjourney 提示词。或者取代你最好的朋友。

![由 ai-forever/kandinsky-2.2 生成的未来主义羊驼群像](/images/posts/fa25e5acc3a7.webp)

微调模型的主要原因之一是，你可以用一个小模型完成通常需要大模型才能完成的任务。这意味着你可以用更低的成本和更快的速度完成同样的任务。例如，70亿参数的 Llama 2 模型在文本摘要方面表现不佳，但我们可以教会它如何做。

在本指南中，我们将向你展示如何创建一个文本摘要器。我们将使用 Meta 的开源大语言模型 Llama 2 7B，并在一个包含对话及摘要的数据集上对其进行微调。完成后，你将能够将聊天记录、电子邮件、网页及其他文档提炼成简洁的摘要。简短而精炼。

## 支持的模型

以下是 Replicate 上可进行微调的 Llama 模型：

- Llama 2 7B 基础版
- Llama 2 13B 基础版
- Llama 2 70B 基础版
- Llama 2 7B 对话版
- Llama 2 13B 对话版
- Llama 2 70B 对话版

如果你的模型需要响应用户的指令，请使用对话版模型。如果你只是完成文本补全，则应使用基础版。

## 训练数据

你的训练数据应保存在 JSONL 格式的文本文件中。

在本指南中，我们将使用 SAMSum 数据集，并将其转换为 JSONL 格式。

## 创建模型

你需要在 Replicate 上为你的训练模型创建一个空模型。训练完成后，新版本将被推送到该模型中。

访问 replicate.com/create 并创建一个名为 "llama2-summarizer" 的新模型。

## 身份验证

通过设置环境变量中的 Token 进行身份验证：

```
export REPLICATE_API_TOKEN=<粘贴你的 Token 到这里>
```

你可以在账户设置中找到你的 API Token。

## 创建训练任务

安装 Python 库：

```
pip install replicate
```

然后启动训练，将目标名称替换为你的用户名和新模型名称：

```
import replicate

training = replicate.trainings.create(
  version="meta/llama-2-7b:73001d654114dad81ec65da3b834e2f691af1e1526453189b7bf36fb3f32d0f9",
  input={
    "train_data": "https://gist.githubusercontent.com/nateraw/055c55b000e4c37d43ce8eb142ccc0a2/raw/d13853512fc83e8c656a3e8b6e1270dd3c398e77/samsum.jsonl",
    "num_train_epochs": 3,
  },
  destination="<你的用户名>/llama2-summarizer"
)

print(training)
```

该函数接受以下参数：

- version：要训练的模型，格式为 {用户名}/{模型}:{版本}。
- input：传递给训练过程的训练数据和参数，由模型定义。Llama 2 的参数可在模型的“训练”选项卡中找到。
- destination：训练版本推送到的目标模型，格式为 你的用户名/你的模型名称

启动训练后，请在浏览器中访问 replicate.com/trainings 以监控进度。

## 运行模型

现在，你可以通过网页或 API 运行你的模型。要在浏览器中使用你的模型，请访问你的模型页面。

要通过 API 使用你的模型，请从训练输出中运行版本：

```
training.reload()

prompt = """[INST] <<SYS>>\
使用输入内容提供对话摘要。
<</SYS>>

输入：
哈利：你是谁？
海格：鲁伯·海格，霍格沃茨的钥匙保管员和猎场看守。当然，你知道霍格沃茨的一切。
哈利：抱歉，不知道。
海格：不知道？天哪，哈利，你难道从来没想过你父母是在哪里学会这一切的吗？
哈利：学会什么？
海格：你是个巫师，哈利。
哈利：我——我是个什么？
海格：一个巫师！而且我敢打赌，等你稍加训练，你会成为一个非常出色的巫师。[/INST]

摘要： """

output = replicate.run(
  training.output["version"],
  input={"prompt": prompt, "stop_sequences": "</s>"}
)
for s in output:
  print(s, end="", flush=True)
```

就是这样！你已经成功微调了 Llama 2，现在可以通过 API 运行你的新模型了。
