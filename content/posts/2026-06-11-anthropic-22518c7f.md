# DXC 将把 Claude 集成到银行、航空公司及其他受监管行业所依赖的系统中

我们宣布与全球最大的 IT 服务公司之一 DXC Technology 建立多年全球联盟。DXC 将培训数万名获得 Claude 认证的前沿部署工程师（FDE）——这些工程师直接嵌入客户组织内部——将 Claude 引入 DXC 为全球最大银行、航空公司、保险公司、制造商和政府机构运营的系统中。

这些由 DXC 运营数十年的系统，在严格的安全与合规要求下，处理企业赖以生存的交易、理赔和运营。在将 Claude 引入这些企业之前，DXC 已在其自身运营中使用了 Claude——其业务覆盖 70 个国家，拥有约 11.5 万名员工，且面临同样严格的要求——包括与 Claude 协作编写了 DXC OASIS（其面向托管服务的全新 AI 原生编排平台）超过 95% 的代码。

DXC 现已加入 Claude 合作伙伴网络，这是我们为将 Claude 带给企业客户而建立的咨询与服务公司网络。

## 将 Claude 引入 DXC 自身运营

DXC 在向客户推广之前，已在其自身系统中使用了 Claude。例如，2026 年 4 月，DXC 推出了 DXC OASIS——用于运行客户 IT 系统的工具，其中 AI Agent 处理大部分日常事务。Claude 现已成为该平台 Agent 工作流的默认基础模型。

Claude 也是 DXC 用于构建该平台的主要工具。该公司估计，Claude 将软件开发速度提升了 10 倍。超过 95% 的代码由 Claude 生成，随后由软件工程师进行审查。OASIS 现已服务于 50 多家 DXC 客户，该公司计划将其推广至全球客户。

" DXC 帮助全球最大的银行、航空公司、保险公司和政府机构将新技术投入实际应用，"Anthropic 首席商务官 Paul Smith 表示。"他们首先在自身运营中验证了 Claude，且面临与客户相同的安全与合规要求。现在，我们正携手将这些环境中的 Claude 引入各个行业，而执行者正是那些已经亲身实践过的工程师。"

## 组建 Claude 认证工程师团队

针对其 FDE 计划，DXC 将从现有开发团队中招募工程师，并通过 Anthropic Academy（我们面向合作伙伴工程师的培训与认证项目）对其进行认证。DXC 在 Anthropic Academy 的基础上增加了自有课程，针对客户运营的关键任务系统提供专项培训。

该联盟将从 DXC 已为客户大规模运营、且 Claude 当前可发挥实际作用的四个领域开始：

- **保险**：DXC 将利用 Claude 提供 Agent 解决方案并实现核心系统现代化，同时考虑每家客户的业务背景、运营模式和战略需求。
- **现代化即服务（MaaS）**：DXC 正使用 Claude 帮助企业客户分析、重构和现代化遗留代码库。与 Claude 协作将帮助他们比传统方法更快、更准确地完成这一过程。
- **网络安全**：针对 OASIS，DXC 正在开发一个基于 Claude Security 构建的始终在线安全工程师子 Agent，该 Agent 将部署在其安全运营中心（SOC）中。
- **应用服务**：最后，DXC 正使用 Claude 开发 OASIS Agent，将 Claude 直接嵌入其为企业客户运营的应用维护与管理环境中。

"50 多年来，DXC 及其前身公司一直运营着支撑世界运转的系统，"DXC Technology 总裁兼首席执行官 Raul Fernandez 表示。"我们深知在这些环境中交付成果需要什么。与 Anthropic 的联盟将信任与经验同最先进的 AI 技术相结合，为我们的客户提供了在其他任何地方都无法获得的价值。我们已在自身运营和全新的 OASIS 平台中使用 Claude。现在，我们正将这一能力直接扩展到为客户运营的关键任务技术系统中。这对 DXC 和整个行业而言都是一个决定性时刻。"
