GPT-5.5 现已登陆 Vercel AI Gateway。

该模型提供两个版本：GPT-5.5 和 GPT-5.5 Pro。两款模型均针对编码、计算机使用、知识工作和科学研究等领域的长期 Agent（智能体）任务进行了调优，且相比上一代模型具有更高的 Token 效率。

GPT-5.5 在 Agent（智能体）编码和长期任务方面表现更强，这类任务要求模型在大型系统中保持上下文，并在整个代码库中贯彻变更。结合计算机使用技能，它可以操作真实软件，将原始材料转化为文档、电子表格或幻灯片演示。

GPT-5.5 Pro 专为对响应质量要求高于延迟的多步骤高要求工作而构建。早期测试显示，在涉及多轮审查和压力测试论证的业务、法律、教育、数据科学和技术研究工作流中，该模型表现优异。

要使用 GPT-5.5，请在 AI SDK 中将模型设置为 `openai/gpt-5.5` 或 `openai/gpt-5.5-pro`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'openai/gpt-5.5', 5  prompt:6    `将我们的用户设置页面从 REST 迁移到新的7     GraphQL 模式，更新受影响的组件和测试，8     并创建一个包含变更摘要的 PR。`,9});
```

AI Gateway 提供统一 API 用于调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商自身的正常运行时间。它包含内置的自定义报告、可观测性、自带密钥支持，以及具备自动重试功能的智能提供商路由。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
