Anthropic的Claude Sonnet 4.6现已登陆AI Gateway，并支持100万Token的上下文窗口。

Sonnet 4.6的智能水平已接近Opus，在智能体编码、代码审查、前端UI质量和计算机使用准确性方面均有显著提升。该模型能主动执行任务、委派子智能体、并行化工具调用，并通过MCP支持实现规模化工具使用。作为一个混合推理模型，Sonnet 4.6能在同一模型中提供近乎即时的响应和深度扩展思考。

要使用此模型，请在AI SDK中将模型设置为`anthropic/claude-sonnet-4.6`。该模型支持`effort`和`thinking`类型为`adaptive`：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'anthropic/claude-sonnet-4.6',5  prompt:6    `Build a dashboard component from this spec with7     responsive layout, dark mode support, and accessibility.`,8  providerOptions: {9    anthropic: {10      effort: 'medium',11      thinking: { type: 'adaptive' },12    },13  },14});
```

AI Gateway提供了一个统一的API，用于调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它内置了可观测性、自带密钥支持，以及具有自动重试功能的智能供应商路由。

了解更多关于AI Gateway的信息，查看AI Gateway模型排行榜，或在我们的模型游乐场中试用。
