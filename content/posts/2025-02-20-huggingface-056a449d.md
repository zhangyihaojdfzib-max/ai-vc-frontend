## 技术细节

我们推出了三个新模型，参数规模分别为256M、500M和2.2B。2.2B模型是视觉和视频任务的首选，而500M和256M模型代表了有史以来发布的最小的视频语言模型。

虽然它们体积小，但在内存消耗方面优于任何现有模型。从Video-MME（视频领域的权威科学基准）来看，SmolVLM2加入了2B参数范围的领先模型家族，而在更小的规模上我们处于领先地位。

### SmolVLM2 2.2B：我们视觉和视频领域的新明星

与之前的SmolVLM家族相比，我们新的2.2B模型在解决图像数学问题、读取照片中的文字、理解复杂图表以及处理科学视觉问题方面表现更好。这体现在模型在不同基准测试中的表现上：

![SmolVLM2视觉评分提升](/images/posts/99b9f5710695.png)

在视频任务方面，2.2B模型性价比很高。在我们评估的各种科学基准中，我们想强调它在Video-MME上的表现，它超越了所有现有的2B模型。

我们能够实现视频/图像性能的良好平衡，得益于Apollo：大型多模态模型中视频理解的探索中发布的数据混合研究成果。

它的内存效率非常高，即使在免费的Google Colab中也可以运行。

```python

!pip install git+https://github.com/huggingface/transformers@v4.49.0-SmolVLM-2

from transformers import AutoProcessor, AutoModelForImageTextToText
import torch

model_path = "HuggingFaceTB/SmolVLM2-2.2B-Instruct"
processor = AutoProcessor.from_pretrained(model_path)
model = AutoModelForImageTextToText.from_pretrained(
    model_path,
    torch_dtype=torch.bfloat16,
    _attn_implementation="flash_attention_2"
).to("cuda")

messages = [
    {
        "role": "user",
        "content": [
            {"type": "video", "path": "path_to_video.mp4"},
            {"type": "text", "text": "Describe this video in detail"}
        ]
    },
]

inputs = processor.apply_chat_template(
    messages,
    add_generation_prompt=True,
    tokenize=True,
    return_dict=True,
    return_tensors="pt",
).to(model.device, dtype=torch.bfloat16)

generated_ids = model.generate(**inputs, do_sample=False, max_new_tokens=64)
generated_texts = processor.batch_decode(
    generated_ids,
    skip_special_tokens=True,
)

print(generated_texts[0])

```

### 使用 MLX 进行推理

要在 Apple Silicon 设备上使用 Python 通过 MLX 运行 SmolVLM2，你可以使用优秀的 mlx-vlm 库。
首先，你需要使用以下命令从该分支安装 mlx-vlm：

```bash
pip install git+https://github.com/pcuenca/mlx-vlm.git@smolvlm

```

然后，你可以使用以下单行命令对单张图像进行推理，该命令使用 SmolVLM2 的未量化 500M 版本：

```bash
python -m mlx_vlm.generate \
  --model mlx-community/SmolVLM2-500M-Video-Instruct-mlx \
  --image https://huggingface.co/datasets/huggingface/documentation-images/resolve/main/bee.jpg \
  --prompt "你能描述一下这张图片吗？"

```

我们还创建了一个简单的视频理解脚本。你可以按如下方式使用它：

```bash
python -m mlx_vlm.smolvlm_video_generate \
  --model mlx-community/SmolVLM2-500M-Video-Instruct-mlx \
  --system "只关注描述此视频片段中发生的关键戏剧性动作或显著事件。除非对理解主要动作至关重要，否则跳过一般背景或场景设置细节。" \
  --prompt "这个视频里发生了什么？" \
  --video /Users/pedro/Downloads/IMG_2855.mov \
  --prompt "你能描述一下这张图片吗？"

```

请注意，系统提示词对于引导模型达到期望行为非常重要。你可以用它来，例如，描述所有场景和转场，或者提供一句话总结正在发生的事情。

#### Swift MLX

通过 mlx-swift-examples 仓库也支持 Swift 语言，我们正是用它来构建我们的 iPhone 应用。

在我们正在进行的 PR 完成并合并之前，你需要从该分支编译项目，然后可以在你的 Mac 上按如下方式使用 llm-tool CLI。

