阿里巴巴的Qwen 3.6 Plus现已登陆Vercel AI Gateway。

相较于Qwen 3.5 Plus，此模型增强了智能体编码能力，从前端开发到仓库级问题解决均有所覆盖，同时提升了多模态感知与推理能力。它具备1M的上下文窗口，并在工具调用、长程规划及多语言任务上表现更优。

要使用Qwen 3.6 Plus，请在AI SDK中将模型设置为`qwen/qwen3.6-plus`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'alibaba/qwen3.6-plus',5  prompt:6    `Refactor this module to separate concerns, update7     the imports across the repo, and verify nothing breaks8     with the existing test suite.`,9});
```

AI Gateway提供了一个统一的API，用于调用模型、跟踪使用情况和成本，并可配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它内置了自定义报告、可观测性、自带密钥支持以及具备自动重试功能的智能供应商路由。
