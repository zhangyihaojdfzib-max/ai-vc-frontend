Chat SDK 现已支持 Telegram，将其单一代码库方法扩展至 Slack、Discord、GitHub 和 Teams，新增了 Telegram 适配器。

团队可以构建支持提及、消息反应、私信和输入状态指示的机器人。

该适配器处理单文件上传，并渲染基础文本卡片，其按钮和链接按钮显示为内联键盘元素，使开发者能够在 Telegram 聊天中直接创建交互式工作流。

开始设置 Telegram 适配器：

```
1import { Chat } from "chat";2import { createTelegramAdapter } from "@chat-adapter/telegram";3
4const bot = new Chat({5  userName: "mybot",6  adapters: {7    telegram: createTelegramAdapter(),8  },9});10
11bot.onNewMention(async (thread, message) => {12  await thread.post(`You said: ${message.text}`);13});
```

Telegram 未向机器人开放完整的历史消息 API，因此消息历史记录依赖于适配器级别的缓存。此外，回调数据限制为 64 字节，且该平台目前不支持模态窗口或临时消息。

阅读文档以开始使用。
