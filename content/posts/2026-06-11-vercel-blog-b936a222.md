Vercel 插件现已在 Grok Build 中可用。

Grok 现在可以在您工作时利用 Vercel 知识。实时活动（包括文件编辑和终端命令）会动态地将相关知识注入上下文，从而使回答与当前平台 API 和推荐模式保持一致。

您可以通过以下两种方式安装：

- 在提示词中添加 `vercel`，Grok 将建议在聊天中安装
- 通过 `/marketplace` 打开 Grok Build 市场，然后搜索并安装 Vercel
