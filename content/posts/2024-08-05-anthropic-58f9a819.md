# Claude 3 Haiku：我们迄今最快的模型

![](/images/posts/f6a117fb2ac2.jpg)

今天我们发布 Claude 3 Haiku，这是其智能级别中速度最快、性价比最高的模型。凭借先进的视觉能力和在行业基准测试中的强劲表现，Haiku 是一款适用于广泛企业应用场景的多功能解决方案。该模型现已与 Sonnet 和 Opus 一同在 Claude API 中提供，并且我们的 Claude Pro 订阅用户可在 claude.ai 上使用。

![](/images/posts/03e546c32696.jpg)

速度对于我们的企业用户至关重要，他们需要快速分析大型数据集，并为客户支持等任务生成及时的输出。对于绝大多数工作负载，Claude 3 Haiku 的速度是其同类产品的三倍，对于不超过 32K Token 的提示词，每秒可处理 21K 个 Token（约 30 页）[1]。它还能快速生成输出，实现响应迅速、引人入胜的聊天体验，并能同时执行许多小型任务。

Haiku 的定价模型采用 1:5 的输入输出 Token 比例，专为通常涉及较长提示词的企业工作负载而设计。企业可以依赖 Haiku 快速分析大量文档，例如季度财报、合同或法律案件，而其成本仅为同等性能级别其他模型的一半。例如，Claude 3 Haiku 仅需一美元即可处理和分析 400 个最高法院案例 [2] 或 2,500 张图像 [3]。

除了速度和性价比，Claude 3 Haiku 还优先考虑企业级的安全性和鲁棒性。我们进行严格的测试，以减少模型产生有害输出和被越狱的可能性，使其尽可能安全。额外的防御层包括持续的系统监控、终端加固、安全编码实践、强大的数据加密协议以及严格的访问控制，以保护敏感数据。我们还定期进行安全审计，并与经验丰富的渗透测试人员合作，主动识别和解决漏洞。有关这些措施的更多信息，请参阅 Claude 3 模型卡片。

从今天开始，客户可以通过我们的 API 或在 claude.ai 上订阅 Claude Pro 来使用 Claude 3 Haiku。Claude 3 Haiku 已在 Amazon Bedrock 上提供，并将很快登陆 Google Cloud Vertex AI。

![](/images/posts/cbc8463f319b.jpg)
