# OpenAI 在 Linux 基金会下共同创立 Agentic AI 基金会

AAIF 旨在通过捐赠包括 OpenAI 的 AGENTS.md 在内的标准，推动开源 Agentic AI 的发展

今天，OpenAI 与 Anthropic 和 Block 共同在 Linux 基金会下创立了 Agentic AI 基金会 (AAIF)⁠(opens in a new window)，并得到了 Google、Microsoft、AWS、Bloomberg 和 Cloudflare 的支持。AAIF 旨在为开放、可互操作的基础设施提供中立的管理，因为 Agentic AI 系统正从实验阶段进入实际生产环境。

作为这项工作的一部分，我们将 AGENTS.md⁠(opens in a new window)——一种为 Agent 提供项目特定指令和上下文的简单、开放格式——贡献给该基金会，以确保其在社区中的长期支持和采用。

### 为什么开放标准很重要

开发者正迅速采用 AI 来构建更强大的 Agentic 系统——从编码助手到工作流自动化和客户服务 Agent。在 2025 年，这些系统已开始从原型转变为在商业和消费环境中处理实际工作的工具。

我们相信，如果有开放标准帮助它们实现互操作性，从实验性 Agent 到实际系统的过渡将能更好地规模化。开放标准使 Agent 更安全、更易于构建，并且在工具和平台之间更具可移植性——同时有助于防止生态系统在这一新类别成熟时出现碎片化。

随着越来越多的 Agent 开始承担实际责任，碎片化的成本也在增加。如果没有通用约定和中立治理，Agent 开发可能会分化成不兼容的孤岛，从而限制可移植性、安全性和进步。

AAIF 正是为应对这一时刻而创建的。

### 构建开放生态系统

在过去一年中，OpenAI 在开放环境中为构建 Agentic 基础设施做出了重大贡献——与开发者、研究人员以及生态系统中的其他公司合作，证明开放协议可以在实践中发挥作用。

我们贡献了核心构建模块，包括 Agents SDK⁠(opens in a new window)、Apps SDK⁠(opens in a new window) 和 Agentic Commerce Protocol⁠(opens in a new window)，以及像我们的 gpt-oss 模型和 Codex CLI⁠(opens in a new window) 这样的开源成果。仅 Codex 就已帮助在 GitHub 上合并了超过两百万个公共拉取请求。

我们也是模型上下文协议 (MCP) 的早期采用者和核心贡献者，将其作为 ChatGPT 中连接器⁠(opens in a new window) 和应用的基础。就在上周，我们宣布与 Anthropic 和 MCP-UI 合作，通过 MCP Apps⁠(opens in a new window) 将 Apps SDK 扩展到所有 MCP 开发者——使得跨平台构建交互式、由 Agent 驱动的体验变得更加容易。

这些努力通过展示开放、可互操作的基础设施可以在工具和社区中获得实际采用，为 AAIF 奠定了基础。

### 捐赠 AGENTS.md⁠(opens in a new window)

AGENTS.md 最初是作为一个常见问题的实用解决方案而诞生的。Codex 需要一种可预测的方式来查找项目特定的指令——例如编码规范、构建步骤和测试要求——以帮助 Agent 在代码库中安全有效地工作。

通过 AGENTS.md⁠(opens in a new window)，我们引入了一种轻量级的 Markdown 文件，设计为与 README.md⁠(opens in a new window) 并存。它提供了一种简单、可互操作的标准，为 AI 编码 Agent 提供一致的项目特定指导来源，使其能够在不同的仓库和工具链中可靠运行。这种基于 Markdown 的约定使得 Agent 行为在不同仓库和构建系统中变得更加可预测。

最初只是一个内部解决方案，但很快在整个生态系统中引起了共鸣。自 2025 年 8 月发布以来，AGENTS.md 已被超过 60,000 个开源项目和 Agent 框架采用，包括 Amp、Codex、Cursor、Devin、Factory、Gemini CLI、Github Copilot、Jules 和 VS Code 等——这反映了随着 Agent 进入生产环境，业界围绕共享的、供应商中立的约定日益达成共识。

将 AGENTS.md 贡献给 AAIF 确保了：

- 开发者和企业拥有一种可移植的方式，为 Agent 提供它们所需的上下文
- 该格式可以在开放环境中，通过众多工具和社区的输入不断发展
- 随着采用率的持续增长，没有任何单一公司控制其发展方向

除了 AGENTS.md⁠(opens in a new window) 之外，AAIF 的共同创始人将各自向基金会贡献一个项目：Anthropic 的模型上下文协议 (MCP)⁠(opens in a new window) 和 Block 的 goose⁠(opens in a new window)。

### Agentic AI 基金会带来的价值

AAIF 提供了一个中立的家园，在这里，Agent 互操作性标准可以协作开发、治理和扩展。我们对 AAIF 的目标很简单：建立一个共享的、中立的平台，用于构建惠及开发者、企业和更广泛开源社区的 Agentic 系统。

AAIF 作为 Linux 基金会下属的一个专项基金运作。Linux 基金会是一个非营利组织，在管理包括 Linux 内核、Kubernetes、Node.js 和 PyTorch 在内的关键开源项目方面拥有数十年的经验。Linux 基金会在中立治理、社区建设和维护广泛采用技术的长期可持续性方面带来了久经考验的专业知识——确保没有任何单一公司控制基础基础设施的发展方向。

由 OpenAI、Block⁠(opens in a new window)、Anthropic⁠(opens in a new window)、Google⁠(opens in a new window)、Microsoft⁠(opens in a new window)、Amazon⁠(opens in a new window)、Bloomberg⁠(opens in a new window) 和 Cloudflare⁠(opens in a new window) 共同创立的 Agentic AI 基金会，旨在通过共享投资、开放开发和社区主导的标准，帮助 Agentic AI 以透明、协作的方式并符合公共利益地发展。

我们欢迎各类贡献者——工具构建者、研究人员、企业和个人开发者——来帮助塑造和扩展这项工作。了解更多关于 AGENTS.md⁠(opens in a new window) 的信息，并在此处⁠(opens in a new window)参与 AAIF。
