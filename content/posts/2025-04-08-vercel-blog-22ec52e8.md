来自 Sentry、Checkly 和 Dash0 的全新原生集成现已登陆 Vercel Marketplace，帮助您更轻松地在一个地方监控、调试和优化应用程序。

- Sentry：实时错误追踪与性能监控，助力更快解决问题
- Checkly：面向前端和 API 的端到端监控与合成检查
- Dash0：以开发者优先的体验构建的日志管理与结构化可观测性。Dash0 还支持原生日志导出，允许您将 Vercel 项目的日志流式传输到外部日志系统，以获得更深入的洞察和集中式监控

Sentry：实时错误追踪与性能监控，助力更快解决问题

Checkly：面向前端和 API 的端到端监控与合成检查

Dash0：以开发者优先的体验构建的日志管理与结构化可观测性。Dash0 还支持原生日志导出，允许您将 Vercel 项目的日志流式传输到外部日志系统，以获得更深入的洞察和集中式监控

此次发布为原生集成引入了日志导出支持——该功能此前仅对可连接账户可用。

这些集成通过 Vercel 提供无摩擦的入门体验、单点登录和统一计费，只需点击几下即可轻松上手。
