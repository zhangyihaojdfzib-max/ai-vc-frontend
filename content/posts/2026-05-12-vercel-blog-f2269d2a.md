您现在可以直接通过CLI管理Vercel防火墙。

使用`vercel firewall`命令，您可以配置自定义规则、IP封禁、系统绕过、攻击模式以及系统缓解措施。

```
vercel firewall rules add --ai "将 /api 的速率限制为每分钟100次请求，按IP计算"
vercel firewall ip-blocks block 1.2.3.4
vercel firewall system-bypass add 10.0.0.1
vercel firewall attack-mode enable --duration 1h
vercel firewall system-mitigations pause
```

通过CLI管理Vercel防火墙功能

基于新的CLI命令，Vercel防火墙技能让Agent（智能体）能够与防火墙交互，并包含安全推出新防火墙规则的最佳实践。

```
npx skills add vercel/vercel-plugin --skill vercel-firewall
```

更新至最新CLI版本并运行`vercel firewall`即可开始使用。了解更多关于Vercel防火墙CLI命令的信息。
