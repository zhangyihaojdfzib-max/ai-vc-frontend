Flask，最受欢迎的 Python Web 应用框架之一，现在无需任何配置更改，即可在 Vercel 上即时部署。

```
1from flask import Flask2
3app = Flask(__name__)4
5@app.route("/")6def hello_world(): 7    return "<p>Hello, World!</p>"
```

一个在 Vercel 上的 "Hello World" Flask 应用

Vercel 的**框架定义的基础设施**现在能够识别并深度理解 Flask 应用程序。此更新消除了在 `vercel.json` 中配置重定向或使用 `/api` 文件夹的需要。

Vercel 上的后端默认使用**流体计算**和**活跃 CPU 计费**。这意味着您的 Flask 应用将根据流量自动扩缩容，并且您只需为代码实际使用 CPU 的时间付费。
