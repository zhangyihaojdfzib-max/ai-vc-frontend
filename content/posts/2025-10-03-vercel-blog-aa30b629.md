# Python包管理器uv现已支持零配置构建

![](/images/posts/6a7ed5a70c8e.jpg)

![](/images/posts/b0eb72f6a159.jpg)

Vercel 现已采用 `uv`（一款用 Rust 编写的快速 Python 包管理器）作为所有 Python 构建安装阶段的默认包管理器。

此项变更使构建速度提升 **30-65%**，并增加了对更多依赖格式的支持。除了 `requirements.txt` 或 `Pipfile`，项目现在还可以通过 `uv.lock` 或 `pyproject.toml` 文件声明依赖。

详细了解 [Vercel 上的 Python 运行时](https://vercel.com/docs/runtimes/python)。
