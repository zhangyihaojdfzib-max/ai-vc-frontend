# Python Vercel Functions 包大小限制提升至 500MB

## 作者

使用 Python 运行时的 Vercel Functions 的包大小限制现已提升至 500MB，将未压缩部署包的最大尺寸从 250MB 提高。

欲了解更多信息，请查阅函数限制文档，或立即在 Vercel 上部署 FastAPI 或 Flask 以开始使用。
