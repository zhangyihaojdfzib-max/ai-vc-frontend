Vercel 插件现已在 VS Code 和 GitHub Copilot CLI 中可用。

GitHub Copilot 现在可按需获取 Vercel 平台知识，具备 Next.js、AI SDK、Vercel Functions 等技能。Vercel 插件还能帮助 Copilot 及时了解最新的 Vercel API 和推荐模式。

要在 Copilot Chat 中安装该插件，请在 VS Code 的扩展面板中搜索 `@agentPlugins vercel`。

要在 GitHub Copilot CLI 中安装该插件，请运行 `npx plugins add vercel/vercel-plugin`。

了解更多信息，请参阅 Vercel 插件文档。
