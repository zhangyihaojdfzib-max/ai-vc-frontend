基于与Gemini 3相同的架构，这两款开源模型均支持函数调用、Agent（智能体）工作流、结构化JSON输出和系统指令。两者均支持高达256K的上下文窗口、140多种语言以及原生视觉能力。

- **26B（MoE）**：在推理时仅激活其260亿总参数中的38亿，旨在实现更低延迟和更快的每秒Token生成速度。
- **31B（Dense）**：推理时所有参数均处于激活状态，旨在追求更高的输出质量。更适合作为微调的基础模型。

要使用Gemma 4，请在AI SDK中将模型设置为 `google/gemma-4-31b-it` 或 `google/gemma-4-26b-a4b-it`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'google/gemma-4-26b-a4b-it',5  6  prompt:7    `Break down this codebase into modules, identify circular8     dependencies, and generate a refactoring plan with9     implementation steps.`,10});
```

AI Gateway提供了一个统一的API，用于调用模型、跟踪使用情况和成本，以及配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它内置了自定义报告、可观测性、自带密钥支持以及具有自动重试功能的智能供应商路由。
