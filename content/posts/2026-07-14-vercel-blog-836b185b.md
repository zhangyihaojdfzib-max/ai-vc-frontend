Chat SDK 现已支持 X，通过新的 X 适配器将其单一代码库方法扩展至 Slack、Discord、GitHub、Teams、Telegram 和 WhatsApp。

团队可以构建能够回复公开提及消息并通过 X API v2 和 X Activity API 进行私信对话的机器人。该适配器自动处理 CRC 验证和 Webhook 签名检查，并可管理长时间运行机器人的 OAuth Token 刷新。

```
1import { Chat } from "chat";2import { createXAdapter } from "@chat-adapter/x";3
4const bot = new Chat({5  userName: "mybot",6  adapters: {7    x: createXAdapter(),8  },9});10
11bot.onNewMention(async (thread, message) => {12  await thread.post(`Hi @${message.author.userName}!`);13});
```

配置适配器以响应 X 上的 @-提及消息

点赞是唯一支持的反应，且由于 X 没有原生流式传输，回复在完成后一次性发布。X 的自动化规则适用于自动消息。

请阅读文档以开始使用，或浏览适配器目录。
