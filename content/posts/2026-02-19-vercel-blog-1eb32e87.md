通过 AI Gateway 现已可调用 Google 的 Veo 模型，生成具有同步音频的逼真视频。立即通过 AI SDK 6 或在 AI Gateway 交互式界面中选择模型进行体验。

Veo 模型以其电影级画质和音频生成能力而闻名：

- **原生音频生成**：自动生成逼真的音效、环境音，甚至与视频内容匹配的对话
- **高达 1080p 分辨率**：可生成 720p 和 1080p 视频
- **逼真画质**：为自然风光、野生动物和电影场景提供真实感
- **图生视频**：为静态照片添加自然运动效果
- **快速模式**：需要快速迭代时提供更快的生成速度

## 两种开始方式

视频生成功能处于测试阶段，目前面向 Pro 版、企业版套餐用户以及付费的 AI Gateway 用户开放。

- **AI SDK 6**：通过编程方式生成视频，使用 AI SDK 6 的 `generateVideo` 方法。

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'google/veo-3.1-generate-001',5  prompt: 'Woman sipping coffee by a rain-streaked window, cozy morning light.',6});
```

- **Gateway 交互式界面**：在每个模型页面内置的可配置 AI Gateway 交互式界面中，无需编写代码即可体验视频模型。您可以比较不同提供商、调整提示词并下载结果，全程无需编码。要访问此功能，请在模型列表中点击任意视频生成模型。

## 可用模型

| `google/veo-3.0-generate-001` | 全质量生成 |

## 简单示例：文生视频（带音频）

描述一个场景即可获得视频。

生成带有自然声音的电影级野生动物视频：此处使用 `google/veo-3.1-generate-001` 并设置 `generateAudio: true`。

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'google/veo-3.1-generate-001',5  prompt:6   `Close-up of a great horned owl7    turning its head slowly in a moonlit forest.`,8  aspectRatio: '16:9',9  providerOptions: {10    vertex: { generateAudio: true },11  },12});
```

## 进阶示例：图生视频（带对话）

一个确保质量的常见工作流程是：首先使用 Gemini 3 Pro Image (Nano Banana Pro) 生成自定义图像，然后使用 Veo 为其注入生命，添加动作和口语对话。

使用支持图像输入的 Veo 模型并配合提示词，可以更好地控制输出。此示例使用支持图生视频的 `google/veo-3.1-generate-001`。

