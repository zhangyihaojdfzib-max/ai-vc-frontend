Vercel Sandbox 现已支持 Node.js 版本 26。

要使用 Node.js 26 运行 Sandbox，请将 `@vercel/sandbox` 升级至 1.10.2 或更高版本，若使用 v2 则升级至 2.0.0-beta.19 或更高版本，并将 `runtime` 属性设置为 `node26`：

```
1import { Sandbox } from "@vercel/sandbox";2
3const sandbox = await Sandbox.create({ runtime: "node26" });4const version = await sandbox.runCommand("node", ["-v"]);5console.log(`Node.js version: ${await version.stdout()}`);
```

立即开始使用，并在[文档](documentation)中了解更多信息。
