# 零配置支持 Nitro

![](/images/posts/4ae5bfc1195a.jpg)

![](/images/posts/c3b8304d07e5.jpg)

Vercel 现已支持 **Nitro** 应用——一个用于构建 Web 服务器的后端工具包，且无需任何配置。

Nitro 为 Nuxt.js、TanStack Start 和 SolidStart 等框架提供支持。

在 Vercel 上部署 Nitro 或查阅 Nitro 的 Vercel 文档。
