- **v0Grok Creative Studio**：v0 团队创建了一个由 AI Gateway 驱动的模板，用于创建和展示 Grok 视频和图像生成。
- **AI SDK 6**：通过 AI SDK 6 的 `generateVideo` 以编程方式生成视频。

**v0Grok Creative Studio**：v0 团队创建了一个由 AI Gateway 驱动的模板，用于创建和展示 Grok 视频和图像生成。

**AI SDK 6**：通过 AI SDK 6 的 `generateVideo` 以编程方式生成视频。

```

## 简单示例：文本到视频

根据文本描述生成视频。

在此示例中，使用 `xai/grok-imagine-video` 生成一段包含 2 只天鹅的视频。请注意，您也可以指定输出的持续时间。

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'xai/grok-imagine-video',5  prompt:6   `Two elegant white swans gliding on a misty lake at dawn, soft golden light,7    reflections on still water, romantic atmosphere, cinematic`,8  aspectRatio: '16:9',9  resolution: '1280x720',10  duration: 3,11});
