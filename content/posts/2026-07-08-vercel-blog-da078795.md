来自SpaceXAI的Grok 4.5现已在AI Gateway上可用。

该模型专为编程、知识工作和STEM领域构建，支持文本和图像输入。

Grok 4.5支持低、中、高三种推理级别，默认设置为高。可通过`reasoning`参数设置级别，以平衡速度与深度。

要使用Grok 4.5，请在AI SDK中将`model`设置为`xai/grok-4.5`：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'xai/grok-4.5',5  reasoning: 'high',6  prompt: '分析此数据集并总结关键趋势。',7});
```

您还可以设置路由规则，在不修改代码的情况下从其他网关模型切换到Grok 4.5。

```
vercel ai-gateway rules add \  --type rewrite \  --source xai/grok-4.3 \  --destination xai/grok-4.5
```

AI Gateway提供统一的API用于调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商级别的正常运行时间。它包含内置的自定义报告、零数据保留支持、API密钥预算、路由规则等功能。

AI Gateway反映提供商定价，不收取额外费用，且不对推理收取平台费用，包括自带密钥（BYOK）请求。
