Grok 4.3 现已在 Vercel AI Gateway 上可用。该模型的知识截止日期为 2025 年 12 月，并拥有 1M Token 的上下文窗口。该模型在准确性、工具调用和指令遵循方面有所改进。

要使用 Grok 4.3，请在 AI SDK 中将模型设置为 `xai/grok-4.3`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'xai/grok-4.3',5  prompt: '分析此数据集并总结关键趋势。',6});
```

AI Gateway 提供统一的 API 用于调用模型、跟踪使用情况和成本，以及配置重试、故障转移和性能优化，以实现高于提供商正常运行时间的可用性。它包含内置的自定义报告、可观测性、自带密钥支持，以及具有自动重试功能的智能提供商路由。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
