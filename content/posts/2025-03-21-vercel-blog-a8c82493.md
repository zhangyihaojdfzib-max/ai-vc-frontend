TheFlags SDK 的 OpenFeature 适配器允许将任何 Node.js OpenFeature 提供程序与 Flags SDK 结合使用。从广泛的标志提供程序中进行选择，同时享受 Flags SDK 与 Next.js 和 SvelteKit 的紧密集成优势。

```
1import { createOpenFeatureAdapter } from "@flags-sdk/openfeature";2import type { EvaluationContext } from "@openfeature/server-sdk";3
4OpenFeature.setProvider(new YourProviderOfChoice());5const openFeatureAdapter = createOpenFeatureAdapter(OpenFeature.getClient());6
7export const exampleFlag = flag<boolean, EvaluationContext>({8  key: "example-flag",9  adapter: openFeatureAdapter.booleanValue(),10});
```

使用 Flags SDK 和 OpenFeature 适配器声明一个功能标志

OpenFeature 是一个开放规范，提供了一种与供应商无关、社区驱动的功能标志 API，可与您喜爱的功能标志管理工具或内部解决方案配合使用。OpenFeature 通过统一 API 公开各种提供程序。

TheFlags SDK 位于您的应用程序和标志源之间，帮助您遵循最佳实践并保持网站快速运行。在您的应用程序中使用 Flags SDK OpenFeature 适配器，从所有兼容的 Node.js OpenFeature 提供程序加载功能标志，包括：
