Meta 的 Muse Spark 1.1 现已在 AI Gateway 上线。

这是一款多模态推理模型，拥有 1M Token 的上下文窗口，专为 Agent（智能体）任务构建，支持文本、图像、视频、PDF 和音频输入。

Muse Spark 1.1 能够跨工具和服务进行规划与编排，可作为主 Agent（智能体）或子 Agent（智能体）运行，并且无需示例即可使用新工具、MCP 服务器和自定义技能。该模型支持并行工具调用、结构化输出以及内置带引用的搜索功能。

要使用 Muse Spark 1.1，请在 AI SDK 中将模型设置为 `meta/muse-spark-1.1`：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'meta/muse-spark-1.1',5  prompt: '阅读这份产品规格 PDF 并实现其所描述的 API。',6});7
```

AI Gateway 提供统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商级别的正常运行时间。它包含内置的自定义报告、零数据保留支持、API 密钥预算、路由规则等功能。

AI Gateway 采用提供商定价，无任何加价，且不对推理收取平台费用，包括自带密钥（BYOK）请求。
