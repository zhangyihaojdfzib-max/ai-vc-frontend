1 分钟阅读
您现在可以通过 Vercel 的 AI Gateway 访问 OpenAI 最新的 GPT-5.2 模型，无需其他提供商账户。
这些模型的表现优于 GPT-5.1 系列模型，在专业知识工作、编码和长上下文推理方面有显著改进。其他亮点包括更少的幻觉、更准确地解读图表和可视化的视觉能力、强大的复杂前端工作能力，以及在处理长文档时更好的信息保留能力。
AI Gateway 上提供 3 个模型：
GPT-5.2 Chat (
openai/gpt-5.2-chat
) 是 ChatGPT 中使用的模型，最适合日常工作和学习。GPT-5.2 (
openai/gpt-5.2
) 适用于涉及编码或长文档的深度工作和复杂任务。GPT-5.2 Pro (
openai/gpt-5.2-pro
) 最适合处理最困难的问题和需要大量推理的任务。
要使用 AI SDK 调用 GPT-5.2 模型，请将模型设置为相应的模型标识符（如上所述）：
import { streamText } from 'ai';
const result = streamText({ model: 'openai/gpt-5.2-pro', prompt: `Create a single-page aurora sky visualizer app in a single HTML file. It should display a realistic animated aurora borealis night sky. Change solar activity level, color palette (green/purple/mixed), and sky clarity (haze, clear, light clouds). The UI should feel tranquil, immersive, and visually realistic, with smooth transitions and subtle ambient motion.` providerOptions: { openai: { reasoningSummary: 'detailed', reasoningEffort: 'high', }, },});
AI Gateway 提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商水平的正常运行时间。它包括内置的可观测性、自带密钥支持以及具有自动重试功能的智能提供商路由。
了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
