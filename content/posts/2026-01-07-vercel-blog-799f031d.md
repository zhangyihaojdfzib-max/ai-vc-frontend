团队现在可以直接通过 Vercel 仪表板、API 和 Terraform 创建、更新和删除安全计算网络。

安全计算网络为您的 Vercel Functions 与后端基础设施之间提供私有连接，并让您能够控制项目的区域部署、寻址、出口和故障转移。

- **自助式网络管理**，无需修改合同或手动配置。
- **直接管理现有的安全计算功能**，包括区域和可用区选择、主/被动故障转移、私有 CIDR 选择、NAT/出口行为，现在均可通过自助流程进行管理。
- **自动化与集成**，通过仪表板、公共 API 和 Terraform 提供完整的网络生命周期支持，使团队能够以交互式或声明式的方式管理网络。
- **即将推出**：通过仪表板、API 和 Terraform 实现自助式站点到站点 VPN 连接、面向 Pro 客户的安全计算以及 PrivateLink 连接。
