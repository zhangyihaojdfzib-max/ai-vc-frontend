您现在可以通过 Vercel 的 AI Gateway 直接访问 OpenAI 的最新模型 GPT-5.1 Instant 和 GPT-5.1 Thinking，无需其他供应商账户。

- GPT-5.1 Instant 在指令遵循、自适应推理方面有所改进，并能提供更温暖、更具对话性的回应。
- GPT-5.1 Thinking 基于 GPT-5 Thinking 构建，具备动态性能调优功能，可优先为简单任务提供速度，为复杂任务提供更深度的推理。

GPT-5.1 Instant 在指令遵循、自适应推理方面有所改进，并能提供更温暖、更具对话性的回应。

GPT-5.1 Thinking 基于 GPT-5 Thinking 构建，具备动态性能调优功能，可优先为简单任务提供速度，为复杂任务提供更深度的推理。

要在 AI SDK 中使用这些模型，请将模型设置为 `openai/gpt-5.1-instant` 或 `openai/gpt-5.1-thinking`：

```
1import { streamText } from 'ai'2
3const result = streamText({4  model: "openai/gpt-5.1-instant",5  prompt: "What are the benefits to adaptive reasoning?" 6})
```

AI Gateway 提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它包含内置的可观测性、自带密钥支持以及具有自动重试功能的智能供应商路由。
