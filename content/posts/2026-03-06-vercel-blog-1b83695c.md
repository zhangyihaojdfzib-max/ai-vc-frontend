OpenAI的Responses API现已通过AI Gateway提供。Responses API是Chat Completions API的现代替代方案。将您的OpenAI SDK指向AI Gateway的基础URL，并使用`creator/modelnames`来路由请求。支持TypeScript和Python。Responses API的所有功能此前已可通过AI SDK和Chat Completions API在AI Gateway上使用，但现在您可以直接使用Responses API。

## 功能概述

- **文本生成与流式传输**：发送提示词，获取响应，实时接收流式Token
- **工具调用**：定义模型可调用的函数，然后反馈结果
- **结构化输出**：将响应约束为JSON模式
- **推理控制**：通过可配置的推理强度级别，控制模型投入的“思考”程度

## 快速开始

安装OpenAI SDK并将其指向AI Gateway。

```
1npm install openai
```

```
1import OpenAI from 'openai';2
3const client = new OpenAI({4  apiKey: process.env.AI_GATEWAY_API_KEY,5  baseURL: 'https://ai-gateway.vercel.sh/v1',6});
```

## 基础示例：文本生成

向任何支持的模型发送提示词并获取响应。

```
1const response = await client.responses.create({2  model: 'openai/gpt-5.4',3  input: 'What is the best restaurant in San Francisco?',4});
```
