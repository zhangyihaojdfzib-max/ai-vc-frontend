# 企业在Cloudflare Agent Cloud中借助OpenAI驱动Agent工作流

关键要点：

- 数百万企业现可直接在Cloudflare Agent Cloud中访问OpenAI前沿模型。
- 借助OpenAI，使用Cloudflare Agent Cloud的企业可以部署由GPT‑5.4等模型驱动的Agent来执行实际工作。
- 企业现可将基于Codex harness构建的Agent部署到Cloudflare。

Cloudflare正在扩大对OpenAI前沿模型（包括GPT‑5.4）的访问，使其可供Agent Cloud的数百万客户使用。Agent Cloud是一个平台，使企业能够部署由OpenAI模型驱动的AI Agent来执行实际工作。例如，企业可以将其与OpenAI结合使用，部署能够自动处理客户响应、系统更新和报告生成等任务的Agent——所有这些都在一个安全、生产级的环境中完成。

Agent Cloud运行在Cloudflare Workers AI（在新窗口中打开）之上，这是该公司在边缘运行AI模型的平台，使企业能够轻松构建和部署AI应用和Agent，从而在全球范围内提供快速、实时的体验。

“通过将OpenAI的强大模型直接引入Cloudflare环境，我们正在拉近智能与最终用户之间的距离。这使得开发者能够构建复杂的、由AI驱动的应用和Agent，这些应用和Agent不仅智能，而且默认情况下具有闪电般的速度和全球可扩展性。无论是使用Codex来简化开发工作流程，还是大规模部署AI应用和Agent，我们都自豪地提供基础设施，使下一代AI原生堆栈成为可能，”Cloudflare首席技术官Dane Knecht表示。

OpenAI在Agent Cloud中的扩展，是Cloudflare将包括Codex等工具在内的最先进AI能力带给企业的更广泛努力的一部分。Codex harness现已在Cloudflare Sandboxes中全面可用，这是一个安全的虚拟环境，开发者可以在其中构建、运行和测试他们的AI应用。它将在不久的将来在Workers AI中也可用。

“Cloud Agent正迅速成为完成工作的基础构建模块，借助Cloudflare，我们让开发者能够更轻松地部署由GPT‑5.4和Codex驱动的生产级Agent，以大规模运行真实的企业工作负载，”OpenAI产品（Codex）负责人Rohan Varma表示。

开发者和客户可以通过访问此链接（在新窗口中打开）在Cloudflare中使用OpenAI。

Cloudflare扩展了OpenAI为全球最大、最成熟的企业（包括Accenture、Walmart（在新窗口中打开）、Intuit、Thermo Fisher（在新窗口中打开）、BNY、State Farm（在新窗口中打开）、Morgan Stanley、BBVA等）提供智能层支持的工作。全球超过100万企业客户直接使用OpenAI，300万每周活跃用户使用Codex，OpenAI API每分钟处理超过150亿个Token。
