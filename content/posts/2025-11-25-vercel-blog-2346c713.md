Node.js 24 现已作为使用 Node 的**构建**和**函数**的运行时可用。

要使用版本 24，请前往**项目设置 -> 构建与部署 -> Node.js 版本**并选择 24.x。这也是新项目的默认版本。

此新版本的亮点：

-   **V8 引擎升级**：Node.js 24 搭载了 V8 JavaScript 引擎版本 13.6，带来了性能提升和新的 JavaScript 特性，例如 `Float16Array` 和 `Error.isError`
-   **全局 URLPattern API**：更简单的 URL 路由和匹配，无需外部库或复杂的正则表达式
-   **Undici v7**：内置的 `fetch` API 受益于更快的 HTTP 性能、改进的 HTTP/2 和 HTTP/3 支持以及更高效的连接处理
-   **npm v11**：它附带更新版本的 npm，提高了与现代 JavaScript 包的兼容性

当前使用的版本是 `24.11.0`，并将自动更新，仅保证主版本号（24.x）不变。

阅读我们的 [Node.js 运行时文档](https://vercel.com/docs/runtimes/node-js) 以了解更多信息。
