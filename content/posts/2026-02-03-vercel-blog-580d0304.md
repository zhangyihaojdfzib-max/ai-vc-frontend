Vercel 现已支持Koa应用，这是一个富有表现力的 HTTP 中间件框架，能让 Web 应用和 API 的开发体验更加愉悦，且无需任何配置。

```
1import Koa from 'koa'2import { Router } from '@koa/router'3
4const app = new Koa()5const router = new Router()6
7router.get('/', (ctx) => {8  ctx.body = { message: 'Hello from Koa!' }9})10
11app.use(router.routes())12app.use(router.allowedMethods())13
14app.listen(3000)
```

一个在 Vercel 上运行的 "Hello World" Koa 应用

Vercel 上的后端服务默认使用Fluid计算和Active CPU定价。这意味着您的 Koa 应用将根据流量自动扩缩容，并且您只需为实际使用的资源付费。

访问Koa on Vercel 文档了解更多详情。
