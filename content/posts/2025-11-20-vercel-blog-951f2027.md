Vercel 的图像优化功能可动态转换源图像，以减少文件大小，同时在访问者的浏览器上保持高质量。

现在，您可以通过提供源图像来使 CDN 缓存失效。

此功能会将源自该源图像的所有转换图像标记为过期。下一个请求会立即提供过期内容，同时后台进行重新验证，对用户无延迟影响。

有几种方法可以使源图像失效：

- 仪表板设置
- Vercel CLI
- 函数 API
- REST API

除了通过源图像使其失效外，如果源文件已不存在，您也可以通过源图像删除缓存。删除缓存可能会在生成新内容时增加延迟，或者如果您的源文件无响应，则可能导致停机。建议您谨慎使用。

此功能适用于所有使用新图像优化价格的套餐。
