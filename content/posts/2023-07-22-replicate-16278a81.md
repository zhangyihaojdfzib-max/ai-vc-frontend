# 本地运行 Llama 2 的全面指南  

- zeke  

我们一直在讨论如何在 Replicate 上运行和微调 Llama 2。但你也可以在 M1/M2 Mac、Windows、Linux 甚至手机上本地运行 Llama。本地运行 Llama 2 的酷炫之处在于，你甚至不需要互联网连接。  

以下是一个使用本地运行的 Llama 2 快速搭建一个关于“为什么羊驼很酷”的网站示例：  

Llama 2 发布才几天，但已经出现了几种本地运行它的技术。在这篇博客文章中，我们将介绍三种可以在自己设备上运行 Llama 2 的开源工具：  

- Llama.cpp（Mac/Windows/Linux）  
- Ollama（Mac）  
- MLC LLM（iOS/Android）  

## Llama.cpp（Mac/Windows/Linux）  

Llama.cpp 是 Llama 在 C/C++ 中的一个移植版本，它使得在 Mac 上使用 4 位整数量化本地运行 Llama 2 成为可能。不过，Llama.cpp 也支持 Linux/Windows。  

以下是一个可以在 M1/M2 Mac 上安装它的一行命令：  

```
curl -L "https://replicate.fyi/install-llama-cpp" | bash
```  

这一行命令的具体操作如下：  

```
#!/bin/bash

# 克隆 llama.cpp
git clone https://github.com/ggerganov/llama.cpp.git
cd llama.cpp

# 构建它。`LLAMA_METAL=1` 允许在 GPU 上执行计算
LLAMA_METAL=1 make

# 下载模型
export MODEL=llama-2-13b-chat.ggmlv3.q4_0.bin
if [ ! -f models/${MODEL} ]; then
    curl -L "https://huggingface.co/TheBloke/Llama-2-13B-chat-GGML/resolve/main/${MODEL}" -o models/${MODEL}
fi

# 设置提示词
PROMPT="Hello! How are you?"

# 以交互模式运行
./main -m ./models/llama-2-13b-chat.ggmlv3.q4_0.bin \
  --color \
  --ctx_size 2048 \
  -n -1 \
  -ins -b 256 \
  --top_k 10000 \
  --temp 0.2 \
  --repeat_penalty 1.1 \
  -t 8
```  

以下是一个适用于 Intel Mac 或 Linux 机器的一行命令。它与上面相同，但没有包含 `LLAMA_METAL=1` 标志：  

```
curl -L "https://replicate.fyi/install-llama-cpp-cpu" | bash
```  

以下是一个在 Windows 上通过 WSL 运行的一行命令：  

```
curl -L "https://replicate.fyi/windows-install-llama-cpp" | bash
```  

## Ollama（Mac）  

Ollama 是一个开源的 macOS 应用（适用于 Apple Silicon），它让你可以通过命令行界面运行、创建和共享大语言模型。Ollama 已经支持 Llama 2。  

要使用 Ollama 命令行界面，请从 ollama.ai/download 下载 macOS 应用。安装完成后，你可以下载 Llama 2，无需注册账户或加入任何等待列表。在终端中运行以下命令：  

```
# 下载 7B 模型（3.8 GB）
ollama pull llama2

# 或 13B 模型（7.3 GB）
ollama pull llama2:13b
```  

然后你可以运行模型并与之聊天：  

```
ollama run llama2
>>> hi
Hello! How can I help you today?
```  

注意：Ollama 建议运行 3B 模型至少需要 8 GB 内存，运行 7B 模型需要 16 GB，运行 13B 模型需要 32 GB。  

## MLC LLM（在手机上运行 Llama）  

MLC LLM 是一个开源项目，它使得在各种设备和平台上本地运行语言模型成为可能，包括 iOS 和 Android。  

对于 iPhone 用户，App Store 上有一个 MLC 聊天应用。MLC 现在支持 Llama 2 的 7B、13B 和 70B 版本，但它仍处于测试阶段，尚未出现在 Apple Store 版本中，因此你需要安装 TestFlight 来试用。请查看此处安装测试版的说明。
