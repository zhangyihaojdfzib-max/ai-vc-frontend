# 五个实验室，五种思维：用小模型构建多模型金融剧

第二份 Build Small 黑客松现场报告：当新兴经济中的每个 Agent（智能体）运行在不同实验室的小模型上，而玩家成为幕后操纵的金融家时，会发生什么。

《千Token森林》的第一个版本是一个天气之神沙盒：五个森林生物在一个微调的 0.5B 模型上交易货物，你通过冲击扰动世界，观察泡沫和崩盘的出现。这是一个不错的玩具。但它更像是一个你观看而非参与的东西。

v2 将其重建为一个你可以操作的游戏。你是森林的庇护者，一个影子金融家：你放贷收息，散布可能真实或虚假的内幕消息，做空市场，行贿，促成联盟，而一位治安官则追查你是否利用不应知晓的信息进行交易。生物们会记住你如何对待它们，并暗中报复。而最大的变化在于底层：每个生物现在都使用不同实验室的小模型进行思考。以下是工程报告。

## 异质性就是产品，而非约束

运行一个 Agent（智能体）议会的显而易见的方式是使用一个模型、多个提示词。v2 运行了四个模型：gpt-oss-20b（OpenAI）、MiniCPM3-4B（OpenBMB）、Nemotron-Mini-4B（NVIDIA），以及我自己微调的 Qwen 0.5B。其意义并非为了新奇而新奇。当参与者真正不同时，市场才变得有趣；而四个实验室的模型，基于不同数据、采用不同后训练方式训练而成，在小模型领域已经算是差异极大。猫头鹰囤积的方式与狐狸投机的方式截然不同。这个议会是一场实时的辩论，而非一个脚本。

将四个不同的模型部署在同一个平台上，揭示了真正的教训：摩擦几乎完全出现在服务层，而非模型层。

- 当前的 vLLM（0.22.1）在加载时 JIT 编译内核，需要 CUDA 工具包（nvcc）存在。一个精简的基础镜像并未附带它，因此所有四个模型都因"找不到 nvcc"而同样失败，直到我将它们基于 CUDA 开发镜像。这不是 gpt-oss 的怪癖；这是该 vLLM 版本的普遍问题。一个镜像修复解除了所有障碍。
- gpt-oss-20b 以其原生的 MXFP4 量化运行，并能在 24GB L4 上轻松容纳，无需高端 GPU。它还使用一种通道格式，将答案包裹在分析前言中，因此消费者需要提取最终的通道。
- MiniCPM3 需要 `trust_remote_code`；Nemotron 则干净加载。每个模型各有其陷阱，但都只是一行配置。

使四个异构模型变得可管理的东西，与使 v1 中单个模型变得可管理的东西相同：一个宽容的 JSON 解析与修复层，每个模型的输出都流经该层。不同的 Token 化器和格式化习惯会产生不同的畸形输出；解析器丢弃无法修复的内容，而模拟永远不会崩溃。构建好这一层后，添加一个模型只是一个配置项，而非一次重构。

## 信息不对称需要防火墙

v2 的戏剧核心是内幕消息。你可以向一个生物耳语一条消息，它可能是**真实的**（对牌堆将抽出的下一轮市场狂热的真实预测，你的真正优势）或**虚假的**（诱饵）。依据真实消息行动并获利会增加你的热度；超过阈值后，治安官会展开调查，最终导致罚款、资产冻结或流放。

为了使这成为一个真正的游戏，消息的真实性必须对生物隐藏。它们看到的是谣言文本；它们绝不能看到标记。这是一个安全属性，而非 UI 细节，而小模型 Agent（智能体）使其更加尖锐：模型可能重复的一切，都是你放入其提示词中的内容。因此，隐藏的标记完全存在于提示词之外（在玩家的账本上），在构建时从公共事件记录中剥离，而叙述者唯一总结的只有公共事件。一个单一的测试会在每一轮扫描每个生物的完整提示词，查找被禁止的 Token。这个测试是整个套件中最重要的一个。当你向一个 Agent（智能体）提供秘密信息时，假设它会泄露，除非测试证明它不能。

## 记忆是廉价的戏剧，只要你加以限制

生物拥有持久的关系：对庇护者以及彼此之间的有符号情感值，受事件影响（你做空了我的庄稼，你偿还了贷款，你让我与对手结盟）。一个变得敌对的生物会拒绝你的贷款并给你更差的报价；结盟的生物会停止互相压价，行为如同卡特尔。

陷阱在于提示词膨胀。原始历史会无限增长，小模型会淹没其中。解决办法是永远不要把历史放入提示词：模型看到一行分桶摘要（"你对 Oona 感到温暖，对庇护者保持警惕"），仅包含最强的几种情感，由整数情感值推导而来。笔记会被保留以追踪痕迹，但加以限制且从不展示。行为偏差部分是涌现的（摘要引导模型），部分是机械的（强烈敌对的生物确定性地拒绝），因此它是可观察和可测试的，而非一种期望。

## 用小模型构建的要点

小模型是可靠的格式生成器，但不可靠的推理器；你通过结构、提示词和小规模微调来缩小差距，而非通过规模。异构的议会比同质的更有趣，一旦服务层稳固，你只需付出配置的代价。给予 Agent（智能体）的秘密信息是一个防火墙问题，而防火墙属于数据流，通过测试来证明，而非通过提示词指令。而持久记忆是让 Agent（智能体）感觉鲜活的最廉价方式，只要提示词只看到有界的摘要。

小模型，大冒险。整个议会是开放的，追踪记录也是开放的。
