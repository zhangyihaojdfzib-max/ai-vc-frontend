# 隆重推出 Gradio 全新数据表格组件！

Gradio 的 `gr.Dataframe` 组件是我们最受欢迎的组件之一，我们已在各类精彩应用中看到它的身影，例如排行榜、仪表盘和交互式可视化。尽管我们已有相当一段时间未对数据表格进行改动，但积压的问题日益增多，一些改进需求也已存在多时。

那么——现在我们无比激动地宣布，Gradio 的数据表格组件迎来一系列全新更新。在过去六周里，我们解决了超过 70 个数据表格相关问题——包括错误修复、改进和功能增强。

### 1. 多单元格选择

现在可以一次性选择多个单元格！轻松复制或删除所选区域内的值。

### 2. 行号与列固定

添加行号列，并使用 `pinned_columns` 参数在浏览宽数据集时保持关键列可见。再也不怕迷失方向！

### 3. 复制按钮与全屏按钮

通过全新的复制按钮，轻松将单元格值复制为逗号分隔格式。需要更好的视野？全屏按钮让您无干扰地进行交互，可通过 `show_full_screen` 参数启用。

### 4. 返回顶部按钮

看看这些数据吧！现在只需一键即可滚动回顶部。

### 5. 无障碍升级与增强样式

改进的键盘导航使 `gr.Dataframe` 比以往更易访问。您还可以通过专用的样式参数掌控数据表格的外观，提升应用的用户体验。

### 6. 行与列选择

在选择事件中访问整行数据，实现更直观的交互与数据操作。

### 7. 静态列

使用 `static_columns` 参数指定不可编辑的列，自定义数据表格的交互性。

### 8. 搜索功能

通过将 `show_search` 参数设置为 `"search"`，利用强大的搜索功能快速找到所需数据。

### 9. 筛选功能

将 `show_search` 设置为 `"filter"`，通过灵活的筛选选项缩小数据集范围，精准聚焦所需内容。

### 10. 改进的单元格选择

体验更流畅、更直观的单元格选择，操作方式完全符合您的预期。

更新后的数据表格已在最新版 Gradio 中上线。通过 `pip install --upgrade gradio` 更新您的安装。

```python
import gradio as gr

df_headers = ["名称", "数量", "最小尺寸（厘米）", "最大尺寸（厘米）", "最小重量（千克）", "最大重量（千克）", "最短寿命（年）", "最长寿命（年）"]
df_data = [
    ["爱尔兰赤狐", 185000, 48, 92, 4.2, 6.8, 3, 5],
    ["爱尔兰獾", 95000, 62, 88, 8.5, 13.5, 6, 8],
    ["爱尔兰水獭", 13500, 58, 98, 5.5, 11.5, 9, 13]
]

with gr.Blocks() as demo:
    df = gr.Dataframe(
        label="爱尔兰野生动物",
        value=df_data,
        headers=df_headers,
        interactive=True,
        show_search="search",
        show_copy_button=True,
        show_fullscreen_button=True,
        show_row_numbers=True,
        pinned_columns=1,
        static_columns=[0],
        column_widths=["300px"]
    )

demo.launch()

```
