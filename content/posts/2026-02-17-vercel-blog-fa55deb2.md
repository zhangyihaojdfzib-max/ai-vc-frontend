您现在可以直接在 GitHub 拉取请求审查中或 Vercel 仪表板上，从 Vercel Agent（智能体）获取针对构建失败的自动代码修复建议。

当 Vercel Agent（智能体）审查您的拉取请求时，它会扫描您的部署以查找构建错误，并在检测到失败时，根据您的代码和构建日志自动提供代码修复建议。

此外，每当检测到构建错误时，Vercel Agent（智能体）还可以在 Vercel 仪表板内自动提供代码修复建议，并建议将代码更改提交至 GitHub 拉取请求，以便在与您的代码合并前进行审查。
