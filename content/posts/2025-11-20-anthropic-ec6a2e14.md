# Anthropic 以1830亿美元投后估值完成130亿美元F轮融资

Anthropic 已完成由 ICONIQ 领投的130亿美元F轮融资。此轮融资使 Anthropic 的投后估值达到1830亿美元。本轮融资由 ICONIQ 与富达管理与研究公司、光速创投共同领投。此次投资反映了 Anthropic 持续的增长势头，并巩固了我们作为面向企业、开发者和高级用户领先智能平台的地位。

本轮重要投资者包括 Altimeter、Baillie Gifford、贝莱德旗下基金、黑石集团、Coatue、D1 Capital Partners、泛大西洋投资、General Catalyst、新加坡政府投资公司、高盛另类投资与成长股权部、Insight Partners、Jane Street、安大略省教师养老金计划、卡塔尔投资局、TPG、普信集团、普信投资管理公司、WCM 投资管理以及 XN。

"从财富500强公司到原生AI初创企业，我们的客户都依赖 Anthropic 的前沿模型和平台产品来完成他们最重要、最关键的任务，"Anthropic 首席财务官 Krishna Rao 表示。"我们看到整个客户群的需求呈指数级增长。此次融资证明了投资者对我们财务表现的非凡信心，以及他们与我们合作继续推动史无前例增长的坚定决心。"

自2023年3月推出 Claude 以来，Anthropic 实现了快速增长。到2025年初（推出不到两年），Anthropic 的年化收入已增长至约10亿美元。截至2025年8月（仅八个月后），我们的年化收入已超过50亿美元——使 Anthropic 成为历史上增长最快的科技公司之一。

Anthropic 的发展轨迹得益于我们顶尖的技术人才、对安全的专注以及前沿研究（包括开创性的对齐和可解释性工作），所有这些都支撑着我们模型的性能和可靠性。每天都有越来越多的企业、开发者和消费者高级用户信任 Claude 来帮助他们解决最具挑战性的问题。Anthropic 目前为超过30万家企业客户提供服务，我们的大客户数量（即每位客户贡献年化收入超过10万美元）在过去一年增长了近7倍。

这一增长覆盖 Anthropic 整个平台，在面向企业、开发者和消费者的产品上均取得进展。对于企业客户，我们的 API 和行业专用产品使其能够轻松地将强大的人工智能集成到关键应用中，无需复杂的集成工作。自2025年5月全面推出以来，开发者已将 Claude Code 作为首选工具。Claude Code 迅速起飞——年化收入已超过5亿美元，使用量在短短三个月内增长超过10倍。对于个人用户，Claude 的 Pro 和 Max 计划为日常任务和专业项目提供了增强的 AI 能力。

"Anthropic 正沿着一条非凡的轨迹发展，融合了卓越的研究、技术领导力和对客户的不懈关注。我们很荣幸能与 Dario 及其团队合作，我们领投其 F 轮融资反映了我们对其价值观以及塑造负责任 AI 未来能力的信念，"ICONIQ 合伙人 Divesh Makan 表示。"企业领导者告诉我们他们亲眼所见——Claude 可靠、建立在可信赖的基础上，并由真正专注于长期发展的领导者引领。"

F 轮投资将扩大我们满足日益增长的企业需求的能力，深化我们的安全研究，并支持国际扩张，同时我们将继续构建可靠、可解释和可操控的 AI 系统。
