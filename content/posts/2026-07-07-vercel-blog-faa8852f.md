Vercel 正在收购 Better Auth，这家公司是开源 TypeScript 身份验证库的幕后团队，该库每周 npm 下载量超过 470 万次，拥有 850 多位贡献者。创始人 Bereket Engida 及核心团队将加入 Vercel，继续推进 Better Auth 和 Agent（智能体）身份方面的工作。

![](/images/posts/77bbe1df1e67.jpg)

## 复制链接到标题开放网络

去年，我们发布了在 Next.js、AI SDK 和 Nuxt 上的方法：默认开放、松散耦合、可移植到任何平台的软件。

Better Auth 将其应用于身份验证：与框架无关，可在任何地方运行，并为开发者提供他们自主拥有的身份验证。

## 复制链接到标题Agent（智能体）身份

当 Agent（智能体）代表你执行操作时，它会在你的身份和权限下运行，因此它所接触的每个服务看到的都是你，而不是 Agent（智能体）。目前没有简洁的方法来限制单个 Agent（智能体）或子 Agent（智能体）能做什么，也无法在不切断其他 Agent（智能体）的情况下单独关闭某一个。

Better Auth 团队一直在构建 Agent Auth（智能体身份验证），以便每个 Agent（智能体）都能拥有自己的身份、作用域和可撤销的权限，而你作为唯一的控制点。Agent（智能体）身份是 Agent（智能体）基础设施的基础。

Better Auth 团队将在 Vercel 继续这项工作，将 Agent（智能体）身份引入 Vercel Connect 和 eve。

## 复制链接到标题Better Auth 的下一步计划

该库将继续保持 MIT 许可下的免费开源状态，保留其名称，团队将继续以相同的开放贡献模式、社区治理和跨生态系统的框架支持来主导开发。

请阅读 Better Auth 博客上的公告。探索 Better Auth 文档和 Agent Auth 协议（智能体身份验证协议），或在 GitHub 上为该项目做出贡献。
