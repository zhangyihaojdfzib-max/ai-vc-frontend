# 发布 Claude Instant 1.2

![](/images/posts/87f43bd2cdbb.jpg)

与 Claude 合作的企业现在可以通过我们的 API 访问最新版本的 Claude Instant 1.2。Claude Instant 是我们速度更快、价格更低但能力依然强大的模型，能够处理包括日常对话、文本分析、摘要和文档理解在内的多种任务。Claude Instant 1.2 在实际应用场景中融合了我们最新模型 Claude 2 的优势，并在数学、编码、推理和安全性等关键领域取得了显著进步。它能生成更长、更有条理的回复，并能更好地遵循格式指令。Instant 1.2 在引用提取、多语言能力和问答方面也有所改进。

Claude Instant 1.2 在数学和编码方面的表现优于 Claude Instant 1.1，在 Codex 评估中达到了 58.7%，而之前模型为 52.8%。它在 GSM8K 基准测试中也获得了 86.7% 的分数，而 Claude Instant 1.1 为 80.9%。

![Claude Instant 1.1 与 1.2 的性能对比](/images/posts/c66af42f37b8.jpg)

我们的最新模型在安全性方面也有所提升。如我们的自动化红队评估所示，它产生的幻觉更少，并且更能抵抗越狱攻击。

![Claude 模型的安全性评估。数值越低越好。](/images/posts/9927fe844803.jpg)

希望使用 Claude Instant 1.2 的开发者现在可以通过我们的 API 调用最新模型（定价信息可在此处查看）。如果您是企业并希望与我们合作，可以在此处表明您的意向。
