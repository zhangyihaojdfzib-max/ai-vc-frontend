我们推出了 Vercel Sandbox CLI，这是一个用于管理隔离计算环境的命令行界面。它基于大家熟悉的 Docker CLI 模型构建，开发者现在可以：

- 为 Node.js (node22) 或 Python (python3.13) 工作负载创建并运行沙盒。
- 在现有沙盒内执行命令。
- 在本地和远程环境之间复制文件。
- 跨项目和团队列出、停止和移除沙盒。
- 以交互方式运行，支持 `--tty`、`--interactive` 以及用于端口转发的 `--publish-port`。
- 通过身份验证令牌、环境变量和超时设置来自动化工作流。

完整的参考文档现已发布在 [Sandbox CLI 文档](https://vercel.com/docs/sandbox/cli) 中。
