Vercel 能够检测并支持 TanStack Start 应用程序，这是一个由 TanStack Router 驱动的、适用于 React 和 Solid 的全栈框架。

创建一个新的 TanStack Start 应用，或在您现有应用的 `vite.config.ts` 中添加 `nitro()`，即可轻松部署您的项目：

```
1import { tanstackStart } from '@tanstack/react-start/plugin/vite'2import { defineConfig } from 'vite'3import viteReact from '@vitejs/plugin-react'4import { nitro } from 'nitro/vite'5 6export default defineConfig({7  plugins: [8    tanstackStart(),9    nitro(),10    viteReact(),11  ],12})
```

在 Vercel 上，TanStack Start 应用默认使用**弹性计算**和**动态 CPU 定价**。这意味着您的 TanStack Start 应用将根据流量自动扩缩容，并且您只需为实际使用的资源付费，无需为闲置的函数时间付费。

访问 [Vercel 上的 TanStack Start 文档](https://vercel.com/docs/frameworks/tanstack-start) 了解更多信息。
