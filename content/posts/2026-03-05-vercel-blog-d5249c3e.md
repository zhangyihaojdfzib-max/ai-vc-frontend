AI Gateway 现已支持按推理设置**供应商超时**以实现快速故障转移。若供应商未在配置的超时时间内开始响应，AI Gateway 将中止请求并回退至下一个可用供应商。

供应商超时功能目前处于测试阶段，仅适用于 BYOK（自带密钥）凭证。请注意，部分供应商不支持流式取消，因此根据供应商策略，超时请求仍可能产生费用。

**基础用法**

通过 `providerOptions.gateway` 中的 `providerTimeouts` 按供应商设置超时时间（毫秒）。

```
1const result = streamText({2  model: 'openai/gpt-5.4',3  prompt,4  providerOptions: {5    gateway: {6      providerTimeouts: {7        byok: { openai: 15000 }, 8      },9    },10  },11});
```

在此示例中，GPT 5.4 的 BYOK 凭证将在 15 秒后超时，并回退至系统凭证。

**多供应商与故障转移的高级用法**

结合 `order` 参数同时控制供应商调用顺序与故障转移速度。

```
1const result = streamText({2  model: 'anthropic/claude-sonnet-4.6',3  prompt,4  providerOptions: {5    gateway: {6      order: ['anthropic', 'bedrock', 'vertex'],7      providerTimeouts: {8        byok: {9          anthropic: 10000,10          bedrock: 15000,11        },12      },13    },14  },15});
```

此示例将首先尝试 Anthropic（10 秒超时），随后回退至 Bedrock（15 秒超时），最后回退至 Vertex（使用网关默认超时设置）。

更多信息请参阅自定义供应商超时**文档**。
