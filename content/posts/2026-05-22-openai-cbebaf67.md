# 为惠及所有人而建

作者：布雷特·泰勒，OpenAI 董事会主席

![为所有人而建](/images/posts/bc0ebe5fa713.png)

OpenAI 已完成其资本重组，简化了公司结构。非营利组织仍保持对营利性实体的控制权，并在通用人工智能（AGI）到来之前，拥有了一条直接获取主要资源的路径。

该非营利组织现更名为 OpenAI 基金会，持有目前估值约 1300 亿美元的营利性实体股权，使其成为有史以来资源最雄厚的慈善组织之一。此次资本重组还赋予基金会额外的所有权，前提是 OpenAI 的营利性实体达到特定的估值里程碑。

OpenAI 的使命——确保通用人工智能（AGI）惠及全人类——将通过其业务和基金会共同推进。OpenAI 作为一家公司越成功，非营利组织的股权价值就越高，非营利组织将利用这些资金来资助其慈善工作。

OpenAI 基金会将初步聚焦于 250 亿美元的承诺，涵盖两个领域：

- **健康与疾病治疗。** OpenAI 基金会将资助旨在加速健康领域突破的工作，以便每个人都能受益于更快的诊断、更好的治疗和疾病治愈。这将从创建开源且负责任构建的前沿健康数据集、以及为科学家提供资金等活动开始。
- **人工智能韧性的技术解决方案。** 正如互联网需要一个全面的网络安全生态系统——保护电网、医院、银行、政府、公司和个人——我们现在需要一个并行的人工智能韧性层。OpenAI 基金会将投入资源支持人工智能韧性的实用技术解决方案，这关乎最大化人工智能的益处并最小化其风险。

这建立在 5000 万美元的“以人为本人工智能基金”以及非营利委员会的建议之上。

## 使命为核心

OpenAI 成立于 2015 年，最初是一家非营利组织；其使命是确保通用人工智能（AGI）惠及全人类。如今，OpenAI 仍然是一家致力于同一使命的非营利组织。

OpenAI 基金会控制着成立于 2019 年的营利性业务，使我们的使命始终处于核心地位。该营利性实体现已成为一家公益公司，名为 OpenAI Group PBC——与 OpenAI 基金会拥有相同的使命——这确保了公司的使命与商业成功能够共同推进。了解更多关于我们结构的信息。

此次资本重组保持了当今行业内以使命为导向的治理结构的最强代表性。

此次资本重组是在与加利福尼亚州和特拉华州总检察长办公室进行了近一年的建设性对话后完成的。作为这些讨论的结果，我们进行了几项调整，我们相信 OpenAI——以及我们服务的公众——因此变得更好。

## 前进之路

OpenAI 基金会和 OpenAI Group 将协同工作，以推进解决人工智能进步带来的难题和机遇的方案。这包括将智能打造成一种人人都能受益的工具，构建安全且对齐的系统，加速科学发现，以及加强全球合作与韧性。

我们相信，世界上最强大的技术必须以反映世界共同利益的方式发展。我们资本重组的完成使我们有能力继续推动人工智能的前沿发展，并通过更新的公司结构确保进步服务于每一个人。
