# 语言模型现已登陆 Replicate  

- bfirsh  

现在，你可以在 Replicate 上部署、运行和微调大语言模型。  

我们提供了 FLAN-T5、GPT-J 和 LLaMA 的官方版本，你也可以推送任何其他自定义模型。同时，我们还发布了语言模型微调的预览版。  

语言模型可以像 Replicate 上的其他模型一样，仅用几行代码即可运行：  

```
import replicate

output = replicate.run(
    "replicate/flan-t5-xl:3ae0799123a1fe11f8c89fd99632f843fc5f7a761630160521c4253149754523",
    input={"prompt": "Answer the following yes/no question by reasoning step by step. Can a dog drive a car?"}
)

for s in output:
    print(s, end="", flush=True)
```

你可以通过 Python、Node.js 或 HTTP API 运行它们，无需搭建服务器或 GPU。  

试试看：  

- Google 的 Flan-T5 XL  
- EleutherAI 的 GPT-J-6B  
- Meta AI 的 LLaMA  

## 推送自定义模型  

你可以将任何语言模型推送到 Replicate。如果你想与世界分享开源模型，可以设为公开；如果你想为产品大规模部署自定义模型，则可以设为私有。  

我们编写了一份指南，教你如何将 Hugging Face 的 Transformers 模型推送到 Replicate。  

## 预览：微调模型  

有了好的提示词，这些模型可以完成许多任务，但在你自己的数据上进行微调后，它们会变得更加强大。你可以创建一个了解你产品的支持机器人，以你的风格撰写邮件，从文本中提取结构化数据，以及实现各种功能。  

我们发布了在 Replicate 云端训练模型的预览版。你可以微调 Flan-T5、GPT-J 和 LLaMA。我们很快会添加对其他模型以及自定义训练脚本的支持。  

```
replicate.trainings.create(
   version="...",
   input={"train_data": "..."},
   destination="bfirsh/hackerbot",
)
```

我们目前只向少数人开放访问权限，并将在未来几周内逐步扩大开放范围。如果你想使用该功能，请通过支持渠道联系我们，并告知你的相关信息及使用意图。
