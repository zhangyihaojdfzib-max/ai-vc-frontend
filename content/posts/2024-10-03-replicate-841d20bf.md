## FLUX1.1 [pro] 是一个新模型

FLUX1.1 [pro] 是 FLUX.1 [pro] 的一个全新、更快、更强大的版本。

它生成图像的速度是其前身 FLUX.1 [pro] 的六倍，同时图像质量更高，对提示词的遵循度更好，输出多样性也更强。截至 2024 年 10 月 1 日，独立基准测试显示，与其他开源模型相比，它生成的图像质量最高。

FLUX1.1 [pro] 在 Replicate 上的定价为每张图像 0.04 美元。

以下是如何使用 Replicate 的 JavaScript 客户端运行 FLUX1.1 [pro] 的示例：

```
import Replicate from "replicate"
const replicate = new Replicate()

const model = "black-forest-labs/flux-1.1-pro"
const input = {
  prompt: "A giant blueberry with a neon sign that says 'World's Juiciest Berries' floating above it",
  aspect_ratio: "1:1",
  output_format: "webp",
  output_quality: 80
}

const output = await replicate.run(model, { input })
console.log(output)
```

### 提示词上采样

FLUX1.1 [pro] 包含一个新的提示词上采样功能。它使用一个大语言模型自动将你的提示词扩展为更长、更详细的版本。你可以使用提示词上采样功能，从你的提示词中获得更多样化的输出。
