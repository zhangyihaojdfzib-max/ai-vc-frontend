# Claude 登陆 Amazon Bedrock：获准用于 FedRAMP High 及 DoD IL4/5 工作负载

今日，我们宣布 Claude 模型已获准通过 AWS GovCloud（美国）区域的 **Amazon Bedrock** 用于 FedRAMP High 及美国国防部（DoD）影响等级 4 和 5 的工作负载。

联邦机构和国防组织现在可以在满足政府严格安全要求的同时，利用 Claude 先进的人工智能能力——这为国防、情报和敏感民用行动等关键任务应用开辟了新的可能性。

## 为最敏感的工作负载启用安全 AI

美国国防部影响等级 4、5 以及 FedRAMP High 授权代表了非密级和受控非密信息云安全认证的最高级别。Claude 已获准在 Amazon Bedrock 中用于需要 FedRAMP 和 DoD 授权的工作负载，使 Anthropic 和 AWS 能够支持涵盖所有非密级政府工作负载的用户。

Claude 3.5 Sonnet v1 和 Claude 3 Haiku 现已可用，同时提供的 Amazon Bedrock 功能还包括 Amazon Bedrock Agents、Guardrails、Knowledge Bases 和 Model Evaluation，这些功能可帮助客户自动化多步骤任务、实施定制化安全防护、定制应用程序并选择最合适的模型。未来可能会添加更多 Claude 模型，包括 Claude 3.7 Sonnet 和 Claude 4 模型。

通过 Amazon Bedrock 中的 Claude，联邦机构现在可以：

*   在安全环境中部署前沿 AI 模型，用于复杂的文档分析和情报综合
*   构建能够处理受控非密信息的 AI Agent（智能体）
*   利用 Claude 的高级智能和 200K Token 上下文窗口，对大型数据集进行全面分析

## 加速政府 AI 应用

我们的目标是让 Claude 模型在公共部门客户运营的每个区域都能实现商业化可用，满足其关键任务工作负载所需的安全认证。Amazon Bedrock 上的 Claude 提供了一项完全托管的服务，在保持所需安全控制的同时，消除了基础设施的复杂性。

政府客户现在可以灵活地在多个安全平台上部署一些能力最强的 AI 系统——包括用于 FedRAMP High 和 IL2 的 Google Cloud Vertex AI，以及用于 FedRAMP Moderate 和 High 与 DoD IL2 和 IL4/5 的 Amazon Bedrock。

## 开始使用

联邦机构雇员、国防承包商和经批准的合作伙伴可以通过 AWS GovCloud（美国）区域的 Amazon Bedrock，使用熟悉的 AWS API 和管理工具访问 Claude 模型。

联系我们的公共部门团队，获取在您的联邦环境中实施 Claude 的指导，同时确保完全符合政府安全要求。
