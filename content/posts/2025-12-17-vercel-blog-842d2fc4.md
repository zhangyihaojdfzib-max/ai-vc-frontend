1 分钟阅读
今天，我们宣布在 Vercel Marketplace 上原生支持 AWS 数据库，包括 Amazon Aurora PostgreSQL、Amazon Aurora DSQL 和 Amazon DynamoDB。
这为开发者提供了一条直接路径，可以在 Vercel 仪表板内直接配置和管理可扩展、可用于生产的 AWS 数据库，无需手动设置，并且：
*   一键支持创建新的 AWS 账户、配置新的 AWS 数据库并将其链接到您的 Vercel 项目。
*   通过简化的区域选择、安全的凭证处理以及从 Vercel 统一监控 AWS 数据库资源，改善了开发者体验。
*   连接字符串和凭证的自动环境变量，安全地存储在您的 Vercel 项目中。
*   为新 AWS 客户提供免费入门套餐，包括 100 美元信用额度，并带有在 AWS 控制台中管理或升级套餐的深度链接。
即将推出：将数据库配置到您现有的 AWS 账户中，将其附加到您的项目，并直接在 v0 内部访问 AWS 数据库。

```javascript
import { Signer } from "@aws-sdk/rds-signer";
import { awsCredentialsProvider } from "@vercel/functions/oidc";
import { attachDatabasePool } from "@vercel/functions";
import { Pool } from "pg";

const signer = new Signer({
  hostname: process.env.PGHOST,
  port: Number(process.env.PGPORT),
  username: process.env.PGUSER,
  region: process.env.AWS_REGION,
  credentials: awsCredentialsProvider({
    roleArn: process.env.AWS_ROLE_ARN,
    clientConfig: { region: process.env.AWS_REGION },
  }),
});

const pool = new Pool({
  host: process.env.PGHOST,
  user: process.env.PGUSER,
  database: process.env.PGDATABASE || "postgres",
  password: () => signer.getAuthToken(),
  port: Number(process.env.PGPORT),
  ssl: { rejectUnauthorized: false },
});
attachDatabasePool(pool);

pool.query("select * from movies where id = $1", [id]);
```

开始使用
1.  导航到 Vercel Marketplace 并选择 AWS
2.  选择“创建新账户”以配置数据库
3.  选择您的数据库类型、区域和套餐（包括为新 AWS 客户提供的含 100 美元信用额度的免费入门套餐），然后点击创建
4.  将其连接到您的项目。Vercel 会自动处理凭证和配置
您也可以通过部署“Movie Fetching Database”模板来尝试一个可运行的示例，以端到端地查看集成效果。
