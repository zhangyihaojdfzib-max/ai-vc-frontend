# Retro Diffusion 的像素艺术模型现已登陆 Replicate

-   andreasjansson

Retro Diffusion 精心打造了一套精美的模型和风格，用于创建游戏资源、角色精灵、图块和其他精彩的复古图形。这些模型专门针对网格对齐、有限调色板的像素艺术进行训练。现在，您可以在 Replicate 上运行它们。

![banner](/images/posts/8eb708ad5e41.webp)

Retro Diffusion 在 Replicate 上提供了四个模型：

-   rd-fast: replicate.com/retro-diffusion/rd-fast
-   rd-plus: replicate.com/retro-diffusion/rd-plus
-   rd-tile: replicate.com/retro-diffusion/rd-tile
-   rd-animation: replicate.com/retro-diffusion/rd-animation

这些模型提供了大量不同的风格预设可供使用，其中大多数支持任意宽度/高度、输入图像、调色板图像、背景移除和无缝平铺。

## rd-fast：快速的像素艺术图像生成

retro-diffusion/rd-fast 针对速度进行了优化，支持 15 种风格，从肖像到 Minecraft 风格的物品。

![rd-fast](/images/posts/c566ea1322fd.webp)

```
import replicate

output = replicate.run(
    "retro-diffusion/rd-fast",
    input={
        "prompt": "knight character, retro game asset, side view",
        "style": "game_asset",
        "width": 128,
        "height": 128,
        "num_images": 1,
        "remove_bg": True,
    },
)
print(output)
```

## rd-plus：高质量且地道的像素艺术图像生成

retro-diffusion/rd-plus 是更高质量的文生图模型，提供场景、地图、UI 和图标等风格。

![rd-plus](/images/posts/a78a15c3ac2e.webp)

```
import replicate

output = replicate.run(
    "retro-diffusion/rd-plus",
    input={
        "prompt": "top-down forest path with trees and rocks, classic jrpg map",
        "style": "topdown_map",
        "width": 256,
        "height": 256,
        "num_images": 1,
        "tile_x": True,
        "tile_y": True,
    },
)
print(output)
```

## rd-tile：生成像素艺术图块集所需的所有工具

retro-diffusion/rd-tile 专注于图块和图块集，包括单个图块、变体和场景对象。

![rd-tile](/images/posts/918a6139184d.webp)

```
import replicate

output = replicate.run(
    "retro-diffusion/rd-tile",
    input={
        "prompt": "grass and dirt path tileset with stones and flowers",
        "style": "tileset",
        "width": 32,  # tile width
        "height": 32,  # tile height
        "num_images": 1,
    },
)
print(output)
```

## rd-animation：风格一致的动画像素艺术精灵生成

retro-diffusion/rd-animation 可生成动画精灵或精灵表，具有一致的帧结构和低帧数，符合常见游戏引擎的要求。

![rd-animation](/images/posts/6e530d297afc.webp)

```
import replicate

output = replicate.run(
    "retro-diffusion/rd-animation",
    input={
        "prompt": "small pirate character with parrot, walking animation",
        "style": "four_angle_walking",
        "width": 48,
        "height": 48,
        "return_spritesheet": True,
    },
)
print(output)
```

这些示例使用 Python，但您也可以使用 Replicate 的 SDK 从 JavaScript 和其他语言调用这些模型。
