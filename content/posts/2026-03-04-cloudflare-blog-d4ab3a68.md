# 从车牌到工牌：网关授权代理的演进

2026-03-04

- Ankur Aggarwal
- Alex Holland

![](/images/posts/8b624b138d15.png)

我们经常谈论“理想”状态——每台设备都安装了像 Cloudflare One Client 这样的托管客户端，提供深度可见性和无缝保护。然而，现实往往不尽如人意。

有时您需要处理公司收购、管理虚拟桌面，或者在严格监管的环境中工作，根本无法在终端上安装软件。即使您无法完全管理设备，仍然需要保护其流量。

要弥补这一差距，就需要将身份验证的挑战从设备转移到网络本身。通过将浏览器的原生代理能力与我们的全球网络相结合，我们可以在任何能够访问互联网的设备上验证用户并执行精细策略。我们构建了**网关授权代理**和**代理自动配置（PAC）文件托管**功能，以自动化此身份验证过程，并简化非托管设备连接至 Cloudflare 的方式。

### 问题所在：有时仅靠IP地址是不够的

早在2022年，我们发布了**代理端点**功能，允许您通过 Cloudflare 路由流量以应用过滤规则。它解决了即时访问需求，但存在一个显著的“身份危机”。

因为该系统依赖静态IP地址来识别用户，这有点像保安只认车不认人。如果一辆车（特定IP）出现，它就被放行。但如果司机换了车或在不同地点工作，保安就会困惑。这带来了几个主要问题：

- **匿名日志**：我们知道IP地址，但不知道具体是谁。
- **脆弱策略**：如果用户搬到新的家庭或办公室，端点就会失效或需要更新。
- **手动维护**：您必须托管自己的PAC文件（即告诉浏览器代理在哪里的“GPS”）——这又给您的团队增加了一项管理任务。

### 解决方案：授权代理

授权代理访问策略设置页面

新的**网关授权代理**在入口处增加了一个“工牌读取器”。我们不再仅仅查看流量来源，而是在执行网关过滤之前，使用类似 Cloudflare Access 的登录方式来验证用户身份。

可以将其理解为从基于车牌号的访客名单，转变为每个人都拥有自己工牌的系统。这带来了几个巨大的好处：

- **真正的身份集成**：与代理端点相关的日志现在能准确显示哪个用户正在访问哪个网站。即使设备上没有安装客户端，您也可以编写特定规则，例如“只有财务团队可以访问此会计工具”。
- **多身份提供商支持**：这对于大型公司或正在进行并购的公司来说是一项超能力。您可以选择向用户显示哪些身份提供商。您可以同时显示一种或多种登录方式（如 Okta 和 Azure AD）。这是竞争对手目前无法提供的灵活性水平。

### 授权代理如何跟踪身份

授权代理使用签名的 JWT cookie 来维持身份，但有一个问题：当您首次通过代理访问新域时，还没有 cookie。这就像进入每栋新大楼时都需要出示工牌。

上面的流程图精确说明了此身份验证过程的工作原理：

- **首次访问域**：当您导航到新域时，网关授权代理会检查是否存在域身份 cookie。如果没有，您将被重定向到 Cloudflare Access，然后检查是否存在现有的 Cloudflare Access 身份 cookie。如果您已通过 Cloudflare Access 身份验证，我们会为该域生成一个安全令牌。如果尚未验证，我们会将您重定向到您的身份提供商进行登录。
- **对用户透明**：得益于 Cloudflare 的全球边缘网络，整个过程在几毫秒内完成。重定向速度极快，用户根本察觉不到——他们只会看到页面正常加载。
- **后续访问即时完成**：一旦设置了 cookie，对该域（及其子域）的所有后续请求都会立即获得授权。无需再次重定向。

由于采用了这种方法，我们可以记录和过滤每个人在其访问的所有域上的流量，并在需要时立即撤销访问权限——所有这些都无需在用户设备上安装任何软件。

### 无需再自行托管PAC文件

我们也在简化设置过程中的“家庭作业”。您现在可以直接在 Cloudflare 上托管您的 PAC 文件，使用**代理自动配置（PAC）文件托管**功能。

### 这适合您的团队吗？

虽然我们仍然推荐 Cloudflare One Client 以获得更大的控制权和最佳用户体验，但授权代理非常适合特定场景：

- **虚拟桌面（VDI）**：用户登录虚拟机并使用浏览器访问互联网的环境。
- **并购**：当您需要快速将两家不同的公司纳入同一安全保护伞下时。
- **合规限制**：当法律或技术上禁止在终端上安装软件时。

### 下一步是什么？

这扩展了我们无需客户端的安全选项，使其能够连接至Cloudflare One。我们目前正在扩展与授权端点相关的支持身份验证方法。敬请期待Kerberos、mTLS以及传统的用户名/密码验证方式，这些将为您提供更灵活的用户身份验证选择。
