# Vercel 插件现已支持 OpenAI Codex 和 Codex CLI

Vercel 插件现已支持 OpenAI Codex 和 Codex CLI。

借助该插件，团队可以访问超过 39 项平台技能、三个专业 Agent（智能体）以及实时代码验证，以协助其开发工作流程。

请在 Codex 应用或通过 Codex CLI 安装：

```
1codex2/plugins
```
