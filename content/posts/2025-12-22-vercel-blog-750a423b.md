1 分钟阅读
您现在可以通过 Vercel 的 AI 网关访问 Z.ai 的最新模型 GLM-4.7，无需其他提供商账户。
GLM-4.7 在代码生成、工具使用和多步推理方面有重大改进，尤其是在处理复杂的智能体任务时。该模型还具备更自然的语气，能提供更好的对话体验，并能生成更精致美观的前端作品。
要通过 AI SDK 开始使用 GLM-4.7 进行开发，请将模型设置为 zai/glm-4.7：
import { streamText } from 'ai';
const result = streamText({ model: 'zai/glm-4.7', prompt: `Create an interactive weather timeline app, fetch forecasts via weather tool, normalize data, render animated charts, cache results, and produce a production build.`});
AI 网关提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商水平的正常运行时间。它包含内置的可观测性、自带密钥支持以及具有自动重试功能的智能提供商路由。
