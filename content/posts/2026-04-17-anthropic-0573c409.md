# Anthropic Labs 推出 Claude Design

今天，我们正式推出 Claude Design，这是一款全新的 Anthropic Labs 产品，让您能够与 Claude 协作，创作出精美的视觉作品，例如设计稿、原型、幻灯片、单页文档等等。

Claude Design 由我们能力最强的视觉模型 Claude Opus 4.7 驱动，现以研究预览版的形式向 Claude Pro、Max、Team 和 Enterprise 订阅用户开放。我们将在今天内逐步向用户开放。

## 与 Claude 一同设计

即使是经验丰富的设计师，也不得不限制探索的广度——很少有时间去制作十几个方向的原型，因此只能局限于少数几个方案。而对于那些有想法但没有设计背景的创始人、产品经理和营销人员来说，创建并分享这些想法可能令人望而生畏。

Claude Design 为设计师提供了广泛探索的空间，也为其他人提供了一种创作视觉作品的方式。描述您的需求，Claude 会构建出第一个版本。然后，您可以通过对话、内联评论、直接编辑或自定义滑块（由 Claude 生成）来完善它，直到满意为止。当获得权限时，Claude 还可以自动将您团队的设计系统应用到每个项目中，确保输出与公司其他设计保持一致。

团队一直在使用 Claude Design 进行：

*   **逼真的原型设计**：设计师可以将静态线框图转化为易于分享的交互式原型，用于收集反馈和用户测试，无需代码审查或提交 PR。
*   **产品线框图和效果图**：产品经理可以勾勒出功能流程，并将其交给 Claude Code 进行实现，或与设计师分享以进一步优化。
*   **设计探索**：设计师可以快速创建多种方向进行探索。
*   **宣传稿和演示文稿**：创始人和客户经理可以在几分钟内从粗略的大纲转变为完整、符合品牌形象的演示稿，然后导出为 PPTX 或发送到 Canva。
*   **营销物料**：营销人员可以创建落地页、社交媒体素材和活动视觉设计，然后请设计师进行润色。
*   **前沿设计**：任何人都可以构建支持语音、视频、着色器、3D 和内置 AI 的代码驱动原型。

## 工作原理

Claude Design 遵循自然的创作流程。

*   **内置您的品牌**：在初始设置期间，Claude 通过读取您的代码库和设计文件，为您的团队构建一个设计系统。之后的每个项目都会自动使用您的颜色、字体和组件。您可以随时间完善该系统，团队也可以维护多个系统。
*   **从任何地方导入**：可以从文本提示词开始，上传图像和文档（DOCX、PPTX、XLSX），或者指引 Claude 访问您的代码库。您还可以使用网页捕获工具直接从您的网站抓取元素，使原型看起来像真实产品。
*   **通过精细控制进行优化**：在特定元素上进行内联评论、直接编辑文本，或使用调整旋钮实时调整间距、颜色和布局。然后让 Claude 将您的更改应用到整个设计中。

![](/images/posts/2421cd0f7f87.jpg)

*   **协作**：设计稿具有组织范围内的共享权限。您可以将文档设为私有，或共享链接让组织内任何人查看，或授予编辑权限，以便同事可以修改设计并在群组对话中与 Claude 一同交流。
*   **随处导出**：可以在组织内将设计稿作为内部 URL 分享，保存为文件夹，或导出到 Canva、PDF、PPTX 或独立的 HTML 文件。
*   **交接给 Claude Code**：当设计稿准备就绪可以构建时，Claude 会将所有内容打包成一个交接包，您只需一条指令即可将其传递给 Claude Code。

在接下来的几周内，我们将简化与 Claude Design 的集成构建，以便您可以将其连接到团队已经使用的更多工具。

## 开始使用

Claude Design 可供 Claude Pro、Max、Team 和 Enterprise 订阅用户使用。访问权限已包含在您的订阅计划中，并使用您的订阅额度，您也可以选择启用**额外用量**来超出这些限制继续使用。

对于企业组织，Claude Design 默认关闭。管理员可以在**组织设置**中启用它。

请访问 claude.ai/design 开始设计。
