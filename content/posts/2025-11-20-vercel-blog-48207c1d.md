您现在可以通过 Vercel 的 AI Gateway 访问 xAI 的最新模型——Grok 4.1 Fast Reasoning 和 Grok 4.1 Fast Non-Reasoning，无需其他提供商账户。这些模型拥有 200 万的上下文窗口，专为 Agent（智能体）工具调用而设计。

Grok 4.1 Fast Reasoning 最适合需要高准确性的结构化推理和 Agent（智能体）操作，而 Grok 4.1 Fast Non-Reasoning 则专为速度优化。

要在 AI Gateway 中使用 AI SDK 调用 Grok 4.1 Fast 模型，请将 `model` 参数设置为 `xai/grok-4.1-fast-reasoning` 或 `xai/grok-4.1-fast-non-reasoning`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'xai/grok-4.1-fast-reasoning',5  prompt:6  `Take the uploaded report and extract the most relevant financial trends.7   Note any outliers and provide a detailed  analysis with citations.`,8});
```

AI Gateway 提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商水平的正常运行时间。它内置了可观测性、支持自带密钥以及智能的提供商路由与自动重试功能。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
