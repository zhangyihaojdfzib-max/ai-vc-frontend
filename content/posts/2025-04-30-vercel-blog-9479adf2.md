为 Vercel Marketplace 构建原生集成的服务商现在可以使用 Checks API 为用户提供更深层次的功能。

借助 Vercel 的 Checks API，您可以在每次部署后定义并运行自定义测试和断言，然后直接在 Vercel 仪表盘中呈现可操作的结果。

作为测试服务商，您可以实现可靠性测试（例如 API 可用性、运行时错误）、性能测试（例如响应时间阈值、负载模拟）或 Web Vitals（例如布局偏移）等检查。这有助于开发人员通过您的集成，更早地在工作流程中发现实际问题。

在构建集成时，请牢记以下最佳实践：

- 提供零配置或低配置的解决方案，使开发人员能够轻松运行检查
- 提供从安装到首次结果的引导式上手体验
- 直接在 Vercel 仪表盘中展示清晰、可操作的结果
- 为高级用户记录扩展或自定义检查的方法
