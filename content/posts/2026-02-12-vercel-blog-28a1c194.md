MiniMax M2.5 现已登陆 AI Gateway。

M2.5 采用先规划后构建的方式，在编写代码前会先分解功能、结构和用户界面设计。它能处理涵盖 Web、Android、iOS、Windows 和 Mac 的全栈项目，覆盖从初始系统设计到代码审查的整个开发生命周期。与 M2.1 相比，它能更好地适应不熟悉的代码库，并使用更少的搜索轮次来解决问题。

要使用此模型，请在 AI SDK 中将模型设置为 `minimax/minimax-m2.5`：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'minimax/minimax-m2.5',5  prompt:6    `Design and implement a multi-tenant SaaS authentication system7     with role-based access control, supporting OAuth providers8     and API key management.`,9});
```

AI Gateway 提供了一个统一的 API 用于调用模型、跟踪使用情况和成本，并可配置重试、故障转移和性能优化，以实现高于提供商水平的运行时间。它内置了可观测性、支持自带密钥（Bring Your Own Key），并具备带自动重试功能的智能提供商路由。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
