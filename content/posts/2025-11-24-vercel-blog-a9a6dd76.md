Vercel 现已支持提交验证，通过要求从 GitHub 部署的提交必须经过密码学验证，从而保护您的部署。

您可以在项目设置中为已连接 GitHub 的项目启用此功能。

在 GitHub 上了解更多关于提交签名与验证的信息，或查阅我们的文档以获取关于此设置的更多说明。
