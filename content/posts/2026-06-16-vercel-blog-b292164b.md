Workflow SDK 现在支持 Vercel 上的 TanStack Start 应用。

TanStack Start 基于 Vite 和 Nitro 构建，因此现有的 workflow/vite 插件可以直接使用。将其与 tanstackStart() 一同添加到 vite.config.ts 中。

```
1import { tanstackStart } from "@tanstack/react-start/plugin/vite";2import { defineConfig } from "vite";3import { workflow } from "workflow/vite";4
5export default defineConfig({6  plugins: [7    workflow(),8    tanstackStart(),9  ],10});
```

将 Workflow SDK 插件添加到 TanStack Start 的 Vite 配置中

接下来，使用 "use workflow" 和 "use step" 在标准 TypeScript 中编写工作流和步骤函数。它们作为持久化、可恢复的操作运行，能够承受重启、休眠数天，并在失败时重试，编译、队列配置和持久化均由插件处理。

阅读 TanStack Start 指南以了解更多信息，并创建您的第一个持久化工作流。
