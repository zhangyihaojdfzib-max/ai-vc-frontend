### Greylock 投资人 Jerry Chen 评估 AWS 与云产业的第二幕

![](/images/posts/668158379ad9.jpg)

独家视频作者：Mark Albertson


本周在拉斯维加斯举行的 AWS re:Invent 大会，众多引人关注的支线情节之一便是亚马逊网络服务公司的“第二幕”问题。

自 AWS 于 15 年前创立以来，公有云已达到相当的规模和成熟度，那么，推动云产业下一波增长的燃料将是什么？知名风险投资公司 Greylock Partners 的风险投资人 Jerry Chen（如图）对此有一些见解。

“亚马逊拥有的、其他公司不具备的一项资产，是其庞大的开发者用户基础，”陈说。“开发者不一定会去构建或消费 SaaS 应用，但他们将会消费像 API 和基础组件这类东西。云产业的第二幕会是什么样子？可能不再是虚拟机或容器。可能是像 Stripe 在支付领域或 Twilio 在消息领域那样的 API。”

陈在 AWS re:Invent 大会期间，与 SiliconANGLE Media 直播工作室 theCUBE 的联合主持人 John Furrier 和 Dave Vellante 进行了交流。他们讨论了科技投资策略的重点，以及 AWS 的领导层变动可能如何影响其未来方向。（*披露信息见下文。）

### 颠覆现有巨头

尽管 AWS 取得了巨大成功，并目前在云市场处于领导地位，但科技行业是由一波接一波的颠覆浪潮驱动的，这些浪潮最终可能影响该公司。陈在多年的初创企业投资中见证了这种模式的演绎。

“投资云或进行一般性投资的关键在于，你要么投资于预算替代——即替代一个已知市场，比如更便宜更好的数据库，要么投资于一个全新的市场，”陈说。“这很棘手，因为初创公司颠覆现有巨头，成为新的巨头，然后它们也可能成为自身成功的受害者。最优秀的公司总是在第一幕仍在进行时，就思考第二幕或第三幕。”

今年的 re:Invent 大会让我们得以一窥 AWS 对其自身第二幕的愿景，这一转型的标志性事件是新任首席执行官 Adam Selipsky 的任命，他接替了现在掌管母公司的 Andy Jassy。

“Adam 是新人，但他也是‘老人’。Adam 是亚马逊的早期员工，”陈指出。“他拥有在 Jassy 领导下的历史和文化积淀，同时他又置身于（AWS）圈子之外过。他实际上懂得运营一家公司意味着什么。这是在 Adam 领导下的第二幕。”
