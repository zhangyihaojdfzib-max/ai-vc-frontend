AI Agent（智能体）现已能够通过 Vercel CLI 自主发现、安装 Vercel 市场集成并获取其设置说明。这使得智能体能够在单一工作流中端到端地配置数据库、身份验证、日志记录等服务。

这些功能由 Vercel CLI 中新增的 `discover` 和 `guide` 命令提供支持。

通过在 `discover` 命令中使用 `--format=json` 标志，CLI 会提供非交互式的 JSON 输出，这对开发者同样有益，使其更容易实现基础设施自动化、编写自定义脚本以及管理 CI/CD 流水线。

在构建应用程序时，智能体首先使用 `discover` 命令探索可用的集成。

```
1vercel integration discover --format=json
```

探索选项后，智能体可以 `add` 一个集成，然后使用 `guide` 命令获取特定集成的入门指南和代码片段。

```
1vercel integration add neon --format=json2vercel integration guide neon
```

Vercel CLI 以适合智能体处理的 Markdown 格式返回此设置文档。这使得智能体能够轻松解析指令、编写必要的集成代码并自主配置项目。

对于需要元数据字段的集成，智能体可以使用 `help` 命令来确定所需的输入，并将其作为选项传递给 `add` 命令。

```
1vercel integration add upstash/upstash-redis --help2vercel integration add upstash/upstash-redis -m primaryRegion=iad1 --format=json
```

CLI 还便于在此过程中暂停以等待人工决策，例如接受服务条款。智能体可以提示开发者进行确认，从而实现需要人工监督某些集成决策的混合工作流。

这些命令会持续通过智能体评估进行测试，以确保可靠的自主行为。

```
1pnpm i -g vercel@latest
```
