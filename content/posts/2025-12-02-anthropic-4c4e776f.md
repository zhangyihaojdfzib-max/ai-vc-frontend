# Anthropic 收购 Bun，Claude Code 达成 10 亿美元里程碑

Claude 是全球面向开发者、初创企业和企业的最智能、能力最强的 AI 模型。Claude Code 代表了一个智能体编码的新时代，从根本上改变了团队构建软件的方式。去年 11 月，Claude Code 实现了一个重要里程碑：在向公众开放仅六个月后，其年度经常性收入就达到了 10 亿美元。今天，我们宣布 Anthropic 将收购 Bun——一个突破性的 JavaScript 运行时——以进一步加速 Claude Code 的发展。

Bun 正在重新定义现代软件工程和开发的速度与性能。由 Jarred Sumner 于 2021 年创立，Bun 的速度远超其主要竞争对手。作为一个集运行时、包管理器、打包器和测试运行器于一体的工具包，它已成为 AI 驱动软件工程的重要基础设施，帮助开发者以前所未有的速度构建和测试应用程序。

Bun 通过优化可靠性、速度和开发体验，改善了 JavaScript 和 TypeScript 开发者的体验。对于使用 Claude Code 的用户来说，此次收购意味着更快的性能、更高的稳定性和新的功能。我们将共同努力，继续使 Bun 成为所有开发者的最佳 JavaScript 运行时，同时在 Claude Code 中构建更优的工作流程。

自 2025 年 5 月全面上市以来，Claude Code 已从一个内部工程实验项目，发展成为包括 Netflix、Spotify、毕马威、欧莱雅和 Salesforce 在内的许多世界级领先企业的关键工具，而 Bun 在其整个发展过程中帮助扩展基础设施起到了关键作用。我们与 Bun 已密切合作数月。我们的合作是 Claude Code 团队快速执行的核心，并直接推动了近期 Claude Code 原生安装程序的发布。我们知道 Bun 团队与我们在 Anthropic 秉持相同的理念，专注于重新思考开发者体验并构建创新、有用的产品。

"Bun 恰恰代表了我们希望引入 Anthropic 的那种技术卓越性，"Anthropic 首席产品官 Mike Krieger 表示。"Jarred 和他的团队从第一性原理出发重新思考了整个 JavaScript 工具链，同时始终专注于实际用例。Claude Code 在短短 6 个月内就实现了 10 亿美元的年度经常性收入，将 Bun 团队纳入 Anthropic 意味着我们可以构建基础设施来增强这一势头，并与 AI 采用的指数级增长保持同步。"

随着开发者越来越多地使用 AI 进行构建，底层基础设施变得比以往任何时候都更加重要——而 Bun 已成为一个必不可少的工具。Bun 每月下载量超过 700 万次，在 GitHub 上获得了超过 82,000 颗星，并被 Midjourney 和 Lovable 等公司采用，以提高速度和生产力。

收购 Bun 的决定符合我们战略性的、审慎的收购方针：我们将继续寻求能够增强我们技术卓越性、巩固我们作为企业 AI 领导者地位的机会，最重要的是，这些机会必须符合我们的原则和使命。

Bun 将帮助我们为下一代软件构建基础设施。我们将共同努力，继续使 Claude 成为编码者以及任何依赖 AI 完成重要工作的人的首选平台。Bun 将保持开源和 MIT 许可，我们将继续投资，使其成为 JavaScript 和 TypeScript 开发者在运行时、打包器、包管理器和测试运行器方面的首选。
