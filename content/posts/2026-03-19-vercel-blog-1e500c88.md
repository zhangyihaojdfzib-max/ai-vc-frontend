您现在可以通过 Stripe Projects 注册并部署至 Vercel。该基于 CLI 的工作流已开放开发者预览，允许团队和 AI Agent（智能体）直接从终端创建基础设施环境。

作为 Stripe Projects 的发布和设计合作伙伴，Vercel 提供了统一的资源调配流程。通过 CLI，您可以：

- 关联现有 Vercel 账户，或系统将为您创建新账户
- 直接从终端配置新的 Vercel 项目
- 在 Vercel 上部署和托管应用程序，作为统一基础设施设置的一部分

关联现有 Vercel 账户，或系统将为您创建新账户

直接从终端配置新的 Vercel 项目

在 Vercel 上部署和托管应用程序，作为统一基础设施设置的一部分

此集成减少了本地开发与云托管之间的摩擦，使得基础设施能够以编程方式配置。

要开始以编程方式配置您的基础设施，请安装 Stripe CLI 插件并初始化您的项目：

```
1stripe plugin install projects2
3stripe projects init my-app
```

选择 Vercel 进行部署并开始构建，或详细了解 Stripe Projects。
