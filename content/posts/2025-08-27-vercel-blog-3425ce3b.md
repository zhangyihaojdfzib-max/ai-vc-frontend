我们发布了@vercel/slack-bolt，这是我们用于将Slack的Bolt for JavaScript部署到Vercel AI Cloud的官方适配器。

Bolt提供了一个类型安全的库，用于响应Slack的webhook事件。然而，Slack的API要求在三秒内得到响应，否则用户将面临超时。这使得在传统的无服务器平台上构建Slack Agent（智能体）变得困难。

我们的适配器利用Fluid compute的流式处理和waitUntil功能，在Slack的截止时间内确认响应，同时您的Agent（智能体）可以在后台继续工作。

```
1import { App } from "@slack/bolt";2import { generateText } from "ai";3import { VercelReceiver, createHandler } from "@vercel/slack-bolt";4
5const receiver = new VercelReceiver();6
7const app = new App({8  token: process.env.SLACK_BOT_TOKEN,9  signingSecret: process.env.SLACK_SIGNING_SECRET,10  receiver,11  deferInitialization: true,12});13
1415app.event("message", async ({ say, message }) => {16  const { text } = await generateText({17    model: "openai/gpt-5",18    prompt: `Respond to the user's message: ${message.text}`,19  });20  21  await say(text);22});23
24export const POST = createHandler(app, receiver);
```
