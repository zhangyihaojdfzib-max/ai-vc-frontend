Workflow Development Kit（工作流开发套件）是一个用于构建持久化、长时间运行流程的框架，现已进入公开测试阶段。

Workflow Development Kit 为异步 JavaScript 带来了持久性、可靠性和可观测性，让您可以轻松构建能够暂停、恢复和保持状态的应用程序与 AI Agent（智能体）。

通过"use workflow"指令，将函数转变为持久化工作流变得非常简单：

```
1export async function hailRide(requestId: string) {2  "use workflow";3
4  const request = await validateRiderRequest(requestId);5  const trip = await assignDriver(request);6  const confirmation = await notifyRider(trip);7  const receipt = await createReceipt(trip);8
9  return receipt;10}
```

主要亮点包括：

- **可靠性**：只需添加"use workflow"即可使异步函数持久化。无需手动配置队列、调度器或 YAML。
- **标记步骤**：使用"use step"进行标记。重试自动进行。
- **持久性**：调用`sleep`可在不占用计算资源的情况下暂停，然后原地恢复。
- **内置可观测性**：每次运行都提供追踪、日志和指标。调试时可暂停、回放和时间旅行。
