# 使用 Vercel CLI 查询可观测性指标

现在你可以在 Vercel CLI 中访问 Observability Plus 指标。

通过新的 `vercel metrics` 命令，查询任意 Vercel 团队或项目的可观测性数据。编码 Agent（智能体）也可以利用这一新命令，更好地分析 Vercel 上应用的性能、可靠性或安全性，并进行问题调试。

![](/images/posts/07ed7570e916.jpg)

![](/images/posts/c06fa6c47d0a.jpg)

该功能以公开测试版形式面向所有拥有 Observability Plus 的团队开放。
