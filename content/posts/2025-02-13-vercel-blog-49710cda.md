Vercel 现已支持将 React Router v7 应用作为框架使用：

```
1import { vercelPreset } from "@vercel/react-router/vite";2import type { Config } from "@react-router/dev/config";3
4export default {5  ssr: true,6  presets: [vercelPreset()],7} satisfies Config;
```

使用 Vercel 预设配置 React Router 应用。

这包括对使用 Vercel Fluid 计算进行服务端渲染的 React Router 应用的支持。此外，Vercel 预设能够智能地将应用包拆分到多个 Vercel Functions 中，并支持自定义服务端入口点。

部署 React Router 到 Vercel，或了解有关 Vercel 上 React Router 的更多信息。
