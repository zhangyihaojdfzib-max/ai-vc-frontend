使用 Fluid 计算的 Vercel Functions 默认限制已提升，执行时间更长，内存和 CPU 也更多。

所有方案下所有项目的默认执行时间现为 300 秒（5 分钟）：

默认值

最大值

Hobby

300 秒（此前为 60 秒）

300 秒（此前为 90 秒）

Enterprise

内存和 CPU 实例大小也已更新：

- Standard（默认）现为 1 vCPU / 2 GB（此前为 1 vCPU / 1.7 GB）
- Performance 现为 2 vCPU / 4 GB（此前为 1.7 vCPU / 3 GB）

Standard（默认）现为 1 vCPU / 2 GB（此前为 1 vCPU / 1.7 GB）

Performance 现为 2 vCPU / 4 GB（此前为 1.7 vCPU / 3 GB）

Basic CPU 实例已移除。Hobby 项目现使用 Standard CPU。

这些提升后的实例由 Active CPU 定价支持，按实际计算时间收费。仅使用内存的时段以显著更低的费率计费，使更长的执行更具成本效益。

您可以查看日志以确定函数是否达到执行限制，并根据需要调整最大持续时间或升级方案。
