# Claude 助力创意工作

创意专业人士期待技术能拓展其工作的可能性。Claude 无法替代品味或想象力，但它能开启全新的工作方式——更快速、更大胆的构思，更广泛的技能组合，以及让创意人员能够承接更大规模的项目。AI 还能通过处理重复性任务和消除手动劳作，帮助承担创意流程中耗费时间的部分。实现这两个目标的关键在于将 Claude 集成到创意行业已经熟悉和信赖的工具中。

今天，我们与包括 Blender、Autodesk、Adobe、Ableton 和 Splice 在内的合作伙伴联盟，发布了一系列连接器——这些工具让 Claude 能够与创意专业人士依赖的软件协同工作，从而拓展创意人员的能力范围。

## 将 Claude 连接到创意工具

连接器允许 Claude 直接访问其他平台和工具。我们正在添加几个新的连接器，旨在让 Claude 更易于用于创意工作：

- **Ableton** 将 Claude 的回答建立在 Live 和 Push 的官方产品文档基础上。
- **Adobe for creativity** 使用户能够利用 Photoshop、Premiere、Express 等 Creative Cloud 应用中的 50 多种工具，让图像、视频和设计栩栩如生。
- **Affinity by Canva** 可自动执行专业创意工作流程中的重复性生产任务——如批量图像调整、图层重命名和文件导出——并直接在应用中生成自定义功能。
- **Autodesk Fusion** 允许拥有 Fusion 订阅的设计师和工程师通过与 Claude 对话来创建和修改 3D 模型。
- **Blender** 为其 Python API 提供了自然语言界面，使用户能够探索和理解复杂设置，并更轻松地访问 Blender 的文档。
- **Resolume Arena** 和 **Resolume Wire** 让 VJ 和现场视觉艺术家能够通过自然语言实时控制 Arena、Avenue 和 Wire，用于现场表演和视听制作。
- **SketchUp** 将与 Claude 的对话转化为 3D 建模的起点——描述一个房间、一件家具或一个场地概念，然后在 SketchUp 中打开进行细化。
- **Splice** 让音乐制作人能够在 Claude 中搜索其免版税样本目录。

## 使用 Claude 进行创意工作

以下是 Claude 可用于创意任务的几种方式：

- **学习和掌握创意工具**：Claude 可以充当复杂软件的按需导师。你可以请它解释修改器堆栈、引导你完成合成技术，或演示一个不熟悉的功能，它会向你展示如何使用。
- **用代码扩展工具**：Claude Code 可以为你正在使用的软件编写脚本、插件和生成系统。你可以请它构建自定义着色器、编写程序化动画脚本或生成参数化模型，它会生成带有文档的代码供你重用和修改。
- **桥接工作流程中的工具**：Claude 可以跨多个应用的项目转换格式、重组数据并保持资产同步，这样你就可以在设计、3D 和音频工具之间移动工作，无需手动交接。
- **实现快速探索和交接**：**Claude Design** 是 Anthropic Labs 的新产品，可用于探索软件体验的想法。Claude 可以可视化选项并根据你的反馈进行迭代。它旨在将结果导出到其他工具，首先从 Canva 开始。
- **处理重复性生产工作**：Claude 可以处理多步骤任务，如批量处理资产、搭建项目框架或跨场景应用程序化更改，从而减少琐碎工作。

## Claude 与 Blender

Blender 是一款免费、开源的 3D 创作套件，广泛应用于从独立游戏开发和动态图形到建筑可视化和电影制作等各个行业。

Blender 开发者创建了一个 MCP 连接器，现已正式可用于 Claude。例如，3D 艺术家可以使用 Blender 连接器分析和调试整个 Blender 场景，或构建自定义脚本以批量对场景中的对象应用更改。通过使用 Blender 的 Python API，该连接器让 Claude 能够直接在 Blender 界面中添加新工具。

Anthropic 已作为赞助人加入 Blender 开发基金，以支持 Blender 项目继续开发其 Python API，正是这个 API 使此类集成成为可能。由于该连接器基于 MCP 构建，除了 Claude 之外，其他 LLM 也可以使用它，这体现了 Blender 对开源和互操作性的承诺。

## 与学生和教育工作者合作

我们还在与艺术和设计项目合作，以支持涉及创意计算的教学大纲。首批三个此类项目是罗德岛设计学院的"艺术与计算"、瑞林艺术与设计学院的"创意人员 AI 基础"以及伦敦大学金史密斯学院的"MA/MFA 计算艺术"项目。学生和教师将获得 Claude 和新连接器的使用权限，他们的反馈将帮助我们了解创意从业者对这些工具的需求。我们期待向他们学习，并在未来将该计划扩展到更多院校。
