您现在可以在 Agent Runs 中检查 eve 项目的子 Agent 活动。

新的 Subagents 标签页会显示每个子 Agent，并按启动它们的轮次进行组织。每一行都显示提示词、持续时间和任何失败信息，所有内容都位于一个共享时间线上。

![轮次详情面板显示一个委派子 Agent 列表，时间线上标注了持续时间；其中一个子 Agent 失败。](/images/posts/f34de4945ed7.jpg)

![轮次详情面板显示一个委派子 Agent 列表，时间线上标注了持续时间；其中一个子 Agent 失败。](/images/posts/530bf0ba3dbe.jpg)

点击任意子 Agent 即可打开其运行记录。该标签页显示与父运行相同的详细信息：轮次、工具调用、元数据、成本和 Token 使用量。

打开 Agent Runs 尝试此功能，或阅读文档了解更多信息。
