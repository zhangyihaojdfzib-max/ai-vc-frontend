我们发布了第一套技能，旨在为AI编程Agent（智能体）提供开源LangChain生态系统的专业知识。这包括使用LangChain、LangGraph和Deep Agents构建Agent（智能体）。在我们的评估集上，这使Claude Code在这些任务上的表现从29%提升至95%。

## 什么是技能？

技能是经过精心编排的指令、脚本和资源，用于提升编程Agent（智能体）在特定领域的表现。重要的是，技能通过渐进式披露动态加载——Agent（智能体）仅在任务相关时才会检索相应技能。这增强了Agent（智能体）的能力，因为历史上，为Agent（智能体）提供过多工具会导致其性能下降。

技能具有可移植性和可共享性——它们由可按需检索的Markdown文件和脚本组成。我们分享的这套LangChain技能可以移植到任何支持技能功能的编程Agent（智能体）中。

## LangChain技能

在`langchain-skills`代码库中，我们维护了一套包含11项技能的集合，大致分为3类：

- **LangChain**：关于如何使用LangChain的`create_agent()`、中间件和工具模式的指导。这是使用经典工具调用Agent（智能体）循环的基础。
- **LangGraph**：关于如何使用LangGraph的原语，并利用其对"人在回路"、持久执行等功能的原生支持的指导。
- **DeepAgents**：关于如何使用我们开源的Deep Agents包并利用其预构建的中间件和文件系统的指导。

## 安装

要安装这些技能，您可以使用`npx skills`：

**本地（当前项目）：**

```bash
npx skills add langchain-ai/langchain-skills --skill '*' --yes
```

**全局（所有项目）：**

```bash
npx skills add langchain-ai/langchain-skills --skill '*' --yes --global
```

要将技能链接到特定的Agent（智能体）（例如Claude Code）：

```bash
npx skills add langchain-ai/langchain-skills --agent claude-code --skill '*' --yes --global
```
