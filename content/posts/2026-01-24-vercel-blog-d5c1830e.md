Clawdbot 是一款由 Claude 驱动的个人 AI 助手，具备持久记忆功能。它能够浏览网页、运行 shell 命令，并跨任何操作系统管理文件。

您可以将 Clawdbot 与 Vercel AI Gateway 结合使用，通过单一端点访问来自多个供应商的数百个模型。AI Gateway 提供了跨模型的统一 API 访问，无需管理单独的 API 密钥。

在 AI Gateway 仪表板中创建一个 API 密钥，然后安装 Clawdbot：

```
1curl -fsSL https://install.clawdbot.com2# 或3npm install -g clawdbot
```

运行初始化向导：

```
1clawdbot onboard --install-daemon
```

选择 Vercel AI Gateway 作为您的供应商，并输入您的 AI Gateway API 密钥。

之后，您可以从数百个可用模型中进行选择。您的 AI 助手现已运行，随时准备帮助您处理系统中的各项任务。

有关 Clawdbot 及更多集成的详细信息，请参阅AI Gateway 文档。
