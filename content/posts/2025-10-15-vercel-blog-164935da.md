您现在可以通过 Vercel 的 AI Gateway 访问 Anthropic 的最新模型 Claude Haiku 4.5，无需其他供应商账户。Haiku 4.5 在编码、计算机使用和 Agent（智能体）任务上的性能与 Sonnet 4 相当，但成本显著更低，速度更快。

AI Gateway 允许您通过统一且一致的 API 调用模型，只需更新一个字符串，即可跟踪使用情况和成本，并配置性能优化、重试和故障转移，以实现高于供应商平均水平的正常运行时间。

要在 AI SDK v5 中使用它，请将模型设置为 `anthropic/claude-haiku-4.5`：

```
1import { streamText } from 'ai'2
3const result = streamText({4  model: "anthropic/claude-haiku-4.5",5  prompt: "Write me a haiku about Claude Haiku 4.5." 6})
```

包含内置的可观测性、自带密钥支持以及具有自动重试功能的智能供应商路由。

为了向 Claude Haiku 4.5 提供高性能和高可靠性，AI Gateway 在底层利用了多个模型供应商，包括 Anthropic、Bedrock 和 Vertex AI。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
