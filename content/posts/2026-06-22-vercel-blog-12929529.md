Sakana AI 的 Sakana Fugu Ultra 现已在 AI Gateway 上线。

Fugu Ultra 基于一组可公开访问的前沿模型构建，而非作为单一模型运行。它协调多个模型，根据问题将任务路由至 1-3 个 Agent（智能体），并将其结果整合为单一答案。

根据推理和科学基准测试，Fugu Ultra 的能力与 Claude Mythos Preview 和 Fable 5 相当。

要使用 Fugu Ultra，请在 AI SDK 中将模型设置为 `sakana/fugu-ultra`：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'sakana/fugu-ultra',5  prompt: 'Review this pull request and flag correctness bugs.',6});
```

AI Gateway 提供统一的 API 用于调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商正常运行时间的可用性。它包含内置的自定义报告、零数据保留支持、API 密钥预算等功能。

AI Gateway 反映提供商定价，不收取额外费用，且不对推理收取平台费，包括自带密钥（BYOK）请求。
