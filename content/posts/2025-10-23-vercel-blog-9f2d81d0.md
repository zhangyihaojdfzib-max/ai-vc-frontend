Vercel Python SDK 现已推出测试版，为众多 Vercel 功能（如 Vercel Sandbox、Blob 和 Runtime Cache API）提供一流的 Python 支持。

要开始使用，请通过 `pip install vercel` 安装 vercel 包。

Vercel Python SDK 允许您通过 Python 代码直接与 Vercel 基础组件交互，例如：

使用 Vercel Sandbox 在隔离的临时环境中运行不受信任的代码：

```

```


通过 `pip install vercel` 开始使用。
