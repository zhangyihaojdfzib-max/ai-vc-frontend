GPT-5.3 Chat（GPT 5.3 Instant）现已在 AI Gateway 上线。

本次更新侧重于语气、相关性和对话流畅度，旨在提供更准确的答案、上下文更贴切的网络搜索结果，并减少不必要的拒绝和附加说明。它还降低了幻觉率，并生成更流畅、更直接的回应。

要使用此模型，请在 AI SDK 中将模型设置为 `openai/gpt-5.3-chat`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'openai/gpt-5.3-chat',5  prompt: 'Best restaurants in SF for a birthday dinner.',6});
```

AI Gateway 提供了一个统一的 API，用于调用模型、跟踪使用情况和成本，以及配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它包含内置的可观测性、自带密钥支持以及具有自动重试功能的智能供应商路由。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
