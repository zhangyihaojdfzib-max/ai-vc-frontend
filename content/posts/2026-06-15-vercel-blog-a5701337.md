Chat SDK 现在可以在 Telegram 适配器上将显式 markdown 和消息渲染为原生富消息。你的机器人将获得真正的标题、列表、表格、任务列表、公式和独立的媒体块，而非扁平化的文本。

```
1bot.onNewMention(async (thread) => {2  await thread.post({3    markdown: [4      "## 部署状态",5      "",6      "| 服务 | 状态 |",7      "| --- | --- |",8      "| API | ✅ 就绪 |",9      "| Web | 🚀 构建中 |",10    ].join("\n"),11  });12});
```

发布一个 markdown 表格，它将渲染为原生 Telegram 富消息。

你将获得：

- 原生格式：标题、列表、表格、任务列表、公式和媒体块。
- 流式传输：私聊显示实时草稿预览并保留最终响应。
- 自动回退：纯文本字符串、原始消息、卡片和标题将完全按之前的方式继续工作。

原生格式：标题、列表、表格、任务列表、公式和媒体块。

流式传输：私聊显示实时草稿预览并保留最终响应。

自动回退：纯文本字符串、原始消息、卡片和标题将完全按之前的方式继续工作。

阅读 Telegram 适配器文档以开始使用。
