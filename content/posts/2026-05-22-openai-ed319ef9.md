# Paradigm

Paradigm 使用 OpenAI 的 API 来改善患者参与临床试验的机会。

临床试验是我们发现新疗法的方式，也是一种可以挽救生命的治疗手段。Paradigm⁠(在新窗口中打开) 正在通过技术打破医疗行业的障碍，让更多人群（如癌症患者）能够参与临床试验，同时减轻医生和护士的文书工作负担，以解决临床工作者职业倦怠问题。

![paradigm](/images/posts/4e41f83cc0da.png)

## 临床试验入组流程存在缺陷

对患者病历的评估是让患者入组可能提供最佳治疗方案的临床试验的主要瓶颈。医疗服务提供者很少有时间去搜索临床试验、了解试验细节，然后匹配并筛选合格的患者。结果是，大多数临床试验的入组患者都来自试验地点附近，这造成了筛选过程中的偏差，并使许多患者无法获得可能挽救生命的突破性治疗。

为了解决这个问题，Paradigm 部署并优化了传统的行业最佳实践、医疗领域专用的 ML 和 NLP 模型，用于提取和解读病历数据。这些模型在由专家临床医生整理的黄金数据集上进行了训练和评估。然而，这种方法既缓慢又繁琐。

由于传统模型的效果有限，临床医生需要手动审查模型输出，以确保达到质量标准。

## 使用 GPT-4 评估临床试验数据集

Paradigm 认为，LLM 具备总结非结构化文本的能力，非常适合他们的应用场景，可以取代构建一次性 ML 模型的方法。他们探索了两条可能的路径：集成一个针对医疗用例训练的自定义 LLM，或者通过 OpenAI 的 API 集成 GPT-4。

Paradigm 原本认为需要专门的医疗模型才能获得良好效果。他们“震惊地”发现，GPT-4 在复杂数据评估任务上的表现甚至超过了训练有素的人类专家团队。

最终，他们选择 OpenAI 的原因如下：

- **准确性**：Paradigm 在其专家整理的黄金数据集上进行了严格的评估。在混合精确率/召回率指标上，GPT-4 比最先进的 ML 模型准确率至少高出 10%。在某些情况下，他们看到了“令人难以置信”的提升。“OpenAI 的准确性优于我们现有部署和优化的行业最佳实践专家训练模型，有时甚至优于我们训练有素的临床医生，”Hirsch 说。“信息越复杂，信息分布的位置越分散，GPT-4 的表现就越好。”
- **易用性**：“从产品角度来看，一切都很简单。API 易于使用，并且很容易集成到我们的技术栈中。”团队还赞赏 OpenAI 的支持包括高质量的 API 文档：“与其他组织合作相比，使用 OpenAI 我们可以更加自主。”
- **多模态输入和长上下文窗口**：这两个功能对病历数据都至关重要。
- **安全性与法规合规性**：“真正让我们决定与 OpenAI 合作的是你们对支持法规合规性的认真态度，包括支持我们遵守 HIPAA 的需求。由于我们服务于医疗服务提供者及其患者，法规合规性对我们来说是不可妥协的要求。”

## GPT-4 提升了产品速度、数据质量和运营成本效益

- **数天内提取新数据元素，而非数月**：GPT-4 彻底改变了 Paradigm 对其核心基础设施的思考方式，完全取代了为单个数据组件逐一构建 ML 模型的过程。这极大地加速了 Paradigm 的发展路线图，使他们能够快速扩展到新的医疗服务提供者合作伙伴和试验类型。
- **专家临床医生模型验证时间减少 90%**：Paradigm 估计，与之前的专用 ML 模型相比，他们只需要 1/10 的数据量来评估 GPT-4 的输出。
- **准确率提升 10%**：凭借比以往更准确的数据——甚至在许多情况下超越人类专家——GPT-4 减少了对模型结果进行专家人工干预的需求。Paradigm 及其医疗服务提供者合作伙伴的医生和护士能够将更多时间用于患者护理，而不是阅读文档。
- **更公平的试验机会**：虽然仍在验证中，但 Paradigm 认为 GPT-4 可以更准确地筛选服务不足的患者参与试验。这些患者的病历中结构化数据较少，非结构化数据（如笔记）较多，而 GPT-4 擅长提取和解读这些数据。

## 每分钟评估数百名患者

展望未来，Paradigm 对如何利用 GPT-4 的自然语言理解能力进一步减轻临床医生负担感到兴奋。临床团队无需编写代码来分析数据，而是可以与 ChatGPT 就患者数据进行对话，以了解其试验资格、缺失信息以及后续步骤。

Paradigm 对如何继续提高患者筛选速度感到兴奋。借助 GPT-4，他们的平台有可能每分钟评估数百名患者。相比之下，一名典型的护士研究协调员每天只能手动审查大约 50 名患者。这些效率提升有望带来一个患者更容易获得临床试验、医生和护士能将更多时间用于患者护理而非文书工作、以及新的救命疗法更快推向市场的世界。
