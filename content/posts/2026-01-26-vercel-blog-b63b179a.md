AI Gateway 现已支持 Claude Code Max 订阅用于 Claude Code CLI。这使得开发者能够在 Anthropic 模型上使用其现有订阅而无需额外费用，同时通过 Vercel 平台获得统一的观测性、使用量追踪和监控功能。

### 设置

在您的 shell 配置文件（`~/.zshrc` 或 `~/.bashrc`）中设置环境变量

```
1export ANTHROPIC_BASE_URL="https://ai-gateway.vercel.sh"2export ANTHROPIC_CUSTOM_HEADERS="x-ai-gateway-api-key: Bearer your-ai-gateway-api-key"
```

将 `your-ai-gateway-api-key` 替换为您实际的 AI Gateway API 密钥。

### 启动 Claude Code

### 使用您的 Claude 订阅登录

如果您尚未登录，Claude Code 将提示您进行身份验证。选择 **选项 1 - 拥有订阅的 Claude 账户** 并使用您的 Anthropic 账户登录。

如果遇到问题，请尝试使用 `claude /logout` 注销后重新登录。

现在，您的 Claude Code 请求将通过 AI Gateway 路由，让您在使用 Max 订阅时能完全掌握使用模式和成本情况。

### 工作原理

当您将 Claude Code 配置为使用 AI Gateway 时，Claude Code 会继续与 Anthropic 进行身份验证。它发送其 `Authorization` 请求头，而 AI Gateway 则充当通往 Anthropic 的直通代理，或在需要回退时充当路由到其他提供商的角色。

由于 `Authorization` 请求头专用于 Claude 订阅凭证，AI Gateway 使用单独的请求头 `x-ai-gateway-api-key` 进行自身的身份验证。这使得两种身份验证机制可以共存。

在[文档](https://vercel.com/docs/ai-gateway/claude-code-max)中了解更多关于如何配置 Claude Code Max 与 AI Gateway 的信息。
