## 原生 SVG 生成

输出可以直接在 Illustrator、Figma 或 Sketch 中打开。你可以编辑路径、重新着色元素、缩放到任意尺寸。无需追踪、清理或转换步骤。

## V4 的新特性

大多数图像生成模型都是针对广泛的普遍偏好进行调整的。V4 采取了不同的方法：它是与设计师合作开发的，并专门围绕设计美学和专业期望进行调整。

在实践中，这体现在几个方面：

**艺术指导的构图。** V4 会对元素的排列方式、颜色之间的相互关系以及视线在画面中的移动路径做出选择。当你将其输出与其他模型的输出并排比较时，V4 的图像往往感觉更像是经过艺术指导的，而不像图库摄影。

**强大的提示词准确性。** 无论你的提示词是几个词还是一个详细的段落（模型最多接受 10,000 个字符），V4 都能紧密遵循，同时保持整体图像的连贯性。

**集成的文本渲染。** V4 将排版视为图像的结构性部分，而不是事后添加的内容。文字与场景互动——连接视觉元素，响应空间环境，并自然地融入设计。这对于海报、包装、编辑版面以及任何文字是构图一部分的作品来说，是一个突出的特点。

**原生矢量输出。** SVG 模型生成真正的可编辑矢量文件，包含干净的路径和结构化的图层。这是 Recraft 独有的功能——没有其他图像生成模型能做到这一点。

## 运行模型

这是一个使用 Replicate 的 JavaScript 客户端的**光栅**示例：

```
import Replicate from "replicate";
const replicate = new Replicate();

const output = await replicate.run("recraft-ai/recraft-v4", {
  input: {
    prompt: "A minimalist poster for a jazz festival. Bold text reads 'BLUE NOTE SESSIONS' at the top. A saxophone player silhouetted against a deep indigo gradient, editorial design.",
    aspect_ratio: "3:4",
  },
});
console.log(output);
// => "https://replicate.delivery/...webp"
```

这是 **SVG 模型**——相同的 API，但输出是 SVG 文件：

```
const svg = await replicate.run("recraft-ai/recraft-v4-svg", {
  input: {
    prompt: "A set of four minimalist coffee shop icons: a steaming cup, a coffee bean, a pour-over dripper, and a croissant. Clean line art, consistent stroke weight, warm brown palette",
    aspect_ratio: "1:1",
  },
});
console.log(svg);
// => "https://replicate.delivery/...svg"
```

在 Python 中：

```
import replicate

output = replicate.run(
    "recraft-ai/recraft-v4-svg",
    input={
        "prompt": "A vintage-style botanical illustration of a monstera leaf, detailed vector art with clean outlines, dark green with lighter highlights, cream background",
        "aspect_ratio": "3:4",
    },
)
print(output)
# => "https://replicate.delivery/...svg"
```

所有四个模型都接受相同的三个输入：

- `prompt` — 你的文本描述（最多 10,000 个字符）
- `aspect_ratio` — 常见比例，如 `1:1`、`4:3`、`16:9`、`9:16` 等
- `size` — 明确的尺寸，如 `1024x1024`、`1536x768` 等（如果设置了宽高比，则忽略此项）

## 你应该使用哪个模型？

- **V4** — 快速光栅输出，适用于网页、社交媒体、概念探索和快速迭代。
- **V4 Pro** — 高分辨率光栅输出，适用于印刷生产、大尺寸显示和细节工作。
- **V4 SVG** — 矢量输出，适用于图标、徽标、品牌资产、插图以及任何需要缩放或编辑的内容。
- **V4 Pro SVG** — 高细节矢量输出，适用于大幅面印刷、详细插图和设计系统组件。
