1 分钟阅读
您现在可以通过 Vercel 的 AI 网关访问 OpenAI 最新的 Codex 模型——GPT-5.1 Codex Max，无需其他供应商账户。
通过一种称为"压缩"的过程，GPT-5.1 Codex Max 经过训练，能够在多个上下文窗口中运行，并处理现实世界的软件工程任务。与之前的 Codex 模型相比，GPT-5.1 Codex Max 速度更快、Token 使用效率更高，针对长时间运行的编码任务进行了优化，并且能够在长时间内保持上下文和推理，无需启动新会话。
要在 AI SDK 中使用 GPT-5.1 Codex Max，请将模型设置为 `openai/gpt-5.1-codex-max`。

```javascript
import { streamText } from 'ai';
const result = streamText({ model: 'openai/gpt-5.1-codex-max', prompt: `Write a CLI tool that scans web server logs, counts 5xx errors per endpoint, and prints the ten endpoints with the most errors using only standard libraries.` providerOptions: { openai: { reasoningSummary: "auto", reasoningEffort: "low" }, },});
```

AI 网关提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它包括内置的可观测性、自带密钥支持以及具有自动重试功能的智能供应商路由。
