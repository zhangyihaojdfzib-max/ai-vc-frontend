# Chat SDK 现已支持 Slack 定时消息

Chat SDK 现支持在 Slack 上发送定时消息，让您能够在未来的某个时间点发送消息。

使用 `thread.schedule()` 并传入您的消息和 `postAt` 日期，例如：

```
1const scheduled = await thread.schedule("提醒：5分钟后站会！", {2    postAt: new Date("2026-03-09T08:55:00Z"),3});4
56await scheduled.cancel();
```

阅读[文档](documentation)开始使用。
