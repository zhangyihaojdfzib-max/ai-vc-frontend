# 开源视频强势回归

-   shridharathi

您的浏览器不支持 video 标签。

Wan 2.2 点燃了整个开源视频社区。

相比 2.1 版本，它实现了巨大飞跃，物理效果逼真、生成速度更快、控制能力更强。并且，由于它是完全开源的，它以惊人的低成本为所有人打开了通往高质量视频工具的大门。

我们已与 Pruna AI 合作，发布了我们自己的优化版 Wan 2.2。您可以通过以下方式进行测试：

*   FAST 图生视频，480p — 每视频 0.05 美元
*   FAST 文生视频，480p — 每视频 0.05 美元
*   FAST 文生视频，720p — 每视频 0.10 美元

您没听错。**5 美分**一个视频。这甚至比 AI 市场上的主流图像生成器还要便宜。

凭借约 30 秒的推理时间，Wan 2.2 可以成为您快速测试视频提示词的主要方式。

别让快速的推理时间误导您——这些 Wan 2.2 优化版本并未牺牲质量。

我们还提供未经优化的 Wan 2.2 i2v 服务，支持 480p 和 720p：

*   图生视频，480p — 每视频 0.40 美元
*   图生视频，720p — 每视频 1.00 美元

使用 Replicate 的 API 运行 WAN 2.2 非常简单。以下是如何使用文生视频模型生成您的第一个视频：

```
import Replicate from "replicate";

const replicate = new Replicate();
const model = "wan-video/wan-2.2-t2v-fast";
const input = {
  prompt: "A serene mountain lake at sunset with gentle ripples on the water",
};

const output = await replicate.run(model, { input });
console.log(output);
```

对于图生视频生成，只需使用 i2v fast 端点并添加图像输入：

```

const model = "wan-video/wan-2.2-i2v-fast";
const input = {
  image: "https://example.com/your-image.jpg",
  prompt: "The leaves rustle gently in the wind",
};

console.log(output);
```
