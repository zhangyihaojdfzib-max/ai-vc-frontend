Vercel 现已支持 **Fastify** 应用程序。Fastify 是一个高度专注于提供最佳开发者体验的 Web 框架，具有极低的开销、强大的插件架构，并且无需配置。

```
1import Fastify from 'fastify'2
3const fastify = Fastify({ logger: true })4
5fastify.get('/', async (request, reply) => {  return { hello: 'world' }})6
7fastify.listen({ port: 3000 })
```

一个在 Vercel 上的 "Hello World" Fastify 应用

Vercel 上的后端默认使用 **Fluid compute** 和 **Active CPU 定价**。这意味着您的 Fastify 应用将根据流量自动扩缩容，并且您只需为实际使用的资源付费。

[在 Vercel 上部署 Fastify](https://vercel.com/new) 或访问 [Vercel 上的 Fastify 文档](https://vercel.com/docs/frameworks/fastify)
