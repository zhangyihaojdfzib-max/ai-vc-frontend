现在您可以在 Vercel CLI 中使用原生 `curl` 语法。`vercel curl` 命令支持完整 URL、裸主机名以及 `--url` 标志，并使用您的 Vercel 认证信息绕过部署保护。

```
1vercel curl https://example.vercel.app/api/hello2vercel curl example.vercel.app/api/users -X POST -H "Content-Type: application/json" -d '{"name":"Ada"}'3vercel curl --url https://example.vercel.app/api/status --compressed
```

基于 URL 对任意部署发起调用

如果您已关联项目，也可以仅传入路径：

```
1vercel curl /api/hello2vercel curl /api/users -X POST -d '{"ok":true}'
```

基于路径对关联项目发起调用

请更新至最新 Vercel CLI 版本并运行 `vercel curl` 开始使用。更多信息请参阅 Vercel CLI 文档。
