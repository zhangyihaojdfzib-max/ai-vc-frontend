# 全球新闻合作伙伴：Le Monde 与 Prisa Media

我们已与国际新闻机构 Le Monde 和 Prisa Media 达成合作，将法语和西班牙语新闻内容引入 ChatGPT。

![新闻 > 合作 > 封面](/images/posts/0c1bc984f7ba.jpg)

我们持续对 ChatGPT 进行改进，并支持新闻行业在向用户提供实时、权威信息方面发挥关键作用。我们很高兴宣布与 Le Monde 和 Prisa Media 及其旗下出版物（如 El País、Cinco Días、As 和 El Huffpost）建立合作伙伴关系。这些合作将使 ChatGPT 用户能够通过 ChatGPT 获取 Le Monde 和 Prisa Media 关于近期事件的高质量内容，同时这些内容也将用于我们模型的训练。

OpenAI 首席运营官 Brad Lightcap 表示："我们致力于通过应用新的人工智能技术并增强内容创作者的机会来支持新闻业。通过与 Le Monde 和 Prisa Media 的合作，我们的目标是让全球 ChatGPT 用户能够以互动且富有洞察力的新方式与新闻建立联系。"

在接下来的几个月里，ChatGPT 用户将能够通过带有来源标注的精选摘要与这些出版商的新闻内容进行互动，并附有增强链接指向原始文章，使用户能够从其新闻网站获取更多信息或相关文章。

Le Monde 首席执行官 Louis Dreyfus 呼应了这一观点，他表示："在我们庆祝 Le Monde 创刊80周年之际，与 OpenAI 的合作使我们能够扩大影响力，并履行大规模提供准确、经过验证、平衡的新闻故事的承诺。与 OpenAI 合作确保我们的权威内容能够被更广泛、更多样化的受众获取和欣赏。"

媒体格局的每一次转变都为 Le Monde 带来了新的机遇。从向数字平台转型到拥抱自由媒体时代，Le Monde 始终抓住这些时刻，强调其对独立性、专业性和新闻诚信的承诺。

自2010年以来，Le Monde 已成为数字媒体的先驱，在坚持核心原则的同时调整其组织结构和运营方式。到2024年，Le Monde 已确立自己作为法国领先新闻媒体的地位，拥有超过60万订阅用户，日均独立访客220万，每月页面浏览量超过6.32亿次。

"我们与 OpenAI 的合作是一项战略举措，旨在确保向人工智能用户传播可靠信息，同时维护我们的新闻诚信和收入来源。"

Prisa Media 董事长兼首席执行官 Carlos Nuñez 补充道："与 OpenAI 联手为我们与受众互动开辟了新途径。利用 ChatGPT 的能力，我们能够以新颖的方式呈现我们深入、高质量的新闻报道，触达那些寻求可信和独立内容的用户。这是迈向新闻未来的明确一步，技术与人类专业知识在此融合，以丰富读者的体验。"

"这是 Prisa Media 数字旅程的新篇章，我们持续巩固自身作为最大西班牙语媒体集团的地位，在核心市场（西班牙、拉丁美洲和美国）运营领先的媒体品牌。我们已实现日均独立访客超过700万，月页面浏览量超过16.5亿次，并明确专注于开发文本以外的数字格式内容，包括音频（每月提供9000万总收听时长和5100万次音频下载）和视频（每月超过1.41亿次视频观看）。"

我们与 Le Monde、Prisa Media 以及 Axel Springer 的合作，有助于赋能新闻机构以新方式触达受众。这些合作建立在我们与美国新闻项目（American Journalism Project）支持创新地方新闻倡议，以及美联社（The Associated Press）贡献于我们模型训练的基础之上。我们的合作伙伴关系彰显了我们的愿景：开发先进的 AI 工具，赋能新闻等行业，并解决其他方式难以企及的问题。
