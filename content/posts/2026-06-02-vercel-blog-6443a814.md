Chat SDK 的 Web 适配器现在已为 Vue 和 Svelte 提供了一流支持，加入了现有的 React 集成。由于该适配器采用 AI SDK UI 消息流协议，因此相同的服务端路由依然有效。每个框架都提供了自己的 useChat，基于对应的 React、Vue 或 Svelte 的 AI SDK 包构建。

```
12import { useChat } from "@chat-adapter/web/vue";3
45import { useChat } from "@chat-adapter/web/svelte";6
7const chat = useChat({ api: "/api/chat" });
```

Vue 和 Svelte 的框架特定 useChat 导入

Vue 和 Svelte 的辅助函数返回一个响应式的 Chat 实例，因此请直接在模板中读取 chat.messages 和 chat.status，而不是进行解构。api 和 threadId 选项与 React 封装保持一致，你的服务端设置也无需更改。

阅读文档以开始使用，或浏览适配器目录。
