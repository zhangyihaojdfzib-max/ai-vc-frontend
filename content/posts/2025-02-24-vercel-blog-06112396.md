了解 Avanade 如何通过采用 Sitecore XM Cloud 和 Vercel 的无头架构实现 90 以上的 Lighthouse 评分。

Avanade 是全球领先的微软专家，为企业提供云、网络安全、ERP 以及人工智能驱动的解决方案。

Avanade 团队启动了一项全面的转型，最终采用了 Next.js、Vercel 和 Sitecore XM Cloud，构建了一个现代、可组合的系统，能够为全球客户提供高度响应的体验。此次迁移帮助替换了由紧密耦合组件和手动部署构成的单体系统，这些系统曾严重影响性能和功能发布。

## 链接到标题看到数字化转型后的价值

Vercel 与 Avanade 以微软为先的运营体系（如 Azure DevOps）集成，实现了更灵活、更原生的云架构。它消除了孤立的内容工作流、延迟的更新以及阻碍团队快速集成新功能或响应市场变化的单体依赖。

新的技术栈结合了边缘计算、自动扩缩容和用户友好的 CI/CD 工作流。Vercel 的边缘网络智能地缓存和分发内容，提升了 Avanade 全球客户的页面加载速度。与 Next.js 的紧密集成使公司能够使用服务端渲染（SSR）、静态站点生成（SSG）和增量静态再生（ISR），在速度和实时数据能力之间实现理想平衡。

Vercel Functions 处理后端逻辑，减少了对专用服务器基础设施的需求。这种方法使架构能够根据流量需求自动扩缩容，并实现了碳排放量降低 42%。

此外，内容在 Sitecore XM Cloud 和 Content Hub 中进行管理，集中了资产管理和编辑工作流。无头架构通过现代 API 桥接了这些平台与前端之间的通信。

- 转化率提升 111%
