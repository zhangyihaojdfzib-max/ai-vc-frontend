Databricks 很高兴宣布收购 Quotient AI，这是一家在 AI Agent（智能体）评估与强化学习领域的创新公司。Quotient 帮助企业监控生产环境中的 Agent 行为，检测关键问题，并利用这些信号持续提升 Agent 性能。随着各组织将 AI Agent 部署到关键业务工作流中，他们需要可靠的方法来确保这些系统在现实世界中按预期运行。通过将 Quotient 纳入 Databricks，我们正在通过持续评估与学习来增强 Genie、Genie Code 和 Agent Bricks，为开发者和组织赋能，使其 AI Agent 随着时间的推移变得更加准确、可靠和专业化。

**解决 AI Agent 的生产环境挑战**
当企业将 AI Agent 从试点推向生产环境时，一个新的挑战随之出现：如何可靠地大规模衡量、调试和改进其性能。由于 Agent 是由模型、记忆、工具、技能和其他组件构成的复杂复合系统，团队不仅需要评估质量，还需要快速洞察故障的根本原因。如果没有一个能够解读系统故障的稳健评估系统，修复问题以提升性能的过程将变得缓慢，产品负责人也难以获得将其推向生产环境的信心。

**Quotient 简介**
Quotient 的创立旨在弥合 Agent 评估与持续学习之间的差距。该团队由曾领导 GitHub Copilot 质量改进的工程师创建，在衡量和改进大规模 AI 系统方面拥有深厚的专业知识。Quotient 平台分析来自生产系统的完整 Agent 执行轨迹，以检测诸如幻觉、推理失败和工具使用错误等问题。这些信号会被自动聚类并转化为结构化的评估数据集和奖励信号，可用于监控和微调 Agent。这种方法使组织不仅能观察 Agent 行为，还能系统地改进它，帮助 AI 系统成为能够从实际使用中持续学习的领域专家。

**Quotient + Databricks**
Databricks 已经为评估和改进 AI Agent 提供了强大的工具。通过整合 Quotient，我们将在整个平台中嵌入一个持续评估和改进层，从而增强这些能力，为客户提供更准确、更可靠的 AI 系统。这将强化 Databricks 的三个关键产品领域：

- **Genie** 是一个 AI Agent，让任何员工都能与其数据对话并从中获取洞察。
- **Genie Code**（同样于今日发布）是一个自主 AI Agent，帮助数据团队规划、构建和运行数据工程、机器学习及分析工作流。
- **Agent Bricks** 让组织能够轻松地基于其数据构建和扩展高质量的 Agent。

随着 AI Agent 在企业中承担更多责任，组织需要能够信赖的系统。通过将 Quotient 的能力与 Databricks AI 研究及平台相结合，客户可以部署不仅能在生产环境中运行，还能随时间不断改进的 AI Agent。

我们很高兴欢迎 Quotient 团队加入 Databricks，我们将继续构建用于大规模开发、部署和改进 Agent 系统的最可靠平台。
