# Ada 使用 GPT‑4 提供全新客户服务标准

Ada 正在推动客户服务支出领域 1000 亿美元⁠(在新窗口中打开)的转型，而这场变革的前沿正是他们基于 AI 原生的客户服务自动化平台。Ada⁠(在新窗口中打开) 成立于 2016 年，目前估值达 12 亿美元，累计融资 2 亿美元；客户包括 Verizon、YETI、Canva 和 Square。

Ada 对 AI 并不陌生——他们自创立之初就是一个 AI 原生平台。第一代产品使用了内部开发和训练的自定义自然语言处理（NLP）模型。但他们注意到，平台能够处理的客户问题数量与真正以令人满意的方式得到解决的查询数量之间存在差距。

## 开发新的客户服务指标：解决率

客户服务领域的行业标准指标——拦截率，衡量的是完全由聊天机器人处理且从未转接给人工客服的客户咨询数量。如果提高拦截率，理论上就能降低客户服务的运营成本。但 Ada 团队发现这个指标存在问题。

"包括 Ada 在内的许多解决方案，可以轻松实现 80-100% 的拦截率，但如果你真的打开那些对话记录，阅读客户的体验，你会发现质量相当差，"Gozzo 说。

通过将精力集中在对话实际解决得有多好这一问题上，Ada 团队知道他们可以在客户服务行业树立一个新标准。

最初使用 GPT‑4，结合他们的历史数据，Ada 构建了一个新的评估框架，能够自动评估对话的解决质量。Ada 的系统根据客户收到相关、准确且安全的回复的程度，对每段对话进行评分——无需人工干预。"在我们的测试中，我们的系统与人工阅读对话的评分一致性达到了 80-90%，"Gozzo 说。

## 实战测试模型并选择 OpenAI

通过理解解决率，Ada 为他们的产品找到了北极星指标。除了使用 OpenAI 的 API 进行评估外，他们还决定与 OpenAI 合作构建新一代 AI Agent（智能体）。

Ada 此前曾使用过 OpenAI 的模型，主要用于为其 NLP 模型生成训练数据。在将 OpenAI 的 API 用于新用例进行测试时，Ada 注意到了几个差异化优势：

- **推理质量**：Ada 构建了一个合成测试框架，模拟数十万终端客户与其系统对话，从而对模型的可扩展性和质量进行压力测试。"我们会定期评估每个主要模型，通常是在它们发布后的几周内，到目前为止，在我们的评估集上还没有任何模型能超越 OpenAI，"Gozzo 说。
- **延迟**：为了确保 Ada 语音用例的最佳性能，低延迟模型至关重要。"对于实时推理，我们看到了 GPT‑4o 在延迟和整体质量上的巨大飞跃，"Gozzo 说。
- **微调**：Ada 使用 OpenAI 的微调 API 来提供关于给定回复中幻觉水平的置信度评分，并在工具链的其他部分使用该评分，以最大限度地减少系统中的幻觉数量。"我们对未来利用微调进一步提高系统推理能力的可能性感到兴奋，也期待这项技术能让更小、更便宜的模型模仿前沿模型的性能，"Gozzo 说。

## 借助 OpenAI 将解决率翻倍

如今，Ada 的 AI Agent（智能体）采用多 Agent（智能体）配置，包括一个中央规划 Agent（智能体）和一组子 Agent（智能体），全部由 OpenAI 的 API 驱动。"任何时候，当你与使用 Ada 的企业沟通时，你的问题都会经过 OpenAI 模型的多轮处理——理解它、反思它、调用工具、引入知识，直到生成答案，"Gozzo 解释道。

Ada 产品的先前版本拦截率为 70%，但解决率仅为 30%。在 Ada 已迁移到新系统的客户中，他们获得了类似的拦截结果，但解决率通常高达 60%，而表现最佳的客户解决率超过 80%。

"从一项技术到下一项技术，我们能够以出色体验自动解决的对话数量翻了一番，"Gozzo 说。对于 Ada 的客户来说，这一指标对 ROI 产生了巨大的下游影响，包括全职等效（FTE）成本节省、客户满意度、留存率以及新用户注册。

## 借助 AI 实现 100% 解决率是可行的

Ada 对解决客户查询的能力充满信心，已将目标锁定在 100% 的解决率——这在几年前还像是天方夜谭。"现在，100% 解决率是一个何时实现的问题，而非能否实现的问题，"Gozzo 说。

市场也发生了变化——当 Ada 谈论自动化解决时，遇到的质疑并不多。"企业越来越精明，"Gozzo 说。"他们正在认真思考 AI 战略，并积极推动将这类技术引入其运营中。"客户可以像对待人工客服一样，对 Ada 基于 LLM（大语言模型）的 AI Agent（智能体）进行入职培训、评估和指导。Ada 未来 12 个月的目标是大幅提高其产品的透明度以及为客户提供的控制杠杆。

这种可能性是相互的。"与我们的其他合作伙伴相比，OpenAI 最突出的一点是纯粹的兴奋感和好奇心，"Gozzo 说。"感觉团队中的每个人都对此充满热情。超越商业关系，这关乎我们如何共同构建一些东西，这真的令人耳目一新。"
