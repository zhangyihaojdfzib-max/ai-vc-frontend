Vercel 微前端现包含两款全新的设置与管理工具：面向编程 Agent（智能体）的 AI 技能，以及新的 Vercel CLI 命令。

**全新 Vercel 微前端技能**：安装微前端技能，让您的 AI 编程 Agent（智能体）通过自然语言提示词引导您完成群组创建。它将自动生成 `microfrontends.json` 文件、配置框架集成并管理项目，全程无需离开编辑器。

```
1npx skills add vercel/microfrontends
```

添加完成后，即可要求您的 Agent（智能体）使用此提示词创建首个微前端群组。

立即开始使用微前端技能。

**全新 CLI 命令**：Vercel CLI 现已包含用于管理微前端群组的命令，您可直接在终端中创建、检查和管理群组，无需打开控制面板。

- vercel microfrontends create-group
- vercel microfrontends inspect-group
- vercel microfrontends add-to-group
- vercel microfrontends remove-from-group
- vercel microfrontends delete-group

vercel microfrontends create-group

vercel microfrontends inspect-group

vercel microfrontends add-to-group

vercel microfrontends remove-from-group

vercel microfrontends delete-group

了解更多信息，请查阅 CLI 文档。
