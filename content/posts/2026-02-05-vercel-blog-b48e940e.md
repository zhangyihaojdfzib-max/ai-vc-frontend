Anthropic的最新旗舰模型Claude Opus 4.6现已在AI Gateway上线。该模型专为驱动处理现实世界工作的Agent（智能体）而构建，在整个开发生命周期中表现出色。Opus 4.6也是首个支持扩展的100万Token上下文窗口的Opus模型。

该模型引入了自适应思维，这是一个新参数，允许模型自行决定何时进行推理以及推理的深度。这种方法能在保持编程、分析和创意任务质量的同时，实现更高效的响应，其性能与扩展思维相当或更优。Opus 4.6可以在单个响应中交错进行思维过程和工具调用。

要使用该模型，请将`model`设置为`anthropic/claude-opus-4.6`。以下示例也使用了自适应思维和`effort`参数。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'anthropic/claude-opus-4.6',5  prompt:6    `Trace this race condition through the event loop, identify all7     affected code paths, and implement a fix with proper test coverage.`,8  providerOptions: {9    anthropic: {10      thinking: { type: 'adaptive' },11      effort: 'max',12    },13  },14});
```

AI Gateway提供了一个统一的API，用于调用模型、跟踪使用情况和成本，以及配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它内置了可观测性、支持自带密钥以及具备自动重试功能的智能供应商路由。
