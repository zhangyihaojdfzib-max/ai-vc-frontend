# 如何为Seedream 5.0编写提示词

- shridharathi

运行Seedream 5.0

字节跳动的Seedream系列模型表现强劲。我们花了大量时间尝试各种提示词。以下是我们的一些发现。

## 美学表现

在深入探讨之前，我们先谈谈生成的图像实际观感如何。Seedream 5.0能生成真正精美的图像——那种放大后细节依然经得起推敲的图片。

一张受彩色胶片启发的肖像，一位年轻人侧目而视，浅景深模糊了周围环境，将注意力引向他的眼睛。细腻的颗粒感和色调暗示使用了高感光度胶片，而大光圈镜头则营造出运动模糊效果，增强了自然随性的纪实风格。

![胶片肖像](/images/posts/2463ab9e6093.jpg)

该模型深刻理解摄影语言。你可以提及特定的胶片型号、镜头特性和布光设置，而它生成的图像会让人感觉正是出自那套设备。

一位女子站在黄昏时分的东京小巷，霓虹灯招牌倒映在潮湿的路面上。使用过期增感两档的柯达Portra 800胶片拍摄。拉面店的钨丝灯光将温暖的橙色洒在她的脸上，而霓虹灯则在她头发上投下冷青色的高光。可见的颗粒感、光源周围的晕影、略微提亮的暗部。她正迈出一步，置身于两种色彩世界之间。

![东京霓虹肖像](/images/posts/ea7194802e56.jpg)

它不仅仅擅长肖像。风景、静物、建筑摄影——该模型处理所有这些题材时都展现出一种有品味、有意图的水平，而非千篇一律。

冰岛冰川河流与火山黑沙交汇的航拍照片，形成了类似血管或闪电的抽象分支图案。拍摄于黄金时段的3000英尺高空，水道在漆黑的沙地上呈现出绿松石般的光泽。画面尺度难以确定——它既可能是毛细血管的显微图像，也可能是三角洲的卫星照片。大画幅相机，极致锐利，没有地平线。

![冰岛冰川河流](/images/posts/2b8197babc54.jpg)

静物：一个吃了一半的石榴放在粗糙的石灰岩表面，由高窗射入的一束午后光线照亮。文艺复兴时期的明暗对比布光——籽粒在深色阴影中如红宝石般闪耀。几颗籽粒滚过石头，留下细小的红色痕迹。氛围介于卡拉瓦乔和现代美食摄影之间。

![石榴静物](/images/posts/f3d1be9ece77.jpg)

一位年迈的渔夫在黎明时分于破旧的木制码头上修补渔网。他的手是焦点——布满疤痕、被阳光晒得黝黑，以熟练而精准的动作穿梭在淡蓝色的尼龙网眼中。在他身后，大海呈现出从锡灰到玫瑰金的柔和渐变。使用徕卡M相机配合50mm Summilux镜头全开光圈拍摄，焦外光斑将港口灯光变成完美的圆形。这幅图像具有塞巴斯蒂昂·萨尔加多肖像作品般的沉静尊严。

![修补渔网的渔夫](/images/posts/e76306461ea1.jpg)

一座粗野主义混凝土建筑倒映在雨后完全静止的水坑中，形成罗夏墨迹般的对称。建筑几何化的立面——重复的矩形窗户和裸露的混凝土——加倍形成了一个抽象图案。一个撑着红伞的人影沿着边缘行走，是这单色场景中唯一的色彩。阴天，平坦的漫射光，使用移轴镜头效果处理边缘的建筑摄影。

![粗野主义倒影](/images/posts/6f4cd0897c9c.jpg)

## 基于示例的编辑

这是最难解释但使用起来最有趣的功能。

与其试图用文字描述复杂的编辑，不如直接向模型展示你想要的效果。给它一组“前/后”图像对——图像1和图像2——然后再给第三张图像。模型会分析前两张图像之间的变化，并将相同的变换应用到第三张图像上。

这里有一个例子。我们从一个普通的白色陶瓷杯开始，然后向模型展示这个杯子经过日本金缮修复后的样子（没有任何文字描述）。接着，我们给它一个完全不同的物体——一个陶瓷花瓶——并要求它应用相同的变换：

参考图像1到图像2的变化，对图像3应用相同的操作

图像1

![普通白色杯子](/images/posts/6bc328bcdb58.jpg)

图像2

![金缮杯子](/images/posts/4e91493eabc0.jpg)

图像3

![普通白色花瓶](/images/posts/bba511fe8a31.jpg)

结果

![金缮花瓶](/images/posts/cebca513d022.jpg)

模型从杯子图像对中学会了“添加金缮图案的金色裂纹”，然后将同样的处理应用到花瓶上——而我们根本不需要用文字描述金缮是什么样子。

这适用于各种变换：

- 材质替换：展示一个物体上木材→大理石的变换，应用到另一个物体
- 场景变化：展示一张照片中白天→夜晚的变换，应用到完全不同的地点
- 风格迁移：展示一张照片转换为木版画，将相同的艺术变换应用到新场景

## 精确遵循指令

Seedream 5.0 的指令遵循能力明显比之前的版本更精准。当你说“蓝色夹克”时，你得到的就是蓝色夹克——不是紫色，也不是蓝绿色。当你指定空间关系、数量或具体细节时，模型会严格遵守。

这对于包含大量特定要求的复杂构图最为重要：

## 文本渲染

Seedream 从 3.0 版本开始就在文本渲染方面表现出色，5.0 版本延续了这一传统。在希望图像中渲染的文本周围使用**双引号**以获得最佳效果：

一张虚构爵士音乐节的大幅排版海报。顶部采用粗体压缩无衬线字体：深海军蓝色的“BLUE NOTE SESSIONS”。下方采用优雅的手写体：“Summer 2026 — Central Park, New York.” 海报列出了四位表演者：“Miles Ahead Quintet / Saturday 8PM” “Sarah Chen Trio / Saturday 10PM” “The Monk Revival / Sunday 7PM” “Coltrane Legacy Orchestra / Sunday 9PM”。一个风格化的金色萨克斯风剪影沿右侧边缘垂直延伸。背景是从午夜蓝到暖琥珀色的渐变。底部采用小型大写字母：“Tickets at bluenote.nyc — All ages welcome.”

![爵士音乐节海报](/images/posts/53533674e70e.jpg)

这是一张包含多种字体、混合大小写、标点符号、特定表演者名称和渐变背景的海报——所有内容都准确渲染。该模型也能很好地处理多语言文本。如果你需要准确渲染中文、日文、韩文或其他文字，Seedream 5.0 可以做到。

## 开始使用 API

以下是如何使用 JavaScript 和 Replicate API 运行 Seedream 5.0：

```
import Replicate from "replicate";

const replicate = new Replicate({
  auth: process.env.REPLICATE_API_TOKEN,
});

const output = await replicate.run(
  "bytedance/seedream-5-lite",
  {
    input: {
      prompt: "Your prompt here",
      // size: "2K" or "4K"
      // aspect_ratio: "1:1", "16:9", "9:16", etc.
      // image_input: ["https://..."] for editing or reference
    }
  }
);

console.log(output);
```

## 提示词使用技巧

我们在测试中总结出以下几点经验：

1.  **使用自然语言，而非关键词列表。** 例如，“一位身着华丽礼服的女孩，撑着阳伞走在林荫小道上，风格类似莫奈的油画” 比 “女孩，雨伞，林荫街道，油画质感” 效果更好。
2.  **使用双引号来渲染文本。** 如果希望图像中出现特定文字，请用双引号将其括起来：例如，设计一张标题为 "Seedream 5.0" 的海报。
