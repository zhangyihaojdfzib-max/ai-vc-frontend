# 欢迎 Pieter 和 Shivon！

我们还有两项团队更新。

![欢迎 Pieter 和 Shivon](/images/posts/ceec240531c3.jpg)

插图：Justin Jay Wang × DALL·E

- **Pieter Abbeel。** Pieter⁠(opens in a new window)是加州大学伯克利分校的教授，专攻机器人学习。早在 OpenAI 成立之前，他就一直在为我们提供建议，现在他正从伯克利休假，全职加入我们。Pieter 及其实验室成员近年来在机器人学习⁠(opens in a new window)和深度强化学习 (RL)⁠(opens in a new window)领域取得了一些最引人注目的进展。我们将共同探索将无监督学习与 RL 相结合的方法，我们认为这有望解决当前 RL 算法中的根本性局限。
- **Shivon Zilis。** Shivon⁠(opens in a new window)是 Bloomberg Beta⁠(opens in a new window)的合伙人，专注于机器智能领域。她对这一领域的发展趋势有着极为广阔的视野（并已⁠(opens in a new window)广泛⁠(opens in a new window)撰文⁠(opens in a new window)论述相关格局）。过去几个月，我们一直向她寻求建议，如今她正式成为 OpenAI 的顾问。

能与他们两位共事，我们无比兴奋。欢迎 Pieter 和 Shivon！
