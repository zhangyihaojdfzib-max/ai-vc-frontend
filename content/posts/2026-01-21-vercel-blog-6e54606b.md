今天我们发布了一套全新的组件，旨在帮助您构建下一代集成开发环境（IDE）、编码应用和后台智能体。

## 链接到标题<Agent />

一个可组合的组件，用于展示配置了模型、指令、工具和输出模式的 AI SDK ToolLoop Agent。

```
1npx ai-elements add agent
```

![](/images/posts/f6950a915160.png)

## 链接到标题<CodeBlock />

基于我们从 Streamdown 中积累的经验，我们大幅改进了代码块组件，增加了对标题、图标、文件名、多种语言的支持，并采用了性能更优的渲染器。

```
1npx ai-elements add code-block
```
