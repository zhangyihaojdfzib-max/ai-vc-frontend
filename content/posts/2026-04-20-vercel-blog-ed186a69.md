月之暗面（Moonshot AI）的 Kimi K2.6 现已登陆 Vercel AI Gateway。

该模型专注于长周期编码任务，具备跨 Rust、Go、Python 等编程语言以及前端、运维和性能优化工作的泛化能力。K2.6 能将简单的提示词转化为具有结构化布局的完整前端界面。

对于跨多个应用程序持续运行的自主、主动型 Agent（智能体），K2.6 在 API 解析、长时间运行的稳定性以及扩展研究任务期间的安全意识方面有所提升。

要使用 Kimi K2.6，请在 AI SDK 中将模型设置为 `moonshotai/kimi-k2.6`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'moonshotai/kimi-k2.6',5  prompt:6    `Build a landing page for a developer tools product,7     including a hero section with scroll-triggered8     animations and generated imagery.`,9});
```

AI Gateway 提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它包含内置的**自定义报告**、**可观测性**、**自带密钥（Bring Your Own Key）** 支持，以及具备自动重试功能的智能供应商路由。

了解更多关于 **AI Gateway** 的信息，查看 **AI Gateway 模型排行榜** 或在我们的 **模型游乐场** 中试用。
