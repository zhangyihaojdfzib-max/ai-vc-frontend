v0Platform API 现已开放公开测试版。v0Platform API 是一个文本到应用的 API——它提供了对 v0 应用生成流程的编程式访问：

- 根据提示词生成 Web 应用代码
- 对生成代码进行结构化解析
- 自动错误修复
- 链接到渲染后的预览

此 API 还支持对 v0.dev 进行编程控制，包括创建和管理聊天与项目。我们很快会将更多 v0.dev 的功能引入 Platform API。

v0Platform API 专为集成到开发工作流、自动化脚本和第三方工具而设计。请查看我们的 TypeScript SDK 和文档以开始使用。
