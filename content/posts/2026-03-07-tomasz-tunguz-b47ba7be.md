GitHub Copilot 开创了AI编程辅助的先河。它率先进入市场，拥有2000万用户。随后，Claude Code 和 OpenAI Codex 于2025年中推出。在六个月内，Copilot的日安装量达到峰值后便开始下滑，而其竞争对手的总安装量则飙升至10万以上。¹

利剑并未斩向落后者，而是砍向了早期的领导者。如果微软都能在六个月内失去市场份额，那么没有人是安全的。

我分析了25家上市软件公司共374个季度的NDR数据。多年来，下降趋势看似平缓。净收入留存率从2022年的125%逐步降至2025年的112%。季度复一季。ChatGPT发布时没有出现断崖式下跌。企业采用Copilot时也未加速下滑。

然后，2026年来了。

![软件行业NDR趋势图，显示2022-2026年第25、50、75百分位数，其中2026年出现急剧下降](/images/posts/28c29fd13b4d.jpg)

第25百分位数在单个季度内从106%跌至101%，现已触及盈亏平衡线。² 最弱的公司最先开始失血。Zoom为98%。Asana为96%。底部四分位区间现已出现收缩。

处于底部四分位区间的公司面临不同的威胁，但它们有一个共同点：产品足够简单，易于被替代。Bill.com（NDR为94%）服务于受宏观经济状况影响的中小企业。³ Zoom（98%）面临着Teams和Google Meet等近乎免费的替代品。Asana（96%）提供的任务工作流，其竞争对手和AI智能体也能复制。以96%的NDR计算，它们每年会损失4%的现有收入。要实现9%的增长，仅需维持现状就需要获得13%以上的新客户。⁴

宏观压力。商品化。竞争。AI。每一把刀切割的方式都不同。底部四分位区间的公司将面临加速的损失。有些可能会陷入彻底的收缩。达摩克利斯之剑仅悬于一根马鬃之上。对于竞争激烈的品类中那些更简单的产品而言，这根马鬃正在磨损。
