今天，我们激动地宣布，LangChain 的 Agent（智能体）工程平台 LangSmith 现已在 Google Cloud Marketplace 上线。Google Cloud 客户现在可以通过其现有的 Google Cloud 账户采购 LangSmith，从而实现无缝计费、简化采购流程，并能够使用现有的 Google Cloud 承诺消费额度。

领先的工程团队已经信赖 LangSmith，为复杂的 Agent（智能体）工作流带来可靠性和可见性。此次在 Marketplace 上线，深化了我们与 Google Cloud 的长期合作。LangSmith 的 SaaS 服务运行在 Google Cloud 基础设施上，并且 LangChain 已与 Google 在 Gemini 模型集成、Agent2Agent (A2A) 协议以及用于数据库的 MCP 工具箱等计划上展开了密切合作。LangSmith 在 Google Cloud Marketplace 上线，是帮助我们共同的客户构建、测试和部署可用于生产的 AI 应用程序的顺理成章的下一步。

## LangSmith 的优势

LangSmith 为企业团队提供了一个统一的平台，用于调试、测试、部署和监控 AI 应用程序。其核心能力涵盖可观测性、评估、提示词工程和部署：

### 可观测性

![](/images/posts/644a6b8bc0f8.png)

LangSmith 通过详细的追踪和实时监控，为应用程序行为提供深入的可见性。工程团队可以检查和调试单个交互，在关键指标上配置警报，并跟踪长期趋势。这一可观测性层帮助企业团队保持对 Agent（智能体）行为的可审计性和可解释性。

### 评估

![](/images/posts/b8382e59a6ce.png)

LangSmith 通过部署前测试和对生产流量的持续反馈，使评估应用程序性能变得简单。团队可以运行实验来比较提示词或模型的变更，通过标注队列收集人工反馈，并组织可复用的数据集以跟踪长期性能。这有助于团队发现性能退化并进行迭代，以持续提升质量。

### 部署

![](/images/posts/1ce318e71586.png)

LangSmith Deployment（原 LangGraph Platform）提供了部署、扩展和管理有状态、长时间运行的 Agent（智能体）所需的基础设施。团队可以通过一键 GitHub 集成在几分钟内部署，通过 30 多个 API 端点暴露 Agent（智能体），并可根据合规要求选择 SaaS、混合部署或完全自托管选项。

### Agent Builder

![](/images/posts/7fdf857c314d.png)

使用 LangSmith Agent Builder，无需编码即可在几分钟内构建强大的 AI Agent（智能体）。从现成的模板开始，连接您已使用的应用程序，让您的 Agent（智能体）处理起草邮件、总结更新和组织信息等日常任务。直接在聊天中或通过 Slack 等工具与您的 Agent（智能体）协作，让帮助触手可及。内置的审批工作流确保您能掌控最重要的操作。这是按您的方式为您服务的自动化。

### 无缝的 Google Cloud 集成

![](/images/posts/d548b91734b2.png)

LangChain 提供了与 Google Cloud 服务的丰富集成生态系统，使 GCP 客户能够轻松地完全在其云环境中构建 AI 应用程序：

-   Vertex AI & Gemini：对使用最新 Gemini 3 Pro 和 Flash 的 Gemini 模型提供一流支持，并可访问 Vertex AI Model Garden 中的 200 多个模型，包括 Claude、Llama 和 Mistral。
-   数据库：与 AlloyDB for PostgreSQL、BigTable、BigQuery、Spanner、Firestore 和 Memorystore 的原生集成，用于向量存储、文档加载和聊天历史记录。
-   计算：完全托管的 SaaS 服务，LangChain 使用 Google Kubernetes Engine (GKE) 等 GCP 服务来托管和运营所有 LangSmith 基础设施和服务。
-   工具：从 Google Finance 查询金融数据，与 Google Drive 中的文档交互，从 Google Trends 搜索趋势。

这些集成意味着 GCP 客户可以利用其现有的数据和基础设施投资，同时通过 LangSmith 获得全面的可观测性和控制力。

### 为何通过 Google Cloud Marketplace 采购 LangSmith

通过 Google Cloud Marketplace 采购 LangSmith 为企业团队带来显著优势：

-   统一计费：所有 LangSmith 费用将显示在您现有的 Google Cloud 发票上，简化供应商管理和财务报告。
-   使用承诺消费额度：LangSmith 的采购计入您的 Google Cloud 承诺消费额度，帮助您最大化现有云投资的价值。
-   简化采购流程：由于 Google Cloud 已是获批准的供应商，团队可以加速采购周期并减少管理开销。

LangSmith 提供多种部署配置以满足您的安全和合规要求，从完全托管的 SaaS（已运行在 Google Cloud 基础设施上），到数据保留在您 VPC 内的混合部署，再到使用我们的 Helm charts 和 Terraform 模块在 GKE 上完全自托管。
