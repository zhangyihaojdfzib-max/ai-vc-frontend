# 使用 Wan2.1 生成风格化视频

- shridharathi

Wan 2.1 是一个快速、开源的大语言模型，能够根据文本或图像生成视频。

使用 Wan 进行视频风格迁移是一种特别有趣的玩法。
无论你是想制作梦幻的吉卜力风格循环动画、粗粝的赛博朋克风格预告片，还是完全属于你自己的独特内容，Wan 都能让你轻松创建风格化视频。你可以直接使用预设风格开始，
也可以使用自己的图像，在几分钟内训练一个自定义风格。

您的浏览器不支持 video 标签。

Replicate 还支持在 Wan 2.1 上使用 LoRA 进行快速推理，因此你可以将风格应用到你的视频中，等待时间更短，且无需额外设置。

以下是入门方法。

## 使用预设风格

你可以在这两个文本到视频的 Wan 2.1 模型上运行预设风格：

•Wan 2.1 文本到视频，480p

•Wan 2.1 文本到视频，720p

要使用某种风格，请键入你的提示词，将 `fast_mode` 设置为 "Fast"，并在 `lora_weights` 输入框中添加你的权重文件链接。此链接可以是来自互联网（如 HuggingFace、CivitAI 或任何其他平台）的任意 safetensors 文件 URL。

`lora_weights` 字段甚至接受你在 Replicate 上找到的经过微调的 Wan 模型。以下是一个使用 `fofr/wan-14b-cyberpunk-realistic` 和 Replicate 的 Python 客户端的示例：

```
import Replicate from "replicate";

const replicate = new Replicate();
const model = "wavespeedai/wan-2.1-t2v-480p";
const input = {
  prompt: "driving a car very fast through a city at night",
  fast_mode: "Fast",
  lora_weights: "fofr/wan-14b-cyberpunk-realistic",
};
const output = await replicate.run(model, { input });

console.log(output.url());
```

以下是一些你可以尝试的预设风格：

- 梵高风格
- 黑巩膜眼睛风格
- 彩虹棱镜光照风格
- 动态模糊风格
- 扁平色彩风格

## 创建你自己的风格

在 Replicate 上，你也可以使用自己挑选的照片来训练 Wan，以创建你自己的风格化视频。以下是使用 `ostris/wan-lora-trainer` 的方法。

1.  选择一个目标模型（或输入新名称以创建一个）。
2.  准备你的训练图像。创建一个包含至少 25 张能体现你风格的图像的文件夹，然后将其压缩成 zip 文件。
3.  将 zip 文件上传到 `input_images` 字段。
4.  选择一个 `trigger_word`（触发词）—— 这是你将在提示词中键入以激活该风格的词语。
5.  保持 `autocaption` 开启。你可以选择性地添加 `autocaption_prefix` 或 `autocaption_suffix` 来更好地引导风格。
6.  保持其他设置为默认（或根据需要调整），然后点击 `运行`。

几分钟后，你将拥有一个基于你的风格训练、可随时生成视频的专属 Wan 版本。

无论你是使用预设风格还是训练自己的风格，Wan 2.1 都能让你轻松创建独具特色的风格化视频。
只需选择一个风格，写一段提示词，然后点击运行。
