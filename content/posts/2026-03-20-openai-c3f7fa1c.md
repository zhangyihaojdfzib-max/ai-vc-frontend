# 参数高尔夫

我们知道，好的想法可能来自任何地方。因此，我们正在举办一项开放式研究挑战，目标是在极端限制下构建最高效的预训练模型。

你的目标：在严格的 16 MB 工件限制（权重 + 训练代码合计）和 8×H100 GPU 上 10 分钟的训练预算内，最小化在固定 FineWeb 数据集上的留出损失。

#### 如何参与

我们提供了一个包含基线模型、固定数据集和评估脚本的 GitHub 仓库。请 Fork 该仓库，在大小和算力限制内改进模型，并通过 PR 提交你的代码、日志、分数和简短说明。一旦审核通过，你的结果将被合并，排行榜将自动更新。你可以通过 Runpod 申请免费算力积分（数量有限，先到先得）。

#### 为何要参与

这项挑战旨在发掘我们希望聘用的杰出研究人员和工程师。表现突出的参与者可能会受邀参加 OpenAI 的工作机会面试，获胜方案可能会被公开报道。

如果你热爱解决此类技术问题，并有兴趣与我们的招聘团队建立联系，请告诉我们你的情况（在新窗口中打开）。
