## 编者按

开源世界从未减速。新模型的发布速度之快，让我应接不暇。有太多事情可做，我们手中掌握了太多新的创意工具。如今，每个人都可以凭借一台笔记本电脑或手机，成为一家初创公司、一个品牌、一个出版商、一个电影工作室。

这些新的AI工具让生成新事物变得更容易。它甚至在某些圈子里成了流行语——“GenAI”（生成式AI）。当神经网络生成某些东西时，确实比它仅仅识别数据中的模式更令人印象深刻，也更显而易见。但它们本质相同：我们要求模型在没有任何模式的地方吐出模式，而它则给出它学会识别的数据。模型是世界的一面镜子。

今天有一篇帖子，来自向量数据库公司 Chroma 的联合创始人 Anton Troynikov，讨论了如何思考AI应用开发。这是很好的建议，你应该读一读。我想在此强调的部分是第3张幻灯片：

什么是LLM？就应用开发而言，LLM是：

一种信息处理系统
以常识性方式
处理非结构化信息
可通过API访问

这一点至关重要。它以常识性方式处理（识别、生成）信息（计算机中的其他一切）——也就是说，通过在世界这面镜子中反映它。Anton是对的，这简直是奇迹！我们可以用它来自动化我们几十年来一直手动完成的许多流程。我们可以解绑智能。这对软件乃至整个世界都是一件大好事。

但我们必须记住，这面镜子是由我们的语言、我们的视频、我们选择分享的事物构成的。它不是对世界的模拟。它是对我们的模拟。

当我们用这面镜子来生成图像、声音、故事、记录、判断以及所有其他曾经仅属于人类的非结构化信息片段时，我们放大了我们自身的倒影。

我们已经在消费市场看到了这一点。比AI自拍更广泛普及的，只有AI恋爱聊天机器人。我们将通过这面镜子看到我们自己，以及彼此。连同它所有的偏见和缺陷。这个反馈循环将持续下去，因为未来的模型将用今天生成的数据进行训练。常识将会改变，世界也将随之改变。

这听起来可能令人沮丧，但这就是人类所为。我们改变自己、我们的环境、我们的关系，然后继续在新的世界体系中做着那些奇怪、有趣、渺小的人类之事，并不断重复。这就是正在加速的过程，我们解锁的每一项新能力都将使这个循环转得更快一些。但关于反馈循环的一点是，它们对初始条件非常敏感。

## 强大的新型图像生成器

FLUX.1 是由 Black Forest Labs 开发的一套令人印象深刻的新型图像生成模型，该公司由前 Stable Diffusion 开发人员新近成立。这些模型采用先进的“流匹配”技术，与标准扩散模型相比，能够实现更快、更可控的生成。

FLUX.1 包含三个变体：

- FLUX.1 [pro]：具有最先进性能的旗舰模型
- FLUX.1 [dev]：用于非商业用途的蒸馏版本
- FLUX.1 [schnell]：最快的模型，在宽松的 Apache 2.0 许可证下发布

这些模型擅长处理复杂的提示词、生成细节文本、组合复杂场景、绘制逼真的手部以及产生多样化的输出。它们支持多种宽高比和高达 200 万像素的分辨率。

FLUX.1 可通过 Replicate API 进行试用。FLUX.1 [schnell] 的开源许可证对开发者和研究人员尤其具有吸引力。

## 图像和视频抠图的新标准

Meta 发布了 SAM 2，这是其 Segment Anything Model 的下一代产品。它能够实现图像和视频中的实时对象分割。

SAM 2 将原始模型的零样本能力扩展到了视频领域。它甚至可以在未见过的领域中分割对象。该模型性能优于先前的方法，同时速度快了 3 倍。

Meta 开源了代码、模型以及包含 51k 个视频上 60 万个标注的新 SA-V 数据集。这为以下应用打开了大门：

- 视频编辑
- 更快的数据标注
- 实时背景移除

## 小巧、可微调且智能十足的语言模型

谷歌发布了 Gemma 2 2B，这是其 Gemma 开源语言模型家族中一个强大的新成员。尽管只有 20 亿参数的紧凑规模，Gemma 2 2B 却提供了超乎寻常的性能，甚至在 LMSYS Chatbot Arena 排行榜上超越了所有 GPT-3.5 模型——尽管有人指出这可能反映了排行榜方法的缺陷，因为该新模型在 MMLU 基准测试中表现不佳。

Gemma 2 2B 的亮点：

- 在其规模类别中达到最佳效果
- 经过优化，可在从边缘设备到云端的广泛硬件上高效运行
- 权重在宽松许可证下提供，并附有免费的 Colab 演示

## 理解和控制大语言模型的新工具

## 来自 fast.ai 创建者的新 Python Web 框架

FastHTML 是一个新的 Python Web 框架，旨在简化交互式应用开发。它允许您使用纯 Python 构建应用，无需单独的 HTML 模板。FastHTML 为身份验证、数据库、缓存和样式等基本功能提供了内置支持。将您的作品部署到 Railway 和 Vercel 等平台非常简单。

## 一种用于大语言模型的分布式训练方法
