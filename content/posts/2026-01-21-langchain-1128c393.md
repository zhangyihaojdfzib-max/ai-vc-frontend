LangSmith Agent Builder 允许任何人通过简单的提示词构建一个智能体。只需要求它为你创建一个市场调研智能体，它就会通过提出相关问题来构建你所需的内容。

但有时你可能希望从现成的方案开始。今天我们推出 **Agent Builder 模板库**，并扩展了我们的工具集成，以帮助你更快地将想法转化为可运行的智能体。

Agent Builder 模板是针对常见任务的预构建智能体，已连接工具并包含智能体指令。它们已准备就绪，可随时部署，且完全可定制。你可以更新智能体的指令、添加工具并设置审批要求。

与传统的工作流自动化不同，你无需规划每一步骤，也无需花费数小时调试更改。只需像对待队友一样给予智能体反馈，它就能学习。

我们与最了解其领域的公司共同构建了这些模板，包括 Tavily、PagerDuty、Exa、Box 和 Arcade，并且我们正在定期添加新模板。[探索模板库](https://smith.langchain.com/agent-builder/templates)

立即试用这些智能体模板：

- **日历简报（Google Calendar）**：每天早晨查看你的日历，并发送包含会议参与者调研的摘要。
- **邮件助手（Gmail）**：对你的邮件进行分类，并起草回复供你审批。
- **事件响应器（PagerDuty）**：分析警报，交叉参考你的运行手册，并推荐行动方案。
- **文档接收审核（Box）**：审核文件提交，并准备摘要供你审批。
- **人才搜寻（Exa）**：根据你的职位描述搜索 LinkedIn，并发送推荐的候选人资料。
- **竞争对手研究（Tavily）**：进行深入的竞争性研究，并提供简洁的报告。
- **社交媒体监控（X + Slack）**：监控 X，并将包含最新消息的每日摘要发送到 Slack。

### 了解 Arcade 的无限可能

目前，Agent Builder 提供了一套现成的工具集成和模板。然而，你的团队可能希望通过 MCP 连接的工具数量几乎是无限的。**Arcade 的 MCP 网关**为 Agent Builder 额外提供了 8,000 种工具，覆盖营销、销售、招聘、客户成功、产品、工程和通用生产力等用例。

为了展示其可能性，Arcade 开发了一套包含 60 多个可立即部署的 Agent Builder 模板的集合，可在其托管的图库中找到。每个模板都包含设置和开始使用智能体的分步指南。[探索 Arcade 模板](https://www.arcade.software/agent-builder-templates)。

### 为你的智能体选择最佳模型

无论你是从提示词还是模板开始，不同的任务可能需要不同的模型。成本、延迟和推理要求会根据你的智能体是在夜间总结邮件还是实时响应问题而有所不同。这就是为什么 Agent Builder 不将你锁定在单一模型中。它支持 OpenAI、Anthropic 和 Google Gemini 模型，以及任何遵循 OpenAI 或 Anthropic 规范的自定义或开源模型。

为了展示其可能性，Baseten 使用他们的 GLM 4.7 模型构建了一个智能体，该模型能够快速响应以实现实时用户交互。将你偏好的模型提供商连接到 Agent Builder，然后你就可以开始构建了。
