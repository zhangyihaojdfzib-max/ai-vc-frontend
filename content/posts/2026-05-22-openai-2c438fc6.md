# Zelma

Zelma 利用 GPT‑4 让教育数据触手可及。

Zelma 是一款基于 GPT‑4 的研究助手，正在让全美家长、教师、学校管理者及政策制定者能够轻松获取教育数据。

他们解决的是什么问题？尽管美国 3–8 年级的每位学生都要参加标准化测试，但关于学生表现的数据在很大程度上仍未被挖掘和充分利用，分散在不同的来源和格式中。测试数据能够提供关于学生在英语语言艺术（ELA）和数学方面学习情况的深刻洞察，涵盖学区和人口群体层面。然而，那些可能塑造教育未来的关键洞察，实际上对大多数家长、记者、教育工作者和政策制定者来说遥不可及。

认识到这一差距后，布朗大学经济学家兼作家 Emily Oster 博士着手改变现状。“Zelma 让家长和政策制定者都能用日常语言即时获得针对性的教育洞察，聚焦于他们最关心的问题，”Oster 博士解释道。Zelma 是如何实现这一点的？Oster 博士在布朗大学的学生研究团队花了一年时间收集数据，并精心将其整理为统一的格式。随后，Zelma 与 Novy（在新窗口中打开）合作，利用 OpenAI 的 API 让数据“活”起来。

最大的挑战是预测人们会如何措辞提问，同时将提示词保持在 Zelma 的知识范围内。“我们通过在 Zelma 的用户体验中进行有意的设计选择来解决这一问题，引导用户提出 Zelma 能够回答的问题，”Oster 博士解释道。关键功能包括：
