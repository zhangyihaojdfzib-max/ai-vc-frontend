Web Analytics 和 Speed Insights 第二版引入了弹性数据接收功能，以提升数据收集的可靠性。新版本包通过动态发现端点，而非依赖单一可预测路径，确保您能捕获更完整的流量与性能数据。

要使用弹性数据接收功能，请更新您的软件包并部署更改。无需其他配置，现有实现将继续如常工作。该功能对所有团队开放，不收取额外费用。

**安装最新版本**

npm install @vercel/analytics@latest

npm install @vercel/speed-insights@latest

这些软件包的许可证已从 Apache-2.0 变更为 MIT，以与其他开源包保持一致。Nuxt 应用程序可利用 Nuxt 模块，通过一行命令安装 Speed Insights 和 Web Analytics。

更新您的软件包以捕获更多数据，或查阅 Web Analytics 文档 和 Speed Insights 文档。
