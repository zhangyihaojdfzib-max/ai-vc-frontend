AI Elements 是一个全新的开源 React 组件库，用于配合 Vercel AI SDK 构建界面，其组件可高度自定义。

它构建于 shadcn/ui 之上，让你能完全控制消息线程、输入框、推理面板和响应操作等 UI 基础元素。

例如，你可以使用 AI SDK 中的 `useChat` 来管理状态和流式响应，并使用 AI Elements 来渲染响应内容。

```
1import { Message, MessageContent } from "@/components/ai-elements/message";2import { Response } from "@/components/ai-elements/response";3import { useChat } from "@ai-sdk/react";4
5export default function Example() {6  const { messages } = useChat();7
8  return messages.map((message) => (9    <Message from={message.role} key={message.id}>10      <MessageContent>11        {message.parts12          .filter((part) => part.type === "text")13          .map((part, i) => (14            <Response key={`${message.id}-${i}`}>{part.text}</Response>15          ))}16      </MessageContent>17    </Message>18  ));19}
```

### 快速开始

要安装这些组件，你可以使用我们的 CLI 进行初始化，选择你需要的组件，导入它们，然后开始构建。

```
1npx ai-elements@latest
```

阅读文档，更快地开始构建更好的 AI 界面。

AI Elements 取代了 ChatSDK，为 AI 界面提供了一套更灵活的 UI 构建模块。ChatSDK 将被迁移到一个专门的 Next.js 模板中。未来的模板将使用 AI Elements，以支持超越聊天模式的更广泛的 AI 原生界面范式。
