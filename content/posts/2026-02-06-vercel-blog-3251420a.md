Vercel Sandbox SDK 新增了两种方法，让文件检索变得简单。

当你在 Vercel Sandbox 中运行代码时，该代码可以生成诸如 CSV 报告、处理后的图像或 PDF 发票等文件。这些文件在隔离的虚拟机内创建，因此需要跨越网络边界进行检索。在此之前，这需要通过自定义管道进行手动流处理。

### 下载文件

如果你想将沙箱中生成的报告下载到本地机器，可以使用 `downloadFile()` 来无缝流式传输内容。

```
1const dstPath = await sandbox.downloadFile(2  { path: 'generated-file.csv', cwd: '/vercel/sandbox' },3  { path: 'generated-file.csv', cwd: '/tmp' }4);
```

### 将文件内容读取到缓冲区

这两种方法都会自动处理底层的流操作。例如，如果你的沙箱运行一个生成 PNG 格式图表的脚本，只需调用一次 `readFileToBuffer()` 即可将其取出，无需手动进行流连接。

```
1​const buffer = await sandbox.readFileToBuffer({ path: 'chart.png' });​
```
