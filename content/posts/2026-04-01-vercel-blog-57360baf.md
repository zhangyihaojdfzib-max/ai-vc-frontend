Chat SDK 现在通过新的 Zernio 适配器支持 Zernio（一个统一的社交媒体 API）。这是一个由 Zernio 团队构建和维护的官方供应商适配器。

团队可以通过单一集成，构建能够在 Instagram、Facebook、Telegram、WhatsApp、X/Twitter、Bluesky 和 Reddit 上运行的机器人。

立即尝试 Zernio 适配器：

```
1import { Chat } from "chat";2import { createZernioAdapter } from "@zernio/chat-sdk-adapter";3
4const bot = new Chat({5  adapters: {6    zernio: createZernioAdapter(),7  },8});9
10bot.onNewMention(async (thread, message) => {11  const platform = message.raw.platform;12  await thread.post(`Hello from ${platform}!`);13});
```

功能支持因平台而异；富卡片在 Facebook、Instagram、Telegram 和 WhatsApp 上可用，而编辑和流式传输目前仅限于 Telegram。

阅读文档以开始使用，浏览目录，或构建你自己的适配器。
