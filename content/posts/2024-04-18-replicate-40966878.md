# 通过 API 运行 Meta Llama 3

-   cbh123

Llama 3 是 Meta 发布的最新语言模型。它拥有最先进的性能，上下文窗口为 8000 个 Token，是 Llama 2 上下文窗口的两倍。

通过 Replicate，您只需一行代码即可在云端运行 Llama 3。

## 在我们的 API 游乐场中尝试 Llama 3

在深入使用之前，您可以在我们的 API 游乐场中先试试 Llama 3。

尝试调整提示词，看看 Llama 3 如何回应。Replicate 上的大多数模型都有这样一个交互式 API 游乐场，可在模型页面上找到：https://replicate.com/meta/meta-llama-3-70b-instruct

API 游乐场是了解模型功能的好方法，并提供多种语言的可复制代码片段，帮助您快速上手。

## 使用 JavaScript 运行 Llama 3

您可以使用我们的官方 JavaScript 客户端运行 Llama 3：

安装 Replicate 的 Node.js 客户端库

```
npm install replicate
```

设置 REPLICATE_API_TOKEN 环境变量

```
export REPLICATE_API_TOKEN=r8_9wm**********************************
```

（您可以在账户中生成 API token。请妥善保管。）

导入并设置客户端

```
import Replicate from "replicate";

const replicate = new Replicate({
  auth: process.env.REPLICATE_API_TOKEN,
});
```

使用 Replicate 的 API 运行 meta/meta-llama-3-70b-instruct。查看模型架构以了解输入和输出的概览。

```
const input = {
  prompt: "Can you write a poem about open source machine learning?"
};

for await (const event of replicate.stream("meta/meta-llama-3-70b-instruct", { input })) {
  process.stdout.write(event.toString());
};
```

要了解更多信息，请查看 Node.js 入门指南。

## 使用 Python 运行 Llama 3

您可以使用我们的官方 Python 客户端运行 Llama 3：

安装 Replicate 的 Python 客户端库

```
pip install replicate
```

```
export REPLICATE_API_TOKEN=r8_9wm**********************************
```

导入客户端

```
import replicate
```

```
# The meta/meta-llama-3-70b-instruct model can stream output as it's running.
for event in replicate.stream(
    "meta/meta-llama-3-70b-instruct",
    input={
        "prompt": "Can you write a poem about open source machine learning?"
    },
):
    print(str(event), end="")
```

要了解更多信息，请查看 Python 入门指南。

## 选择使用哪个模型

Replicate 上有四个 Llama 3 模型变体，各有其优势。Llama 3 提供两种参数规模：700 亿和 80 亿，并包含基础模型和经过对话调优的模型。

-   meta/meta-llama-3-70b-instruct：700 亿参数模型，针对对话完成进行了微调。如果您想构建一个具有最佳准确性的聊天机器人，请使用此模型。
-   meta/meta-llama-3-8b-instruct：80 亿参数模型，针对对话完成进行了微调。如果您正在构建聊天机器人，并且希望它更快、更便宜，但可以接受牺牲一些准确性，请使用此模型。
-   meta/meta-llama-3-70b：700 亿参数基础模型。这是在进行对话完成指令微调之前的 700 亿参数模型。
-   meta/meta-llama-3-8b：80 亿参数基础模型。这是在进行对话完成指令微调之前的 80 亿参数模型。
