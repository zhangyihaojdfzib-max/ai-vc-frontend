Qwen 3.5 Plus 现已上线 AI Gateway。

该模型具备 1M 的上下文窗口，并内置自适应工具调用功能。Qwen 3.5 Plus 在 Agent（智能体）工作流、思考、搜索以及跨多模态情境使用工具方面表现卓越，非常适合网页开发、前端任务以及将指令转化为可运行代码。与 Qwen 3 VL 相比，它在科学问题解决和视觉推理任务上展现出更强的性能。

要使用此模型，请在 AI SDK 中将模型设置为 `alibaba/qwen3.5-plus`：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'alibaba/qwen3.5-plus',5  prompt:6    `Analyze this UI mockup, extract the design system,7     and generate a production-ready React component8     with responsive breakpoints and theme support.`,9});
```

AI Gateway 提供了一个统一的 API，用于调用模型、跟踪使用情况和成本，并可配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它内置了可观测性、支持自带密钥（Bring Your Own Key），并具备带自动重试的智能供应商路由功能。

了解更多关于 [AI Gateway](https://example.com) 的信息，查看 [AI Gateway 模型排行榜](https://example.com) 或在我们的 [模型游乐场](https://example.com) 中试用。
