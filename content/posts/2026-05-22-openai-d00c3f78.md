# Superhuman

Superhuman 借助 OpenAI 开启了电子邮件的新时代。

![Screenshot 2024 03 12 At 1128 27am](/images/posts/e31d5d373669.png)

向在线工作的转变使得大约 10 亿专业人士每天花费近 3 小时管理电子邮件⁠(opens in a new window)。这相当于每年约 1 万亿小时的人类时间花在电子邮件上。

Superhuman 的目标是减少管理收件箱所花费的时间。他们利用 OpenAI 的 API 构建了一套下一代 AI 电子邮件产品，为用户节省时间、创造价值并提高参与度。

## 爆满的收件箱

在任何公司中，很大一部分员工（Superhuman 估计在 30–50% 之间，涵盖财务、销售和支持等岗位）每天都在处理电子邮件，与爆满的收件箱作斗争。电子邮件很容易被忽视，导致延误，甚至更糟的是被错误处理——从而引发错误甚至声誉损害。

Superhuman 最初通过一款热门产品解决了这个问题，该产品提供了一系列内置的电子邮件自动化功能，并与 Gmail 和 Outlook 等常见电子邮件服务集成。随后 AI 的出现，为公司将其愿景提升到新高度并彻底改变电子邮件体验提供了机会。

## 借助 OpenAI 重新构想电子邮件体验

Superhuman 正在基于 OpenAI API 驱动的 Superhuman AI 上快速推进。他们已经推出了大量由 GPT 驱动的功能，例如：

- AI 写作，将书面提示词转化为完整的电子邮件
- 用你的语气重写，以用户个人的写作风格和语气重新措辞电子邮件
- 用你的语气写作，将口述的陈述转化为完整的电子邮件，让用户只需说几句话就能撰写邮件
- 自动摘要，始终在用户收件箱中为每封邮件和每个邮件线程显示最新的单行摘要
- 即时回复，让用户通过从上下文相关的回复选项中选择，一键回复电子邮件

从一开始，与 OpenAI 合作就是 Superhuman AI 路线图的核心部分，原因如下：

- 创业文化和支持："OpenAI 本身也是一家初创公司，因此深知如何以极快的速度工作，"Vohra 说。"我们与 OpenAI 团队的各个成员（包括工程师、市场推广人员和 executives）保持着持续的沟通。"
- 隐私优先：由于 Superhuman 嵌入在机密邮件线程中，客户对保护其数据的需求是可以理解的。因为 OpenAI 为符合条件的用例提供零数据保留（ZDR），Superhuman 得以满足其严格的隐私和合规要求。
- 优化的性能和成本：OpenAI 的 API 提供了访问一系列具有不同性能和成本特征的模型的机会，这使得 Superhuman 能够做出明智的权衡。"OpenAI 是该领域的明确领导者，其模型在质量上领先于次优模型六个月到一年以上。这在模型输出质量和模型运行速度两方面都是如此，"Vohra 补充道。

## 以 2 倍速度实现收件箱归零

Superhuman 推出的 Superhuman AI 取得了巨大成功，用户采用率高，工作流程更快：

- AI 功能的高选择率：超过 85% 的 Superhuman 用户选择了新的 GPT 驱动功能。
- 不断增长的参与度：使用数据显示，用户不仅选择了这些功能，而且使用频率还在持续增长。例如，用户平均每周使用 AI 写作功能 25 次，自推出以来这一数字增长了 55%。
- 收件箱处理速度翻倍："我们的 AI 功能帮助客户处理收件箱的速度比以前快了大约两倍，"Vohra 说。
- 电子邮件写作速度提升 2 倍：Superhuman 的即时回复功能利用 OpenAI 的 API 起草回复，用户只需点击发送即可。"在测试阶段，用户写邮件的速度比以前快了两倍，"Vohra 解释道。"如果按我们用户发送的邮件数量来算，仅这一项功能每周就能为用户节省一个多小时。"

## 按下电子邮件的"快进"键

"想象一下，醒来时看到收件箱里每个邮件线程都有一条一句话摘要和一封草拟好的回复。只需点击发送即可，"Vohra 说。这在几年前可能听起来有些牵强。但 Superhuman 已经利用 OpenAI 的 API 构建并推出了这种端到端的体验。每封邮件都附带自动 AI 生成的摘要，用户可以从多个 AI 起草的回复中选择并即时发送。"得益于这些 AI 功能，Superhuman 的用户群在过去一年中快速增长，每周新用户数量翻了一番以上。"
