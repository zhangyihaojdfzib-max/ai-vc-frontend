通过Vercel的更新，将功能标志更深入地集成到您的工作流程中

## Flags Explorer：在Vercel中管理您的标志

Vercel工具栏中的Flags Explorer让您清晰查看所有功能标志，并允许您直接从工具栏覆盖其值，从而简化测试和实验。此功能现已对所有计划的用户全面可用。

## Flags SDK：在代码中使用标志

Flags SDK是一个开源库，为开发者提供在Next.js和SvelteKit应用程序中集成功能标志的工具。

```
npm i flags
```

flags与Statsig等流行提供商、其他第三方工具或您自己的自定义解决方案兼容。无论您是运行A/B测试、逐步推出新功能还是个性化用户体验，该SDK都提供了一种灵活的方式在运行时控制功能，而无需重新部署。

```
1import { flag } from 'flags/next';2
3export const exampleFlag = flag({4  key: 'example-flag',5  decide() {6    return false;7  },8});
```

使用Flags SDK导入和评估标志。

Flags SDK现在通过新的适配器架构支持直接从多个提供商加载功能标志，更多提供商即将加入。这简化了使用Flags SDK将代码连接到功能标志提供商的过程。例如，`@flags-sdk/statsig`包允许您在Vercel的Flags Explorer中直接管理您的Statsig标志。
