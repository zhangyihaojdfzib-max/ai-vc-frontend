# Indeed 借助 OpenAI 为数百万求职者提供情境化职位匹配

Indeed⁠(opens in a new window) 的使命是帮助人们找到工作，它是全球排名第一的招聘网站¹。每月有超过 3.5 亿独立访客² 访问 Indeed，与超过 350 万家雇主和 3200 多万个职位建立联系。更重要的是，Indeed 上每三秒就有人成功获得录用³。

自 Indeed 成立以来，人工智能就通过诸如“邀请申请”等功能，为数以百万计的求职者和雇主在平台上建立联系提供动力。“邀请申请”功能会根据求职者的简历、Indeed 个人资料和其他资质，向他们发送基于人工智能的职位推荐。人工智能——特别是生成式人工智能——的进步，正在以新颖且令人兴奋的方式帮助求职者与职位进行匹配。利用 OpenAI 的 GPT 模型和微调能力，Indeed 增强了“邀请申请”功能中的个性化语言，以更好地解释为何候选人的背景或先前工作经验使其非常适合某个职位。

## 职位匹配的早期人工智能实验

Indeed 早期的人工智能模型有效地将求职者与雇主的职位发布相匹配，并为这些匹配提供了简要的解释。但 Indeed 的产品和工程团队相信，OpenAI 的 GPT 模型能够做得更多，通过处理自然语言来提供更好的背景信息，解释候选人为何收到特定的职位推荐。

为职位推荐增添更具洞察力的“原因”，成为 Indeed 团队与 OpenAI 合作的目标。

Indeed 的工程团队于 2023 年开始与 OpenAI 合作，遵循 OpenAI 的 Chat Completions API 文档，并对其少样本提示进行迭代。在整个开发过程中，团队进行了严格的 A/B 测试，并在实施阶段和上线后持续监控和评估每一项变更。

## 微调 OpenAI API 以实现个性化匹配解释

Indeed 发现，使用少样本提示训练模型来生成回应，能有效增加解释中相关的“原因”，但在 Indeed 的巨大规模下，这导致了高昂的 Token 消耗。为了提高效率，OpenAI 和 Indeed 合作微调了一个较小的 GPT 模型，该模型能够提供性能相似的结果，但 Token 使用量减少了 60%。

作为微调过程的一部分，他们利用 GPT‑4 进行数据增强，并制定了具体的内容指南。团队还建立了一个标注操作，包括标注 LLM（大语言模型）生成的输出、为促进自动评估创建基准事实，以及添加上下文以帮助模型理解细微差别。

进一步的测试验证了职位推荐中个性化的价值，增加了合格候选人的数量。为了规模化提供个性化职位推荐，Indeed 与 OpenAI 合作，采用了于 2024 年 1 月配置的专用实例。Indeed 工程团队成功地将微调后的 GPT 模型部署到这些实例上，从而能够为数百万更多的求职者提供个性化的就业机会。

“几十年来，我们一直投资于自己的人工智能匹配技术，以帮助求职者和雇主建立联系。无论我们的匹配技术多么出色，可解释性是任何成功推荐系统的关键。将 OpenAI 的 GPT 解释与 Indeed 自有的专有人工智能和庞大的市场数据相结合，使我们能够更快地将更多人与工作联系起来——这对求职者、雇主和社会来说都是共赢，”Indeed 首席执行官 Chris Hyams 表示。

## GPT 个性化带来可验证的增长

GPT 开发的个性化消息的有效性通过一个多阶段实验得到了验证，该实验在几个月内扩展到每天近 2000 万条消息。

“我们一直在测试改进‘邀请申请’体验并为求职者寻找相关机会的方法。微调 OpenAI 的 GPT 模型帮助我们提供了更个性化的推荐，即使我们每天的消息量从 100 万条扩展到 2000 万条，”Indeed 平台产品经理 Horatio Lun 表示。

Indeed 团队测试了传统职位匹配版本的“邀请申请”与由 GPT 驱动、为申请人提供额外定制化背景信息的版本。性能提升立竿见影：

*   已开始的职位申请数量增加 **20%**
*   下游成功率提升 **13%**，这表明不仅更多候选人可能申请，而且雇主也认为这些申请人更合适，最终导致更多录用

“重要的是，我们能够以投资回报率为正的方式利用 OpenAI，因此我们看到很多机会继续投资于这种新基础设施，以帮助我们增加收入，”Indeed 首席执行官 Chris Hyams 表示。

而“邀请申请”只是 Indeed 如何利用 OpenAI 技术取得成功的一个例子。Indeed 有近十几个产品正在使用 OpenAI，以提供更个性化和更具吸引力的体验，帮助求职者发现新机会，并帮助雇主更快地招聘。
