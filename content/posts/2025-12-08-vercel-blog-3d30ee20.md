1 分钟阅读
Vercel 现已提供统一仪表板，可呈现需要您团队处理的所有安全问题。当检测到关键漏洞或安全相关任务时，仪表板会自动将受影响的项目分组，并指导您完成保护项目所需的步骤。

当需要采取行动时，此视图会以横幅形式显示，并可通过仪表板搜索随时访问。

大多数 CVE 会通过 WAF 规则和其他保护措施自动处理，但当需要用户操作时，它们会在此处显示。

**自动检测需要用户干预的安全漏洞** - 当平台识别出无法通过 Vercel 自主保护完全缓解的漏洞或配置时，会在此处显示并提供清晰的说明。

**基于所需操作的项目分组** - 当前类别包括未修补的依赖项、需要手动修复、未受保护的预览部署。随着新增保护措施和检查，未来会出现更多分组。

**支持自动修复** - 在可能的情况下，Vercel Agent 提供一键式自动升级和 PR。

**支持手动修复** - 对于需要手动更新或无法访问 GitHub 的情况，我们提供直接指令，例如：
```
npx fix-react2shell-next
```
### 以更少精力保持安全
统一仪表板帮助团队在关键时刻快速行动，将所需的修复集中在一处，并在所有项目中维持更强的安全态势。

探索仪表板以查看任何所需的更新。
