Vercel Flags 的版本历史现在可以通过 Vercel CLI 使用新的 `vercel flags versions` 命令进行查看。

运行 `vercel flags versions <flag>` 可以打印某个标志的完整修订历史，包括每个修订版本的作者、消息、时间戳以及变更的环境。使用 `--environment` 可以筛选到特定环境，使用 `--limit` 和 `--cursor` 可以分页查看，或者添加 `--json` 以便脚本处理。

```
vercel flags versions my-flag
vercel flags versions my-flag --environment production
```

打印某个标志的修订历史，可选择筛选到单个环境

要比较某个修订版本与其前一个版本，请运行 `vercel flags versions diff <flag> --revision <n>`。该差异是语义化的，会展示针对目标规则、发布百分比和条件的字段级变更。新增、删除和修改会分别以彩色 `+`、`-` 和 `~` 显示。

```
vercel flags versions diff my-flag --revision 42
```

比较修订版本 42 与其前一个版本

请更新到最新版本的 Vercel CLI，并在 Vercel Flags CLI 文档中了解更多信息，以开始使用。
