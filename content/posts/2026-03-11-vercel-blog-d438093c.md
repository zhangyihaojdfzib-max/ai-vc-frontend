Vercel CLI 现已支持编程式功能标志管理，让团队无需打开仪表盘即可通过终端直接创建和管理功能标志。

```
1vercel flags create my-flag
```

添加 Flags SDK 技能

基于此基础，**Flags SDK 技能**让 AI Agent（智能体）能够通过自然语言提示词生成和管理功能标志。

该技能在底层利用了 CLI，使 Agent（智能体）能够实现服务器端评估，从而防止布局偏移并保持机密性。利用 SDK 的适配器模式，Agent（智能体）可以连接多个提供商并评估用户细分，而无需重写核心功能标志逻辑。

```
1npx skills add vercel/flags
```

添加后，尝试使用此提示词向您的 Agent（智能体）发出指令，以创建您的第一个功能标志。

```
1Add a feature flag for setting up a new promotion banner
```

开始使用 Flags SDK 技能生成功能标志。
