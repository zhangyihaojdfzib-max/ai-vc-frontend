# Vera 到货：英伟达首款为 Agent 打造的 CPU 抵达顶级 AI 实验室

Agentic AI 一直以来都需要一种不同的 CPU。英伟达 CEO 兼创始人黄仁勋在三月圣何塞 GTC 大会上推出了答案——独立的 Vera CPU，这将成为英伟达下一个数十亿美元的业务。

上周五，这款 CPU 从英伟达实验室交付到了客户手中。

首批英伟达 Vera CPU 于上周五抵达全球三家领先的 AI 实验室——旧金山的 Anthropic、Mission Bay 的 OpenAI、帕洛阿尔托的 SpaceXAI——随后于周一交付至圣克拉拉的 Oracle 云基础设施。英伟达超大规模与高性能计算副总裁 Ian Buck 亲自送达。

"Agentic AI 正在 AI 工厂中创造一个新的 CPU 时刻——随着模型从回答问题转向采取行动，Vera 专为保持这种工作大规模运行而设计，"Buck 表示。

英伟达的 Ian Buck 亲手将首批英伟达 Vera CPU 交付给我们的合作伙伴 @AnthropicAI、@OpenAI、@SpaceX 和 @OracleCloud。🎉

核心理念：想象一下，你的工作效率能提高 10 倍。你的电脑能跟得上吗？Agentic AI 对我们用于完成各种工作的基础设施提出了前所未有的需求——从制作幻灯片到编译和测试软件、分析数据、搜索文件，甚至运行模拟。

AI Agent 并非仅靠 GPU 运行。每个 agent 沙箱、每次工具调用、每个编排层、每次长上下文检索操作——这些都是 CPU 的工作。Vera 是一类新型 CPU，其设计出发点正是这一现实。

这种并发、实时任务的挑战给 CPU 带来了压力，而传统的核心密度导向设计从未优先考虑过这一点。Vera 配备了 88 个英伟达定制的 Olympus 核心、1.2 TB/s 的内存带宽以及提升 50% 的单核性能。在持续负载下，任务完成速度更快——提高了整个 AI 工厂的效率，帮助用户通过更快的响应完成工作。

## Vera 回到南湾与 OCI

周一，在 Oracle AI 客户卓越中心，来自 OCI 的团队——包括负责整体产品管理的 Karan Batta 以及首席客户与合作伙伴成功官 Gary Miller——参观了开箱后的 Vera CPU 系统。背景中，一台英伟达 GPU 机架正在处理来自全球的 OCI 客户工作负载。该中心是 Oracle 客户前来测试各种 AI 工作负载的地方。

![](/images/posts/613390bbf609.jpg)

Buck 解释了 Vera 将如何提供帮助。

"当 AI 模型被提出一个问题时，答案往往并非事先准备好、随时可用的。模型实际上必须生成一些 Python 代码才能得出正确答案，"Buck 说道。这正是 Vera CPU 擅长的任务。"这就是我们看到 CPU 需求飙升的原因，"Buck 继续说道。

OCI 团队也观察到了这一趋势。

"OCI 计划从 2026 年开始部署数十万个英伟达 Vera CPU，因为 agentic AI 需要大规模持续性能，"Batta 表示。"Vera 的架构专为高吞吐量推理工作负载而设计，提供了 OCI 为驱动下一代企业 AI 所需的效率、密度和占地面积。"

OCI 是首家超大规模部署 Vera 的云服务提供商。对于企业客户而言，这意味着生产级 agentic AI 基础设施的规模是当今其他云服务提供商无法比拟的。

OCI 团队渴望将 Vera 投入使用，为客户提供另一个系统来定制和验证他们的 agentic AI 及工作负载，Miller 表示。"我真的很期待来到这里的人们的反应，并共同努力从 Vera 中获得最大价值，"他说。

## Vera 带来的价值

Vera 是英伟达极致协同设计故事的一部分，与英伟达 Rubin GPU、BlueField 4 DPU、Spectrum-X 和 MGX 机架架构并列。

除了驱动独立的 CPU 系统外，Vera 还是 Vera Rubin NVL72 的主机处理器，通过第二代英伟达 NVLink-C2C 与一对 Rubin GPU 配对。

在这些系统中，Vera 和 Rubin 共享统一内存架构，使加速计算保持高利用率。

Vera 快速的 CPU 核心和互连处理编排、控制和数据移动，以传统基础设施 2 倍的能效为 GPU 提供数据。
