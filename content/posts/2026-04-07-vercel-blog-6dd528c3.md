Chat SDK 现已支持 Liveblocks，通过全新的 Liveblocks 适配器，机器人能够在 Liveblocks 评论线程中读取并作出响应。这是由 Liveblocks 团队构建和维护的官方供应商适配器。

团队可以构建能够在 Liveblocks 房间内发布、编辑和删除评论、使用表情符号回应以及处理 @提及 的机器人。

立即尝试 Liveblocks 适配器：

```
1import { Chat } from "chat";2import { createLiveblocksAdapter } from "@liveblocks/chat-sdk-adapter";3
4const bot = new Chat({5  userName: "mybot",6  adapters: {7    liveblocks: createLiveblocksAdapter({8      apiKey: "sk_...",9      webhookSecret: "whsec_...",10      botUserId: "my-bot-user",11      botUserName: "MyBot"12    }),13  },14});15
16bot.onNewMention(async (thread, message) => {17  await thread.post(`You said: ${message.text}`);18});
```
