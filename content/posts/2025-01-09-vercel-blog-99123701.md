`x-vercel-ip-postal-code` 标头现已纳入 Vercel 的地理定位功能，可提供与请求者公网 IP 地址关联的邮政编码。该功能补充了现有的 `x-vercel-ip-country`、`x-vercel-ip-city` 和 `x-vercel-ip-country-region` 等标头。

此信息应作为一般参考，而非精确定位工具。由于基于 IP 的地理定位存在固有局限性和可变性，其准确性可能有所差异。对于关键应用，我们建议结合其他数据源（如 Geolocation API）使用。

`x-vercel-ip-postal-code` 标头可在 Vercel Functions（包括 Edge Middleware）中访问。以下是一个 TypeScript 示例：

```
1export function GET(request: Request) {2  const postalCode = request.headers.get('x-vercel-ip-postal-code');3  return new Response(`Postal Code: ${postalCode}`);4}
```

邮政编码也可通过 `@vercel/functions` 包获取：

```
1import { geolocation } from '@vercel/functions';2
3export function GET(request: Request) {4  const geo = geolocation(request);5  return new Response(`Postal Code: ${geo.postalCode}`);6}
```

有关标头和地理定位的更多信息，请参阅 Vercel 的请求标头文档。
