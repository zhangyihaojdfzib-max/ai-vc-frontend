Vercel 官方的 MCP（模型控制协议）服务器现已上线，在 mcp.vercel.com 开启公开测试。该服务器提供了一个带有基于 OAuth 授权的远程接口，使 AI 工具能够安全地与您的 Vercel 项目进行交互。

该服务器集成了 AI 助手（如 Claude.ai、Claude Code 和桌面版 Claude）以及 VS Code 等工具，可实现以下功能：

-   搜索和浏览 Vercel 文档
-   管理项目和部署
-   分析部署日志

Vercel MCP 完全实现了最新的 MCP 授权和可流式 HTTP 规范，以增强安全性和性能。
