# Match Group

![Match Group 1](/images/posts/5327667cd3e2.png)

Match Group 是全球在线约会领域的领导者，旗下产品组合包括 Tinder、Hinge 和 OkCupid。他们正在将新时代的 AI 融入其产品和内部运营中，发现 ChatGPT Enterprise 正让他们的员工更快乐、更具创造力、更成功。

![matchgroup](/images/posts/a42d0fc03bf6.png)

## 打造优秀产品，技术之外有更多考量

Match Group 不断致力于改进产品，打造能激发新连接的奇妙用户体验。为此，他们也需要持续帮助员工成长和建立联系。

然而，当团队成员无法顺畅沟通时，协作就会陷入停滞。有时是字面意义上的语言障碍：“我们在首尔和日本的工作室需要与美国团队协作，” Kantor 说。这些团队的许多成员英语并不流利，这使得深入的技术对话充满挑战。

更多时候，连接上的隔阂源于技能和知识的差异，阻碍了员工充分表达创造力或在跨团队、跨角色间无缝协作。设计师和产品经理可能对新项目有想法，但缺乏有效构建原型的编码技能。写作能力较弱的团队成员可能难以构思有力的提案或创建高质量的文档。如今，借助 AI，所有这些挑战都可以被克服。

## 利用 ChatGPT Enterprise，打造更快乐、更高效的公司

Match Group 迅速评估了 ChatGPT Enterprise 的潜力，发现它不仅有助于打造更好的产品，还能赋能员工队伍。他们的发现令人鼓舞：“当我们所有人真正开始讨论我们的 AI 战略，并通过使用 ChatGPT 来理解模型的推理能力时，我们的想象力就天马行空地展开了，” Wu 说。

Match Group 开始探索集成 OpenAI 的 API 来构建新产品和功能，例如帮助用户更轻松地创建更深入、更真实的个人资料。同时，他们迅速采用 ChatGPT Enterprise 并将其推广给数千名员工。ChatGPT 带来的成果甚至超出了领导层最初的预期。“ChatGPT 有很多显而易见的用例，但随着更多员工获得访问权限，他们发现了令人惊喜的用例，真正将其提升到了一个新的水平，你会意识到我们只是触及了可能性的表面，” Kantor 说。

Match Group 观察到员工使用 ChatGPT Enterprise 来改进：

- **跨职能协作。** 技术主题的协作已成为 ChatGPT Enterprise 最有价值的用途之一，产品经理和设计师找到了开发原型的新方法——即使他们不知道如何编码。
- **产品可用性的焦点小组。** 设计团队尝试使用 GPT‑4 的多模态能力来运行模拟用户研究。通过上传线框图并要求 ChatGPT 模仿特定用户角色，设计师可以在要求“用户”浏览界面并提供反馈时提出问题。
- **工程效率。** 工程师们使用 ChatGPT Enterprise 来修复错误、编写文档、设计新的 API 等。“让我大开眼界的一个时刻是，一个团队在一天内就拼凑出了一个 iOS 应用来原型化一个功能，” Wu 分享道。“让我震惊的是，当我得知创建这个原型的工程师以前从未创建过 iOS 应用——他们是在使用 ChatGPT 即时组装这个应用。”
- **知识共享。** Match Group 正在探索创建自定义 GPT，供企业工作空间内的所有员工访问。例如，创建一个能够通读并搜索所有内部文档的 GPT，可以帮助团队成员更好地理解从信任与安全政策到技术决策的一切。“ChatGPT 营造了一种自主学习文化，” Wu 说。
- **翻译与国际协作。** “使用 ChatGPT 进行翻译，让我们的国际团队能够以一种全新的方式真正与全球其他员工建立联系，实现了超越语言和文化的惊人协作量，” Kantor 说。

“与 OpenAI 客户团队的合作非常具有协作性、支持性且富有成效，让我们能够在正确的时间与正确的人进行正确的对话，” Wu 说。随着 ChatGPT Enterprise “为我们整个员工队伍赋能”，Match Group 计划在 OpenAI 的 API 基础上，将 AI 融入在线约会旅程的每个环节。“Match Group 的每个团队要么正在积极构建 AI 以改进产品，要么正在积极设计和思考 AI 将如何提供帮助，” Kantor 说。“AI 不仅将成为我们的员工和团队日常使用、以提高生产力和幸福感的东西，我们还将找到负责任地将 AI 添加到我们产品中的方法，让我们的客户更容易建立联系。”
