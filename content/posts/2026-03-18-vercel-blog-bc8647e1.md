MiniMax M2.7 现已在 Vercel AI Gateway 上提供两种版本：标准版和高速版。M2.7 在软件工程、Agent（智能体）工作流和专业办公任务方面，相比之前的 M2 系列模型有了重大提升。

该模型原生支持多 Agent（智能体）协作、复杂技能编排以及动态工具搜索，用于构建 Agent（智能体）工作流。M2.7 还在生产调试和端到端项目交付方面进行了改进。

高速版在延迟敏感型用例中，以约每秒 100 Token 的速度提供相同性能，成本为标准版的 2 倍。

要使用 M2.7，请在 AI SDK 中将模型设置为 `minimax/minimax-m2.7` 或 `minimax/minimax-m2.7-highspeed`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'minimax/minimax-m2.7-highspeed',5  prompt:6    `Analyze the production alert logs from the last hour,7     correlate them with recent deployments, identify the8     root cause, and submit a fix with a non-blocking9     migration to restore service.`,10});
```

AI Gateway 提供统一的 API 用于调用模型、跟踪使用情况和成本，以及配置重试、故障转移和性能优化，以实现高于提供商级别的正常运行时间。它包含内置的可观测性、自带密钥支持，以及带自动重试的智能提供商路由。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
