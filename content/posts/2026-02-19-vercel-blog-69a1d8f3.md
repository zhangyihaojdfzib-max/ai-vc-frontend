通过阿里云的Wan模型生成风格化视频并转换现有素材，现已在AI Gateway平台开放使用。您可通过AI SDK 6或AI Gateway交互式平台选择相应模型立即体验。

Wan模型能够生成具有流畅动态的艺术视频，并能利用现有内容保持视频一致性：

- **角色参考（R2V）**：从参考视频/图像中提取角色外观与声音以生成新场景
- **极速变体**：更快的生成速度，支持快速迭代
- **灵活分辨率**：支持480p、720p与1080p输出

## 两种入门方式

视频生成功能处于测试阶段，目前面向专业版、企业版套餐及付费AI Gateway用户开放。

- **AI SDK 6**：通过编程方式生成视频，使用AI SDK 6的`generateVideo`接口

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'alibaba/wan-v2.6-t2v',5  prompt: 'Watercolor painting of a koi pond coming to life.',6});
```

- **Gateway交互平台**：在嵌入各模型页面的可配置AI Gateway交互平台中无需代码即可体验视频模型。您可以对比不同服务商、调整提示词并直接下载生成结果。访问方式：在模型列表中点击任意视频生成模型。

## 可用模型

| 模型 | 描述 |
|------|------|
| alibaba/wan-v2.6-t2v | 文生视频：根据文本提示生成视频 |
| alibaba/wan-v2.6-i2v | 图生视频：为静态图像添加动态效果 |
| alibaba/wan-v2.6-i2v-flash | 快速图像动画生成 |
| alibaba/wan-v2.6-r2v | 参考生视频：从参考内容中迁移角色 |
| alibaba/wan-v2.6-r2v-flash | 快速风格迁移 |
| alibaba/wan-v2.5-t2v-preview | 历史版本 |

## 基础应用：带音频的文生视频

根据文本描述生成风格化视频。

您可以通过详细提示词配合Wan模型指定风格以获得理想输出。以下示例使用`alibaba/wan-v2.6-t2v`：

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'alibaba/wan-v2.6-t2v',5  prompt:6   `Animated rainy Tokyo street at night, anime style,7    neon signs reflecting on wet pavement, people with umbrellas8    walking past, red and blue lights glowing through the rain.`,9  resolution: '1280x720',10  duration: 5,11});
```

## 进阶应用：参考生视频

利用从参考图像或视频中提取的角色生成新场景。

本示例使用2张狗狗参考图像生成最终视频：

![](/images/posts/38d76b0543a4.jpg)

使用`alibaba/wan-v2.6-r2v-flash`时，您可指示模型在提示词中调用特定人物/角色。Wan模型建议在多参考视频生成场景中使用`character1`、`character2`等标识符以获得最佳效果。

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'alibaba/wan-v2.6-r2v-flash',5  prompt:6   `character1 and character2 are playing together on the beach in San Francisco7    with the Golden Gate Bridge in the background, sunny day, waves crashing`,8  resolution: '1280x720',9  duration: 5,10  providerOptions: {11    alibaba: {12      referenceUrls: [shibaImage, yorkieImage],13    },14  },15});
```
