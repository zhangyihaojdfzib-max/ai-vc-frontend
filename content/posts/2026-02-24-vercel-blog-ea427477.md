GPT 5.3 Codex 现已登陆 AI Gateway。GPT 5.3 Codex 将 GPT-5.2-Codex 的编码优势与 GPT-5.2 的推理深度集于一身，形成单一模型，其速度提升 25%，且 Token 使用效率更高。

该模型专为长时间运行的智能体工作而构建，能够处理贯穿整个软件生命周期的研究、工具使用和多步骤执行任务，涵盖从调试、部署到产品文档和数据分析。此外，您可以在任务中途引导它，而不会丢失上下文。对于 Web 开发，它能更好地理解描述不充分的提示词，并默认生成更具功能性、更接近生产就绪的输出。

要使用此模型，请在 AI SDK 中将模型设置为 `openai/gpt-5.3-codex`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'openai/gpt-5.3-codex',5  prompt:6    `Research our current API architecture, identify performance7     bottlenecks, refactor the slow endpoints, add monitoring,8     and deploy the changes to staging.`,9});
```

AI Gateway 提供了一个统一的 API，用于调用模型、跟踪使用情况和成本，以及配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它包含内置的可观测性、自带密钥支持以及具有自动重试功能的智能供应商路由。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
