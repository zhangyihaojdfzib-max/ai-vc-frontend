Chat SDK 现在通过新的 `vendor-official` 适配器支持 Velt。

构建能够在 Velt 评论线程中阅读和回复的机器人，直接在您的团队已经工作的地方：文档、文本编辑器和画布。标记机器人后，它将在同一线程中回答，并以文档名称、URL 和锚定文本为基础进行回复。

```
1import { Chat } from "chat";2import { createVeltAdapter } from "@veltdev/chat-sdk-adapter";3
4const bot = new Chat({5  userName: "mybot",6  adapters: {7    velt: createVeltAdapter(),8  },9});10
11bot.onNewMention(async (thread, message) => {12  await thread.post(`You said: ${message.text}`);13});
```

配置适配器以响应 Velt 文档中的 @-提及

该适配器将 Velt 文档映射到 Chat SDK 频道，将评论注释映射到线程，将单个评论映射到消息。

阅读 Velt 文档以开始使用。
