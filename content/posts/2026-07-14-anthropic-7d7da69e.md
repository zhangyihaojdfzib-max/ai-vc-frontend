# 推出 Claude for Teachers（教师版）

![推出 Claude for Teachers（教师版）](/images/posts/044646b63e65.jpg)

我们推出 Claude for Teachers（教师版），为美国经过认证的 K-12 教育工作者提供免费访问高级 Claude 功能的权限、教学技能库，以及与循证课程的直接连接，这些课程已映射至全美 50 个州的学术标准。

## 我们为何为教师构建

数十年的研究表明，差异化教学、掌握式学习以及小组教学等实践能够可靠地提升学生成绩，但教师往往缺乏时间和资源来实施这些方法。预算紧张，班级规模可能过大而无法满足每位学生的个性化需求，备课工作常常延续到夜晚。在资源匮乏的学校，这种压力最为沉重。Claude for Teachers（教师版）旨在弥合教育最佳实践与教师每周可用时间之间的差距。

早期证据表明，虽然面向学生的人工智能工具效果参差不齐且取决于实施方式，但面向教师的人工智能工具可以强化教学实践并改善学生成果。这正是 Claude for Teachers（教师版）的目标：支持卓越教学背后的技艺，保护教师最珍视的东西——与学生相处的时间。

## 连接循证课程与 K-12 生态系统

Claude for Teachers（教师版）连接至 Learning Commons，使 Claude 能够访问全美 50 个州的学术标准——在每个标准之下，还有构成该标准的更细粒度学习能力以及学生通常掌握这些能力的顺序。因此，当 Claude 起草教案时，它是有支架支撑且符合教学标准的。Claude for Teachers（教师版）还引入了来自 Illustrative Mathematics 的 OpenSciEd 和 IM v.360 等可信课程资源。

截至目前，教育工作者现在可以将 Claude 连接至整个 K-12 工具生态系统。

- ASSISTments：生成自动评分、符合标准的数学题目，用于练习和评估。
- Brisk Teaching：创建互动学生活动和符合标准的课程，将您的想法在数秒内转化为课堂就绪材料。
- Canva Education：将课程材料转化为课堂就绪的设计和互动学习体验。
- Coteach：创建基于 K-12 课程的高质量数学图表。
- Diffit：为每位学生创建和调整教学材料。
- Eedi：生成诊断性问题，超越对错判断，揭示学生思维并深化洞察，支持英语和西班牙语。
- MagicSchool：使教学内容达到课堂就绪状态。
- Snorkl：获取关于您的班级、作业和学生进度的洞察，为您的教学提供信息。
- TeachFX：基于真实的课堂对话，获得个性化的教学反馈。

## 工作原理

一旦通过验证，美国的 K-12 教育工作者即可访问带有 Learning Commons 连接器的 Claude for Teachers（教师版），以及一套基于学习科学量身定制的教学技能。这些技能是与 Learning Commons 共同开发的，围绕教师告诉我们最重要的任务而设计。这些技能经过评估以确保严谨性、教学一致性和课堂实用性，并通过早期来自课堂教师（包括布鲁克林 Prospect Schools 等学校的教师）的反馈进行了优化。

- **从高质量教学材料中规划课程。** 提出课程需求，Claude 会利用广泛使用的课程（已映射至您所在州的标准）以及其下的细粒度学习组件和进度，然后起草一份计划和学生用材料，您可以修改后带入课堂。
- **为课堂中的每位学习者进行差异化教学。** 要求 Claude 为不同准备水平的学生调整材料，它会为您制定差异化计划，并为每个熟练水平的学生提供个性化学生用材料。支架使材料对学生更易理解，而准备充分的学生则可以接受进一步挑战。

Claude for Teachers（教师版）包含 Claude Code 和 Cowork，这意味着 Claude 可以自主推进工作。一些示例：

- **分析班级数据以规划教学。** 将包含数据（花名册、诊断结果、出勤记录、您自己的笔记）的文件夹交给 Claude，它会清晰呈现每位学生的情况，使教师能够更好地调整教学。您可以控制哪些数据被共享；您共享的任何数据都不会用于模型训练。
- **安排重复性任务。** 一次性交办一项任务——例如，查看每天的课堂反馈单以了解学生掌握情况，并调整第二天的计划以匹配——它会在每个上学日下午 4 点运行。当您开车回家时，Claude 仍在工作。

从两位教师 Zac 和 Karina 那里了解更多，他们解释了如何在各自的课堂中使用 Claude for Teachers（教师版）。

## 专为教育工作者打造，保障学生数据安全

Claude for Teachers（教师版）仅供教育工作者使用，符合 Claude 的 18 岁以上政策。它附带专为 K-12 隐私保护而构建的**教师条款**。Claude for Teachers（教师版）的数据不用于模型训练目的，学生信息受我们的 K-12 数据处理附录保护，该附录符合 FERPA 要求。

我们还在与美国教师联合会合作，使我们的条款和隐私实践与他们正在制定的黄金标准保持一致：

"我们一直与 Anthropic 合作制定一项黄金标准，为 K-12 教育中的安全性和隐私性设定行业最佳实践，"美国教师联合会主席 Randi Weingarten 表示。"重要的是，Anthropic 在其新的 Claude for Teachers（教师版）中承诺遵守这些原则——这是一款由教育工作者为教育工作者设计的工具，旨在为教学提供辅助，并有望为他们留出更多时间用于学习核心中的人际关系。"

您可以在此处阅读更多关于 Claude for Teachers（教师版）的 K-12 数据隐私标准。
