来自月之暗面的Kimi K2.7 Code现已在AI Gateway上线。

K2.7 Code是一款为长周期编程任务构建的编码模型，可泛化应用于前端开发、DevOps和性能优化等场景。该模型采用原生多模态架构，支持文本和视觉输入，并始终以思考模式运行。

要使用K2.7 Code，请在AI SDK中将模型设置为`moonshotai/kimi-k2.7-code`：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'moonshotai/kimi-k2.7-code',5  prompt: '重构此服务以消除N+1查询。',6});
```

在提示词旁传入图片以使用模型的多模态输入：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'moonshotai/kimi-k2.7-code',5  messages: [6    {7      role: 'user',8      content: [9        { type: 'text', text: '详细描述这张图片。' },10        {11          type: 'image',12          image:13            'https://exampleimage.com',14        },15      ],16    },17  ],18});
```

AI Gateway提供统一API用于调用模型、追踪使用量和成本，并配置重试、故障转移及性能优化，以实现高于提供商级别的可用性。它内置自定义报告、零数据留存支持、API密钥预算等功能。AI Gateway采用提供商定价且不加价，推理过程不收取平台费用，包括自带密钥（BYOK）请求。

了解更多关于AI Gateway的信息，查看AI Gateway模型排行榜，或在我们的模型游乐场中试用。
