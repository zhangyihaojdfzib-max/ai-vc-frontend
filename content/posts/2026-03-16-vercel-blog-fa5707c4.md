您现在可以在 Vercel 上部署 LiteLLM 服务器，为开发者提供通过 OpenAI 兼容网关访问任何受支持提供商（包括 Vercel AI 网关）的 LLM 能力。

```
1from litellm.proxy import proxy_server2app = proxy_server.app
```

基础的 LiteLLM 网关应用

要通过 Vercel AI 网关路由单个模型，请在 `litellm_config.yaml` 中使用以下配置：

```
1  - model_name: gpt-5.4-gateway2    litellm_params:3      model: vercel_ai_gateway/openai/gpt-5.44      api_key: os.environ/VERCEL_AI_GATEWAY_API_KEY
```

在 LiteLLM 中通过 Vercel AI 网关路由模型
