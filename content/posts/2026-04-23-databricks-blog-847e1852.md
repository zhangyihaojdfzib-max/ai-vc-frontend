Databricks 很高兴与 OpenAI 合作开发其最新前沿模型 GPT-5.5。GPT-5.5 是 OpenAI 在企业的 Agent（智能体）工作、复杂文档推理以及长周期编码 Agent（智能体）领域中最强大的前沿模型。GPT-5.5 现在也为 OpenAI 的编码 Agent（智能体）Codex 提供支持。

## GPT-5.5 功能与优势

GPT-5.5 是迄今为止最智能的前沿模型，也是迈向全新工作方式的下一个步骤。它能更快地理解你试图完成的任务，并能自行承担更多工作。OpenAI 的编码 Agent（智能体）Codex 现在由 GPT-5.5 驱动，为开发者工作流提供了更强的推理和执行能力。

使 GPT-5.5 在编码方面表现出色的相同优势，也使其在计算机上的日常工作中同样强大。由于该模型能更好地理解意图，它能够更自然地完成知识工作的完整闭环：查找信息、理解关键内容、使用工具、检查输出结果，并将原始材料转化为有用的成果。

它可以编写和调试代码、在线研究、分析数据、创建文档和电子表格、操作软件，并在不同工具之间切换，直到任务完成。你无需精心管理每一步，只需向 GPT-5.5 提供一个杂乱、多步骤的任务，并信任它去规划、使用工具、检查工作、从歧义中恢复并持续推进。

## GPT-5.5 树立了最先进的性能标杆

为了理解这些改进如何转化为真实的企业工作负载，我们在 OfficeQA 上评估了 GPT-5.5。OfficeQA 是 Databricks 的基准测试，针对客户日常执行的、以文档为中心的多步骤分析任务。OfficeQA 基于 89,000 页的美国财政部公报构建，衡量模型跨文档检索信息、解读复杂表格以及基于真实企业数据进行精确计算的能力。

当提供正确的文档时（OfficeQA Pro LLM with Oracle PDF + Web Search），GPT-5.5 得分为 64.66%，较 GPT-5.4 的 57.14% 有显著提升，实现了约 13% 的改进，并在该基准测试上达到了新的最先进水平。这测试了当检索已处理完毕时模型能力的上限。在完整的 Agent（智能体）工作流评估中（OfficeQA Pro Agent Harness），模型必须自行找到正确的文档、解析它们并使用 Codex Agent（智能体）框架计算答案，GPT-5.5 得分为 52.63%，高于 GPT-5.4 的 36.10%。这相当于错误率降低了 46%，表明 GPT-5.5 的提升并非只是理论上的；它们在真实、端到端的企业工作流中同样有效。
