- shridharathi

**太长不看版：** Ideogram 3.0 是文生图模型的一次重大更新，在真实感、风格控制和布局生成方面都有巨大改进。它提供三个版本，现已在 Replicate 上线。

- Ideogram 3.0 Turbo
- Ideogram 3.0 Balanced
- Ideogram 3.0 Quality

## 关于 3.0 版本

Ideogram 刚刚发布了其文生图模型的 3.0 版本，功能丰富。该模型提供三个变体：“Turbo”、“Balanced”和“Quality”。

“Turbo”模型非常适合快速迭代，而“Quality”模型则在精度至关重要时提供最高保真度的结果。“Balanced”选项则介于两者之间，为大多数用例提供了一个很好的折中方案。

所有这些模型都能生成细节丰富的图像结果，提供更强的控制力、更好的文本渲染能力，并显著提升了真实感。

以下是使用 Replicate 的 JS 客户端运行这些模型的方法：

```
import { writeFile } from "fs/promises";
import Replicate from "replicate";
const replicate = new Replicate();
```

```
const input = {
    prompt: "The text \"V3 Turbo\" in the center middle.",
    aspect_ratio: "3:2",
};
const turbo_output = await replicate.run("ideogram-ai/ideogram-v3-turbo", { input });
await writeFile("turbo_output.png", output);
```
