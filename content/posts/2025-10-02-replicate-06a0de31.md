# IBM的Granite 4.0现已登陆Replicate

IBM发布了Granite 4.0，这是其最新系列专为速度和低成本打造的开源小语言模型。

Granite 4.0模型采用混合架构，比传统模型占用更少内存，因此您可以在普通的消费级GPU上运行它们，而无需昂贵的服务器硬件。它们在文档摘要、RAG（检索增强生成）系统和AI Agent（智能体）方面表现良好。

`ibm-granite/granite-4.0-h-small`是一个拥有300亿参数的长上下文指令模型，现已可在Replicate上使用。

## 通过API运行Granite 4.0

您可以立即在Replicate上开始使用Granite模型。以下是如何通过API运行它们：

```
curl -s -X POST \
  -H "Authorization: Bearer $REPLICATE_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d $'{
    "version": "ibm-granite/granite-4.0-h-small",
    "input": {
      "messages": [
        {
          "role": "user",
          "content": "Explain the key benefits of using open-source models in business environments"
        }
      ]
    }
  }' \
  https://api.replicate.com/v1/predictions
```

以下是使用Replicate的JavaScript客户端的示例：

```
import Replicate from "replicate";

const replicate = new Replicate({
  auth: process.env.REPLICATE_API_TOKEN,
});

const output = await replicate.run(
  "ibm-granite/granite-4.0-h-small",
  {
    input: {
      messages: [
        {
          role: "user",
          content: "Explain the key benefits of using open-source models in business environments"
        }
      ]
    }
  }
);
```

以下是使用Replicate的Python客户端的示例：

```
import replicate

output = replicate.run(
    input={
        "messages": [
            {
                "role": "user",
            }
        ]
    }
)
```

# Granite性能卓越

Granite模型围绕混合设计构建，结合了两个关键思想：Mamba-2的线性扩展效率与Transformer的精确性。

Mamba-2是一种状态空间模型，能够线性处理序列，这与传统Transformer随序列长度呈二次方扩展不同。这使得它在处理非常长的输入（例如包含数十万个Token的文档）时效率更高。Transformer模块通过更好地支持需要长上下文推理的任务来对此进行补充。

部分Granite 4.0模型还采用了MoE（专家混合）路由策略。MoE设置将模型分割成多个“专家”。模型不会一次性运行所有参数，而是将每个输入仅路由到它实际需要的专家进行处理。例如，Granite 4.0 Small总共有320亿参数，但一次推理请求仅激活其中的90亿参数。

这两种方法相结合，使得Granite模型能够快速处理长上下文，并在更普通的硬件（如消费级GPU）上运行，而不会牺牲性能。

Granite模型是为实际工作而设计的，不仅仅是演示。它们轻量且高效，非常适合以下场景：

- 总结长文档，如合同或技术手册。
- 构建从大型数据集（如CRM或知识库）中提取答案的系统，而无需将输入切分成小块。
- 为复杂工作流同时运行多个AI Agent（智能体）。
- 在本地设备或边缘硬件上部署模型，这些场景下带宽或云访问受限。

## Granite是开源的

Granite模型基于Apache 2.0许可证发布。这意味着您可以不受限制或隐藏费用地将其用于商业和非商业项目。您还可以按需修改模型——进行微调、添加适配器或在私有数据集上训练——并按照您自己的条款发布这些更改。这种开放性使得Granite成为需要合规性、安全性或定制化的公司的实用选择。

欲了解更多详情，请查阅IBM关于部署、微调和集成模式的文档。如果您正在使用LangChain，IBM还构建了一个LangChain与Replicate的集成，以便更轻松地使用Granite模型。
