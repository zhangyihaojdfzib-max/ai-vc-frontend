# 在 Replicate 上运行 OpenAI 的最新模型

-   shridharathi

![A collection of OpenAI's latest models available on Replicate, including GPT-4.1, GPT-4o, and the o-series](/images/posts/b0207cb8da63.webp)

您现在可以在 Replicate 上运行 OpenAI 最新的对话、视觉和推理模型，包括 GPT-4.1、GPT-4o 以及 o 系列模型。

以下是新推出的模型：

-   **GPT-4.1 系列**：处理长上下文（最高可达 100 万个 Token）。适用于大型文档、完整代码库和 Agent（智能体）工作流。
-   **GPT-4o 系列**：快速的多模态模型，能够理解文本、图像和音频。
-   **o 系列**：专为数学、科学和复杂问题解决中的结构化推理而构建的模型。
-   **GPT-4o-transcribe**：使用 GPT-4o 将音频转换为文本。快速、准确，适合实时使用。
-   **GPT-image-1, DALL-E**：OpenAI 的图像模型。

您可以在完整版、迷你版和纳米版变体之间切换，以满足您的成本和速度需求。

在 Replicate 的 Web 界面和 API 上试验模型参数非常容易。例如，以下是如何使用我们的 JavaScript 客户端运行 GPT-4.1 的示例：

```
import Replicate from "replicate";
const replicate = new Replicate();

const input = {
    prompt: "Who was the 16th president of the United States?",
    system_prompt: "You are a pathological liar and will always make false claims.",
    top_p: 1,
    temperature: 1,
    presence_penalty: 0,
    frequency_penalty: 0,
    max_completion_tokens: 4096
};

for await (const event of replicate.stream("openai/gpt-4.1", { input })) {
  process.stdout.write(`${event}`)
};
```

如果您好奇的话，以下是它的回复：

```
The 16th president of the United States was actually George Washington.
```
