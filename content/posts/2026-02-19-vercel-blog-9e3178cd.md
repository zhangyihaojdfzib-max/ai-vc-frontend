Kling视频模型现已上线AI Gateway，包含最新的Kling 3.0系列模型。通过Kling先进的视频模型，您可以从文本、图像或动作参考生成电影级视频，现可通过AI Gateway和AI SDK使用。

Kling模型以其图像生成视频能力和多镜头叙事功能著称：

- **图像生成视频能力**：擅长将静态图像动画化为视频片段
- **逼真运动与物理模拟**：以连贯的动作、面部表情和物理交互表现闻名
- **高分辨率输出**：支持最高1080p生成（专业模式）
- **多镜头叙事**：Kling 3.0可根据单一叙事提示词生成多场景视频
- **音频生成**：为视频创建同步音效与环境音频
- **首尾帧控制**：可指定起始帧与结束帧，实现精准场景转场

## 两种入门方式

视频生成功能处于测试阶段，目前面向专业版、企业版套餐及付费AI Gateway用户开放。

- **AI SDK 6**：通过编程方式生成视频，使用AI SDK 6的`generateVideo`接口

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'klingai/kling-v2.6-t2v',5  prompt: 'A chef plates a dessert with caramel drizzle. Kitchen ambiance.',6});
```

- **Gateway Playground**：在嵌入各模型页面的可配置AI Gateway Playground中，无需代码即可体验视频模型。可对比不同供应商、调整提示词、下载生成结果。访问方式：点击模型列表中的任意视频生成模型。

## 基础示例：带音频的文生视频

根据文本描述生成视频。

此示例使用`klingai/kling-v3.0-t2v`模型，仅通过简单文本提示词生成樱花树视频。

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'klingai/kling-v3.0-t2v',5  prompt:6   `Cherry blossom petals falling in slow motion through golden sunlight,7    Japanese garden with a stone lantern, peaceful atmosphere, cinematic`,8  aspectRatio: '16:9',9  duration: 5,10  providerOptions: {11    klingai: {12      mode: 'pro',13    },14  },15});
```

## 进阶示例：多镜头视频

仅通过单条提示词生成包含多个场景的叙事视频。利用Kling 3.0的多镜头功能，模型可智能切换镜头以讲述完整故事：

提示词需编写为包含多个独立场景的叙事文本以获得最佳效果。`shotType: 'intelligence'`让模型自主决定最优镜头构图，`sound: 'on'`为整段视频生成同步音频。请注意，由于此功能为Kling特有，提示词需置于`providerOptions`中。Kling 3.0系列模型支持此功能，本例使用`klingai/kling-v3.0-t2v`。

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'klingai/kling-v3.0-t2v',5  prompt: '',6  aspectRatio: '16:9',7  duration: 10,8  providerOptions: {9    klingai: {10      mode: 'pro',11      multiShot: true,12      shotType: 'intelligence',13      prompt:14       `Elephants walk across a golden savanna under gathering storm clouds.15        Lightning cracks in the distance. Rain begins to fall heavily.16        The herd finds shelter under acacia trees.17        The storm clears revealing a double rainbow.`,18      sound: 'on',19    },20  },21});
```

## 进阶示例：首尾帧控制

通过提供首帧和尾帧图像，精确控制视频的起始与结束画面。此功能特别适用于延时效果或精准场景转场：

这两张图像被设定为起始帧与结束帧。

![](/images/posts/5e41ecce3bb8.jpg)

使用AI SDK 6时，可通过`image`和`lastFrameImage`参数设置首尾帧。此示例使用`klingai/kling-v3.0-i2v`模型。

```
1import { experimental_generateVideo as generateVideo } from 'ai';2
3const { videos } = await generateVideo({4  model: 'klingai/kling-v3.0-i2v',5  prompt: {6    image: startImage,7    text: `Time-lapse of a pink peony flower blooming.8     The tight bud slowly unfurls, petals gently separating and opening outward.9     Smooth organic movement. Soft natural lighting.`,10  },11  duration: 10,12  providerOptions: {13    klingai: {14      lastFrameImage: endImage,15      mode: 'pro',16    },17  },18});
```
