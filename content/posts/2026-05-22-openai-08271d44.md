# OpenAI 现已获得 FedRAMP Moderate 授权

为政府扩展安全 AI。

OpenAI 已为 ChatGPT Enterprise 和 API 平台获得 FedRAMP 20x Moderate 授权⁠（在新窗口中打开），这标志着在向美国政府机构提供前沿 AI 方面迈出了重要一步，满足了联邦工作所需的安全、隐私和治理期望。

公务员不应等待安全访问那些正在改变其他经济领域的相同先进 AI 能力。各机构已经在利用 AI 来加速审批、起草居民通信、推进前沿科学、总结复杂信息、支持公共卫生分析、加速软件开发、翻译服务，并帮助员工在政策和项目材料中寻找答案。此次 FedRAMP 20x Moderate 授权扩大了可以使用 OpenAI 托管产品的任务范围，具体取决于各机构的政策和授权决定。

得益于 FedRAMP 20x，这一里程碑无需在速度和严谨性之间做出选择。20x 流程将授权转向云原生安全证据、关键安全指标、自动化验证以及对云服务产品运营方式的持续可见性。

## 我们如何实现这一目标

FedRAMP 20x 由 GSA 于 2025 年 3 月宣布，为云服务提供商展示安全配置和实践创建了更快的路径。OpenAI 的安全和工程团队通过 KSI 实施、证据收集、验证、审查周期和评估材料，将 ChatGPT Enterprise 和 API 平台带入了 20x Moderate 路径。

我们与 FedRAMP 团队的密切合作对于实现这一里程碑至关重要。他们的合作帮助将 FedRAMP 20x 的承诺转化为一条实用、以安全为重点的授权路径，其速度符合现代政府技术的要求。

## FedRAMP Moderate 扩展政府能力

各机构将不再需要在尖端 AI 和可信部署环境之间做出选择。对于联邦机构而言，FedRAMP 20x Moderate 为采用 OpenAI 的托管 AI 产品用于内部、运营和任务支持用例开辟了更清晰的路径。项目团队可以使用 ChatGPT Enterprise 来加速研究、起草、翻译、分析和知识工作。技术团队可以使用 OpenAI API 将 AI 功能构建到现有系统、副驾驶、案件管理工具和公民服务工作流中。各机构现在可以在我们的 FedRAMP 环境中访问我们最强大的模型，包括 GPT‑5.5。此外，各机构很快可以通过其 FedRAMP ChatGPT Enterprise 工作区访问其 Codex Cloud 环境，并通过与 FedRAMP 账户管理和后端基础设施的集成来使用 Codex 应用程序。

对于安全、隐私和采购团队而言，该授权为机构审查创建了可重复使用的 FedRAMP 授权数据。各机构可以在 OpenAI 的信任门户⁠（在新窗口中打开）中评估云服务产品、其最低评估范围、共担责任期望、支持的功能以及支持证据和验证材料，而无需从头开始。

## 如何参与

各机构可以在 FedRAMP 市场⁠（在新窗口中打开）中找到 ChatGPT Enterprise 和 API 平台，并通过 fedramp@openai.com⁠ 联系 OpenAI 以获取软件包访问权限。各机构可以直接与 OpenAI 接洽，通过 OpenAI 授权的公共部门经销商 Carahsoft 进行采购，或评估其他符合其要求的采购途径。OpenAI 可以帮助各机构确定合适的部署和采购路径；团队可以使用我们的销售表格⁠（在新窗口中打开）、发送邮件至 gov-gtm@openai.com，或与他们的 OpenAI 客户团队合作以获得支持。

OpenAI 将继续使用重大变更通知流程扩展支持的功能，缩小 FedRAMP 与商业产品体验之间的差距，并与政府客户合作，以公共部门任务所需的控制、安全和保证来提供前沿 AI。
