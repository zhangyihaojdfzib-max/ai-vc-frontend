Gel Data 团队将加入 Vercel，以帮助我们投资 Python 生态系统。在 Python 核心开发者 **Yury Selivanov** 和贡献者 **Elvis Pranskevichus** 的带领下，他们将为 AI Cloud 带来世界级的 Python 支持。

## 对 Python 社区的承诺

Python 是全球使用最广泛的编程语言之一，驱动着复杂的后端、机器学习管道、数据基础设施，并且越来越多地应用于 AI 领域。我们的客户希望在 Vercel 上获得强大的 Python 支持，而随着我们持续构建 AI Cloud，我们也希望 Python 能变得对所有人更好。因此，我们正在投资生态系统本身：

*   **Python 软件基金会赞助**：我们将以“维护级”赞助商的身份加入，这包括在美国 PyCon 大会上的亮相，以及支持 PSF 推动 Python 语言和社区发展的使命。
*   **赞助一位 Python 核心维护者**：我们承诺通过 Python 软件基金会赞助 **Serhiy Storchaka** 一年。Serhiy 是过去十年中按提交量排名前三的 Python 核心开发者之一，其贡献涵盖核心解释器、标准库改进和性能优化。
*   **会议和社区支持**：我们承诺赞助关键的 Python 会议和本地聚会，帮助确保社区拥有聚集、构建和学习的资源与空间。我们计划于二月在旧金山举办首次 Vercel + Python 黑客松。详情即将公布，请密切关注我们的活动日历。

## uvloop 和 asyncpg 背后的团队

Yury 是 **uvloop** 的创建者，这是一个高性能的 Python 事件循环，每日下载量达 300 万次。它是 **FastAPI** 以及许多其他需要速度和可靠性的 Python 应用程序的基础。他还为 Python 贡献了 async/await 语法以及其他七项 Python 增强提案，并在 asyncio 成为 Python 标准库的一部分之前就影响了其设计。

Elvis 和 Yury 还共同创建了 **asyncpg**，这是一个深度嵌入 Python 生态系统的 PostgreSQL 库。

多年来，他们共同致力于为全球开发者推动生态系统向前发展。他们将继续为开源 Python 项目做出贡献，同时帮助我们在 Vercel 上为 Python 开发者构建强大的支持。

## 这对 Vercel 意味着什么

随着我们持续为下一代 Web 应用程序和 AI Agent（智能体）构建基础设施，我们正在超越最初的 JavaScript 和 TypeScript 根基。

在接下来的几个月里，我们将在 Vercel 上投资 Python 支持。这包括：

*   改进 Python 框架支持
*   使 Python 部署像 JavaScript 框架一样简单快捷
*   回馈社区所依赖的工具和库

我们采取“公开构建”的方式。当我们在 Vercel 上发布对 Python 支持的改进时，我们将分享我们的学习成果以及我们处理这些问题的方法。我们也希望在此过程中听到您的反馈。这仅仅是个开始，我们迫不及待地想看到 Yury、Elvis 和团队将构建出什么。

基于 Postgres 的数据平台 Gel Data 将于 2026 年 1 月 31 日停止服务。我们不会将该平台商业化，但其开源组件将继续可用。此次收购的重点在于 Python 专业知识和社区投资，而非进入数据库市场。我们通过 Vercel 市场与优秀的数据库提供商合作，并且没有改变这一计划的打算。

## 为长远而构建

这是我们承诺支持 Python 的第一步。我们致力于为生态系统做出贡献，支持那些使其变得伟大的人们，并帮助 Python 变得更强大。

我们相信最好的基础设施建立在**开放的基础**之上。无论是像 Next.js、Nuxt 和 SvelteKit 这样的 JavaScript 框架，还是现在的 Python 工具和库，我们的角色都是支持开发者已经喜爱的东西并使其变得更好。

我们很荣幸欢迎 Yury、Elvis 和 Gel Data 团队加入 Vercel。我们也很高兴能与 Python 社区一起开始构建。为了开放的 Web。
