# 将您的ChatGPT体验延续至WhatsApp之外

现在关联ChatGPT账户，让对话继续

![将您的ChatGPT体验延续至WhatsApp之外](/images/posts/319475d210e9.png)

我们非常高兴地看到超过5000万用户在WhatsApp上通过ChatGPT聊天、创作和学习。消息传递的简洁性和熟悉感使其成为日常创意和好奇心的天然家园。

遗憾的是，由于WhatsApp的政策和条款变更，ChatGPT将于2026年1月15日后不再在WhatsApp上可用。虽然我们更愿意继续在WhatsApp上为您服务，但我们正专注于让所有用户尽可能顺利地完成过渡。您可以在ChatGPT中继续您的对话——ChatGPT可在iOS、Android、网页以及MacOS上的ChatGPT Atlas中使用——您还将发现语音对话、深度研究和文件上传等更多功能。

如何在ChatGPT中继续聊天
1. 在Android、iOS或桌面端下载ChatGPT应用⁠(opens in a new window)。
2. 创建ChatGPT账户⁠(opens in a new window)并登录。
3. 将您的账户与WhatsApp关联：您可以通过WhatsApp中1-800-ChatGPT的联系人资料，点击其中的URL来关联您的ChatGPT账户。

关联后，您的手机号码将与您的ChatGPT账户绑定，您过去的WhatsApp对话将出现在您的ChatGPT历史记录中。

我们建议您尽快关联账户以保留您的历史记录。2026年1月15日后，您的WhatsApp对话将不会自动转移。WhatsApp不支持聊天记录导出。ChatGPT将在该日期前继续在WhatsApp上可用，我们会在此期间发送提醒。

如果您需要帮助，请访问我们的帮助中心⁠(opens in a new window)获取详细步骤。
