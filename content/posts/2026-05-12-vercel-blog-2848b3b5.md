Claude Opus 4.7 的快速模式现已在 AI Gateway 上以研究预览形式提供。

快速模式在保持完整 Opus 4.7 智能的同时，输出 Token 生成速度提升约 2.5 倍。这是一项早期的实验性功能。

要启用快速模式，请在 anthropic 提供商选项中传递 `speed: 'fast'`，并使用 `anthropic/claude-opus-4.7`。

```
import { streamText } from "ai";
const { text } = await streamText({  model: "anthropic/claude-opus-4.7",  prompt: "分析此代码库结构并制定添加用户认证的计划。",  providerOptions: {    anthropic: {      speed: "fast",    },  },});
```

您可以通过 AI Gateway 在 Claude Code 中使用快速模式，方法是在您的 shell 配置文件或 `~/.claude/settings.json` 中设置 `CLAUDE_CODE_SKIP_FAST_MODE_ORG_CHECK` 和 `CLAUDE_CODE_ENABLE_OPUS_4_7_FAST_MODE` 变量。

```
export CLAUDE_CODE_ENABLE_OPUS_4_7_FAST_MODE=1export CLAUDE_CODE_SKIP_FAST_MODE_ORG_CHECK=1
```

```
{  "env": {    "CLAUDE_CODE_SKIP_FAST_MODE_ORG_CHECK": "1",    "CLAUDE_CODE_ENABLE_OPUS_4_7_FAST_MODE": "1"  }}
```

快速模式的定价为标准 Opus 费率的 6 倍。
