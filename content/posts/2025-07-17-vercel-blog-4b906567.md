Grepnow 支持模型上下文协议（MCP），使 AI 应用能够通过标准接口查询上百万个公共 GitHub 仓库。无论您是在 Cursor 中构建、使用 Claude，还是集成您自己的 Agent（智能体），Grep 现在都可以作为一个可通过 HTTP 访问的可搜索代码索引。

## 什么是 Grep MCP 服务器

MCP 是一种向大语言模型（LLM）暴露工具的协议。Grep 的新 MCP 服务器提供了一个用于搜索公共 GitHub 仓库的端点。通过 Grep MCP 服务器，AI Agent（智能体）可以发出搜索查询，并检索匹配特定模式或正则表达式的代码片段，这些结果可按语言、仓库和文件路径进行过滤。

它由与 grep.app 相同的基础设施支持。结果通常在几分之一秒内返回，并且代码片段会根据相关性进行排序。

## 如何在您的 AI 客户端中配置它

设置 MCP 服务器通常很简单。一旦您的客户端知晓 MCP 端点，它就可以内省可用的工具并直接调用它们。每个工具都在一个机器可读的模式中定义，这使得 Agent（智能体）和应用的集成变得可预测。
