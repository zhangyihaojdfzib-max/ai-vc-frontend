您现在可以为 Vercel Blob 生成带时限的签名 URL。签名 URL 是一种具有过期时间的限定范围 URL，允许您上传、下载、检查或删除特定对象，而无需授予对整个 Blob 存储的访问权限。

每个 URL 限定于单一操作（put、get、head 或 delete）、单一路径名以及您选择的过期时间（最长 7 天）。签名覆盖了操作和约束条件，因此为 GET 签名的 URL 不能重用作 PUT。

```
1import { issueSignedToken, presignUrl } from '@vercel/blob';2
3const token = await issueSignedToken({4  operations: ['get'],5});6
7const { presignedUrl } = await presignUrl(token, {8  pathname: 'invoices/2026-q1.pdf',9  operation: 'get',10  validUntil: Date.now() + 5 * 60 * 1000, 11});12
1314<img src={presignedUrl} />
```

签发一个 Token，生成一个 5 分钟的读取 URL，让浏览器直接渲染对象。

## 从浏览器直接上传

上传 URL（put）支持多部分传输，因此浏览器可以将大文件直接流式传输到 Blob 存储，无需经过服务器中转。

```
1import { presignUrl } from '@vercel/blob';2
3const { presignedUrl } = await presignUrl(token, {4  pathname: 'user-uploads/avatar.png',5  operation: 'put',6  validUntil: Date.now() + 15 * 60 * 1000,7});8
910await fetch(presignedUrl, { method: 'PUT', body: file })
```

生成一个 15 分钟的上传 URL，让浏览器将文件直接写入 Blob。

## 条件删除

删除 URL 接受 ifMatch 选项，因此删除操作仅在对象自您签名 URL 以来未被覆盖时生效：

```
1import { presignUrl } from '@vercel/blob';2
3const { presignedUrl } = await presignUrl(token, {4  pathname: 'tmp/session.json',5  operation: 'delete',6  validUntil: Date.now() + 60 * 1000,7  ifMatch: '"a1b2c3"', 8});9
1011await fetch(presignedUrl, { method: 'DELETE' })
```

如果自您签名 URL 以来 ETag 已更改，则删除操作不执行任何操作。

签名 URL 与 OIDC 配合使用。您的服务器通过 OIDC 向 Blob 进行身份验证，生成签名 Token，并为浏览器生成严格限定范围、带时限的 URL，这样您长期有效的 BLOB_READ_WRITE_TOKEN 永远不会离开服务器。

将 @vercel/blob 更新至 2.4.0 版本，并阅读文档以开始使用。
