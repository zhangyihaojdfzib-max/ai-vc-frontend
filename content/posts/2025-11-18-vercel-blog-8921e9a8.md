今日发布的 Gemini 3 Pro Preview 模型现已通过 Vercel AI Gateway 和 v0.app 提供。感谢 Google，Vercel 在过去几周内已在 v0、Next.js、AI SDK 和 Vercel Sandbox 上对 Gemini 3 Pro Preview 进行了测试。

我们注意到，该模型在编码、多模态推理和工具使用方面的关注度有所提升，尽管其各方面均有改进。

根据我们的测试，Gemini 3 Pro Preview 在遵循指令和响应一致性方面带来了显著改进。在我们的 Next.js 评估中，其正确率较前代模型提升了近 17%，跻身排行榜前两名。

### 我们在 Vercel 如何使用 Gemini 3 Pro Preview

在 Vercel 的 Next.js 评估中，Gemini 3 Pro Preview 与 openai/gpt-5-codex 并列，成功率为 42%，在一致性和推理质量上超越了 anthropic/claude-opus-4.1、zai/glm-4.6、moonshotai/kimi-k2-turbo 等模型。

Gemini 3 Pro Preview 在前端生成和多轮 Agent（智能体）流程中表现突出。它能高精度地遵循开发者级别的提示词，生成稳定、结构良好的 UI 组件。其 100 万 Token 的上下文窗口有助于在长生成链中保持保真度。我们在 v0 的测试中，观察到生成速度和质量均有提升。
