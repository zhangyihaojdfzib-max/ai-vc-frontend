您现在可以直接通过 Stripe Projects 使用共享支付令牌（SPT）注册或升级到 Vercel Pro 套餐。Agent（智能体）和开发者无需离开工作流程，即可通过 Stripe CLI 以编程方式管理套餐变更。

### 链接到标题 新增功能

- 直接从 Stripe CLI 配置或升级到 Vercel Pro
- 支持升级和降级流程
- 基于共享支付令牌，实现安全、简化的计费

直接从 Stripe CLI 配置或升级到 Vercel Pro

支持升级和降级流程

基于共享支付令牌，实现安全、简化的计费

这一功能基于我们在开发者预览版中推出的 Stripe Projects，实现了端到端的配置和计费一体化。您无需在不同控制面板之间切换，现在可以直接在终端中处理基础设施设置和套餐管理。

### 链接到标题 快速开始

如果您已经在使用 Stripe Projects 并通过 `stripe projects billing add` 设置了计费，只需运行 `stripe projects add vercel/pro` 即可从 CLI 升级您的 Vercel 套餐。

如果您是 Stripe Projects 的新用户，请安装插件并初始化您的项目：

```
1stripe plugin install projects 2
3stripe projects init my-app 4
5stripe projects add vercel/pro
```
