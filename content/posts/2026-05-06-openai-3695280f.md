# Canva

Canva 的 AI 驱动 Magic Studio 已被使用 50 亿次，且仍在增长。

Canva 是一个视觉沟通平台，每月有超过 1.75 亿人使用它来制作演示文稿、视频、文档、网站、社交媒体图形等。全球大多数知识工作者缺乏设计培训，但 Canva 结合了易用的界面、庞大的素材库和节省时间的工具，使任何人都能创建视觉上引人注目的内容。

## 用 AI 开创全新的创作流程

Canva 相信 AI 可以改变并加速创作流程，使设计对所有技能水平的人来说都更加直观和快速。为了将 AI 快速交到用户手中，Canva 选择 OpenAI 作为基础模型，为其 AI 套件 Magic Studio 开发多项工具。

Canva 的目标是让 Magic Studio 成为一个统一的创意套件，将可用的最佳 AI 技术整合到单一平台中。虽然 Canva 多年来一直在其产品中利用 AI，但集成生成式 AI 工具是一个微妙的挑战，因为这需要在增加更多功能的同时不使产品变得复杂。毕竟，Canva 的声誉建立在为即使是非专业设计师也能提供简单方式制作高质量内容的基础上。

Magic Studio 必须确保各种模态——文本、图像、视频——能够有效协同工作，而不会让用户感到不知所措。OpenAI 的 GPT‑4 擅长将多模态整合到单一工作流中，使 Canva 能够在保持直观用户体验的同时，提供先进的 AI 产品选项。

"提供强大且安全的 AI 技术对于 Canva 推动设计世界发展的能力至关重要。OpenAI 使我们能够快速加速 Canva 的 AI 愿景，"Canva AI 产品负责人 Danny Wu 表示。

## 构建 Magic Studio，一个集成的 AI 套件

利用 GPT‑4 的创作能力驱动 Magic Studio 中的多项工具：

- Magic Write 是一个由 OpenAI API 驱动的 AI 写作工具。除了根据提示词生成文本外，Magic Write 还经过优化，支持写作过程的每一步，包括大纲撰写、段落生成、重写、释义、语法检查和总结。使用 Magic Write，跨语言和内容类型的用户已撰写了超过 100 亿个单词。
- Magic Design，Canva 的设计生成工具，将 OpenAI API 与 Canva 自身的 AI 设计引擎以及超过 1 亿个素材和模板的素材库相结合。要生成演示文稿、社交媒体帖子和视频，用户只需输入他们想要创建内容的提示词。
- Magic Switch 一键即可将一种设计转换为无数其他格式。Canva 利用 OpenAI API 和视觉技术来帮助解析和理解内容设计，从白板到演示文稿。Magic Switch 可以总结、翻译、重新排列，甚至将其转换为另一种格式，如电子邮件或歌词。用户只需描述他们想要的内容，例如"将此海报转换为法文演示文稿"或"将此文档改为横向"。而 OpenAI 的 GPT‑4 模型支持超过 100 种语言的即时翻译。

## 通过协作激发创造力

Canva 的内部 AI 开发团队知道，通过与 OpenAI 直接协作，他们可以走得更远、更快。他们希望确保构建的 AI 工具既遵守客户的信任和安全指导原则，又为创新打开大门。

OpenAI 强大的安全实践和控制措施，加上两个平台各自信任与安全团队的协作，使 Canva 有信心部署安全且负责任的 AI。"我们非常尊重 OpenAI 的使命及其章程，"Wu 表示。

Canva 与 OpenAI 的紧密合作不仅限于 Magic Studio。原生 DALL-E 应用使得在 Canva 中直接生成高质量图像变得极其简单。或者，如果你在 ChatGPT 中，Canva GPT 使得生成视觉资产就像对话一样简单，只需描述你想要创建的内容。这个自定义 GPT 已被证明是 Canva 获取新用户的强大来源。

## 更多进展即将到来

迄今为止，全球用户使用 Canva AI 产品的次数已超过 50 亿次。

早在 2018 年，Canva 就开始设想 Magic Studio 是只有"遥远未来"的技术才能实现的东西。Canva 和 OpenAI 的进步使这一愿景比预期更快成为现实。然而，对于他们迄今取得的成就，Wu 认为，就 AI 设计生成的可能性而言，"我们只走了 1% 的路。"
