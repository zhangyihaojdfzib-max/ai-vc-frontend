Edge Config 读写操作在 Pro 计划中正从基于套餐的定价转向按单位计费。您将继续支付相同的实际费率，但从下一个计费周期开始，将按使用量直接计费，使成本与您的使用情况直接挂钩。

新的费率如下：

- Edge Config 读取：每次读取 $0.000003（原为每 100 万次读取 $3）
- Edge Config 写入：每次写入 $0.01（原为每 500 次写入 $5）

Edge Config 读取：每次读取 $0.000003（原为每 100 万次读取 $3）

Edge Config 写入：每次写入 $0.01（原为每 500 次写入 $5）

按单位计费能更平滑地适应不同团队规模和使用模式。这也帮助 Pro 计划团队使用 Edge Config 时，不会立即消耗掉月度包含使用额度的大部分。
