# Anthropic 收购 Stainless

AI 的前沿正在从回答问题的模型转向能够行动的 Agent（智能体）——而 Agent（智能体）的能力取决于它们能够连接的系统。今天，Anthropic 收购了 SDK 和 MCP 服务器工具领域的领导者 Stainless，以进一步扩展这一连接能力。

Stainless 成立于 2022 年，自我们 API 推出之初就为所有官方 Anthropic SDK 的生成提供支持。数百家公司依赖 Stainless 生成 SDK、CLI 和 MCP 服务器——这些库、命令行工具和连接器让开发者和 Agent（智能体）能够使用 API。Stainless 将 API 规范转化为 TypeScript、Python、Go、Java、Kotlin 等多种语言的 SDK。每一个 SDK 都快速、可靠，并且在其语言中具有原生体验。

"Stainless 从一开始就塑造了开发者体验 Claude API 的方式，与他们合作非常愉快，"Anthropic 平台工程负责人 Katelyn Lesse 表示。"Agent（智能体）的价值取决于它们能连接什么。我们很高兴将 Stainless 团队引入 Anthropic，以提升 Claude 连接数据和工具的能力。"

"我创办 Stainless 是因为 SDK 应该得到与它们所封装的 API 同等的关注。Anthropic 是最早与我们共同押注这一理念的团队之一，"Stainless 创始人兼 CEO Alex Rattray 表示。"过去几年，我们一直在关注开发者基于 Claude 构建的成果，这使得我们团队的合并成为一个容易的决定。团队可以继续做我们热爱的工作，在最重要的平台上。"

Anthropic 创建了 MCP 以实现 Agent（智能体）的连接能力。通过整合 Stainless 和 Anthropic 团队，Claude 平台持续推动开发者体验和 Agent（智能体）连接能力的前沿发展。
