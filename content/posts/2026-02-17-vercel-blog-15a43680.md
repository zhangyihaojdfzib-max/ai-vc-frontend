使用Vercel Sandbox创建的快照现在具有可配置的过期时间，取代了之前7天的限制，同时默认期限也有所延长。

```
1import { Sandbox } from '@vercel/sandbox';2import ms from 'ms';3
4const sandbox = Sandbox.create();5
6sandbox.snapshot({ expiration: ms('1d') })
```

过期时间可配置为1天到无限期。若未指定，快照的默认过期时间为30天。

您也可以在CLI中配置此选项。

```
12sandbox snapshot sb_1234567890 --stop3 45sandbox snapshot sb_1234567890 --stop --expiration 14d6 78sandbox snapshot sb_1234567890 --stop --expiration 0
```
