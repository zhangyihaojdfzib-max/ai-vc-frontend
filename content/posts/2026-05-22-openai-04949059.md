# OpenAI 将收购 Astral

加速 Codex 发展，为下一代 Python 开发者工具提供动力

今天，我们宣布 OpenAI 将收购 Astral⁠（在新窗口中打开），将强大的开源开发者工具引入我们的 Codex 生态系统。

Astral 构建了一些使用最广泛的开源 Python 工具，通过 uv、Ruff 和 ty 等现代工具帮助开发者更快地推进工作。这些工具支撑着数百万开发者的工作流程，并已成为现代 Python 开发的基础。作为我们开发者优先理念的一部分，在交易完成后，OpenAI 计划支持 Astral 的开源产品。通过将 Astral 的工具和工程专业知识引入 OpenAI，我们将加速 Codex 的工作，并扩展 AI 在整个软件开发生命周期中的能力。

## 将 Codex 扩展到编码之外

自今年年初以来，Codex 的用户数已增长 3 倍，使用量增长 5 倍，每周活跃用户超过 200 万。

我们对 Codex 的目标是超越单纯生成代码的 AI，转向能够参与整个开发工作流程的系统——帮助规划变更、修改代码库、运行工具、验证结果以及长期维护软件。Astral 的开发者工具直接嵌入在该工作流程中。在交易完成后，通过将这些系统与 Codex 集成，我们将使 AI Agent（智能体）能够更直接地与开发者每天依赖的工具协同工作。

## 强化 Python 生态系统

Python 已成为现代软件开发中最重要的语言之一，为从 AI 和数据科学到后端系统及开发者基础设施的方方面面提供动力。

Astral 的开源工具在该生态系统中已发挥关键作用：

- uv 简化了依赖项和环境管理
- Ruff 提供了极快的代码检查和格式化功能
- ty 帮助在代码库中强制执行类型安全

这些工具共同帮助开发者管理项目、保证质量，并在开发过程的早期捕获错误。随着 Astral 加入 OpenAI，我们将继续支持这些开源项目，同时探索它们与 Codex 更无缝协作的方式——使 AI 系统能够在整个 Python 开发工作流程中运行。我们的目标很简单：帮助开发者更快地推进工作，同时保持其工作流程所依赖的可靠性和性能。

## 下一步计划

本次收购的交割需满足常规交割条件，包括获得监管批准。在交割完成前，OpenAI 和 Astral 将保持独立运营。

交割完成后，Astral 团队将加入 OpenAI 的 Codex 团队，随着时间的推移，我们将探索更深度的集成，使 Codex 能够更直接地与开发者已使用的工具交互，助力 Codex 发展成为整个开发生命周期中的真正协作者。我们的目标始终如一：构建帮助人们更快创造、学习和构建的 AI 系统，并使强大的开发工具惠及全球更多人。我们很高兴在交割完成后欢迎 Astral 团队加入 OpenAI。
