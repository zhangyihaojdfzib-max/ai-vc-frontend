1 分钟阅读
您现在可以通过 Vercel 的 AI 网关访问亚马逊的最新模型 Nova 2 Lite，无需其他提供商账户。Nova 2 Lite 是一款面向日常工作负载的推理模型，能够处理文本、图像和视频以生成文本。

要使用 Nova 2 Lite，请在 AI SDK 中将 `model` 设置为 `amazon/nova-2-lite`。扩展思维功能默认是禁用的。要为该模型启用推理功能，请在 `providerOptions` 中设置 `maxReasoningEffort`。推理内容会被编辑并如此显示，但用户仍需为这些 Token 付费。

```javascript
import { streamText } from 'ai';
const result = streamText({ model: 'amazon/nova-2-lite', prompt: `Derive a correct, optimal algorithm to detect all unique cycles in a directed graph, explaining each logical step and validating edge cases.`, providerOptions: { bedrock: { reasoningConfig: { maxReasoningEffort: 'medium' // low, medium, or high accepted }, }, },});
```

AI 网关提供了一个统一的 API，用于调用模型、跟踪使用情况和成本，以及配置重试、故障转移和性能优化，以实现高于提供商水平的正常运行时间。它包括内置的可观测性、自带密钥支持以及具有自动重试功能的智能提供商路由。
