# 如何通过API运行Yi聊天模型

- nateraw

Yi系列模型是由01.AI开发者从头训练的大语言模型。今天，他们发布了两款新模型：Yi-6B-Chat和Yi-34B-Chat。这些模型基于基础模型Yi-6B和Yi-34B扩展而来，并针对聊天完成任务进行了微调。

Yi-34B目前在大多数基准测试中保持最先进水平，超越了Llama-70B等更大的模型。

## 如何通过API运行Yi-34B-Chat

Yi-34B-Chat已在Replicate上发布，您可以用几行代码在云端运行它。

您可以使用我们的JavaScript客户端运行：

```
import Replicate from "replicate";

const replicate = new Replicate({
  auth: process.env.REPLICATE_API_TOKEN,
});

const output = await replicate.run(
  "01-ai/yi-34b-chat:914692bbe8a8e2b91a4e44203e70d170c9c5ccc1359b283c84b0ec8d47819a46",
  {
    input: {
      prompt:
        "Write a poem about Parmigiano Reggiano.",
    },
  }
);
```

或者，使用我们的Python客户端：

```
import replicate
output = replicate.run(
    "01-ai/yi-34b-chat:914692bbe8a8e2b91a4e44203e70d170c9c5ccc1359b283c84b0ec8d47819a46",
    input={"prompt": "Write a poem about Parmigiano Reggiano."}
)
# 01-ai/yi-6b-chat模型可以在运行时流式输出。
# predict方法返回一个迭代器，您可以遍历该输出。
for item in output:
    print(item, end="")
```

或者，您可以直接使用cURL等工具调用HTTP API：

```
curl -s -X POST \
  -d '{"version": "914692bbe8a8e2b91a4e44203e70d170c9c5ccc1359b283c84b0ec8d47819a46", "input": {"prompt": "Write a poem..."}}' \
  -H "Authorization: Bearer $REPLICATE_API_TOKEN" \
  "https://api.replicate.com/v1/predictions"
```

您还可以使用其他客户端库（如Go、Swift、Elixir等）运行Yi聊天模型。
