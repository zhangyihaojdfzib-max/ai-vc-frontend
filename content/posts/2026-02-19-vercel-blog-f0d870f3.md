谷歌的Gemini 3.1 Pro预览版现已在AI Gateway上线。

此次模型发布带来了软件工程和Agent（智能体）工作流质量的全面提升，并增强了在金融和电子表格应用等现实任务中的可用性。Gemini 3.1 Pro预览版引入了更高效的跨用例思考能力，在保持性能的同时减少了Token消耗。

要使用此模型，请在AI SDK中将模型设置为`google/gemini-3.1-pro-preview`。该模型支持`medium`（中等）思考级别，以便在成本、性能和速度之间进行更精细的权衡控制。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'google/gemini-3.1-pro-preview',5  prompt:6    `Review this pull request for security vulnerabilities,7     suggest fixes, and update the test suite to cover edge cases.`,8  providerOptions: {9    google: {10      thinking_level: 'medium',11    },12  },13});
```

AI Gateway提供了一个统一的API，用于调用模型、跟踪使用情况和成本，以及配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它内置了可观测性、支持自带密钥（Bring Your Own Key）以及具备自动重试功能的智能供应商路由。
