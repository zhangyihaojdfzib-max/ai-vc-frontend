阿里巴巴的 Qwen 3.7 Plus 现已在 Vercel AI Gateway 上可用。Qwen 3.7 Plus 和 3.7 Max 对已付费的 AI Gateway 用户免费，截止时间为太平洋时间 2026 年 4 月 6 日中午 12:00。

该模型将视觉和语言统一为单一的 Agent（智能体）基础，具备 GUI 和 CLI 操作、编码和生产力工作流（支持全模态输入）以及包括感知和推理在内的视觉 Agent（智能体）任务能力。它旨在跨多种 Agent（智能体）框架进行泛化。

要使用 Qwen 3.7 Plus，请在 AI SDK 中将模型设置为 `alibaba/qwen-3.7-plus`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'alibaba/qwen3.7-plus',5  prompt: `Click through the checkout flow and flag any UI bugs you find.`,6});
```

AI Gateway 提供统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于提供商的服务可用性。它包含内置的自定义报告、零数据保留支持、按延迟和成本动态排序提供商等功能。AI Gateway 反映提供商的定价，不加价，并且不对推理收取平台费用，包括自带密钥（BYOK）请求。
