# OpenAI 收购 Software Applications Incorporated（Sky 的开发商）

提升人们在电脑上使用 AI 的方式

![一幅抽象图像，带有柔和的蓝色渐变和温和的光线过渡，唤起平滑波浪或天空运动的质感。](/images/posts/177d2b3607f0.png)

AI 的进步不仅仅关乎智能的提升——更在于通过能够理解上下文、适应你的意图并无缝工作的界面来释放智能。这正是我们激动地宣布 OpenAI 已收购 Software Applications Incorporated（Sky 的开发商）的原因。

Sky 是一款面向 Mac 的强大自然语言界面。借助 Sky，AI 能与你协同工作，无论你是在写作、规划、编程还是管理日常事务。Sky 能理解你屏幕上的内容，并利用你的应用程序采取行动。

我们将把 Sky 深厚的 macOS 集成能力与产品工艺融入 ChatGPT，该团队的所有成员都将加入 OpenAI。

“我们正在构建一个未来，ChatGPT 不仅能回应你的提示词，还能帮助你完成任务。Sky 与 Mac 的深度集成加速了我们的愿景，即将 AI 直接带入人们日常使用的工具中。”——Nick Turley，副总裁兼 ChatGPT 负责人

“我们一直希望电脑能更赋能、更可定制、更直观。借助大语言模型，我们终于能将各个部分整合在一起。这就是我们构建 Sky 的原因——一种悬浮在桌面上的 AI 体验，帮助你思考和创造。我们很高兴加入 OpenAI，将这一愿景带给数亿用户。”——Ari Weinstein，Software Applications Incorporated 联合创始人兼首席执行官

请继续关注更多更新，我们将着手整合 Sky 的功能。

披露：与 Sam Altman 关联的一只投资基金曾持有 Software Applications Incorporated 的被动投资。此次收购由 Nick Turley 和 Fidji Simo 主导，并已获得 OpenAI 董事会独立交易与审计委员会的批准。
