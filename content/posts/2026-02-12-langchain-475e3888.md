Interrupt - LangChain 主办的 Agent（智能体）大会 - 是构建者们学习实际生产环境中有效方案的地方。今年，我们将汇聚 1000 多名开发者、产品负责人、研究员和创始人，共同探讨 Agent 的未来发展方向以及如何构建它。

5月13日至14日，旧金山 Midway 场馆，期待您的参与。门票现已开售。

## Interrupt 大会亮点前瞻

今年的议程基于我们过去一年与客户、合作伙伴以及像您一样的构建者共同积累的经验而设计。这意味着您将带着可立即应用的实战经验、实践技能和新方法满载而归。

### 主要亮点包括

**塑造 Agent 未来的主题演讲**：Harrison Chase（LangChain 联合创始人）和 Andrew Ng（DeepLearning.AI）将分享未来趋势及其对您当前构建的 Agent 的意义。更多主题演讲嘉宾即将公布。

**实战经验分享**：直接聆听来自 Clay、Rippling、Monday.com 等领先企业和 AI 原生初创公司的 AI 团队分享 Agent 进入生产环境后的真实有效经验。

**LangChain 新产品首发**：见证最新的产品发布，并直接与背后的工程师交流。请带上您的问题。

**实践工作坊**：与 LangChain 产品专家一起提升技能。您将学习使用 LangSmith 构建、调试和交付可靠 Agent 的最佳实践。

**开放讨论**：从鼓舞人心的演讲转向与面临相同挑战的构建者们进行真实对话。

**赞助商展示**：结识构建 Agent 基础设施的公司。观看现场演示并直接与其团队交流。今年的赞助商包括 Cisco、Focused、Kong、Redis、Tavily、Oracle、D. E. Shaw、Cloudflare、Vapi、Daytona、Arcade 和 LiveKit。
