# 现已全面推出：LangSmith Agent Builder

LangSmith Agent Builder 现已全面推出——让任何人都能为复杂的日常任务构建 Agent（智能体），无需编写代码。

一个 Agent（智能体）应该像你的得力助手：你提出需求，它来想办法完成，并根据反馈不断改进。这正是 Agent Builder 背后的理念。

只需用你自己的话描述目标即可开始。Agent Builder 会规划方法，引导你从最初的想法到部署一个准备好接手你工作的 Agent（智能体）。它会创建详细的指令，选择所需的工具，甚至在需要时调用子智能体。

它与传统的工作流自动化不同。你无需规划每一步、摆弄“如果-那么”的分支逻辑，或者小心翼翼地处理依赖关系。你只需像对待队友一样给予反馈，Agent（智能体）就会利用其记忆进行学习。它从底层就是为作为 Agent（智能体）而构建的，具备规划和迭代能力，直到完成任务，或在执行敏感任务前寻求许可。

*   **每日简报**：生成包含公司和利益相关者研究，以及来自你 CRM 的相关客户背景信息的会前简报。只需说“给我发送每日会议简报”即可开始。

*   **项目跟踪**：根据 Notion 或 Google Docs 中的产品需求文档创建 Linear 任务，然后为关键里程碑创建日历事件。只需说“将我的产品需求文档转化为 Linear 任务”即可开始。
