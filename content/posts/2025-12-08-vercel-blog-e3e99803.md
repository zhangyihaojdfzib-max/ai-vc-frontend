1 分钟阅读  
今天我们正式推出 Rust 运行时 Beta 版的一流支持。  
此次原生支持的新版本，作为社区 Rust 运行时的演进，带来了 Vercel Functions 的全部优势，包括 Fluid 计算（支持 HTTP 响应流和 Active CPU 定价）以及环境变量限制从 6KB 提升至 64KB。  
Rust 部署将自动与 Vercel 现有的日志、可观测性和监控系统集成。  
要开始使用，请创建一个 Cargo.toml 文件和一个处理函数，如下例所示：

```toml
[package]
name = "rust-hello-world"
version = "0.1.0"
edition = "2024"

[dependencies]
tokio = { version = "1", features = ["full"] }
vercel_runtime = { version = "2" }
serde = { version = "1.0", features = ["derive"] }
serde_json = "1.0"

[[bin]]
name = "hello"
path = "api/hello.rs"
```

```rust
use serde_json::{Value, json};
use vercel_runtime::{Error, Request, run, service_fn};

#[tokio::main]
async fn main() -> Result<(), Error> {
    let service = service_fn(handler);
    run(service).await
}

async fn handler(_req: Request) -> Result<Value, Error> {
    Ok(json!({
        "message": "Hello, world!",
    }))
}
```

立即使用我们的入门模板 Rust Hello World 或 Rust Axum 部署到 Vercel，或在 Function 文档中了解更多信息。
