DeepSeek V4 现已在 Vercel AI Gateway 上可用。

该模型有两个版本：DeepSeek V4 Pro 和 DeepSeek V4 Flash。两个模型默认均支持 100 万 Token 的上下文窗口。

DeepSeek V4 Pro 专注于智能体编程、形式化数学推理以及长周期工作流。它能处理跨技术栈的功能开发、漏洞修复和代码重构，并支持在 MCP 工作流和智能体框架等工具链中使用工具。此外，它还能撰写清晰、结构良好的长文文档。

DeepSeek V4 Flash 在推理能力上接近 V4 Pro，在较简单的智能体任务中表现稳定，同时参数量更小，响应速度更快，API 成本更低。它非常适合高并发工作负载和对延迟敏感的使用场景。

要使用 DeepSeek V4，请在 AI SDK 中将模型设置为 `deepseek/deepseek-v4-pro` 或 `deepseek/deepseek-v4-flash`。

```
1 import { streamText } from 'ai';
2
3 const result = streamText({
4   model: 'deepseek/deepseek-v4-pro',
5   prompt:
6     `审计此代码库中不安全的并发访问模式，
7      提出引入适当同步机制的代码重构方案，
8      并将更改以 PR 形式提交，附带迁移计划。`,
9 });
```

AI Gateway 提供统一的 API，用于调用模型、跟踪使用情况和成本，并配置重试、故障转移以及性能优化，以实现高于提供商自身的可用性。它内置了自定义报告、可观测性、自带密钥支持，以及带有自动重试的智能提供商路由。

了解更多关于 AI Gateway 的信息，查看 AI Gateway 模型排行榜，或在我们的模型游乐场中试用。
