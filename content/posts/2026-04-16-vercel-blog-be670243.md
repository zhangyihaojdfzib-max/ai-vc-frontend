Anthropic 的 Claude Opus 4.7 现已登陆 Vercel AI Gateway。

Opus 4.7 针对长时间运行的异步 Agent（智能体）进行了优化，能够以可靠的 Agent（智能体）执行能力处理复杂的多步骤任务。该模型在知识工作者任务上表现出性能提升，尤其是在需要视觉验证自身输出的场景中。

Opus 4.7 在使用图像处理库分析图表和图形（包括像素级数据转录）的程序化工具调用方面也更加强大。它支持高分辨率图像，这对于计算机使用、截图理解和文档分析工作流非常有用。Opus 4.7 现在改进了记忆能力，那些在多轮对话中维护结构化记忆存储的 Agent（智能体）能够实现更可靠的回忆，并在无需额外提示词的情况下减少事实遗漏。

要使用 Claude Opus 4.7，请在 AI SDK 中将模型设置为 `anthropic/claude-opus-4.7`。您还可以尝试新的努力级别：`'xhigh'`。

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'anthropic/claude-opus-4.7',5  prompt: 'Explain the halting problem in one paragraph.',6  providerOptions: {7    anthropic: {8      thinking: { type: 'adaptive' },9      effort: 'xhigh',10    },11  },12});
```

Opus 4.7 还引入了任务预算功能。任务预算允许您通过 `taskBudget` 为 Agent（智能体）的一轮对话设置总 Token 预算。模型会看到一个剩余 Token 的倒计时，并利用它来优先安排工作、提前规划，并在预算消耗时优雅地结束任务。对于 Opus 4.7，思维内容现在默认被省略。要接收思维内容，请将 `display` 设置为 `'summarized'`：

```
1import { streamText } from 'ai';2
3const result = streamText({4  model: 'anthropic/claude-opus-4.7',5  prompt: 'Research how this codebase handles authentication and suggest improvements.',6  providerOptions: {7    anthropic: {8      thinking: { type: 'adaptive', display: 'summarized' },9      effort: 'high',10      taskBudget: { type: 'tokens', total: 50000 },11    },12  },13});
```

AI Gateway 提供了一个统一的 API 来调用模型、跟踪使用情况和成本，并配置重试、故障转移和性能优化，以实现高于供应商的正常运行时间。它包括内置的自定义报告、可观测性、自带密钥支持以及具有自动重试功能的智能供应商路由。
